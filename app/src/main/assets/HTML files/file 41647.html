The documentation in the following official tutorial is missing an important line. http://developer.android.com/training/location/location-testing.html

newLocation.setElapsedRealtimeNanos(SystemClock.elapsedRealtimeNanos()); is crucial. After modifying the following function, the mock location worked fine.

public Location createLocation(double lat, double lng, float accuracy) {
        // Create a new Location
        Location newLocation = new Location(LocationManager.GPS_PROVIDER);

        newLocation.setLatitude(lat);
        newLocation.setLongitude(lng);
        newLocation.setProvider(LocationManager.GPS_PROVIDER);
        newLocation.setAccuracy(1);
        newLocation.setTime(new Date().getTime());
        newLocation.setElapsedRealtimeNanos(SystemClock.elapsedRealtimeNanos());
        return newLocation;
    }
