How about if you select it directly from the information_schema? 

select * from information_schema.user_privileges;

Now to parse out the result set, you can either use a client or pipe it to sed to create a csv file, i.e.

mysql -uroot -p -e "select * from information_schema.user_privileges" | sed 's/\t/","/g;s/^/"/;s/$/"/;s/\n//g' > grants.csv

References: http://dev.mysql.com/doc/refman/5.0/en/user-privileges-table.html
