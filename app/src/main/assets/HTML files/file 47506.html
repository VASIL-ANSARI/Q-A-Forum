You always create a new Fragment inside your displayView method. Instead, you should use FragmentManager to find your fragment by Id or by Tag, and then put it inside your container view.

Please refer http://developer.android.com/reference/android/app/FragmentManager.html#findFragmentByTag(java.lang.String)
