Your problem is most likely caused by importing "too far" into a set of modules (all the way to an actual member instead of a module or submodule):

import module_name.sub_module.array_import

rather than either

  1. from module_name.sub_module import array_import
  2. import module_name.sub_module

but without code it's impossible to tell.

EDIT

@Dave, makes a great point that you also might be missing the module entirely. Try a different version or better yet use whatever the library you're attempting to use replaced the old version with.
