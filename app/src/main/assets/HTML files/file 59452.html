the only difference I see is using vm with two last functions in your second code sample. When you use

function grupoVisible(canchaComplejo){
   return vm.grupoMostrado === canchaComplejo;
}

this function is private to the current JS code of your controller (since your controller is immediately executed - (function(){})()) and is not accessible from the HTML. When you write it as 

vm.grupoVisible = function(canchaComplejo) {
    return vm.grupoMostrado === canchaComplejo;
};

it may be later accessed in HTML via ng-controller="CanchaController as cc" and then cc.grupoVisible, for example:

<div ng-controller="CanchaController as cc">
  <button ng-if="cc.grupoVisible()">SOME BUTTON</button>
</div>
