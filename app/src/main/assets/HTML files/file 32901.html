You are right you try call another instance of ClassA.

You can achieve this by passing ClassA as a parameter to ClassB, something like that:

@interface ClassB : NSObject
@property (nonatomic, copy) ClassA classA;

//.. other stuff
@end

After that when you create ClassB just pass ClassA to it:

ClassA myClassA = [ClassA new];
//set up class a here

ClassB classB = [ClassB new];
classB.classA = myClassA;

and in ClassB you can call update on classA:

[self.classA updateView:value];
