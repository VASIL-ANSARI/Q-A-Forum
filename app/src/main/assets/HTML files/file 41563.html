The caller still has a reference to the Date (and can modify their reference). You can prevent that if you clone() it in the constructor. Like,

public ImmutableClass(Date d)
{
  this.d = (Date) d.clone();
}

Also, a caller could modify the Date you return in getD so you should clone() there as well;

public Date getD()
{
  return this.d != null ? (Date) this.d.clone() : null;
}

Alternatively, for a Date, store the number of milliseconds it wraps like

final class ImmutableClass {
    private final long d;

    public ImmutableClass(Date d) {
        this.d = (d != null) ? d.getTime() : 0;
    }

    public Date getD() {
        return new Date(d);
    }
}
