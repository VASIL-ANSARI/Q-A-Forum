If you are looking for setting easing parameter, you may pass it as a separate argument like here:

$('your-selector').animate({
    'your-property': 'value'
}, 8000, 'linear');

...or with options object like here:

$('your-selector').animate({
    'your-property': 'value'
}, {
    duration: 8000,
    easing: 'linear'
});
