Check out this question - basically, when you call serialize on .net controls, the jQuery serialize routine uses the name attribute of the control to identify the data. Since this is a .net control, the name property will not be as you expect it to be. If you want a quick fix and you are using .net 4+, you can set the ClientIDMode on the control to correctly set the name client side:

<asp:DropDownList ID="MeetingLength" runat="server" ClientIDMode ="Static">
    <asp:ListItem Text="15 Minutes" Value="15" Selected="true" /> 
    <asp:ListItem Text="30 Minutes" Value="30" /> 
    <asp:ListItem Text="45 Minutes" Value="45" /> 
    <asp:ListItem Text="1 Hour" Value="60" /> 
    <asp:ListItem Text="1.5 Hours" Value="90" /> 
    <asp:ListItem Text="2 Hours" Value="120" /> 
</asp:DropDownList>
