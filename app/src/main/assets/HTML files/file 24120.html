And I want to redirect it to a server but with a specific header

First of all, you should realize what is the difference between HTTP Response Headers and HTTP Request Headers.

header("Content-Type: image/png");
header("Location: http://www.someotherserver.com/image.png");

Those are both Response headers that your server sends to the client.

Since you are instructing the client with the location header to request a new resource, the Content-Type header will be thrown away by the client - nothing to display in response to that request before following the redirect, so content type of no interest.

The client makes a new request to the (other) server in the location header - so any additional request headers for that request have to come from the client. Your server has no way of influencing what additional headers the client might send with that request. (Unless you write your own client, that passes received response headers on with his next request.)
