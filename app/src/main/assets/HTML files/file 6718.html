Use app.yaml for admin areas because login: admin is an easy and clean solution.

If you want moderator functionality on public pages you can use users.is_current_user_admin().

Update: You can't use login: admin to restrict access to some functions in one file. If you want to use it you have to create a separate admin.py file or something.

You could also try to create a function decorator like @admin_required. Here's an example for @login_required.

Here's the part that you'd have to change:

user = users.get_current_user()
if not user:
   self.redirect(users.create_login_url(self.request.uri))
   return
else:
   handler_method(self, *args)

to

if not users.is_current_user_admin():
    self.redirect(denied_url)
    return
else:
    handler_method(self, *args)
