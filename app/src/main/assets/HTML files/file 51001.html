Solution 1:

You should store the value of the parameter in a variable an check which key is available in $_GET:

$queryValue = '';

// Fetch parameter for omdbapi
if(isset($_GET['t'])) {
    $queryValue = $_GET['t'];
}

// Fetch parameter for Google CSE
if(isset($_GET['g'])) {
    $queryValue = $_GET['g'];
}

// Optional:
if(empty($queryValue)) {
    die('No parameter given.');
}

// Use the value stored in $queryValue

Solution 2: (If you don't have the possibility to use a variable like $queryValue)

$_GET['g'] = isset($_GET['g']) ? : $_GET['t'];
$_GET['t'] = isset($_GET['t']) ? : $_GET['g'];
