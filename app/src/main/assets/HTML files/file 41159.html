In jQuery, you can obtain the value of an attribute such as title using:

$(selector).attr("title");

In your example, the select item's title could be found with:

var title = $("#select_choice").attr("title");

Now that you have the title, you can add your text to the DOM prior to the select. For example:

$("#select_choice").prepend($("<span>" + title + "</span>");

For more details on each of these APIs, see the relevant section in the jQuery online documentation.
