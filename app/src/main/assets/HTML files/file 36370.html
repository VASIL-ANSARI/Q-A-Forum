When you define a plugin as $.fn[pluginName], that makes it a method on jQuery objects. If you want it to be an ordinary function rather than a method, you should define $[pluginName]:

$[pluginName] = function (options) {
    ...
};

However, this doesn't make sense for a plugin that does this.each(...) -- if you don't apply it to any elements, what do you expect this to refer to?
