Add injector annotations.

There are a few ways to do this. When using the prototype pattern you can easily just add $inject = [...] to your prototypes.

Since you are not using this pattern, the inline array annotations would be the easiest path.

Example:

.directive('noScroll', ['$document', function($document) {
    // . . .
}])
.controller('CardsCtrl', 
    ['$scope', 'TDCardDelegate', '$timeout', 
    function($scope, TDCardDelegate, $timeout) {

    // . . .
}]);
