Not the direct answer to the question, but a possible alternative solution - use the dateutil datetime parser in a "fuzzy" mode and simply format the extracted datetime object via .strftime():

In [1]: from dateutil.parser import parse

In [2]: s = "From stephen.marquard@uct.ac.za Sat Jan  5 09:14:16 2008"

In [3]: parse(s, fuzzy=True).strftime("%b %Y")
Out[3]: 'Jan 2008'
