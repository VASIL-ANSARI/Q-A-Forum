You can use rxJava library, for that you have to use specific call adapter factory during Retrofit instance creation: .addCallAdapterFactory(RxJavaCallAdapterFactory.create()) . Advantage of this method is that you can easily handle request in background and be notified about errors.

Observable<List<Place>> pendingResult = myApi.getPlaces();
pendingResult.subscribeOn(Schedulers.io)
.map(places -> {
    // pre- process places somehow
})
.subscribe(places -> {
// process result
}, error -> {
// handle error
}}
