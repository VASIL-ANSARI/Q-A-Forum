You are free to automate Excel from any Office applocation (Outlook in your case). The following articles describes the required actions in depth:

  • How to automate Microsoft Excel from Microsoft Visual C#.NET
  • How to automate Microsoft Excel from Visual Basic

You just need to decide what kind of application you need to develop - an add-in or just a VBA macro code. If you need to distribute it on multiple PCs - the add-in is the right way to go. VBA macros are not designed for distributing.
