If you trying parse/read the XML, you can Linq XML. Following example shows you how to parse and access elements/attributes. You might want to add some error checking (null check or valid values etc.). You would check against your schemaTypeName.Name compared with a.Value in the first lookup below.

XDocument doc = XDocument.Load("base.xsd");
var element = doc.Descendants()
                 .FirstOrDefault(e => e.Attributes()
                 .Any(a => a.Name=="name" && a.Value == "StringNotNull")) as XElement;

var minLength = element.Descendants()
                       .FirstOrDefault(e => e.Name.LocalName == "minLength")
                       .Attribute("value").Value;
var maxLength = element.Descendants()
                       .FirstOrDefault(e => e.Name.LocalName == "maxLength")
                       .Attribute("value").Value;

For schema validation, good option would be to use actual schema validator.
