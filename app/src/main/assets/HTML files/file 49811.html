The feature method is provided by the rspec-rails gem. So, you probably just need to install rspec-rails.

Feature specs are high-level tests .. [that] drive the application .. via its external interface .. The feature and scenario DSL correspond to describe and it, respectively. These methods are simply aliases that allow feature specs to read more as customer and acceptance tests. https://www.relishapp.com/rspec/rspec-rails/docs/feature-specs/feature-spec
