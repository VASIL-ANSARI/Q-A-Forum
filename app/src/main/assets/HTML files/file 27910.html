You shouldn't be swallowing all types of exceptions with a rescue => e. That line of code should almost never exist. Your exception handler should only capture the types of exceptions it can meaningfully recover from.

Change your Parent so that it doesn't swallow all exceptions, and then catch them in your Child class:

class Parent

  def commit
    # ...
  end 

end

class Child < Parent
  def commit
    begin
      super
    rescue UniqueError => e
      # ...
    end
  end
end
