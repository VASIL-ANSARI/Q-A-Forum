You can declare a method to be called in the click event

 $('.dropmenu').click(function (e) {
     var el = this;
     e.preventDefault();
     toggleMenu(el);
 };
 var toggleMenu = function (el) {
     $(el).parent().find('ul').slideDown();
 }

So from the console all you would have to do is 

 toggleMenu('.dropmenu')
