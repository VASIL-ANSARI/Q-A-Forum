The alternative to the imp module that (I believe) should be available in GAE is __import__(). It's in fact what the 'import' statement calls to do the actual importing.

user_language = getattr(__import__('localizable.%s' % language), language)

or

user_language __import__('localizable.%s' % language, {}, globals(), [''])

(passing a non-empty fourth argument to __import__ causes it to return the right-most module in the first argument, instead of the left-most. It's a little hacky, so people tend to prefer the first solution over the second.)
