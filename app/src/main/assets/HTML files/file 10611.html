If you always navigate from ListActivity -> FavoritesActivity, you could launch the FavoritesActivity from ListActivity using 

startActivityForResult (Intent intent, int requestCode) .

If the user changes there favorites, pass a boolean 'updated' back using 

 setResult (int resultCode, Intent data)

then retrive it in the ListActivites 

 onActivityResult(int requestCode, int resultCode, Intent data)

If the boolean is true, re-query your cursor.

Another option would be to update a SharedPreference that you write in FavoritesActivity and later read in your ListActivity (much like static-variable).
