Q: Why remote: true not working in form_for?

As I was using multipart: true but Images can't be created by ajax form by default. I've to include jquery.remotipart gem. How to embed gem in rails app.

My Form:

<%= form_for(@image, format: :js, remote: true,:html => { class: "form-horizontal", multipart: true }) do |f| %>

Remember to add //= require jquery.remotipart immidiately after //= require jquery_ujs in app/assests/javascripts/application.js file

Resource interpreted as Script but transferred with MIME type text/html this was just warning don't effect any thing.
