The cb function in the context of the fn function is a local function that can only be called within that context. So if you call the fn function and then the cb, ie. fn(); cb();, then the global cb will be called not the local cb.

A callback is a function that is passed as an argument to another function and it's executed when an operation is complete. So your code probably should be:

function fn(callback) {
    // do something
    if (typeof callback === 'function') {
        callback();
    } else {
       cb();
    }
}

function cb() {
    console.log("Original");
}

fn(); // "Original"

fn(function() {
    console.log("Overriden");
}); // "Overriden"
