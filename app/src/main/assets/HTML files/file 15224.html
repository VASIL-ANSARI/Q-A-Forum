"How do I load these objects" - there can be multiple way as per your need-

  1. you can use concurrent threads to load thing.

  2. You can go for lazy loading - first load those thing which needs to be show on first screen(considering you are loading couple of data which is not needed on first screen).

  3. You can show a loading screen and let user know that loading is going on.

you can use a UIActivityIndicator or UIProgressBar to show that currently loading is on.
