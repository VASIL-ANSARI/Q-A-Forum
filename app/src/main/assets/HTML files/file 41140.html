If last day of month is your intention use one of the following.

If you are using SQL Server 2012, you can use following.

select eomonth(dateadd(m,-6, '2014-11-30 00:00:00'))

If you are using earlier versions of SQL Server you can use following.

select dateadd(m, datediff(m, -1, dateadd(m,-6, '2014-11-30 00:00:00')), 0) - 1
