Sure, in build dev from dist serve html file in the right location what component links. In build progress use gulp-jade in simple task for example:

gulp.task('html:build', () =>
  gulp.src(PATH.src.jade)
    .pipe(jade())
    .pipe(gulp.dest(PATH.dest.app.component))
);
