Since the computers in question already have python, you can install the NLTK for your own use without admin privileges. The NLTK itself is pretty small, but you can further trim it down to produce a customized installation that only contains the modules you need for this task.

You don't say what resource you were planning to use as a dictionary/thesaurus replacement, but perhaps its format is simple enough to process without the nltk. If so, you can just get the file from nltk_data on your own computer and copy it to other systems.
