Ahhh-hah, without seeing your code I'm thinking you might find use in the .not selector. http://api.jquery.com/not-selector/

$("#foo .bar").not(this).hide();

This will basically grab anything BUT the thing you pass it, then do whatever to it (in this case hide). The nice thing about this is that doing it like so makes your UI expandable, if you plan to add more panels/windows later on.
