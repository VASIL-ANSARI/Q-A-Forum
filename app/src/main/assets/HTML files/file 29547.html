This code sets up a one-to-one relationship between the Comment and the Attachment model:

has_one :attachment

This has nothing to do with Paperclip.

I assume that within your Attachment model you have something along the lines of the following:

has_attached_file :attach

You need to interact with the :attach attached to the Attachment model:

@comment.attachment.attach

When you're trying to attach this to your email you should be using something like this:

attachments[comment.attachment.attach_file_name] = File.read(comment.attachment.attach.path)
