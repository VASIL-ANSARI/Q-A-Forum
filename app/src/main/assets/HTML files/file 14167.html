Cribbing from another answer, this example shows one quick and dirty way to solve your problem.

As an additional point, some of your code may be simplified as follows:

public float sumProteins(String[] items) {
    float sum = .0f;
    for (int i = 0; i < items.length; i++) {
        Cursor c = database.rawQuery(
                "select SUM(food_protein) from food where food_name='"
                        + items[i] + "'", null);
        if (c.moveToFirst()) {
            sum += c.getFloat(0);
        }
    }
    return sum;
}

You could further improve the code by building a query string like select SUM(food_protein ) from food where food_name=items[0] OR food_name=items[1] OR ... so you only need to perform a single query.
