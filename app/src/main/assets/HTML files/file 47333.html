I would say that you should rather reorganise your database to have a LaborYear and LaborMonth. Having 12 separate fields for months seems to be needless duplication. So have: 

class LabourYear < ActiveRecord::Base
    has_many :labor_months
end


class LabourMonth < ActiveRecord::Base
    belongs_to :labor_year
end

This way you can simply ask:

LaborYear.find_by(year: '2015').labor_months.pluck(:hours).inject(:+)
