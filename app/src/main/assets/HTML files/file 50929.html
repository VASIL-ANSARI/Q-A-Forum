I would create a separate property to get what you want

[NotMapped]
public ICollection<int> ProductIds {
   get {
      return Products.Select(p => p.ProductId)
                     .ToList();

}

Or if you really want it to be an array....

[NotMapped]
public int[] ProductIds {
    get {
       return Products.Select(p => p.ProductId)
                      .ToArray();

    }
