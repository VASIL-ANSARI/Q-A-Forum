Workaround

Update CSS

a, a:visited, a:hover {
    color: #2B80A9;
    text-decoration: underline;
    cursor: auto;
    margin-right: 1%;
}

notice

a, a:visited, a:hover

instead of

.anchorStyle:link

Get rid of: choiceAnchor.setStyleName(style.anchorStyle()); as you have defined your CSS to be applied to the anchor (a, a:visited, a:hover) directly so no need to specifically call setStyleName()
