You can not directly pass value from code behind to js file although you can pass to .aspx (html), you can then call .js file function to pass value to it. You need public property/attribute in code behind for that.

In code behind

public stirng YourPublicProperty  { get; set; }

In javascript

var inJs = '<%= YourPublicProperty %>';
someFunctionInJs(inJs);

Or

someFunctionInJs('<%= YourPublicProperty %>');

In .js file

function someFunctionInJs(para)
{
   alert(para);
}
