As per your example, you want to apply CSS to first and last column alone.So just add a reusable CSS class name to the column and define the style for it.

<col class="columnStyle">
<col>
<col  class="columnStyle">

In your CSS file:

col.columnStyle {
    width: 0px;
    /* specify required width here */
}

This will apply the specific width to only to those columns and not the middle one.
