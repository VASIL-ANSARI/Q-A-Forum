Though it is not a solution for all applications with transparent windows, it works for me since I only want this single application to not use Compton shadows.

Solution:

  1. Create a Compton config file.
  2. Add shadow-exclude = [ "name = 'WindowTitle'" ]; where WindowTitle is the title of the window.
  3. Execute pkill compton in terminal.
  4. Start Compton and make sure it is linked to the config file with the --config parameter.

That's it.
