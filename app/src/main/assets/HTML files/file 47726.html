http://www.jsonschema2pojo.org/ . Copy paste your response in this website. Select schema as "Json" and Annotation style as "GSON" . This will generate a response class needed for GSON.

Then parsing the response into GSON is straight forward.

Gson gson = new Gson()    
YourReponseClass object  =  gson.fromJson(json, YourReponseClass);

This will create the needed response object for you
