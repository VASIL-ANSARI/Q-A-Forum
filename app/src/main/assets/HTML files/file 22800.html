This is what the step option is used for. Have a look at the documentation:

http://api.jquery.com/animate/

$("#bar").animate({
    width: '50%'
}, 
{
    duration: 3000,
    step: function(now, fx){
        // update here, now is the current value
        // something like this
        $(this).text(Math.floor(now) + "%");
    }
});
