Can't you just define a plain function in each of the three JS files and then add fourth file / script section that calls them on window load in the correct order?

<script src="1.js"></script>
<script src="2.js"></script>
<script src="3.js"></script>
<script type="text/javascript">
window.onload(function() {
    one(); //Function from 1.js
    two(); //Function from 2.js
    three(); //Function from 3.js
    four(); //Function from 4.js
})
</script>

Note that the JS files should not use window.onload but instead just define the functions mentioned in the code example.
