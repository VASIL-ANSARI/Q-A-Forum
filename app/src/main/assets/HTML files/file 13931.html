When you use square brackets to access an element of a jQuery object you get the actual DOM element, not a jQuery object. DOM nodes don't have a text or html method.

You can use the eq method to get a jQuery object containing the element at the specified index:

$(function(){ 
    var $dvObj = $('div.desc');
    alert($dvObj.eq(0)); 
    alert($dvObj.eq(0).text());
    alert($dvObj.eq(0).html());
});
