If you are trying to update inventory alone then the payload array should look like this 

$inventory = array (            
    "variant"=>array (
        "id"=>{{'your_variant_id'}},
        "inventory_quantity"=>{{'your_inventory_quantity'}}
    )               
);

Your URL should look like this

$url = "variants/".{{'your_variant_id'}}.".json";

Call the above url with PUT method along with $inventory array as parameter.

I have a working demo, which is using the above method to manage inventory.

You can see that demo at http://sarkutils.in/swpanel with demo:password
