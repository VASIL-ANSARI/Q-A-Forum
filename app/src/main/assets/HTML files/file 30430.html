There's no other option. Just the alias via using or a struct wrapper. Personally I'd choose a struct wrapper, because it provides better type safety and you can define overloaded operators as needed. It's more work, but better than copy&pasting a single line of code across the whole code base, which would be a lot more error-prone and less maintainable.

Note that wrapping it with a class would make it a reference type, while a struct is a value type (same as long and other primitive types).
