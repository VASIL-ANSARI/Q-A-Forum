Generally, when dealing with methods, functions, subroutines etc, you have to match the arguments you pass when calling them.

For instance, if you have (or declared) a function like this:

void myFunction(int a, int b) { ... }

You have to call it passing 2 ints (no less, no more)

myFunction(10, 20);

There are some languages that accepts overload of methods, in these cases you could have something like this:

void myFunction(int a) { ... }
void myFunction(int a, int b) { ... }
void myFunction(int a, int b, int c) { ... }

This way youÂ´d have 3 possible ways to call your function:

myFunction(10);
myFunction(10, 20);
myFunction(10, 20, 30);
