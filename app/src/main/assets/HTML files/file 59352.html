The bulk of your question is answered in this SO post. I will use that as the basis.

The point that remains is how to set up the object[] parametersArray with a delegate instance.

To do this, you need to create a MethodInfo for your handler method. Using reflection on the calling code, for instance:

var handlerInfo = typeof(MyClass).GetMethod("MyHandlerMethod", 
                     BindingFlags.Instance | BindingFlags.NonPublic);

and then you can create the argument to pass (we know parameters[1] contains the delegate type):

var arg2 = Delegate.CreateDelegate(parameters[1].ParameterType, this, handlerInfo);

and you can complete the calling code with 

object[] parametersArray = new object[] { "MethodName", arg2 }; 
result = methodInfo.Invoke(classInfo, parametersArray);
