$ is just a symbol in javscript - it has no magic powers or interpretation. It can be assigned to be a variable or a function or anything you want. Same with $$ of $1 or any legal symbol in javascript that you want to use like that.

In the jQuery library, $ is a shortcut for the main jQuery function and thus:

$("#" + gridId + " .k-scrollbar-vertical")

is a function call and

$("#" + gridId + " .k-scrollbar-vertical").scrollTop(0);

is both a jQuery function call and then a method call to .scrollTop(0) on the returned object.
