You can use media queries in your CSS to add styles that will only be used if the webpage is being printed. In your case, I would use such styles to:

  1. Change the grey background to white
  2. Add black outlines and white backgrounds to your textboxes
  3. Change the text color to solid black
  4. Adjust fonts to compensate for the printed page

Media queries are normally used to change a website for smaller screens, but they're also perfectly applicable for changing it for print. You can read more about using media queries specifically for print here.
