Making the dropdown show on load is trivial, you just add the class open to the containing li. What you really need is a way to persist some data to tell you when to open the menu

The below might be the simplest:

<ul class="nav nav-pills"> 
<li class="<?php echo ($someVar == 'someVal') ? 'open' : ''; ?>">

See https://jsfiddle.net/f0yzyryL/2/

You could also do this by reading and writing cookies, using localstorage, reading url parameters, using location hash..... etc :)
