Generally, the rules for popups are as follows:

  • If you do a window.open outside of a user triggered event, it will be blocked, and display a given message. In general, popups will work transparently as part of a click event.
  • If you want a pop-over, leave it be...
  • if you want a pop-under, simply window.focus() after you do your window.open(...);

var w = window.open(...);
window.focus();

Note: The target attribute will still work in a browser. Though, you wouldn't want a link to pop under your current page.
