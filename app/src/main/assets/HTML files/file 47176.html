You would .append() to the table, or .after() to the last row. Right now you're trying to .append() the last row, which is effectively trying to nest a new row INSIDE of your row.

Should be as simple as:

$('#matters_table').append('<tr><td class="classhere"><input type="text"></input></td></tr>');

Or:

$('#matters_table tr:last-child').after('<tr><td class="classhere"><input type="text"></input></td></tr>');

Personally, I feel appending to the table is cleaner.
