The second problem it that when you decrease it output print(n - 1) not print(n)

I believe your solution is that.

def updown(n, m):
    while n >= m:
        print(n)
        n = n - m

    while n <= 7:
        print(n)
        n = n + 1

However if you want to start with even number the solution is that:

def updown(n, m):
    if n % 2 != 0:
        n = n - 1
    while n >= m:
        print(n)
        n = n - m

    while n <= 7:
        print(n)
        n = n + 1
