If you set deleteAfterServicing=true, this will clean up your images folder.

From the docs:

deleteAfterServicing

Whether the image should be deleted after successful download from a client.

The default value is true.

Alternatively, as the default is true, you can simply remove the deleteAfterServicing setting:

<appSettings>
    <add key="ChartImageHandler" 
         value="storage=file;timeout=1;webDevServerUseConfigSettings=false;"/>
</appSettings>
