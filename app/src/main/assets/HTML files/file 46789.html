Instead of using regular expressions, you could also just loop through the string and build a new one (new_string):

s = "hello_string_processing_in_python"

new_string = ""
capitalize_next = False
for character in s:
    if capitalize_next:
        new_string += character.upper()
        capitalize_next = False
    elif character == '_':
        capitalize_next = True
    else:
        new_string += character

print new_string
