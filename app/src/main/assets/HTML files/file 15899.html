function (){
    alert('a function');
}

is a function statement, so it requires a name.

(function (){
    alert('a function');
}())

The () change the statement to an expression, so it's OK.

And you could also use the below ways.

(function (){
    alert('a function');
})();

!function (){
    alert('a function');
}();

+function (){
    alert('a function');
}();
