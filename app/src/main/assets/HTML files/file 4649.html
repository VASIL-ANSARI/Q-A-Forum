The default user is IIS APPPOOL\DefaultAppPool but it's hidden by default so you won't see it in the security UI. You can still add permissions for this user though, you just can't search for it.

You can also do it programmatically:

icacls c:\inetpub\wwwroot /grant "IIS APPPOOL\DefaultAppPool":(OI)(CI)(RX)
