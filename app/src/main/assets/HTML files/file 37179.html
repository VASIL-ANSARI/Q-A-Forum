It's no problem at all to use memcpy here. However, you will need to use VirtualProtect to change the protection flags for the destination to PAGE_EXECUTE_READWRITE. Restore the original protection when you've finished writing.

If you are targeting x86/x64 that's all you need to do. For other architectures you need a call to FlushInstructionCache.

It is also a mistake to store a pointer to a DWORD. Don't cast the return value of GetProcAddress. It is a pointer.
