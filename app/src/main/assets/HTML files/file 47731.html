If you wrap the images and overlay in a parent element with position: relative you will be able to use position: absolute on the overlay element to position it on top of the images.

Here's a quick example: https://jsfiddle.net/hmpxdezy/1/

Based off your jsFiddle, is it div.custom you want to position over the top of div#modInistagram?

If so, here is an updated fiddle to give you an idea of how to achieve what you want with your existing markup: https://jsfiddle.net/24157Lmf/1/

You need to add position: relative to main#content and add position: absolute to div.custom. From there you can center align the overlay element using a similar method to my original jsFiddle with left: 50% and a negative margin-left with a value that is half the width of the element you want to center.
