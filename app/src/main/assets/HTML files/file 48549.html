I was having the exact same problem until I found this: https://community.serif.com/forum/18904/how-do-i-add-the-slick-slider-to-webplus. Martin has a great answer that simply involves adding some css to the .slick-center class. Adding your orange border under the first code block should work!

.center .slick-center h3 {
    -moz-transform:scale(1.08);
    -ms-transform:scale(1.08);
    -o-transform:scale(1.08);
    -webkit-transform:scale(1.08);
    color:#e67e22;
    opacity:1;
    transform:scale(1.08);
 }
.center h3 {
    opacity:0.8;
    transition:all 300ms ease;
}
