I don't know if i understand what you want to do. Is you want to center the container you should add text-align center to the body and fix width and add margin auto to the container:

body {
  background-color:#EFE9E1;
  height:100%;
  text-align:center;
}
#container {
  background-color:#EFE9E1;
  min-width:1024px;
  overflow:hidden;
  position:relative;
  width:1024px;
  margin:0 auto;
}
