Inside a plugin method, the selected elements are referenced by this. So it should be:

(function($) {      
    $.fn.extend({          
        animateleft: function(amount) {              
            this.animate({
                left: amount
            }, 300, function() { }); 
            return this;    
        }      
    });  
})(jQuery);

$() just calls the jQuery function with no parameter, which returns a new (empty) jQuery object.
