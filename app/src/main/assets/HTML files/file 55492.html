I'm not sure why you want to do this, but you can unbind click event of your button element after 768px.

$(window).scroll(onScroll);

function onScroll(){
    if($(window).scrollTop >= 768){
        $(".resp-menu-btn").off('click');
        $(window).off('scroll');
    }
};

If you want to bind other function to this element, you can re-bind it to another function for click event.
