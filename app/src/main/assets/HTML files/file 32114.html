You have your user object with current_user. You could create an instance method on User that will update the points value.

class User
  def reward_points(points)
    self.rating += points
    self.save
  end
end

So in your create action, if the award saves then call current_user.reward_points(@points), you probably don't even want @points to be an instance variable, instead just do points and pass that to the method.
