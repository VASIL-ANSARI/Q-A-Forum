Try changing your css to this:

* {
   margin: 0;
   padding: 0;
}

html,body {
   font-family:     Verdana, "DejaVu Sans", "Bitstream Vera Sans", "Lucida Sans", Arial, Helvetica, sans-serif;
   font-size:       90%;    /* Allow IE/Win to resize the document */
   color:           black;
   background:      white;
   border:          none;
   width:           100%;
   height:          100%;
}

Some browsers (like Google Chrome) have predefined margins and paddings. The * symbol selects everything in the document, by using * you get rid of those pesky predefined margins and paddings.

Furthermore I've changed the width and height of the body and html tags to stretch to 100% of the screen size.
