There is no way to disable it from keyboard directly. but, instead You can always use Textwatcher on your edittext to check what you are writing each time you change text inside edittext.

   yourEdittext.addTextChangedListener(new TextWatcher() {
        @Override
        public void beforeTextChanged(CharSequence charSequence, int start, int count, int after) {
            String oldText = yourEdittext.getText().toString();
        }

        @Override
        public void onTextChanged(CharSequence charSequence, int start, int before, int count) {
          String newText = yourEdittext.getText().toString();
        }

        @Override
        public void afterTextChanged(Editable editable) {
          String newText = yourEdittext.getText().toString();
        }
    });
