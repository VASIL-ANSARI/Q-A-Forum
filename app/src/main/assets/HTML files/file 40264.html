That error means you do not have DoDirectPayment enabled on the account you're attempting to use it with.

I see this a lot when people sign up with Payments Pro ever since PayPal released the new version of it.

If you've already signed up for Pro and are expecting it to work on your live account, then what probably happened is you got signed up for "PayPal Payments Pro 2.0" which uses the PayFlow API's instead of DoDirectPayment.

DoDirectPayment works with "PayPal Website Payments Pro 3.0" which is actually the older version.

Payments Pro 2.0 is technically supposed to give you full access to both PayFlow and DoDirectPayment, however, I've seen cases where that doesn't work out due to the PayPal account version that you're on.

What it all boils down to is that you need to make sure Payments Pro is active on your live account in the first place, and then figure out exactly which version of Pro it is.
