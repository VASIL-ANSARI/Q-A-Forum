Your while loop starts like this:

while(!gTermination)
{
    DWORD res=WaitForSingleObject(hMutex,INFINITE); 
    ...
}

So each player is blocking on hMutex, then you set the flag to true, then you release hMutex, so they finish their loops. A simple fix would be something like this:

while(!gTermination)
{
    DWORD res=WaitForSingleObject(hMutex,INFINITE);
    if(gTermination)
    {
        ReleaseMutex(hMutex);
        break;
    }
    ...
}
