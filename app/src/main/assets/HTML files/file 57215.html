You could use object-fit: contain; Note, the current IE11 and Edge don't support it, works fine on all other modern browsers though, see the support tables.

Use object-fit: cover; if you need the image to fill the container completely.

For polyfill, check this out: Polyfill for CSS object-fit property.

.img-container {

  height: 200px;

  width: 50%;

  overflow: hidden;

  background: yellow;

  margin-bottom: 5px;

}

.img-container img {

  width: 100%;

  height: 100%;

  object-fit: contain; /* or cover */

}
<div class='img-container'>

  <img src="http://placehold.it/350x150" />

</div>
