Signed integer types are already using two's complement to represent negative numbers, you don't need to do anything to them.

If you display a negative number in hex format, you see the two's complement in action:

int number = -3092;
Console.WriteLine(number.ToString("x"));

Output:

fffff3ec

If you want to convert a positive number to a negative number, which makes it use two's complement, use the - negation operator:

number = -number;
