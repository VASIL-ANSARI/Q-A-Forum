Each call to readline() skips one line if not assigned to a variable, just remove those calls, and since this empties most of the if-else blocks, you can simplify it to:

// to be a bit more efficient
String separator = System.getProperty("line.separator");
while (( line = input.readLine()) != null)
{
    if (!(line.startsWith("#") || 
          line.startsWith("*") ||
          line.isEmpty() )) 
    {
        contents.append(line);
        contents.append(separator);
        System.out.println(line);
    }
}
