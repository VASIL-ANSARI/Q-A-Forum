Why are you using fb.dialog() to post a status?

Instead of calling:

fb.dialog(AppTestingActivity.this, "feed" , params, new DialogListener() {

do:

mAsyncRunner.request("me/feed", params, "POST", new RequestListener() {

to post to your wall.
