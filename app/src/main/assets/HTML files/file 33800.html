you can achieve it by adding following lines in onResume method of your activity ,

    getSupportActionBar().setDisplayHomeAsUpEnabled(true);

    if(this.getResources().getConfiguration().orientation == Configuration.ORIENTATION_PORTRAIT) 
    {
    // code to do for Portrait Mode
        getSupportActionBar().setHomeButtonEnabled(true);
    } else {
    // code to do for Landscape Mode         
        getSupportActionBar().setHomeButtonEnabled(false);
}

Note: use getSupportActionBar() while using actionbarsherlock or appcompat library. and use getActionBar() when you don't support lower versions.
