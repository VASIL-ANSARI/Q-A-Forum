Thanks for the comments... I was gonna do a jsfiddle but the site was temporarily down... It is back now, but I solved the issue. The issue was this line in the HTML:

<link rel="stylesheet" type="text/css" href="~/templates/preview.css" />

For some reason, Chrome/Safari need some extra time to parse that. I changed it to:

<link rel="stylesheet" type="text/css" href="http://www.fulldomain.com/templates/preview.css" />

and everything works now.
