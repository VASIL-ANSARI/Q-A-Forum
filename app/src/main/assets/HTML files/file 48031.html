You have to add * to your regex.

Try this regex :

       cs.toString().matches("^[a-zA-Z. ]*$")

As you did not specify * in your regular expression, it tries to match only one character.

Update:

You can further simplify your code and change 

    if(cs.equals("")){ // for backspace
           return cs;
     }
     if(cs.toString().matches("[a-zA-Z. ]*")){
         return cs;
     }
     return cs;

to

    if(cs.toString().matches("[a-zA-Z. ]+")){
         return cs;
     }
     return "";

The difference is now you are using + instead of *.

     * matched 0 or more occurrences
     + matched 1 or more occurrences

So there is no need to explicitly check for empty string.
