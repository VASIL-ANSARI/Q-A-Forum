Let's start saying you don't need to call pow() at all, but in your case you misplaced the braces. It should be

pow(height, 2)

Assuming that height is a numeric variable. It's not clear if you want to include somehow the units of measures of phisical quantities (inches or pound) into your variables. That's a more challenging task, but meanwhile you can use this:

float weight = 0, height = 0;
cout << "Enter weight(lbs): ";
cin >> weight;
cout << "Enter height(inches):";
cin >> height;

float BMI = weight * 703.0 / (height * height);
cout << "BMI = " << BMI << '\n';
