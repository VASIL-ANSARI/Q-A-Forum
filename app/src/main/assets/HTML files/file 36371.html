In response to comment... Your syntax is still wrong. if you are trying to see if the FULL is populated and the same, that's one part. Then you want to see if that also happens to be within a range. This is what it should look like:

$sql = "SELECT * FROM transfers WHERE full='".$full."' and full BETWEEN 1 AND 1,000,000";

If you want to do this same thing but check against a balance, you don't need to match anything. All you need to do is take the input and compare it to the balance. So, ignoring your syntax, this is what your statement should look like.

$sql = "SELECT CASE WHEN ".$full." < [balance] THEN 'sufficient funds'
 WHEN ".$full." > [balance] THEN 'insufficient funds'
END AS 'result', [balance]
 FROM transfers

With that, you are checking the input amount for the possible results (add as many as you need), and returning the current balance you just checked against.
