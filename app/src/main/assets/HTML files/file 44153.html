You don't need to unset it just array_filter() it out like this:

<?php

    $my_array = [TRUE => 1, FALSE => 2, NULL => 0]; 
    print_r($my_array);
    $my_array = array_filter($my_array, "strlen", ARRAY_FILTER_USE_KEY);
    print_r($my_array); 

?>

output:

Array ( [1] => 1 [0] => 2 [] => 0 )
Array ( [1] => 1 [0] => 2 )

Side Note:

A key can't actually be NULL. You can also see this in the manual:

The key can either be an integer or a string. The value can be of any type.

Means your "NULL" key gets casted to an empty string. You can see this with a var_dump():

array(3) { [1]=> int(1) [0]=> int(2) [""]=> int(5) }
                                   //^^^^ See the key is just an empty string
