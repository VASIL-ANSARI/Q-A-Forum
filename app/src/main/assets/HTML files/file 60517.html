Make sure You assign your delegate in cellForRowAtIndexPath for the cell. Also, I would recommend to keep weak reference to the delegate.

protocol CustomCellDelegator: class {
    func callSegueFromCell()
}

In your CustomTableViewCell replace 

var delegate:CustomCellDelegator!

with 

weak var delegate: CustomCellDelegator?

After this you do not need to check if Your delegate is nil or not, your app will not crash.
