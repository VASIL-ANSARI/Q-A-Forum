For the diagonal, the row index is equal to the column index, so you can do:

dimension=int(input('Enter dimension of triangle: '))

if dimension < 2:
    print('Invalid input')

else:
    for r in range(dimension):
        for c in range(dimension):
            # If we are on the diagonal
            if r == c:
                num = "1"
            # Otherwise
            else:
                num = "0"

            print(num, end=" ")

        print()
