All you have to do is:

  1. Change your Layout root to a RelativeLayout

  2. Make sure all CardViews fall nicely under each other with the required spacing between them (margin) using the layout_below property

  3. Declare the "line views" inside the root RelativeLayout but AFTER the CardViews in XML so that when you lay them out they appear on top of the CardViews

  4. Use layout_below and layout_above referencing the ImageViews on top and under the line. For example...

<View
    android:layout_width="2dp"
    android:layout_below="@id\id_of_the_image_view_at_the_top"
    android:layout_above="@id\id_of_the_image_view_below" />

Then make sure you set you margins so you can centre them horizontally
