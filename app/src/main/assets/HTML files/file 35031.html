search returns nothing. But the event handler needs to be a function. So either you do U.addEvent(U.$('searchBox'), 'click', search); or make search() return an event handler.

Basically right now you are immediately invoking the search handler and your form does nothing.

You also have the click handler on an input field. That will be pretty much useless, since you don't want anything to happen when the user just clicks the input field.
