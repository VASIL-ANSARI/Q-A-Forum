I don't know the exact architecture of your database, but what you'll want to do is use

http://api.jquery.com/change/

to detect a change in selection of a checkbox, then use

http://api.jquery.com/jQuery.ajax/

and pass all the parameters to a separate .php file to get the result off another page and print it out

So, an example.

<select class="order">
    <option value="name">Name</option>
    <option value="date_poster">Date Posted</option>
</select>

<div id="results">
</div>

<script>
    $("select.order").change(function(){
        var orderBy = $(this).val();
        $.ajax({
          url: "your_external_file.php",
          data: "orderBy="+orderBy,
          success: function(html){
            $("#results").append(html);
            //Add whatever you want to do here.
          }
        });
    });
</script>

Although I haven't tested it, it should work as long as you name the files correctly
