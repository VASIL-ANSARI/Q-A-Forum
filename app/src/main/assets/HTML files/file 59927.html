You have display: none; style for img with class load_more_img. That's why loading image isn't showed (As I understand you want to show it after click "show more"). Also you render _load_more.plus value which is path of the image to the content of <a></a> tag (don't know why you need this, but it's strange).

After all I think you want to make <a></a> element with image in backround from _load_more.plus path and with href with path to open more items. And when items are still loading you want to show load_more_img. In this case you should make some refactoring of your code.

EDITED:

 $('.load_more').click(function() {
