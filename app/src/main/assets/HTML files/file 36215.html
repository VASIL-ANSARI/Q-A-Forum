If you're using PHP 5.5+ you can use

$uniqueCids = array_unique(
    array_column(
        $myArray,
        'cid'
    )
);

For PHP < 5.5, you can use

$uniqueCids = array_unique(
    array_map(
        function($value) {
            return $value['cid'];
        },
        $myArray
    )
);

Then just do

$uniqueCidCount = count($uniqueCids);
