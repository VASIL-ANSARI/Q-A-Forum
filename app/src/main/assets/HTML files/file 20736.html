In case you have formatting code (HTML) in the table cells and you don't want them to show up, you could also take the HTML and display it in a dialog box, or you could (when the table is being generated in a PHP script) add attributes to the with the plain content and use $(elem).prop(prop-name) function to get the value of that property.
