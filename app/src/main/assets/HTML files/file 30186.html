Technically you only need to install the SDK Platform files for the minimum API Level you plan to target your App. So if you plan to target your app for ICS (4.0.3) and above, you should at the minimum have the SDK Platform files for API Level 15.

But nothing will happen if you chose to download every single component listed on the SDK Manager. What matters is how you setup your project for your app. In eclipse it's through a wizard, so it's pretty self explanatory.
