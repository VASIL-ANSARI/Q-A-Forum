Maybe you should try using conditional navigation instead:

http://www.mkyong.com/jsf2/conditional-navigation-rule-in-jsf-2-0/

<navigation-rule>
    <from-view-id>start.xhtml</from-view-id>
    <navigation-case>
        <from-outcome>test</from-outcome>
        <if>#{BeanTools.testt}</if>
        <to-view-id>finish.xhtml</to-view-id>
    </navigation-case>
</navigation-rule>

I'm not sure if you can use boolean types with from-outcome...I've only ever used/seen Strings.
