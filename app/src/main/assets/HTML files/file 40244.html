From what i'm seeign, your WordPress function outputs your image inside .thumbnail. so do something this

 .thumbnail {
 width: 800px;
 height: 500px;
 overflow: hidden; /* Cuts off whatever goes beyond the given width */
}


 .thumbnail img {
  width: 100%; /* 100% equals 800px as defined for .thumbnail */
  height: auto; /* Allows the image a breathing option so it doesn't deteriorate */
}

Note: The containing element already has a given width(800px) so the image won't go beyond 800px
