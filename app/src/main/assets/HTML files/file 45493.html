Do you happen to be calling init like below?

 export default Ember.Component.extend({
  init: function(){
    //do stuff here...
  },

If so, make sure you are calling this._super(); at some point in time within your init call. 

export default Ember.Component.extend({
 init: function(){
    //do stuff here..
    this._super();
    //do stuff here...
 },

If that is not the case, please post your code.
