myList.getItemAtPosition(position) gets you the "Object" that represents the row at that position. Simply put, that is defined by your CursorAdapter. This is actually a Cursor object and should be casted as follows:

Cursor myCursor = (Cursor) myList.getItemAtPosition(position);

You then have to get the String from the correct position in the Cursor (which should be in whatever order you put it into the adapter):

String myDesiredString = myCursor.getString(cursorPos);

where cursorPos is an integer that represents the position you wish to get.

For more information, here is an excellent similar question: getItemAtPosition() How to get readable data....

As a side note, I personally prefer using my own adapter which extends BaseAdapter so that I have more control over the data and can even make the adapter itself return the data [such as defining a method like public String getEventName(int position)]. However, the simplest solution for this context is what I described above.
