I ended up using the grid lines that were already in my code as baselines for the rectangles. This is the code that makes the grid lines work:

function make_y_axis() {        // function for the y grid2 lines
    return d3.svg.axis()
    .scale(y)
    .orient("left")
    .ticks(5)
}

And here is how they're called:

svg.append("g")      // Draw the y grid2 lines
    .attr("class", "grid2")
    .call(make_y_axis()
        .tickSize(-width, 0, 0)
        .tickFormat("")
    );

This creates my grids, and in CSS up top, I erase the lines, so they are there, but can not be seen.

.grid2 .tick {
    stroke: none;               
}

And to almost finish it off, I actually add the rectangles like so:

svg.selectAll(".grid2")
    .selectAll(".tick")
        .append("rect")
        .attr("width", width)
        .attr("height", maxY - 4.03)
        .attr("class", function(d, i) {
            return ((i) % 2) == 1 ? "zebraGrey" : "zebraNone";
        })
        .attr("stroke", "none");

And add the styling:

.zebraNone {
    opacity: 0;
}
.zebraGrey {
    background-color: #000;
    opacity: 0.05;
}

And that's it.
