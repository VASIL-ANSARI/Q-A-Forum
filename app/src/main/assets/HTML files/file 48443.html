If you're using R (I use Rstudio), you can add the Leaflet package (https://rstudio.github.io/leaflet/ it has some prerequisites). First you load the CSV as a dataframe, then you add a marker layer from it with a one liner (see https://rstudio.github.io/leaflet/markers.html)

For the layer control follow this example https://rstudio.github.io/leaflet/showhide.html
