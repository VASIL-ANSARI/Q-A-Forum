Possibly you are expecting the string to be modified in place. You need to assign the result back to the string.

Here is an example of how you could split your data

>>> s = """P1\tTitle\tName
... P2\tTitle\tName"""
>>> for row in s.splitlines():
...     print row.split("\t")
... 
['P1', 'Title', 'Name']
['P2', 'Title', 'Name']
