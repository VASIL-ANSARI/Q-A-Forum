Here is your issue:

Intent openMenu = new Intent("com.mysterygame.au.Menu");
          startActivity(openMenu); } } };

It should be:

Intent openMenu = new Intent(Pentagram.this, Menu.class);
startActivity(openMenu); } } };

When you are using Intent, you need to insert the class name as I did above, not the way you did. You basically include the current Activity you are in, and the one you want to start, which is why you use Pentagram.this (the Activity you are in) and Menu.class the Activity you want to start.
