Steps

  1. When you load the page, hide all the menus.
  2. Now for the handler, when you click on it, you have two cases.
    1. See if the submenu is visible, if so, hide the sub-menu.
    2. If the submenu is not visible, then hide everything and show just the submenu

Use this code:

$(document).ready(function() {
    $(".gen-sel > li > ul").hide();
    $(".gen-sel > li > a").click(function(){
        if ($(this).next().is(":visible"))
            $(this).next().hide();
        else
        {
            $(".gen-sel > li > ul").hide();
            $(this).next().show();
        }
    });
});â
