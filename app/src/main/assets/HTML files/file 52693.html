Use Frame.BackStack property. It returns an IEnumerable that you use to enumerate the entries in back navigation history for a Frame.

You can get the containing Frame by traversing the VisualTree.

Sample : 

private void Button_Click(object sender, RoutedEventArgs e)
{
    var leaf = (DependencyObject)sender;
    while(leaf.GetType() != typeof(Frame))
    {
        leaf = VisualTreeHelper.GetParent(leaf);
    }

    Frame f = leaf as Frame;
    if (f != null)
    {
        JournalEntry firstItem = null;
        foreach (JournalEntry item in f.BackStack)
        {
            firstItem = item;
            break;
        }

        this.NavigationService.Navigate(firstItem.Source);
    }
}

Addional methods if you are navigating from within the Page, then

  1. You can have your Page class have a parameterized constructor receiving the Frame object.

  2. You can use a while loop with going backward until you reach the first pageentry.

  3. Use a PageFunction. A PageFunction is a Page which can return a value.
