I would use this technique instead:

jQuery(document).ready(function() {

    console.log( "ready!" );

});

Reference: https://learn.jquery.com/using-jquery-core/document-ready/

The reason is the DOM is not accessible when the window has loaded, the DOM has to be processed as well. That is why we wait for the 'document' to be ready (DOM: Document Object Model)
