You need to add default_date: false

See here: https://github.com/nazar-pc/PickMeUp#user-content-configuration-options

Allows to keep empty value until date selected, but open PickMeUp at this default date (false for no default date)

Update

Here's a workaround that will remove the selected class entirely, along with the highlighting of today's date:

before_show: function() {
    $(".pmu-today").removeClass("pmu-selected pmu-today");
}

Example: http://jsbin.com/kavusibomu/edit?html,js,output
