Your indentation is messed up and you've added an extra end in there because of it.

The private declaration should be at the same level as class.

A general template looks like:

class Example
  def method
    # ...
  end

private
  def private_method
  end
end

What you have:

class Example
  def method
    # ...
  end

  private
    def private_method
    end
  end
end
