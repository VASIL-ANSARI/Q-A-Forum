To answer your two questions:

1) The only way to do this by creating a variable and using that:

var name = 'type';
$('.' + name + ', #' + name).on(...)

2) You can use this like so:

$('.type, #type').on('change', function() {
    selected = $(this).val();
    show = selected != 0 ? true : false;
    $(".div").toggle(show);
});
