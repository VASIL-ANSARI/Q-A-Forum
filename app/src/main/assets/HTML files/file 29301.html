Android TextView does not support the <table> tag and its associated tags. See here for a list of HTML tags that are supported by TextView.

To display this you need to use a WebView. First, add the WebView to your layout:

<WebView
    android:id="@+id/webView"
    android:layout_width="fill_parent"
    android:layout_height="wrap_content" />

Then load the table HTML in the associated Activity:

String html = "<table> ..... </table>";

WebView webView = (WebView)findViewById(R.id.webView);
webView.loadData(html, "text/html", "utf-8");
