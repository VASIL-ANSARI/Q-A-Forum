It sounds like you are using the default option verticalCentered:true and therefore your elements inside your sections will be wrapped by the plugin. You will need to use delegation or use your javascript inside the callback afterRender as suggested in fullPage.js docs:

afterRender()

This callback is fired just after the structure of the page is generated. This is the callback you want to use to initialise other plugins or fire any code which requires the document to be ready (as this plugin modifies the DOM to create the resulting structure).

Also adviced in fullPage.js FAQs:

My other plugins don't work when using fullPage.js

Short answer: initialize them in the 'afterRender' callback of fullPage.js.

Explanation: if you are using options such as 'verticalCentered:true' or 'overflowScroll:true' of fullPage.js, your content will be wrapped inside other elements changing its position in the DOM structure of the site. This way, your content would be consider as "dynamically added content" and most plugins needs the content to be originally on the site to perform their tasks. If you use the afterRender callback to initialise your plugins, then those elements are not going to suffer from any modification in the DOM structure anymore and will be consider to be originally in its place for the plugins you want to use from now on.
