If all your point is readability, I'd suggest to take 2 variables - 

query_statement = "LOWER(id) LIKE ? OR " + 
                  "LOWER(name) LIKE ? "


query_variables = s,s

and then use these two variables in the actual query :

u = User.where("#{query_statement}, #{query_variables}")
