Try replacing the autoloader code from your question with this:

require_once 'Zend/Loader/Autoloader.php';
Zend_Loader_Autoloader::getInstance();

Just calling Zend_Loader_Autoloader::getInstance() will have the Zend Framework autoloader register itself with spl_autoload_register which is all that Yii::registerAutoloader() tries to do.

Registering ZF's autoloader like this should allow you to autoload Yii classes and ZF classes together without any problems.
