Besides the aforementioned stuff (rename your enum!), you simply have to declare the enum outside of your Question class. You don't have to make a new file for this (although you can if you want), simply put it above your class like so:

enum QuestionType: String {
    case Text  = "TEXT"
    case Image = "IMAGE"
}

class Question: NSObject {
    //...
}

Now you can use the QuestionType enum in your MyViewController class.
