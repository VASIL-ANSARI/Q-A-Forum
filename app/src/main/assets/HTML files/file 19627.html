If you are including you JavaScript in your head tags, that might be the cause because the DOM is not fully loaded when the function is called. 

<html>
   <head>
       <script></script>
   </head>
   <body>
   </body>
</html>

My suggestion is to do this instead.

<html>
   <head>
   </head>
   <body>
       All your html goes here.
       FINALLY.
       <script></script>
   </body>
</html>
