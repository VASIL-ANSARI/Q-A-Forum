There is no need to create another input field in html if data is not put by user. You can add child to your xml file.

add this line 

$xml->reports->addChild('time', date("Y-m-d H:i:s",time()-60*60));

this will add date which is one hour back as you suggested in comments. You can change date format to fit your needs. To find how to do it you can check Date() in manual
