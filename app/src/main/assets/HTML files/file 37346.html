I had a similar problem with one of my application. The trick I found (and I say the trick) is, depending your fragment, hide the unused items and show the needed ones.

In the Activity where you manager your fragment, just manage your items as well :

private Menu mMenu;

public void onCreateOptionsMenu(Menu menu,MenuInflater inflater)
{
    inflater.inflate(R.menu.note_menu, menu);
    mMenu = menu;
    ...
}

private void displayMenu1(boolean display)
{
    mMenu.findItem(<ID>).setVisible(display);
    mMenu.findItem(<ID2>).setVisible(display);
}

private void displayMenu2(boolean display)
{
    mMenu.findItem(<ID>).setVisible(display);
    mMenu.findItem(<ID2>).setVisible(display);
}

I'm sure this is not the best solution, and didn't think of any better solution since that.
