try this, assuming your HDI is in a column called 'HDI' and you are trying to create a new column that is equal to the HDI, or 0 if HDI is < .5

def adj_hdi(row):
    hdi = row['HDI']
    if hdi>.5:
        return hdi
    else:
        return 0
mydataframe['ADJ_HDI'] = mydataframe.apply(lambda row: adj_hdi(row), axis = 1)
