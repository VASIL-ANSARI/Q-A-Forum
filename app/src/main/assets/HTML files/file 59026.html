To adjust a page that way you need to modify it using JavaScript.

Regarding your example, to add the sum after the price and quantity you could have a script like this:

let price = Number.parseFloat(document.querySelector(".price1 > span").textContent);
let quantity = Number.parseFloat(document.querySelector(".quantity > span").textContent);
let a = document.querySelector(".a");

let sum = document.createElement("div");
sum.textContent = price * quantity;

a.appendChild(sum);

This script can be executed with the command line of the browser debugging tool.

Though note that this change is only temporary. It will be gone again the next time you access the website or reload it. To apply the script everytime the page is loaded you can use a browser add-on like Greasemonkey, which allows to automatically execute your script on specific pages.
