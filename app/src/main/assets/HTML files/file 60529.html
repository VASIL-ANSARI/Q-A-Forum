So you basically need to append a span before each line. Here is the codepen link. I am currently using jQuery's insertBefore() method for it.

See below for the explanation of the code :- $('.code') will give you the pre tag. Now the jQuery .find() method will give you all the spans inside the pre tag. Now jQuery .each() function is basically a for loop ( in simple terms ) which will loop for all the span tags inside the pre tag. .insertBefore() function simply inserts whatever there is in the selector to the element mentioned inside the function.

var iCount = 1;

$('.code').find('span').each(function(){

  

  $( "<span>"+iCount+"| </span>" ).insertBefore( this);

  iCount++;

});
