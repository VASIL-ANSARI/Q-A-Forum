I got two solutions for this problem

  1. copy texture attached to the FBO using glGetTexImage uint * pixels = new uint[FBO_WIDTH * FBO_HEIGHT]; glBindTexture(GL_TEXTURE_2D, someFBO->texture()); glGetTexImage(GL_TEXTURE_2D, 0, GL_RGBA, GL_UNSIGNED_BYTE, pixels);

  2. share resources as Mateusz Grzejek said: in qt you must put a parameter to the constructor shareWidget of QGLWidget, read more here QGLWidget e.g. GLWidget(QWidget *parent = 0, QGLWidget * shareWidget = 0); and then when you creates two GL widgets use style like this

GLWidget *glWidget = new GLWidget(this); GLWidget *glWidget2 = new GLWidget(this, glWidget);
