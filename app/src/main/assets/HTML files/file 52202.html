It's not a good practice to mutate template context. however you can try options below:

  1. Execute python in template and use dict.update method:

    {% do field.update(data=email) %}

  2. Try to use with statement:

    {% with data = field["data"] %}
    {% set data=email %}
{% endwith %}

  3. Try to access your data thought ['data'] instead of '.' notation:

    {% set field["data"]=email %}
