The pattern is more clear in an image with color.

You need a column and row of perfect d-trees with height h.

A perfect d-tree of height h has d^h leaf nodes, and since the leaf nodes of a column tree maps to leaf nodes of distinct row trees, you need d^h row trees and d^h column trees.

The leaf nodes form a (d^h)x(d^h) matrix and can be mapped trivially.
