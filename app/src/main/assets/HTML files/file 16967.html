For those hitting the same problem: I have not found a solution. I have however successfully connected an app to the Google API and can access all the info and methods I want.

Decided to go about this with omniauth. It's really simple, there's a railscast episode on how omniauth works, and a great walk through of using omniauth to connect to Google here. It describes specifically the act of connecting to the Google Calendar, but all you would need to do is change the client.discovered_api setter and the client.execute call in the sessions_controller to which ever you want. Decent documentation and code examples at developers.google.com/google-apps/calendar/v3/reference/

Best of luck.
