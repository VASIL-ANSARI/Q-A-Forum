The error means the site is blocking the method you're trying to connect to it.

Basically the admins of the site are blocking the specific method you're trying to use OR the site is on 'read only', you can however remove this block and here's a link to teach you how to do it:

http://www.checkupdown.com/status/E405.html

As for the 401 error it can indicate old cached data, a missing cookie, or a security conflict. 401 errors may also indicate a server error.
