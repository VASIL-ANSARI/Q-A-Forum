Did you try using display()? According to the docs:

The display() method is a shortcut to output the template directly.

So, in your controller, you should be able to do something like this:

$template = $twig->loadTemplate('index.html');
$htmlString = $template->display(array('the' => 'variables', 'go' => 'here'));

And then pass the $htmlString back to the view.
