You violated here almost every rule of OOP. there shouldn't be variables like _query in your class.

What you really need is just these five lines:

public function runQuery($sql, $bind_value = array()) {
    $stmt = $this->_pdo->prepare($sql);
    $stmt->execute($bind_value);
    return $stmt;
}

which have to return a statement, not set it as a class variable.

get() function is also useless and wrong and you better avoid it. If you really want a function to automate select queries, you have to look at the existing query builders, like AuraSQL.
