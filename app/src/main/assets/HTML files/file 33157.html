http://www.cplusplus.com/reference/numeric/adjacent_difference/

  • adjacent_difference requires an output iterator as its third parameter.
  • it returns an iterator

So you'll be wanting to do something like:

auto dif_iter = adjacent_difference(prices.begin(),prices.end(),back_inserter(dif));

And then dif will be filled with the difference, and iter will be an iterator to where the differences begin.

By the sounds of it you'll want to iterate over all the differences to show the price change each day. So you'll be wanting to put that in a loop:

for( auto iter = dif_iter; iter != dif.end(); ++iter )
{
  cout << ... some information with *iter ...
}
