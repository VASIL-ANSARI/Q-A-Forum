How do I create a custom archive page that my client can edit?

The Solution

  1. I created a page called archive-mycustomposttype.php which wordpress automatically uses as the archive page for your custom post type, if it exists.
  2. I then created a new section in wordpress's customizer and added controls with text, textarea and image settings.
  3. I called the saved values via in the archive-customposttype.php template.

I hope this helps
