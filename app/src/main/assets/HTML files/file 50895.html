If your variables are set as Strings, which is likely the issue here, convert them to numerical values.

You have three options:

  1. Use the function parseFloat() (or parseInt() for Integer values)

  2. Add the plus sign before the variables, so instead of widthDimension write +widthDimension

  3. Use the Number() function, e.g. Number(widthDimension) (as noted by Sterling Archer in the comments)
