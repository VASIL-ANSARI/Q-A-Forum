It seems, that you don't have a database server running and you try to connect to a .sqlite-File. If you have setup a database server, you can establish a database connection like this:

Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/");

And this line
String query = "insert into EmployeeInfo(EID,name,surname,age) values(?,?,?,?)";
could look like
String query = "insert into EmployeeInfo(EID,name,surname,age) values(textField.getText(),textField_1.getText(),textField_2.getText(),textField_3.getText())";
