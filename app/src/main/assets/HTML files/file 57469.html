Use the "toArray" method instead of the "serialize" method.

var arr = $("#sortable").sortable('toArray');
$.each(arr, function(index, value) {
    alert(value);
});

BTW, a String of the following form is not JSON. In fact, it is not even close to JSON.

item[]=51&item[]=41&item[]=49&item[]=35&item[]=58&item[]=45&item[]=32&item[]=36&item[]=26&item[]=27&item[]=33&item[]=31&item[]=29

A JSON string for an array would look like:

[51, 41, 49, 35, 58, 45, 32, 36, 26, 27, 33, 31, 29]
