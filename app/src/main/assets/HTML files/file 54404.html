You can assign your infoWindow like this:

var directionsDisplay = new google.maps.DirectionsRenderer({
  infoWindow: myInfoWindow,
  map: map
});

And you can create editableInfoWindow. See here. http://googlemaps.googlermania.com/google_maps_api_v3/en/editable_infowindow.html

Another answer is that just disable infoWindow:

var directionsDisplay = new google.maps.DirectionsRenderer({
  suppressInfoWindows: true,
  map: map
});

And you can parse the directionResult, then add infoWindows by yourself. Read this page: http://googlemaps.googlermania.com/google_maps_api_v3/en/map_example_direction_customicon.html
