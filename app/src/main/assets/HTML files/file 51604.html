You can do this by adding :

// Span last child
span:before {
    content: '';
    display: inline-block;
    margin-right: 10px;
}

// Span first child
span:after {
    content: '';
    display: inline-block;
    margin-left: 10px;
}

To your First-child-span ou your Last-child-span button type. Without adding a class to these two types of button I don't think this would be possible.
