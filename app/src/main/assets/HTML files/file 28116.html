I see you are calling your window by the same name and yet they are still opening in a new window, it could be because your onclick function hasn't get specific scope to save the reference to the window.

If you were to change onclick to a function you can cache the window.

//js
var myWindow;
function myClick(){
    myWindow = window.open("http://www.google.com", "infoWindow", "width=300, height=250");
    return false;
}

//html
<a href="#" onclick="myClick()">info1</a>
