You can set it to whatever you want by using set_unixfrom:

>>> from email.mime.multipart import MIMEMultipart
>>> msg = MIMEMultipart('csv')
>>> msg.set_unixfrom('From user')
>>> print msg
From user
Content-Type: multipart/csv; boundary="===============9221413516749323109=="
MIME-Version: 1.0

From nobody is hardcoded value if unixfrom is not set for a given message.
