Why do you want JSNI code for that?

You can use the following

Checkbox checkbox = new Checkbox("Some-Label");
checkbox.getElement().setAttribute("indeterminate", true /*Or false*/);

EDIT

Just checked some docs and blog posts. Apparently, GWT doesn't support indeterminate in checkboxes. Apparently there is a work around.

Whenever you want the indeterminate value, set the checkbox to null. Then use CSS tricks to style the checkbox.
