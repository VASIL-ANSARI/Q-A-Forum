There were a few things wrong,

Firstly, your HTML code was not formatted quite correctly opacity: 1;"> on both image divs. It's not even needed so I have removed it. Then you were only fading one image in and out the whole time and I assume you were laying them ontop of one another which mean that the text/links were always visible and on-top of one another. In the demo I have absolutely positioned the .hover elements to overlap. Then I moved the active class to the first .hover as the next() call you were making was targeting the .mainportfo_title div of the first .hover. ll that completed, I made the active fadeOut and the next fadeIn. Also all that z-index stuff isn't needed if one is hidden while another is shown. If you wanted more slides then that's also possible as they are all display:none to begin with.

HTML:

<div class="porto_container">
    <div id="fade-group">
        <div class="hover active"> <a href="#">
            <img width="250" height="250" alt="image1" src="http://dummyimage.com/400x260/000/fff.png&text=Image+01"/>
            <div class="mainportfo_title">image 1</div>
        </a> 
        </div>
        <div class="hover"> <a href="#">
            <img width="250" height="250" alt="image2" class="attachment-post-thumbnail" src="http://dummyimage.com/400x260/000/fff.png&text=Image+02"/>
            <div class="mainportfo_title">image 2</div>
        </a> 
        </div>
    </div>
</div>

CSS:

.hover {
    position:absolute;
    top:0;
    left:0;
}
.hover {
    display:none;
}
.hover:nth-child(1) {
    display:block;
}

JS:

setInterval(function () {
    var $active = $('#fade-group .active');
    // Select element following the element with the class .active within #fade-group
    var $next = ($active.next().length > 0) ? $active.next() : $("#fade-group .hover:first");
    // Fadeout the active img element
    $active.fadeOut(1500, function () {
        // Active img element
        $active.removeClass("active"); // Remove active class
        // Next img element 
        $next.addClass("active"); // Add active class
    });
    $next.fadeIn(1500);
}, 3000);
