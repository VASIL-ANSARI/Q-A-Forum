You can update the height of your container element in a window.resize event handler:

$(window).bind('resize', function () {
    $('#container').height($(this).height());
});

This is of-course very simplified, you'll most likely have to do a bit of math to figure-out the exact height of the container.

I would also add a timeout so this only fires once per resize (different browsers send different numbers of resize events as the browser resizes):

var timer;
$(window).bind('resize', function () {
    clearTimeout(timer);
    timer = setTimeout(function () {
        $('#container').height($(this).height());
    }, 50);
});

As the address bar is hidden, resize events are fired since the view-port is changing sizes.
