Transpose then reshape:

>>> t.transpose(1, 2, 0).reshape(4, 3)
array([[ 1,  5,  9],
       [ 2,  6, 10],
       [ 3,  7, 11],
       [ 4,  8, 12]])

Edit: alternatively, you can reshape then transpose:

>>> t.reshape(3, 4).T
array([[ 1,  5,  9],
       [ 2,  6, 10],
       [ 3,  7, 11],
       [ 4,  8, 12]])
