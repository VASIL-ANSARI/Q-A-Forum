Are you using the iOS8 SDK? iOS8 introduced UITextEffectsWindow - this gets created in addition to the application window so @app.windows.size == 2 and not 1. You can check this by running your app and in the REPL looking at the 2 window objects.

(main)> UIApplication.sharedApplication.windows[0]
=> #<UIWindow:0x11280d700>
(main)> UIApplication.sharedApplication.windows[1]
=> #<UITextEffectsWindow:0x10b6127a0>

I would suggest modifying the default spec to one which tests that the application window has a default rootViewController for example.
