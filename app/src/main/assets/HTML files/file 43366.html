Using plain HTML is certainly one way, as suggested by LJ_1102, and for this needs, maybe a simpler one too.

The process to create this effect with webGL would be as follows:

create canvas, draw one square (two triangles in total) over entire canvas, take mouse position on the canvas and calculate percentage of how far left/right is it from one of the edges; and for that square use some fragment shader similar as this one bellow.

uniform float percentage;
uniform vec2 canvas_size;
uniform sampler2D left_image;
uniform sampler2D right_image;

void main() {
    vec2 uv = gl_FragCoord.xy/canvas_size;
    if ( uv.x<=percentage )
        gl_FragColor = texture2D(left_image,uv);
    else
        gl_FragColor = texture2D(right_image,uv);

}
