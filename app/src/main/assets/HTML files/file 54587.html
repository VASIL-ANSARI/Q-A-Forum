Not entirely sure what you're asking for, but the code should look something like this to make sense:

<th>
    <select name="first_allocation[<?=$rowId?>]">
        <option>Select Faculty</option>
        <? foreach ($faculties as $faculty) { ?>
            <option value="<?=$faculty?>"><?=$faculty?></option>
        <? } ?>
    </select> 
</th>
<th>
    <select name="second_allocation[<?=$rowId?>]">
        <option>Select Other Faculty</option>
        <? foreach ($faculties as $faculty) { ?>
            <option value="<?=$faculty?>"><?=$faculty?></option>
        <? } ?>
    </select>
</th>

That way you would receive something like this when the user submits the form:

$_POST = array(
    'first_allocation' = array(
        232 => 'BCS-IIIB-CSC509-Class',
        372 => 'BCS-IIIB-CSC509-Class',
        // ...
    ),
    'second_allocation' = array(
        232 => 'BCS-IIIB-CSC509-Class',
        372 => 'BCS-IIIB-CSC509-Class',
        // ...
    ),
);

Where 232 and 373 are the respective values of $rowId, changing on each iteration of the outside loop.

If you already have the selected faculties and want to mark them as such use this:

<option value="<?=$faculty?>"<?=($first_allocation[$rowId] === $faculty ? ' selected' : '')?>>
    <?=$faculty?>
</option>
