I don't think the standard route type is set to use dot as a separator. You could use a regexp route instead:

$routes = array(
    'twitter_output' => new Zend_Controller_Router_Route_Regex(
        'twitter/([\w]+)\.([\w]+)',
        array('controller' => 'twitter', 'action' => 'index'),
        array(1 => 'username', 2 => 'format'),
        'twitter/%s.%s'
    )
);
