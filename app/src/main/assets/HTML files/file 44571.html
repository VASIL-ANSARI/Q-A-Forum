You can use the regex feature of NetBean's Find in Projects function, accessible via Ctrl+Shift+F.

A regex such as catch\s*{ should be sufficient to take you to all of your catch blocks through repeated iterations, and with the line you want to add placed in your clipboard, you can add it easily to the block with copy + paste.

If you averaged even a few seconds for each replacement, the whole process would take you less than ten minutes.
