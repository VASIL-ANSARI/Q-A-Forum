If you put the line "app:tabMode="scrollable", then the tablayout width will not fill the entire screen. The best solution is this. Put this line in XML

    app:tabMode="fixed"

and in Java, do

    if(tabLayout.getTabCount() > MAX_NUMBER) {
       tabLayout.setTabMode(TabLayout.MODE_SCROLLABLE);
    }

MAX_NUMBER is some number depending on your requirement. I have used 3, you may use 3,4 or whatever suits your requirement. This will ensure that if there are only a few tabs, they will occupy the entire width, and be scrollable if not.
