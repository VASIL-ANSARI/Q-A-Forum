Ive created an easy to read version of your code here that just focuses on the query that is failing.

function add_to_cart(){
   var quantity = $("#quantity span").html();
   console.log(quantity);    // 1
}

Initially your code is correct and a value of 1 is logged, however as soon as you increment/decrement the value it fails. This is due to your click event handlers

$(this).parent().find('.value')

selects the spans parent div and overwrites its contents. Appending the same selectors to your increment/decrement functions should solve the issue.
