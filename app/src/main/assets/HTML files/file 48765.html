You can't store the different types.

You can store dynamic instances by polymorphic reference (std::reference_wrapper) or (smart) pointer, iff your classes share a common base.

Otherwise you should use static polymorphism. You could look at boost::variant<GLSurface<float>, GLSurface<int>> for example.

To operate on these uniformly, you will end up doing something that's a hybrid between static and runtime polymorphism: type erase.

With the type erasure technique you basically bring (unrelated) types under a shared ad-hoc interface.
