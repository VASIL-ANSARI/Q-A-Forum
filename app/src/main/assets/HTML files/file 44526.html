You can make an ajax call trigger with JQuery after clicking the button. Then, you can append the result returned by you PHP API via ajax to your page.

Example:

HTML: 

<form action="process.php" method="POST">
    <button id="button">Send</button>
</form>
<div class="appendText"></div>

JQuery:

$("#button").click(function(){
 var myData = $this.text();
 $.ajax({
            url: "process.php",
            type: "POST",
            data: {
                data: myData

            },
            dataType: "json",
            success: function(data){
                $(".appendText").append(data); 
            }
    });
