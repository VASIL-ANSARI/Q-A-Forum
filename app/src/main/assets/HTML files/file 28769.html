First of all, since your id must be unique, you don't need to do a while loop.

And since you are returning an ajax call, JSON might be the best data format (and the default for jQuery).

So in your getdata.php:

if (isset($_POST['id']) === true && empty($_POST['id']) === false) 
{
    require '../connect.php';

    $query = "SELECT * FROM `members` WHERE `memberNo` = '"
            . mysql_real_escape_string(trim($_POST['id'])). "' ";
    $result = $db->query($query);
    $row = $result->fetch_assoc()
    // setting the appropriate content type
    header('Content-Type: application/json');

    echo (json_encode($row));
}

And you process this in the success callback:

var id = $('input#id').val(); //get value of input box
$.post('./ajax/getdata.php', {"id": id}, function(data)
{
    $('input#name').val(data.firstname);
    $('input#age').val(data.age);
    $('input#email').val(data.email);
}
