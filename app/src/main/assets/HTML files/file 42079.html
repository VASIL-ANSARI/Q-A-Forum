If you want courses with an active module, I would go for an exists:

select c.*
from course c
where exists (select 1
              from CourseHasModul chm join
                   Modules m
                   on chm.IdModule = m.IdModule
              where chm.IDCourse = c.IDCourse and
                    m.datainici <= CURDATE( ) AND m.datafi >= CURDATE( )
             );

However, this query should return results similar to your query (albeit with no duplicates for courses that have multiple active modules).
