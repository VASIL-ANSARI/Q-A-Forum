I am able to solve the issue.

One of the dll causing reference issue, but build is getting succeed.

Solved by following these steps

  1. Checked in the temporary folder(C:\Windows\Microsoft.NET\Framework\v4.0.30319\Temporary ASP.NET Files) where VS Creates dlls for each page( i have opted for individual assemblies while publishing)
  2. Before creating dlls, it generates .Compiled files and then converts each file to a dll.
  3. we found one of the page is not getting converted to dll, instead it created a file with .delete extension and halted.
  4. Now we ran our application from VS and tried open the page which is not able getting converted to dll.
  5. Now we got the issue as an exception.

Hope this will help some one else.
