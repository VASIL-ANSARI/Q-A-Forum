I don't see why your given example wouldn't work, however you have some redundancies in your code which will create unnecessary overhead. As you are already selecting all desired elements using $(".avatar"), and jQuery calls like this return an array, there is no need for you to cast the result to an array and iterate over its members manually. We can call the animate method on all elements returned by $(".avatar") like so:

$(document).ready(function(){
    $(".avatar").animate({height: "100px", width: "100px", left: "-=50px",top: "-=50px"}, "slow");
});
