Its case where click is binded two times. See for a possibility where you have done that. eg. in js file and as well as inside script tag in html document.

Also if container is dynamic, you should use live instead of click.

$("selector").live("click", function() { /*body*/ })

You can try this.

$(document).ready(
function(){
    $("body").click( "#btn_toggle_transcript",
        function(){
            if(!$("#transcript_container").hasClass('opened')) {
              $("#transcript_container").slideDown(1000).addClass('opened');
            } else {
              $("#transcript_container").slideUp(1000);
            }
        }
    );
}
);
