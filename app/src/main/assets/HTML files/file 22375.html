You can modify jsDatePick js file, and add that feature like this:

if (this.oConfiguration.disableNotFriday){ 
    if (parseInt(oDay.getDay()) != 5  ){
      disabledDayFlag = true;
      aDayDiv.setAttribute("isJsDatePickDisabled",1);
}
}

You also need to add to configuration:

this.oConfiguration.disableNotFriday    = (aConf["disableNotFriday"] != null) ? aConf["disableNotFriday"] : false;

and on javascripty call object like this:

new JsDatePick({
        useMode:2,
        target:"inputField",
        dateFormat:"%d-%M-%Y",
        disableNotFriday:true
    });

Example is set here: http://jqfiles.webege.com/calendar.html
