You want an outer container which sets the positioning and width. Then absolutely position within it. Then set the one on the right according to its right position and width; the one on the left according to the sides.

<div style="position:relative; width: 100%; height: 2em;">
  <div style="position: absolute; right: 0; width: 30px; top: 0; bottom: 0;">...</div>
  <div style="position: absolute; left: 0; right: 30px; top: 0; bottom: 0;">...</div>
</div>
