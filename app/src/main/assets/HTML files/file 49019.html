Simple barboning could be by order of decreasing size :

pip uninstall mingwpy

pip uninstall scipy

pip uninstall numpy

You may then try re-installing the 'vanilla' light version of numpy using http://www.lfd.uci.edu/~gohlke/pythonlibs/#numpy

Then, if it's not enough:

pip uninstall scikit-image

pip uninstall pywin32

==> You should be 33% of the initial size at this point (rough guess, let us know)
