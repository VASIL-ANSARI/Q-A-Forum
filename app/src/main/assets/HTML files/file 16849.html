To explain I will copy and paste code -

private static void Receive(Socket client) {
    try {
        // Create the state object.
        StateObject state = new StateObject();
        state.workSocket = client;

        // Begin receiving the data from the remote device.
        client.BeginReceive( state.buffer, 0, StateObject.BufferSize, 0,
            new AsyncCallback(ReceiveCallback), state);
    } catch (Exception e) {
        Console.WriteLine(e.ToString());
    }
}

The receive function above is called when a successful connection with the server is established. Notice the call to BeginReceive it takes a new AsyncCallback functiona as a parameter. Whenever there is data sent by the server this function (ReceiveCallBack) will be called which will do something with is it. This just works!
