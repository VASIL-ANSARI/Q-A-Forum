First of all you need a database that both can talk with. In that database you would store a login token for a specific user Object. You generate the token via RijndaelAES or something comparable. Then you just link to the next site where you take the token as request parameter.

Be sure to limit the lifetime of the token to about 10 minutes or something like that. Define a place where you say "buildTokenForTransfer" and "destroyTokenAfterTransfer".
