The NSAffineTransform class provides methods for creating, concatenating, and applying affine transformations.

rotateByDegrees: Applies a rotation factor (measured in degrees) to the receiverâs transformation matrix.

rotateByRadians: Applies a rotation factor (measured in radians) to the receiverâs transformation matrix.

Example:

{
    NSAttributedString * myString = [[NSAttributedString alloc] initWithPath:mypath documentAttributes:NULL];
    NSAffineTransform* xform = [NSAffineTransform transform];
    [myString drawAtPoint:NSMakePoint(10,10)];
    [xform rotateByDegrees:-90.0];
    [xform concat];
    [myString drawAtPoint:NSMakePoint(-150,0)];
    [xform invert];
    [xform concat];
    [myString drawAtPoint:NSMakePoint(15,15)];
    [myString release];
}
