This happens because we don't recalculate the position of the DIV after the menu is invoked. What I would do is disable the compact menu and set the button to only use the expanded (full) menu, which is auto centered.

So change <a class="addthis_button_compact"></a> to <a class="addthis_button_expanded"></a>
