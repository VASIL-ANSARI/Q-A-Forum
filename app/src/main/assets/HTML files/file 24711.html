The img element actually defines height and width attributes (which you want to set on the server side to help with rendering). As you use thumbnails to display the size, you will not be able to utilize them directly for holding this extra information.

For elements that don't define them you can use data-* attributes to hold the information - you can do this on the img elements as well. jQuery has the data functions exactly for these.

Example:

<img src="..." height="100" width="100" data-actual-height="300" 
                                        data-actual-width="400" />
