In first look at thi segment of your code:

if (lastPosition.Y < destinationPosition.Y)
{
    north = true;
    south = false;
    east = false;
    west = false;
}

if (destinationRectangle.Y > lastPosition.Y)
{
    north = false;
    south = true;
    east = false;
    west = false;
}

The (lastPosition.Y < destinationPosition.Y) and the (destinationRectangle.Y > lastPosition.Y) is same conditions, so your will have south = true in both situations.

Second, why are you using four boolean variables? Instead use one variable with four possible states. Enums is good for this:

enum Direction
{
    None = 0, // default
    North = 1,
    South = 2,
    East = 3,
    West = 4
}

And instead of four boolean use one Direction variable. Less variables is less troubles.

And last. I'm not understand properly how are you moving your sprites? I mean how are you detect direction before changing coords? In my opinion it should by like that:

  1. In Update() detect in what direction you need to move and change Direction varialbe and basing on it change and update sprite state.
  2. In Draw() anly draw sprite in current state.
