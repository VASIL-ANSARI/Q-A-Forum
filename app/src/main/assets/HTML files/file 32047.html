In your local environment, you're running the Ruby process from the Rails root directory. This is why you're able to locate the file using '../hub/data/comment_file.csv' (or probably just './data/comment_file.csv'

However, Heroku and many other hosts will run it from other locations.

You'll have to specify the full path of the file:

File.join(Rails.root, 'data/comment_file.csv')
