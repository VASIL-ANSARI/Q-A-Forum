You can use zip on the ndarray and pass it to numpy.array:

In [36]: a = np.array([[1,2,3], [4,5,6]])

In [37]: b = np.array(zip(*a))

In [38]: b
Out[38]:
array([[1, 4],
       [2, 5],
       [3, 6]])

In [39]: b*2
Out[39]:
array([[ 2,  8],
       [ 4, 10],
       [ 6, 12]])

Use numpy.column_stack for a pure NumPy solution:

In [44]: b = np.column_stack(a)

In [45]: b
Out[45]:
array([[1, 4],
       [2, 5],
       [3, 6]])

In [46]: b*2
Out[46]:
array([[ 2,  8],
       [ 4, 10],
       [ 6, 12]])
