I don't believe you can animate transform using the animate function. You use CSS transitions though, and add them using jQuery. For example:

$('h2').css({
    'transform': 'translateY(-100px)',
    'transition': 'transform 3s ease-in'
});

DEMO: http://jsfiddle.net/StathisG/pa3f61bc/

Apart from the above, the problem with your function is that you cannot pass variables into the animate function this way.

So, instead of this:

return $(element).animate({
    property : value
}, timing);

You can do the following:

var animation_options = {}
animation_options[property] = value;
return $(element).animate(animation_options, timing);

DEMO: http://jsfiddle.net/StathisG/0bq56kaz/

Also, I removed the setTimeout that you had there, since it doesn't seem to make sense (at least in this example).
