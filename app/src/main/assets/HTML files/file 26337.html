As first step I'd suggest you check this link Using Custom Renderers.

Now regarding your question, you can implement your own TableCellRenderer (in this case extending DefaultTableCellRenderer) as follows:

public class CustomCellRenderer extends DefaultTableCellRenderer {
    @Override
    public Component getTableCellRendererComponent(JTable table, Object value, boolean isSelected, boolean hasFocus, int row, int column) {
        if(column == 3 && ((Integer)value).intValue() > 90) {
            setBackground(isSelected ? Color.red : Color.red.brighter());                
        } else {
            setBackground(isSelected ? /* some color if selected*/ : /* some color if not */);
        }
        setForeground(isSelected ? Color.white : Color.black);
        setText(value != null ? value.toString() : "<null>");
        return this;
    }
}

Then set up a new CustomCellRenderer to your table before of populate it:

vitalSignJTable.setDefaultRenderer(Object.class, new CustomCellRenderer());
