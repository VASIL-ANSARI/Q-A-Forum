It's possible using ajax .

You have to write some thing like a timed request on your jsp to invoke a separate request to check the status of the submit action.

setTimeout(function() {
    $.get("checkStatus",function(response){
       //update the message
    });
}, 3000);

Put the above code inside a while loop. Make sure that the loop starts only once you clicked the submit button.

Since the ajax requests are asynchronous the loop will not wait for the request to finish so make sure to provide enough timeout.

Once the processing is done break out of the loop by changing.

The status check response I use json so that I can send more information like status of the action and the message or any other data you want to pass.

Complete code :

var isLopp = true;
$("submitButton").on("click", function() {
    $(this).attr("disable", true);
    while (isLopp) {
        setTimeout(function() {
            $.get("checkStatus", function(resp) {
                if(resp.status="done"){
                    isLopp=false;
                }
                $("message").text(resp.msg);
            },"json");
        }, 3000);
    }
});

Instead if while loop you can also use window.setInterval("javascript function", milliseconds);
