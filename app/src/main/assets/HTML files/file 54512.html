Your main problem is that you've misplaced most of the contents of your Donor class. What you have is structured like this:

public class Donor
{
    private String lastName;
}

public String getLastName()
{
    return lastName;
}

While what you want is more like this:

public class Donor
{
    private String lastName;

    public String getLastName()
    {
        return lastName;
    }
}

All the members of the class need to be inside the curly braces that delimit the contents of the class, not just the fields.

Additionally you seem to be calling super() with arguments in a constructor that's meant for a class that doesn't have a superclass with a constructor that takes arguments. That doesn't make sense.
