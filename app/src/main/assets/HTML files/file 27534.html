Your argument list needs to be comma separated, then you can pass several of them in one operator<< call.

OpenCV does this. Here is how it is to be used (search for comma-separated): http://docs.opencv.org/modules/core/doc/basic_structures.html?highlight=comma-separated#mat

Have a look at their code of cv::Mat_ here: https://github.com/Itseez/opencv/blob/master/modules/core/include/opencv2/core/matx.hpp line 840

It uses an auxiliary object, your task is to find out how it works.
