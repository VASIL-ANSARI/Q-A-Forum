You should be able to fix the issue by adding text-align: center to #chicken as below:

#chicken {
    margin: 0;
    width: 100%;
    height: 100%;
    position: absolute;
    top: 0;
    left: 0;
    text-align: center; /* add this */
}

Another way you can do it is by adding this rule:

#chicken a{
    display: block;
    text-align: center;
}
