You can use the T Type Operator to call static methods on a class. So you can just call String.format() as you usually would. eg.

${T(java.lang.String).format('0x%06X', intColor)}

The only slight limitation here is you can't perform the bitwise and from within Thymeleaf. You'll either need to do this before you add it to the model, or maybe look at Integer.toHexString() which will allow you to format your number as if it were unsigned:

'0x'+${T(java.lang.Integer).toHexString(intColor).toUpperCase()}

This second approach requires some additional work to get the leading 0x and to uppercase the characters.
