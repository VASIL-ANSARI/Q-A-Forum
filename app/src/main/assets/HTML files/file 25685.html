Before you call [tableView scrollToRowAtIndexPath:indexPath..., do this:

BOOL visible = NO;
for (UITableViewCell *cell in [tableView visibleCells])
    if ([indexPath isEqual:[tableView indexPathForCell:cell]]) {
        visible = YES;
        break
    }

If you need to, store visible as a member variable of your view controller so you can check its value when you handle some delegate method.
