I'm not familiar with the technical details of HTAs, but if they run the JavaScript using Microsoft's JScript, and if they do so without the usual browser-based security, you could use the various objects available to Windows Scripting Host, including (most notably) the FileSystemObject in order to enumerate the files and directories.

The rest is just straight-forward DOM manipulation, and if you're allowed to use jQuery in the HTA (you have the jquery tag on your question), there are plenty of treeview plug-ins for it, such as jsTree.
