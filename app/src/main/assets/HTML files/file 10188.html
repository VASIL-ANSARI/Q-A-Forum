There could be a number of reasons.

Does the slideshow wait for all the images to load before it starts? If that's the case, consider putting the slideshow script at the bottom of the html page, if possible. That way, all the other content loads first.

Is the slideshow animated? Do images slowly move across the screen every 2500 seconds? If this is the case, the code you're using may have too small of a value for its setTimeout() function. Consider raising it, or risking it and using CSS3 transitions for animating.
