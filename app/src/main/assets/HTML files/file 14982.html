This doesn't answer my exact question, but I was able to bypass it by adding -jobconf stream.recordreader.compression=gzip to my Hadoop command (source where I learned this):

hadoop jar /usr/lib/hadoop/contrib/streaming/hadoop-streaming-*.jar \
    -jobconf stream.recordreader.compression=gzip \
    -D mapred.reduce.tasks=0 \
    -file map_to_old_format.py \
    -mapper map_to_old_format.py \
    -input /mydata/* -output output/newdata
