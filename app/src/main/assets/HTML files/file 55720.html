To get a random element from array:

function randChoice(array){
    return array[Math.floor(Math.random()*array.length)];
};

There are 2 method to generate a mesh from random points that I know of: convex hull and alpha shape. Creating a mesh by repeating picking 3 random points would almost surely result in spaghetti.
