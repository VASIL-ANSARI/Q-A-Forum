int* Arrpoint = &DataArr does not make a copy of DataArr. It simply makes Arrpoint point to DataArr, so whatever changes you do through Arrpoint will be reflected in DataArr.

You must copy the array to do what you want in this case.

int a[5]={5, 4, 3, 2, 1};
int* b = malloc(5*sizeof(*b));       // making new array b
for (int i=0; i<5; i++) b[i]=a[i];  // copying a to b
print(a, 5);                       // display a (unsorted)
sort(b, 5);                       // your sort function
print(b, 5);                     // display b (sorted)
print(a, 5);                    // display a (unsorted)
