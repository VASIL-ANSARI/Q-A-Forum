I think you can use UITableView and custom UITableViewCell for this purpose. On click of "Add More", add one entry to NSMutableArray and just reload tableview. And no need to manage index separately, you can get it directly by indexPath.row

Also you don't need to manage scroll view size as tableview automatically adjust is content height according to row.

If you are using same kind of view,then go with this approach, as its easy and will save your time.
