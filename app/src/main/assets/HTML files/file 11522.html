To connect to mysql through CLI with a specified user you need to add the option -u at the command line, so : 

mysql -u user -p

where p parameter ask for password.

You get that error most likely because there is something wrong in the table mysql.user , how do you have created the user ?

Try creating your user in this way : 

GRANT ALL ON yourdb.* TO youruser@'%' IDENTIFIED BY 'yourpassword';
