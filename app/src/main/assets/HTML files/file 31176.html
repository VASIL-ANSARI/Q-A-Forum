At a high level:

  1. Store your PDFs outside of your webserver document root. Make them completely inaccessible to direct browser access.
  2. Write a PHP script to handle download requests. This PHP page can check the session user ID to ensure the user is requesting a file that they are allowed to access.
  3. Use header() calls and readfile() to then send the appropriate PDF file to the user.

Feel free to come back and post a question when you've researched and worked on this, and have a specific question with code.
