In order to access any wro4j related details, you have to make the Context instance available. This can be easily achieved using WroContextFilter:

<filter>
  <filter-name>WroContextFilter</filter-name>
  <filter-class>ro.isdc.wro.http.WroContextFilter</filter-class>
</filter>
<filter-mapping>
  <filter-name>WroContextFilter</filter-name>
  <url-pattern>/*</url-pattern>
</filter-mapping>

This filter is responsible for setting the Context object to the current request cycle.
