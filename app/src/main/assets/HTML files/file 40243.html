What you need is Clear the floated elements that are inside the header, with that you can remove the overflow:auto too. Try this:

div.header {
  margin: 8px 8px 8px 8px;
  width: 100%;
  background-color: #fff;
  display: block;
  position: relative;
  border: 1px solid #dedede;
  border-radius: 4px;
  box-shadow: 0 2px 2px -1px rgba(0, 0, 0, 0.055);
}
div.header:after {
  content:" ";
  display:block;
  clear:both;
}
