What you are looking for are stacked if-statements:

SELECT title, IF('title' = 'Mr.', 1,
    IF (title = 'Mrs.', 2, 
        IF (title = 'Dr.', 3, 0)
    )
) 
AS title_number 
FROM users...

title | title_number
