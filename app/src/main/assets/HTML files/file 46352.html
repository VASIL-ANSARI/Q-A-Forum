The jQuery Validation plugin is automatically triggered by certain user events. However, since Select2 is dynamically changing a hidden input value in the background, no validation is automatically triggered.

You simply need a change handler that programmatically triggers a validation test whenever the value of the hidden element is changed.

$('input[name="tags"]').on('change', function() {
    $(this).valid();    
});
