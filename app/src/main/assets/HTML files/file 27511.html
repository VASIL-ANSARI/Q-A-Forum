How about:

$(function(){

  $('select').focus(function(){

    $(".nav-btns").hide();

  });
});

This should bind the focus event to all of your select elements, and then hide the element with the class .nav-btns.

For undoing the change on an 'unfocus':

$(function(){

  $('select').focus(function(){

    $(".nav-btns").hide();

  }).blur(function(){

    $(".nav-btns").show();

  });
});
