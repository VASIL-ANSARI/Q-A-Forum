You have the right idea, but need some minor changes to make it work.

The problem is that your code is creating multiple independent timers. The image updates are not coordinated with each other. So one solution is to create a single central timer that updates the image. In my example, you add images (url) to an array buffer. The timer then pulls them off (shift) in the order in which they were added. There are a number of way you could do this, but this should get you started int he right direction.

Run the code snippet to try 

var images = [];



// append images to the buffer

function addImage(url) {

  images.push(url);

  if (!window.image.src) updateImage();

}



// central timer 

function updateImage() {

  if (images.length > 0) {

    window.image.src = images.shift();

    window.caption.innerHTML = (new Date()).toTimeString();

  }

}



// start the timer

setInterval( updateImage, 4000);
