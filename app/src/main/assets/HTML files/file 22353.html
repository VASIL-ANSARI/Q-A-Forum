It seems you have the same subquery in all the conditions.

You can use concatenation in this way to save code:

//common subquery
$conditions = array('AND' => array(
                        'autori LIKE' => "%$a%", 'editore LIKE' => "%$e%",
                        'titolo LIKE' => "%$t%"));

//for example...  
if(...){
    $conditions+=array('AND' => array('anno <=' => "$an2"));
}else{
    $conditions+=array('OR' => array('anno BETWEEN ? AND ?' => array($an,$an2)));
}

$u = $this->paginate('CdBiblio',$conditions);

You can also change the comparisons:

if ($an==''&&$an2=='')

for

if (!$an && !$an2)

Oh and... i would use real words for variables. It is always better to understand what is what rather than creating variables with no sense, such as $a. This makes things easier for future modifications of the code as well as preventing coding mistakes.
