Since you are using Json.Net. You can use a service, such as http://json2csharp.com/ to generate a DTO object that you can deserialize. After doing that, you get several classes, one of them is called RootObject. You can rename this to whatever you desire. Then you can do something like:

var dtoObj = JsonConvert.DeserializeObject<RootObject>(response);

Where response is you Json in string format. This method will give you the advantages of DTO objects, instead of dealing with JArray. To access your data it would be dtoObj.Result.Matches.

Your other option is to use dynamic.

var dynObj = JsonConvert.DeserializeObject<dynamic>(response);

The benefit of dynamic is you don't have to maintain DTO objects, however you lose some of the enforced typing and structure. But with dynamic you would access matches by dynObj.result.matches. Sometimes with dynamic you have to explicitly cast it to some sort of Enumerable, which is annoying, so I prefer the DTO method above.
