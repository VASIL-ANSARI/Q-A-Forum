In the view controller your game scene appears add

[self setInterstitialPresentationPolicy:ADInterstitialPresentationPolicyManual];

in viewDidLoad

You also need the view controller to call 

[self requestInterstitialAdPresentation];

So you can create a protocol in GameScene.h with a method like showInterstitialAd. Like:

@protocol GameSceneDelegate <NSObject>
- (void) showInterstitialAd;
@end

Add a delegate property to GameScene.h

@property (nonatomic, weak) id <GameSceneDelegate> delegate;

Have the view controller as it's delegate and implement something like and declare it conforms to the GameSceneDelegate protocol. 

-(void)showInterstitialAd{
  [self requestInterstitialAdPresentation];
}

In the viewDidLoad of GameSceneController set the gameScene.delegate = self;

When you want to show the add in GameScene call [self.delegate showInterstitialAd];
