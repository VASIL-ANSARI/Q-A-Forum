Here are some tips:

  • Separate the cases where you get an open bracket from the ones where you get a closing bracket by using something like if/else if statements

  • When you get an open bracket, you want to push it on the stack and that should be it

  • When you get a closing bracket, you want to verify the top of the stack to see if there is an associated opening bracket of the same type (if this is relevant for you), and remove that opening bracket by popping the stack.

  • When the loop finishes, the stack should be empty, or else there is an error

You probably want this structure:

for (size_t i=0; i<ie.size(); i++)
{
    if (/*conditions for open brackets*/) 
    {
        //Push the bracket on the stack
    }
    else if (/*conditions for closing brackets*/)
    {
        //Check the top of the stack if there is an opening bracket
        //If so, pop the stack. If not, error.
    }
}
