Problem is you use one dropdown mechanism in another dropdown mechanism (that works the same way).
You need to get your hands dirty to make this work.
First of all you submenu is showing on hover (which is not good as you know) instead of opening on click (delete CSS rule(s) which responsible for that).
If menu still dont work well, then write use jQuery click method for .dropdown-submenu .dropdown-toggle class. Inside method type something like this:

e.preventDefault();
$(this).closest('.dropdown-submenu').children('.dropdown-menu').toggleClass('opened');

add to CSS something like this:

.dropdown-menu.opened {display: block;}
