I'm assuming initgraph takes a char* parameter, and you passed it a string literal, where it decays to a const char*. String literals are immutable, and by accessing it through a char* it would become possible to try to modify it and invoke UB.

I don't see why it would take a string by non-const pointer unless it intended to modify it, so you probably want to save the literal in an array.

char str[] = "blahblah";
initgraph(..., str);
