If you check the documentation, you'll se that func viewWithTag(_ tag: Int) -> UIView? returns one view with the tag, not all views. What I would do is create IBOutletCollection if you load the views from storyboard or just array of the views, like this:

@IBOutlet var buttons: [Button]! // for IBOutlet collection
// or
var buttons: [Button]!

then you just need to iterate through the array:

for button in buttons {
    if button.backgroundColor == self.accentColor {
        button.backgroundColor = UIColor.MDColor.white
        button.title?.textColor = UIColor.MDColor.grey800
    }
}
