"W\xF6rth".valid_encoding?
=> false

I think you're trying to convert ISO-8859-1 to UTF-8.
In ISO-8859-1 "Ã¶" is 0xF6.

"W\xf6rth".force_encoding('iso-8859-1').encode('utf-8')
