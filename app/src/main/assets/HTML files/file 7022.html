As an aside: you're probably better off doing what you're doing already for the owner of the room in a before_create rather than a before_save so that it's only done on the initial creation of the room (not on any subsequent updating).

Now to your question...

The action of joining the room will be a click by the user on some link (probably a POST form/link). You should map that to an action in your controller which does something like this (I assume they're logged in, and that you have current_user available):

def join_room
  @room = Room.find params[:id]
  current_user.join_room @room
end

Then in the User#join_room method you'd have it create whatever record(s) you need.
