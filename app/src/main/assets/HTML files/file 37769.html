If the image is not in the same folder as your js file, you'll have to navigate to the image folder to retrieve the image.

From what you say, your folder structure looks something like this:

/root/ 
   /js/myJSFile.js
    /css/myCSSFile.css
    /img/closed.gif

You can access your closed.gif image using a relative link like so:

ddtreemenu.closefolder="../img/closed.gif"

The ../ tells it to exit the current folder, /js/, and then it can navigate to the closed.gif file the rest of the way.
