I would do, these are called list comprehensions:

>>> original = ['state1', ',', 'state2,st6']
>>> [[item] for string in original for item in string.split(',') if item]
[['state1'], ['state2'], ['st6']]

or if you don't want nested lists:

>>> [item for string in original for item in string.split(',') if item]
['state1', 'state2', 'st6']
