it's much more readable as separate lines (especially if you line them up):

SUCCESS  = 0
FAIL     = 0
TARGET   = os.getcwd()

SUCCESS, FAIL, TARGET, TIMEOUT, MAXDEPTH, ORIGINAL, PATHS = 0, 0, os.getcwd(), 2, 0, '', {}
