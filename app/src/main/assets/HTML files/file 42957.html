Since both pieces of code are correct and would work in most circumstances, I can only think of these reasons why the second one would not work:

  1. There's an <a> tag or a form post in the hierarchy that is causing the browser to go to a new page before the results of your code can be seen.

  3. There's a script error right before your second block of code that keeps it from executing.
