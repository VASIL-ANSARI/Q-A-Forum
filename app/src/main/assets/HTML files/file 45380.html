The empty string in st-sort-default will be ignored. From line 282 in smart-table.debug.js:

if (attr.stSortDefault) {
   sortDefault = scope.$eval(attr.stSortDefault) !== undefined ?    scope.$eval(attr.stSortDefault) : attr.stSortDefault;
}

I would suggest you to use st-sort-default="default". However, any string will do the same.

There is a specific check in line 308 for reverse:

if (sortDefault) {
   index = attr.stSortDefault === 'reverse' ? 1 : 0;
   sort();
}
