when going out of range it means the value is not indexed example

int[] myArray = {1, 2, 3, 4, 5, 6 };
console.writeline(myArray[6]);

you would get the going out of range exception for the reason that the index starts with Zero

Index: 0 1 2 3 4 5

Array: 1 2 3 4 5 6

So when you are using (myArray[6]); your actually pulling the 6th element in the index. If you notice it only goes to 5.
