If you miss it - you can use my code snippets to easily create one:

  1. Put my snippets in %USERPROFILE%\Documents\Visual Studio 2013\Code Snippets\Visual C#\My Code Snippets
  2. Type vm[tab] or vm[Esc][Tab] if you have ReSharper installed to drop the full bindable base snippet in your app. You could also use a bb snippet for just the INotifyPropertyChanged implementation. Once you find a version you like - you can also export it to a VS item template (VS File menu).
