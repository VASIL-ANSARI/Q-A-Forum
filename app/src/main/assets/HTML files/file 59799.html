So as I understand you just want to accept numbers or?

const regex = /([0-9])/g;
regex.test("123"); //true
regex.test("123a"); //false

What you can also do is except every "letter":

const regex = /([^A-Za-z])/g;

But be careful using this! It also recognizes whitespaces and other symbols.

Or are you talking about unicode chars? That could be a little bit tricky with javascript.
