Ofcourse you wont get what you want because in your PostVent.php the one you are displaying is the id and the hidden value is the name. It should be:

echo '<option value="' . $row['id'] . '">' . $row['name'] . '</option>';

Syntax of <option> is <option value='[Specifies the value to be sent to a server]'>[Label or display value]</option>

SELECT id, name FROM states ORDER BY name
