Use class selector instead of id. id supposed to be unique. You can not have same id for multiple elements. In this case class will work perfectly.

<a href="#" class="readmore">... Read More</a>
<input type="hidden" class="idd" name="idd" value="<?php echo $row['n_id'];?>" >

$(".readmore").click(function(e){
    e.preventDefault(); // Only if you dont want it to reload
    var id=$(this).next('.idd').val();
    alert(id);
});
