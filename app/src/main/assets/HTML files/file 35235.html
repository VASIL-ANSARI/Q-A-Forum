Do not use an array. Arrays when defined must have a constant length and so if you are adding new entries into an array it is better to use an Arraylist as you can change the length of an Arraylist when adding new elements rather than an array which has a fixed length.

List<int[]> newArr = new ArrayList<int[]>();
