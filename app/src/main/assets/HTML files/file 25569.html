You are using the ClientID in your jQuery call, which might not always be the same. You need to retrieve the ClientID of that control from ASP.NET when the page is loading in order for that code to work more consistently.

Something like this:

$("#<%= UCCalender11_textDate1.ClientID %>").datepicker({

Note that you need to change UCCalender11_textDate1 to whatever the TextBox control's ID value is in ASP.NET (if that's not it).
