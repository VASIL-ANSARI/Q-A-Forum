I managed to change the CURL TIMEOUT option by finding the php file which contains the CurlHttpAdapter. The path for this file (in the GEOCODER PHP installation) was:

vendor\egeloen\http-adapter\src

I commented these two lines:

//$this->configureTimeout($curl, 'CURLOPT_TIMEOUT');
//$this->configureTimeout($curl, 'CURLOPT_CONNECTTIMEOUT');

and added this line:

curl_setopt($curl, CURLOPT_TIMEOUT,0);

That solved this problem..
