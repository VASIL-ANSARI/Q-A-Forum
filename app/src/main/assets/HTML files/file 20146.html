This variable, myContact, is local to BeginContact:

MyContact myContact = { contact->GetFixtureA(), contact->GetFixtureB() };

But here, you're storing a pointer to it for future reference:

bodyHoldUD.climbingHoldContact = &myContact;

When BeginContact returns, the myContact variable ceases to exist, and when you later attempt to access its fixtureA your computer says EXC_BAD_ACCESS because whatever was there has been overwritten with something else.
This is the same problem as returning a pointer to a local variable.

Note that the example you built on stores MyContacts in the vector, not MyContact*s.

You should be able to fix this one by just changing the climbingHoldContact member to not be a pointer.
