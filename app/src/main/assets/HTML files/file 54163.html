If i want to return a, b, and c. how I should do that?

Option 1

Define a struct and return an object of the struct.

struct MyStruct
{
   int a;
   int b;
   int c;
};

Now you can use:

return MyStruct{10, 20, 30};

Option 2

Use std::tuple.

using MyData = std::tuple<int, int, int>;

and then

return MyData{10, 20, 30};
