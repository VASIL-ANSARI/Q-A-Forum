Unfortunately, perspective seems not to be supported yet (at least in chrome where I tested it).

But you can use other transformations, just use the css attribute transform (or the prefixed ones):

img {
  -webkit-transform: skew(-10deg, 5deg);
  -moz-transform: skew(-10deg, 5deg);
  -o-transform: skew(-10deg, 5deg);
  -ms-transform: skew(-10deg, 5deg);
}

Check out this example (tested in chrome): http://jsfiddle.net/uynXh/5/
