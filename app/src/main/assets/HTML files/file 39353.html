Add Google play service from adt-bundle-linux-x86_64-20130729/sdk/extras/google/google_play_services/libproject.

Don't forgot to mention these meta data in manifest:

<meta-data
            android:name="com.google.android.gms.version"
            android:value="@integer/google_play_services_version" />

then clean and build your project and run it again. I hope this will help you.
