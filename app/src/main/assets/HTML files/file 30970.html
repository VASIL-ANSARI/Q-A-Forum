The current set of Environment properties that you can access in your App Engine application is given over here. I don't think there is a property that refers to what you are trying to get.

However, I would suggest that you print out the request headers in the incoming request object, there could be something that could give you the hint there:

Enumeration headers = req.getHeaderNames();
while (headers.hasMoreElements()) {
    String name = (String)headers.nextElement();
    String value = req.getHeader(name);
    resp.getWriter().println(String.format("%s: %s", name, value));
}

Additionally, I would suggest to look at the Backend API. This should give values of particular interest to you too: https://developers.google.com/appengine/docs/java/javadoc/com/google/appengine/api/backends/BackendService
