You can call the authentication URL like the following , which is i used for Calender API

Dim Uri As String = "https://accounts.google.com/o/oauth2/auth?access_type=offline&response_type=code&client_id=" & clientId & "&redirect_uri=" & redirecturi & "&scope=https://www.googleapis.com/auth/calendar&login_hint=" & mainUser & "&include_granted_scopes=true"
        Response.Redirect(Uri)

you have to give appropriate values for :

clientId,mainUser,redirecturi. where, mainUser is the mail id which is used for authentication, redirecturi is the URI of our site/ page their we handles the response. and it must be same in the passing url and also in the developer console
