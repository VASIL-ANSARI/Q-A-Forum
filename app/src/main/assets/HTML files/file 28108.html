Before your first loop

allCells[i].document.getElementsByName(allCells)

there should be something like

allCells = document.getElementsByName( [[ NAME ]] );

And 

document.getElementsByName(allCells)

can't work, because allCells isn't set to anything.

After you've set allCells to the List of Elements, you can iterate over them like that:

allCells[i].style.backgroundColor = "white";

And JavaScript is case sensitive so you have to write backgroundColor instead of backgroundcolor.

Update: To assign a function to an event, you have to write

.onclick = yourFunction;

not

.onclick = yourFunction();

On the long run it would be even better to use this:

.addEventListener("click", yourFunction);
