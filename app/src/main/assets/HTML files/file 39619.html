The root cause of the problem is that Tkinter isn't designed to have two root windows. Doing so has some unexpected side effects such as what you're seeing. In effect, you can think of the two root windows as two separate processes or threads which cannot share information. Your IntVar belongs to the first window, but you're trying to use it in the second.

The fix is to never create more than one instance of Tk. If you need more windows, create instances of Toplevel. Doing that, you can share the same IntVar among as many windows as you want.
