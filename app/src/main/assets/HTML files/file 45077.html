I'm not 100% sure, hadn't the time to read the docs completly. But the String Functions like "LOWER" are only usable in the WHERE part of the query. In the Select part it makes no sense, because the conversion to lower case is easyly done in the Result set object.

For example when your JPA Query returns Pojo MyEntity then you can implement it this way:

public class MyEntity {
  private String value;

  public String getValue(){}
  public void setValue(String value) {}
  public String getValueLowerCase() {
    return value.toLowerCase();
  }
}
