If you set the tabindex property for the <p> element, it will allow it to have focus. Once it has focus, the document.activeElement property will return the p element instead of the div.

Here is a fiddle which shows that it works. The code changed is below

<div id='body-text' class='body-text' contenteditable=true data-placeholder='Body Contents'>
    <p id="1343" tabIndex="1">I have a problem<br>I can't handle it</p>
    <p id="1976" tabIndex="2">Please help me</p>
    <p id="1999" tabIndex="3">Thank you in advance</p>
</div>
