You need to cast it to T like this:

public static <T extends A & B> T f(boolean b) {
    if (b)
        return (T) new C();
    return (T) new D();
}

< what_should_I_write_here > x = f(< user_inputted_boolean >);

You need to assign a concrete type to a variable, indeed don't forget that due to Type Erasure the generics don't exist anymore at Runtime they exist only at compile time, such that only concrete type like this can work:

C x = HelloWorld.<C>f(true);

As you may have noticed, you won't be able to do much with A & B that is why Multiple Bounds are rarely used in practice, such that you should create an interface that extends A and B and use this interface instead, like this

public interface AB extends A, B {}

public static class C implements AB {}

...

I know that it is much less flexible but it will be hard to do much better.
