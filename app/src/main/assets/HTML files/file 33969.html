Just modify the $characters array to include all the characters you want.

Code: 

<?php
    $characters = array('a', 'b', 'c', 'd');
    foreach($characters as $first){
        foreach($characters as $second){
        echo $first.$second.'<br/>';
        }   
    }
?>
