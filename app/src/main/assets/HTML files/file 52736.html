scanf does the job:

  int max;
  scanf("%d", &max); 

  for (int i = 0; i < max; i++)
  {
    int number;
    scanf("%d", &number); 
    printf("%d\n", number);
  }

This allows you to enter:

3
11 12 13

and you will get this output:

11
12
13

Storing the numbers in an array and displaying them in reverse order is left as an exercise to the reader.
