If the script tag comes before the elements, you'll need a dom ready function, and preventing the default scrolling to the top when using a hash as a href is usually good idea. When clicking the delete button, use closest() to find the closest parent that mathes the bottomsurgicalhistory class :

$(function() {
    $(".delete-button").on('click', function(e){
        e.preventDefault();
        $(this).closest('.bottomsurgicalhistory').remove();           
     });
});
