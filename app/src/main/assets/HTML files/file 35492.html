The problem you're having is that you're using perspective as a seperate CSS property. If you look here and here, you'll see that perspective is simply a "command" you can pass to transform.

Another point: if you look at this you'll see as of version 16 of firefox the -moz is no longer needed.

The fix is relatively simple, just do exactly as you've done for chrome (webkit) compatibility:

.book {
    position: relative;
    transform: perspective(100px) rotateY(-2deg);
    -webkit-transform: perspective(100px) rotateY(-2deg);
    outline: 1px solid transparent;
    box-shadow: none;
    margin: 0;
}
