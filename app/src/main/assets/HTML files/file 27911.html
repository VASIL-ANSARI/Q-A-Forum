If your input consists only of capital letters, then the letter was already provided if and only if you reach the end of your inner loop, i.e. if the letter is not found in the alphabet because it has already been replaced by _. One way to check for this is to check if i equals 26 after your inner loop. A slightly cleaner way would be to use a boolean to indicate if you did break out of the inner loop.

Now if your input can be any character, it would be best to use a separate array to keep track of the letters already provided instead of replacing letters in alphabet. You could have:

const char alphabet[26] = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
int count[26] = {0};

and

if (letter == alphabet[i]) 
{
    ++count[i];
    break;
}

to keep track of exactly how many times each letter was provided.
