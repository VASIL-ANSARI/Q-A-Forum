I recommend you use the ToneGenerator class. It requires no audio files, no media player, and you can custom the beep's volume, duration in milliseconds and Tone type. I like this one:

ToneGenerator toneGen1 = new ToneGenerator(AudioManager.STREAM_MUSIC, 100);             
toneGen1.startTone(ToneGenerator.TONE_CDMA_PIP,150);

You can see into the ToneGenerator object, and choose another beep type besides TONE_CDMA_PIP, 150 is the duration in milliseconds, and 100 the volume.
