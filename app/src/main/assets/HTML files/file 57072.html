For current sample data, it's simple pivot case:

select
    id,
    max(if(taxper=0, value, null)) as `0%`,
    max(if(taxper=5, value, null)) as `5%`,
    max(if(taxper=10, value, null)) as `10%`
from table1
group by id

If there are a lot of values for taxper, then you may need this:

SET @sql = NULL;
SELECT
  GROUP_CONCAT(DISTINCT
    CONCAT(
      'MAX(IF(taxper = ''',
      taxper,
      ''', value, NULL)) AS `',
      taxper, '%`'
    )
  ) INTO @sql
FROM Table1;
SET @sql = CONCAT('SELECT id, ', @sql, ' FROM Table1 GROUP BY id');
PREPARE stmt FROM @sql;
EXECUTE stmt;
DEALLOCATE PREPARE stmt;
