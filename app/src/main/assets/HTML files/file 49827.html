This only works if the due_date column contains timestamp values. Otherwise this query is meaningless.

I believe all you need is a couple sets of well places parentheses.

$tasks_done = ProjectHasTask::all(array('conditions' => 
    array('(status = ? AND project_id = ?) AND (due_date <= ? AND due_date >= ?)', 
    'done', $project_id, $from_date->getTimestamp(), $to_date->getTimestamp())));
