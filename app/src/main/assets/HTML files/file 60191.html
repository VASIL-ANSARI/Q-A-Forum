Try with:

RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ send.php?url=$1&name=&submit=submit [NC,L]

This way you avoid to rewrite existing files and directories.

With -f you test for files, and with -d for directories. http://httpd.apache.org/docs/2.2/mod/mod_rewrite.html#rewritecond
