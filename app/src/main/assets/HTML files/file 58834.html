I can't comment so I'll put this in an answer. There's nothing wrong with your code, runs fine for me. Try overriding onStop() to check if it's being called when you're pressing the home button or at some other time.

@override
protected void onStop(){
    super.onStop();
    Log.v("test", "stopped");
}

onStop()is called if a higher priority app needs the memory. https://developer.android.com/guide/components/activities.html#Lifecycle.

If you really need the app to always remember when the MediaPlayer was paused maybe save it in a file or in SharedPreferences.
