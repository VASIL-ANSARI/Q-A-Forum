As others have commented, your example code (which you should have condensed to a minimal example) isn't actually invoked. The way I've returned a result from Python is by setting a Vim variable, like this:

strValue = "just a test"
vim.command("let python_result = '%s'" % str(strValue).replace("'", "''"))

For a boolean, just return 0 / 1 as a number, simplifying the logic:

vim.command("let python_result = %d" % int(boolValue))
