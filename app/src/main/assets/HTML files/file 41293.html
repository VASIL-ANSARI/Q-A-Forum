I suppose that your system don't have MinGW compiler!! If that's the case, download it by visiting their site. If you have already done that, the only thing I can think of is that you have not edited the path variable value in your system properties.

For this, follow these simple steps :

  1. Go to "My Computer" or "This PC" or whatever name you have given to it.
  2. Right click and select "Properties".
  3. Click on "Advanced system settings" present on left hand side.
  4. Click on "Environment Variables" present at the bottom under "Advanced" tab in the "System Properties"
  5. Select the "Path" variable in the "System variables" and select the "Edit" option.

  6. At the end of the contents written in "Variable Value", add this

    ;C:\MinGW\bin

and your .c files should compile and run.
