You need to fix this on the Android side rather than in the PHP.

You need to change the default handling of URLs when they're loaded by overriding shouldOverrideUrlLoading

You can do this as follows:

webview.setWebViewClient(new WebViewClient() {
    public boolean shouldOverrideUrlLoading(WebView view, String url){
        view.loadUrl(url);
        return false;
   }
});
