Add chart-options to your html:

<canvas id="line" class="chart chart-line" chart-data="data"
chart-labels="labels" chart-legend="false" responsive="true" 
chart-click="onClick" scaleShowVerticalLines= "false"
chart-options="options">
</canvas>

and add chart-options to your js file. It would look like this:

$scope.options = {scales: {
                     xAxes: [{
                          gridLines: {
                               display: false
                          }
                       }],
                     yAxes: [{
                          gridLines: {
                               display:false
                          }
                       }]
                     }
                  };
