if you are on a linux box, there is http://sourceforge.net/projects/littleutils/ which offers

  1. Includes several basic file property query utilities.
  2. Includes several file renaming utilities. Includes several lossless image file recompressors.
  3. Includes several compressed file recompressors.
  4. Includes a duplicate file-finding utility.
  5. Includes a secure temporary file creation utility.

if you are running wordpress there is even a handy dandy plugin, that works with it, http://wordpress.org/extend/plugins/cw-image-optimizer/
