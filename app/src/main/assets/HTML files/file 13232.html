You can stop current animation in jQuery.

Or you can restore original values yourself.

To do both you'll have to be able to select element that is currently animated. To facilitate this, I suggest adding a special class (like currentlyAnimated) in your click handler. When you click another element, check if there are elements with this class. If there are, stop/revert animation on them. After that, you remove this class from those elements and add it to the current one.
