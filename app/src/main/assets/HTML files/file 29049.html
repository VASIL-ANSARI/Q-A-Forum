You haven't setup validation only jpa. You need hibernate-validator for validation.

<dependency>
    <groupId>org.hibernate</groupId>
    <artifactId>hibernate-validator</artifactId>
    <version>4.3.1.Final</version>
</dependency>

Is the dependency for the validator.

Next to that the BindingResult must directly follow the @ModelAttribute annotated attribute, so switch the ModelMap and BindingResult attributes in your controller method also.
