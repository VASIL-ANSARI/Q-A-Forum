The first question is: what model of RFID reader you are using?

If I'm assuming correctly, you probably have an MC9090 (or equivalent) from Motorola/Symbol

If that's the case, their SDK (called EMDK) includes documentation for several RSSI related classes and properties including RssiRangeFilter.

The most recent revision for .NET developers is EMDK v2.6 Update 2

To download the documentation for the API, right click and save this file. On Windows 7, it may not display properly at first, so if that happens, check out this solution.

Hope that helps!
