You only need to solve half of it and put same number in the mirror image by changing the indices.

You can do this:

  • fill the diagonal
  • Loop through rows: r = 1 -> colmax -1 //square matrix!
    • Loop c = r -> colmax-1 times again
    • set value at [c][c+1] = [c][c-1] + [c][c+1]

The index at [c][c+1] runs parallel to the diagonal with offset r.

The index at [c][c-1] is is the previous value in the row

The index at [c+1][c+1] is the next value in the column

After you are done, fill the other half by copying what you calculated.
