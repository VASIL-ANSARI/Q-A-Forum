Data that you are offering to series property ( as data attribute ) is in wrong format.

One of the way to provide data to series is using following format.

series: [{
            type: 'pie',
            name: 'Browser share',
            data: [
                ['Firefox',   45.0],
                ['IE',       26.8],
                {
                    name: 'Chrome',
                    y: 12.8,
                    sliced: true,
                    selected: true
                },
                ['Safari',    8.5],
                ['Opera',     6.2],
                ['Others',   0.7]
            ]
        }]

Please follow this link : http://www.highcharts.com/demo/pie-basic
