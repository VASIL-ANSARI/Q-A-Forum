How about this for a solution:

Whenever a new object is added to the canvas, export the canvas data to JSON:

    savedposition = canvas.toJSON();

In your function to change the BG, just load the savedposition JSON string back on to the canvas, then render the background:

    canvas.loadFromJSON(savedposition,canvas.renderAll.bind(canvas));

There's a fiddle here: http://jsfiddle.net/DanBrown180/sbT3d/
