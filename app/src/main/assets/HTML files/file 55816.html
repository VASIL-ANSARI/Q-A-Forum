You could probably simplify the whole thing like this:

@helper myHelper(){
    int totalItems = 0;
    <ul>
        @for(var i = 0; i < 25; i++)
        {
            totalItems = i;
            @if (totalItems % 10 == 0)
            {
                </ul><ul>
            }
            <li>List Item #@totalItems</li>
        }
    </ul>

    <p>Last list contains @totalItems list items!</p>
}

By getting rid of all of the extra helper stuff, and using modulo (%) instead of resetting at every 10 count. You may need to make some modifications, I am not very familiar with razor syntax.
