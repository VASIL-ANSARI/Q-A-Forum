it doesn't work because getdata ask a function as a second parameter. In the second example you'are executing the function (and passing its returning value) instead of passing the function itself.

if you write

$( '#someid' ).click( function() {
    getdata( 'listing' , showlist, 'string' , 1 );
});

without parens () it still works, but as you can see, you can't pass anymore the data parameter along the function showlist
