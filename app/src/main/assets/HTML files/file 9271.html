Actually my initial solution was wrong, it gives you:

[{1=>[1, 3]}, {1=>[2, 3]}, {2=>[1, 3]}]

I believe this is closer to what you were looking for:

x = [[1,1,3], [1,2,3], [2,1,3]]
h = {}

x.each do |a|
  if h[a[0]].nil?
    h[a[0]] = []
  end
  h[a[0]] << [a[1], a[2]]
end

Which gives you a hash of arrays:

{1=>[[1, 3], [2, 3]], 2=>[[1, 3]]}
