Codeigniter has it own encryption class, that use a hash algorithm combined with a key that you can generate yourself. You should store the password already encrypted in the database. Then, when you want, you retrieve and decrypt.

Eg. Creating user or changing password:

$password_to_db = $this->encrypt->encode($this->input->post('password');

Retrieving password:

$user = $this->user_model->get_user($user_id);

if($password_from_login_form == $this->encrypt->decode($user->password)){
    /* ... login function ... */
}

Check Encryption Class manual to see how to load and configure it.
