1. Try this code to get superuser access (you'll need a rooted phone with superuser installed):

    try {
    // Perform su to get root privledges
    Process p = Runtime.getRuntime().exec("su");

    DataOutputStream os = new DataOutputStream(p.getOutputStream());

    os.writeBytes("<..... here you may write commands ....>\n");

    // Close the terminal
    os.writeBytes("exit\n");
    os.flush();
} catch (IOException e) {
    toastMessage("not root");
}

  2. Once you have superuser access, you may start adbd listening on some TCP port using:

    setprop service.adb.tcp.port 5555
adbd &
setprop service.adb.tcp.port -1

  3. Open a TCP connection to the selected port and issue remount command.
