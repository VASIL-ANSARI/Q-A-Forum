First, Grails is used to create web based applications, so you will need a controller that will respond to your browser request.

Secondly, if your controller is in a different package than the class of your Java program that has your public static void main(String[] args) method then you will need to import that class into your controller.

Finally, you do not need to create another controller inside src/java. Controllers exist within the grails-app/controllers directory structure.

Following the instructions provided in the linked post in your question should give you all you need to accomplish calling your Java program from a Grails controller.
