Try this :

Open your .profile file in gedit.

gedit ~/.profile

Paste the following at the bottom of your .profile file.

# Gradle
if [ -d "$HOME/usr/local/gradle" ]; then
    export GRADLE_HOME="$HOME/usr/local/gradle"
    PATH="$PATH:$GRADLE_HOME/bin"
fi

Finally, source your .profile and test gradle.

source ~/.profile

which gradle

gradle -version
