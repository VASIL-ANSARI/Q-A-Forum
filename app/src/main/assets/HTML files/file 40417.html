there is a way to do this but only if the common task that you want to put in a method do NOT need instance of the class. that is the Class method using the + sign (vs instance method, which uses -).

in ClassA add this class method (add declaration)

.h (public API)

+(void)ShowMessages;

.m :

+(void)ShowMessages{
NSLog(@"I am a message");
NSLog(@"And me too");
}

in classB you can use the class method of ClassA:

[ClassA ShowMessages];
