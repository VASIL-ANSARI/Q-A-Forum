If you already have a domain class such as:

class Book {
  String name
}

Create a controller

class BookController {
  def list() {
    [books: Book.list()]
  }
}

And a GSP grails-app/views/book/list.gsp

<html>
  <head></head>
  <body>
    <ul>
      <g:each in="${books}">
        <li>${it.name}</li>
      </g:each>
    </ul>
  </body>
</html>
