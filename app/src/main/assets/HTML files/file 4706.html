You could bind each Button's IsDefault property to its corresponding TabItem's IsSelected Property. Each time the selected tab item changes, another button would become the default, then.

Something like that:

<TabControl>
  <TabItem x:Name="tab1" ... />
  <!-- ... other tabs ... -->
</TabControl>
<Button x:Name="button1" IsDefault="{Binding IsSelected, ElementName=tab1}"/>
<!-- ... other buttons ... -->
