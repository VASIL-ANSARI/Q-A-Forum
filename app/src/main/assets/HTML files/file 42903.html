Split the URL using PHPs explode() function:

$link = get_permalink();
$temp = explode("/",$link);
print_r($temp);

The get_permalink() function returns the URL, and print_r prints the array. Get your part which you want to check in if condition like;

For example: 

Array ( [0] => www.xyz.com
        [1] => xxx
        [2] => services
      )
