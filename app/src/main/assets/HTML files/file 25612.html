You can achieve this with CASE

UPDATE YourTable
SET    name = CASE
                WHEN id = 1 THEN 'johnny'
                ELSE 'james'
              END
WHERE  id IN ( 1, 2 )

Or another possibility (SQL Fiddle)

UPDATE YourTable a
JOIN
   ( SELECT 1 AS id, 'johnny' AS name
     UNION ALL
     SELECT 2 AS id, 'james' AS name
   ) b ON 
   b.id = a.id
SET a.name = b.name;
