You need to change the y-value of the bottom separator frame to view.bounds.size.height-0.5f if you in fact want it positioned at the bottom. Correct me if I'm wrong, but I think you're trying to achieve a true 1px horizontal line. I'm not 100% sure about this but it's possible that you'd run into a problem using 0.5f heights on non-retina display devices. To be safe you could define the height first using the device scale property like so:

float height = 1/[[UIScreen mainScreen]scale];

UIView *bottomSeparatorView = [[UIView alloc] initWithFrame:CGRectMake(0.0f, view.bounds.size.height-height, CGRectGetWidth(view.frame), height)];
bottomSeparatorView.backgroundColor = DEFAULT_COLOR_PLACEHOLDER;
bottomSeparatorView.autoresizingMask = UIViewAutoresizingFlexibleWidth|UIViewAutoresizingFlexibleTopMargin;
[view addSubview:bottomSeparatorView];
