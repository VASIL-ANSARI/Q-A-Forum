A quick answer is yes, there is. You just need to overwrite the Object template and there iterate over your model properties, sending them to another template, which generates all your desired elements.

1 - Call the EditorFor helper, for the entire Model (where you want all your properties to be rendered):

@Html.EditorFor(x => x, "path_to_custom_Object.cshtml")

2 - Iterate your properties (in Object.cshtml):

foreach (var prop in ViewData.ModelMetadata.Properties)        
{
      @Html.Editor(prop.PropertyName, "path_to_custom_Template.cshtml")
}

3 - Build all your elements (in Template.cshtml):

@Html.LabelFor(x => x)
@Html.TextBoxFor(x => x)
@Html.ValidationMessageFor(x => x)
