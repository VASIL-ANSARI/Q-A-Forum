Don't use it in viewdidappear! It is a method which is called more than once in a ViewController. Instead use it in viewDidLoad which is executed only once that is when the view controller is loaded.
And you have to call it in the 

func didReceiveNotification(notification: NSNotification)

And after the function use:

deinit {
NSNotificationCenter.defaultCenter().removeObserver(self)
}
