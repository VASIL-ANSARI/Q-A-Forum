You can simplify this code a lot. I stuck with adding css through jQuery since thats how you were doing it but this would be better just using classes. Also added color just so you could see the changes. Instead of adding the overhead of constantly adding and removing events with one we can just use a single handler that tracks the state of the div with a class.

HTML:

<div id="container">
    <div id="inner">This is the inner div</div>
</div>

JS:

 function handler1 () {
     var inner = $("#inner");
     if (!inner.hasClass('added')) {
        inner.css('height', '+=65px').addClass('added');
     }
     else {
        inner.css('height', '-=65px').removeClass('added');
     }
 }

 $("#inner").on("click", handler1);

CSS:

#container {
    width: 200px;
    background: red;
}

#inner {
    background: yellow;
}
