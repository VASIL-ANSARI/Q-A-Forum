The first thing you need is to give your textbox a fixed size font. without this it will (and should) always contain a jumbled mess. I recommend Consolas. From there on you may want to write a helper function that formats a piece of code to a fixed length. But first things first!

A helper function could be as simple as

    public string padded(string input, int length)
    {            return input.PadLeft(length);         }

or if your input is always an int: 

    public string padded(int input, int length)
    {            return input.toString().PadLeft(length);         }

or if you want to suppress zeroes completely, as your example suggests: 

    public string padded(int input, int length)
    {            return input.toString("#").PadLeft(length);         }
