why you're accessing:

 model.subModel[i]

when that is just of type:

subM*     subModel;

you're accessing memory that is not supposed to be accessed.

On a side note, your code is overcomplicated and dangerous. You can cut most of the pointers, that you don't actually need.

std::vector<openGLmodel::subM>* subModels = new std::vector<openGLmodel::subM>; 

std::vector<Vec3>* positions = new std::vector<Vec3>;
std::vector<Vec3>* normals = new std::vector<Vec3>;

becomes

std::vector<openGLmodel::subM> subModels;    
std::vector<Vec3> positions;
std::vector<Vec3> normals;

and your structure

struct openGLmodel
{

    struct subM
    {

        Vec3 positions;
        Vec3 normals;
        int  nbrVertices;

        int        nbrTriangles;
        std::vector<int> triangleIndices;
    };

    dim_type  subModelCount;
    subM*     subModel;
}

Should you need to reuse existing vectors and not copy them around, you can just use a reference to an vector and initialize it in the constructor's initialization list.
