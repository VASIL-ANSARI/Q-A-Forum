Define a CSS class e.g. "collapsed" which would hide list items in the list element. Then toggle the class on click on a list element:

listElement.onclick = function() {
  this.classList.toggle("collapsed");
}

Js fiddle - http://jsfiddle.net/gh424/2/

Of course clicks on elements inside the list will trigger click on the list itself and it will toggle. If it is not desired behavior then think of wrapping the list into div e.g.:

<div class="container">
    <div class="toggler">Toggles the lsit</div>
    <ul class="collapsed">
        <li>1</li>
        <li>2</li>
        <li>3</li>
        <li>4</li>
        <li>5</li>
    </ul>
</div>
