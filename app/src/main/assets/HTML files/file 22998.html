i've not actually solved the problem - it is still evident on this simplified page for anyone who is interested... http://www.haroldstreet.org.uk/test

But the following might be useful work-round to someone...

First i upgraded my ad code to the lastest version and set it up to show alternate ads (from amazon in this case). This was a useful alternative to just having empty ad spaces - see this example... http://www.haroldstreet.org.uk/waypoints/download/?list=humps

Then for my more important /munros/ page - i've added the following line to the google ad code. 

   google_page_url = "http://www.haroldstreet.org.uk/waypoints/download/?list=munros&area=04";

This seems to prompt google back into giving closely related ads, and restores the Google adds to my /Munros/ page.

So its not a fix, but a reasonable work-round
