Whenever you have a set of similar objects donÂ´t create a variable for every instance of those objects, instead put them into one single collection. Then you can easily loop this list and manipulate the objects within it:

var buttons = new List<Button>();

// put the buttons into the list using buttons.Add

for (var b in buttons)
{
    if (b.text == "true")
    {
        b.textcolor = blue
    }
    else 
    {
        b.textcolor = red
    }
}

You can also use a GridView to put all those buttons into a ragular grid.
