You can just create the Species class like this:

public class Specie
{
    public int Id { get; set; }

    public int TaskId { get; set; }

    public Task ParentTask { get; set; }

    public ICollection<Specie> CurrentSpecies { get; set; }

    public ICollection<Specie> HistSpecies { get; set; }
}

If you have to create the Task object and the collections within this object then I would suggest you rethink your design. Essentially the task object should then become the aggregate root.
