You can iterate indefinitely using while True, then stop whenever your conditions are met using break:

count = 0
while True:
    count += 1
    if count == 10:
        break

So for your second example you can try:

while True:
    ...
    if f - f2 < 0:
        # use new variables
        f, E = f2, E2
    else:
        break
