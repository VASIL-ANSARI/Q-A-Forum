Changed my project/build.gradle file to look like this. (Removed the compile 'com.android.support:support-v4:18.0.0' bit).

dependencies {
  compile project(':vpi')
  compile 'com.actionbarsherlock:actionbarsherlock:4.4.0@aar'
  compile 'org.jsoup:jsoup:1.7.2'
  compile fileTree(dir: 'libs', include: '*.jar')
}

This seems to make gradle DebugAssemble work. I did run gradle clean DebugAssemble too, not sure if that had anything to do with this working.

Also had to add a local.properties file to the project root which contained.

sdk.dir=/Applications/Android Studio.app/sdk

After doing this, importing the settings.gradle file from Android Studio worked.
