You need to create an NSManagedObject subclass (in Editor menu or manually). Then, you can use the following code (Employee being your subclass)

let entityDesc =  NSEntityDescription.entityForName("Employee", inManagedObjectContext: managedContext)!

let entity = Employee(entity: entityDesc, insertIntoManagedObjectContext : managedContext)

To access all your fields. Creating a subclass is not mandatory though, you can always use your NSManagedObject as a dictionary and do

if let time = newUser.valueForKey("time") as? NSDate {
   //use date here
}

However it is ugly.
