I think the issue is just how you construct $newRouteObjs because you're passing in [] which is an unindexed array. Simply index it:

foreach ($routes as $key => $val)
{ 
    $routeObj = new Route($routes[$key]);

    //add [$key] as opposed to []
    $newRouteObjs[$key] = $routeObj;
}

return $newRouteObjs;
