  1. If you have a value to insert for that particular column, insert it. If you don't have a value (for this particular record) then you need to state that explicitly (by inserting a NULL value).
  2. If the value of 'HUN2' is most likely to be a particular value for most records or atleast to begin with, (but, might change thereafter), then you might set that value as default.

Based on business rules, you can decide which of the two options mentioned above you'd like to go with.
