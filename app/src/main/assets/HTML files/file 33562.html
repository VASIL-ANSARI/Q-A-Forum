The argument passed to the Where() method cannot be a string, it must be a Func<TSource, Boolean>. Your method that generates the query will need to be defined as:

Expression<Func<Car, bool>> query = null;

Then whenever you want to set the value of query you would set it using:

query = car => car.Budget >= 20000 && car.Budget <= 34999;

Now you can use the query variable in your Where() method call.
