The original issue

To make the section where your comment appears scale with content length, you need to replace the following html: 

<p id ="content" class="minimize more"  style ="width :500px; height :100px; white-space :pre ">

to (min-height)

<p id ="content" class="minimize more"  style ="width :500px; min-height :100px; white-space :pre ">

Your p, who's normaly an element with a display:inline property, is turned to display:block somehow.

The issue in the comments

Text taken from a text area doesn't contains html. Line breaks, extra spaces (more than 1) and tabs are encoded as such and ignored by browsers. As such, you need to replace them with html to display them correctly.

as such,
\r\n, \r and \n (line-break) become <br />
\t (tabs) become &nbsp;&nbsp;&nbsp;&nbsp;
(spaces) become &nbsp;
