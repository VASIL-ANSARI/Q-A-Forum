You can create the look by giving your list items a dashed border-bottom. As a simple example:

HTML:

<ul>
    <li><a href='#'>Title</a></li>
    <li><a href='#'>Element 1</a></li>
    <li><a href='#'>Element 2</a></li>
    <li><a href='#'>Element 3</a></li>
    <li><a href='#'>Element 4</a></li>
</ul>

CSS:

ul{
    list-style:none;
    padding-left:10px;
    padding-right:10px;
    width:140px;
    border:1px #eee solid;
    border-radius:10px;
}

li{
    border-bottom: 1px #ccc dashed;
}

a{
    padding-top: 10px;
    padding-bottom: 10px;
    color: rgb(107, 107, 107);
    text-decoration: none;
    width: 100%;
    height: 100%;
    display: inline-block;
}

li:last-child{
    border-bottom: none;
}

li:first-child{
    border-bottom: solid grey;
}
li:first-child a{
    color: rgb(237, 106, 61);
    font-weight: bold;
    font-size: 20px;
}

See https://jsfiddle.net/72vvksws/3/ for a working example.
