This is an alert to developers when they make the Parse calls that would block the main thread.

To see where this is happening, add a symbolic breakpoint on warnBlockingOperationOnMainThread if you use a Parse API released from 2015+.Otherwise, put it on warnParseOperationOnMainThread.

It'll break on that function while you running your code, and will show you a stack trace which should help you to find the blocking function.
