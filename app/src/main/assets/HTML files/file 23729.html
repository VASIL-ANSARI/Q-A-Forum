The problem is you bind to the elements based on the classes they have at binding time. And initially $("li.off") is empty.

Use 

$(function(){
    $('#sections')
    .on('click', "li.on", function() {
        $(this).addClass('off').removeClass('on');
    })
    .on('click', "li.off", function() {
        $(this).addClass('on').removeClass('off');  
    });
});
