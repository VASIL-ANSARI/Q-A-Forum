You can use GroupBy o[0] fist and then it will give you grouped result. Each group you can then order the items by d[3], Since grouped ordered grouped list you can use SelectMany method to flattens it back to a list what you expected.

var sortedRecords = records
                    .GroupBy(o => o[0])
                    .Select(g => g.OrderBy(d => d[3]))
                    .SelectMany(g=>g)
                    .ToList();
