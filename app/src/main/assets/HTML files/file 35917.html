Go to your S3 console and make sure the bucket is set to "Do not enable website hosting" in "Static Website Hosting".

I'd recommend you to store your s3 credentials only in ENV variables and dump the s3.yml file. It's more secure.

You could setup your paperclip defaults in the environment files (production.rb, development.rb, test.rb, etc.) like this:

  config.paperclip_defaults = {
    :storage => :s3,
    :s3_credentials => {
      :bucket => ENV['AWSBucketName'],
      :access_key_id => ENV['AWS_ACCESS_KEY_ID'],
      :secret_access_key => ENV['AWS_SECRET_ACCESS_KEY']
    }
  }

you won't need to repeat those in your model (game.rb)
