You can't access the Node class from outside because it is declared private inside your LZWBinTree class. If you want to make it accessible, change the access modifier. From:

private:
    class Node{ ...

To:

public:
    class Node{ ...

Once you do that, you can create instances of your node class by accessing it through the class it is in, as follows:

LZWBinTree::Node n;

I'm not quite sure how dynamic allocation can help you in this case.
