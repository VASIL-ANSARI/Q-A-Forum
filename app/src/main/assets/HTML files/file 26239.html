You probably want a Range, which you can get with window.getSelection().getRangeAt(0)

As the link describes, that object has properties which describe the beginning and end of the selection in terms of the elements which the selection starts and ends in, and a numeric offset from the first textual "space" in those elements.

To pass that over the wire, you'll need a consistent identifier for the start and end elements. If every element in the document has an ID which is the same no matter who's viewing it, then you could use that. Otherwise, you'll need to determine some custom serialization to uniquely identify elements within your markup.
