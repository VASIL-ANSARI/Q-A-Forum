Marc B is right, you can't really hide the parent and show the children. The next best thing is to throw in a toggle element under the true parent element. A simple example might be: 

$('.parent-toggle').click(function () {
  $(this).closest('.parent').find('.child').show()
  $(this).closest('.parent').find('.parent-toggle').hide()
});

Here's a working example. Note that visibility: hidden doesn't remove the space the parent element in question takes up. I've used display: none to completely hide the element, but change as you wish.
