I suppose the screen is stable. I think you can follow these steps:

  1. Create a background. Use HSV color space.
  2. subtract image between your "destination frame"(any include object frame) and background.
  3. After step 2 you'll get a different image, and set a threshold at channel H and V to binary image.
  4. Remove noise by contour area size.
  5. Smooth binary image use Gaussblur (maybe filter size 15*15). you'll get a more complete object image.
  6. Make object image as mask you can delete the object and recover use background image.

Good luck!
