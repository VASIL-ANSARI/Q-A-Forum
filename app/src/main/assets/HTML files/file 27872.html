If I understand you correct you can check $.holdReady() method to acheive the result; so just after load jquery run

jQuery.holdReady(true);

later attach your handlers which should run only after load with 

jQuery(function($) {
    App.initialize();  //will run only after window.load;
}

and in your load handler setup:

jQuery(window).load(function () {
    jQuery.holdReady(false);  //fire all domready holded events
});
