You can create different layout files for Mobile phones and tablets. For tablets you should create extra layout folder in res folder which is called layout-large (for tablets), For Mobile phone there is default layout folder, and at the same time for landscape mode you can create one more folder named as layout-large-landscape (tablets) for Mobile phones layout-landscape.

Also in Manisfest file make a change

<supports-screens android:anyDensity="true" />
