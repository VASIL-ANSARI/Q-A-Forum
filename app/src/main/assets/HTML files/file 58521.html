This is the code for getting the value of the input every time the user types something. And then send it to the controller.

  $(document).ready(function(){
    $("#input").keyup(function(){
      console.log($(this).val());

      $.ajax({
        url: "controller",
        type: "get",
        data:{text: $(this).val()},
        success: function(response) {
          //here is where you treat the JSON
        },
        error: function(xhr) {
          //if stuff goes bad
        }
      });  
    });
  });

the "#input" is the id of the input field.
