$tt1=mysqli_query($con,"SELECT MAX( 'task_id' ) AS max, MIN( task_id) AS min FROM task ");
$row   = mysqli_fetch_row($tt1); 

echo $row['max'];
echo $row['min'];

You can mysqli_fetch_row() instead mysqli_fetch_array() or mysqli_fetch_assoc() because you try to get one row, and in select query add alias to columns, to call result with the alias: MAX( 'task_id' ) AS max ==> $row['max'].
