I can use the following code to find the set of rows in the data who have Buying set to 1, and whose Close is greater than the set Cut_Off_Price.

import pandas as pd

df = pd.read_csv('data.txt')
buy_signal = df[df.Buying & (df.Cut_Off_Price != 0) & 
                (df.Close > df.Cut_Off_Price)]
print(buy_signal)

df['Signal_Buy'] = buy_mask + 0 #add 0 to convert from bool to int
