Here's your regex:

preg_match_all('/Q[0-9]+?\.\s([^\?]+\?)/', $contents2, $matches);

Your strings will be in $matches[1]:

array(2) {
  [0]=>
  array(2) {
    [0]=>
    string(29) "Q1. What is the value of 2+3?"
    [1]=>
    string(31) "Q2. What is the value of 10+30?"
  }
  [1]=>
  array(2) {
    [0]=>
    string(25) "What is the value of 2+3?"
    [1]=>
    string(27) "What is the value of 10+30?"
  }
}

Assumptions:

  • The question is always preceded with Q##. where ## is any combination of numbers.
  • The question always ends with a ? mark.

Regex breakdown:

  • Q[0-9]+? - Match starts with the letter Q, followed by 1 or more characters that are digits between 0 and 9. Non-greedy with ?.
  • \. - a period . follows the Q## portion of the match
  • \s - a space follows the period . portion of the match
  • ( - begin a group selection
  • [^\?]+ - match one or more characters that are not a ? character (the \ escapes the regex character)
  • \? - include the question mark at the end of the match
  • ) - end of group selection

Bonus: in case the question is not phrased with a ? mark at the very end (e.g. "What is 2 + 2? Pick the correct answer."), to instead match to the consistent o1 following the question, you can use:

preg_match_all('/Q[0-9]+?\.\s(.+?)o1/', $contents2, $matches);

This strategy instead matches any character after the Q##. in a non-greedy manner until the o1 portion been found. It's intentionally left out of the group so the matches don't contain the o1.
