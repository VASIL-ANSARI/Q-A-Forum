We need to explicitly create an object of class Thread when we are performing multhreading by implementing Runnable interface. In the case of implementing Runnable interface, we gotta create Thread object. Having said that, since here the ThreadDemo class already extends Thread, we need to do it in the start method, while we must have done it while implementing Runnable.
