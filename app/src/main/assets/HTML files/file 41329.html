Use the STUFF function:

SELECT STUFF(
    (SELECT ',' + CODE 
     FROM LG_xxx_CLCARD 
     WHERE CODE < 'KUR' AND CODE > 'CARI1' 
     ORDER BY CODE 
     FOR XML PATH ('')), 1, 1, '')

If you need the single quotes, then the following will do the trick:

SELECT STUFF(
    (SELECT ',' + '''' + CODE + '''' 
     FROM LG_xxx_CLCARD 
     WHERE CODE < 'KUR' AND CODE > 'CARI1' 
     ORDER BY CODE 
     FOR XML PATH ('')), 1, 1, '')
