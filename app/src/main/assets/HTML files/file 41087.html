From Apple's documentation

An SKAction object is an action that is executed by a node in the scene

You will need to either add soundEngine to the scene with

self.addChild(soundEngine)
soundEngine.playSoundtrack()

or pass the scene into the playSoundtrack method and run the action on the scene

soundEngine.playSoundtrack(self)

func playSoundtrack(SKScene *scene) {
    scene.runAction(SKAction.repeatActionForever(SKAction.playSoundFileNamed("theme.mp3", waitForCompletion: true)))
}

Of course, if you choose the latter it may be better to simply run the action from the SKScene subclass.
