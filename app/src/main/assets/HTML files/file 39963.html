In test.py you are importing the module package, not the module.module module. In this package's __init__.py you're doing a star import from the module.module module. So when you rebind module.foo in test.py, it's the package's 'foo' you are rebinding, not the module's one. Wrt/ the other objects, you are mutating them, not rebinding them, which is a different operation.

If in test.py you replace import module with from module import module you'll get a different behaviour.
