You have to implement the __repr__ method in your class. Example:

class CellClass:
  def __init__(self, value):
    self.value = value
  def __repr__(self):
    return 'Cell value: %s' % (self.value)
c = CellClass('cell text')

Output:

In [2]: c
Out[2]: Cell value: cell text
