You load the data in asynchronous way which means that the HTML loads first (all the DOM elements are shown) and when you get the response from the server, then the object $scope.eventData gets values.

I believe the best way to handle this is by showing a loader png. Just show the loader.png before calling the $http.get("/GetEvents") and hide it inside the .then().

Another trick:

use ng-if directive to show a 'loading' text inside your DOM elements until you get the response from the server, like so:

<div>
  <h3 ng-if="eventData.heading != ''">{{eventData.heading}}</h3>
  <h3 ng-if="eventData.heading == ''">Loading..</h3>

  <p ng-if="eventData.description !=''">{{eventData.description}</p>
  <p ng-if="eventData.description ==''">Loading..</p>
</div>

Hope helps, good luck.
