I think you are missing a point. You are settings the result in your $scope as items, but you use item in your view. If you expect to receive more result, you should use ng-repeat, otherwise, rename it to $scope.item and you should see the result. To use with ng-repeat, you need to change the div to following

<div ng-repeat="item in items">
  <h2>{{item.movieTitle}}</h2>
  <p>{{item.movieId}}</p>
  <p>{{item.movieDescription}}</p>
</div>
