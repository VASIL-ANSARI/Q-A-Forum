There's an open issue about it in fullpage.js issues forum.

No solution was implemented yet. The default behavior of fullpage.js is the following one:

fullPage.js will animate to the top if you come from a section above the destination one. It will animate to the bottom if you come from a section below the destination one.

Why? Because the feature was originally though to be used for footers, and in that particular scenario this behavior is the expected one.
