You'll need to specify a launchMode for DashBoardActivity in your manifest. Choosing the correct mode depends on the behavior that you want. For example, singleInstance will ensure that only one instance of DashBoardActivity ever gets created:

<activity [...] android:launchMode="singleInstance" />

However, that also means that you can only use the activity for one purpose. A similar option that's less restrictive is singleTop.
