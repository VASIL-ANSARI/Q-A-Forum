It's a problem with folder path containing spaces (or quotes), I guess. Either JBOSS_HOME or JAVA_OPTS .

==> Work-around: Install JBoss on a folder whose path does not contain spaces, parenthesis, quotes, Unicode characters.

Example of JBoss home:

  • C:\Program Files\jb-eap-6 --> Bad (spaces)

  • C:\Program Files (x86)\jb-eap-6 --> Worse (spaces & parenthesis)

  • C:\jb-eap-6 ==> OK , no spaces no parentheses.

  • D:\apps\jb-eap-6 ==> OK , no spaces no Unicode

If the problem still remains, apply that work-around for JAVA_HOME and JAVA_OPTS as well.
