By looking at your code, I guess you haven't initialised btnDisplay thats why your app getting crashed. Initialise btnDisplayin addListenerOnRadioBox() method before setting onClick() callback.

public void addListenerOnRadioBox() {
        radioSexGroup = (RadioGroup) findViewById(R.id.radioSex);
        //Initialise button 
        btnDisplay = (Button) findViewById(R.id.buttonDisplay);
        btnDisplay.setOnClickListener(new OnClickListener() {

            @Override
            public void onClick(View v) {

                int selectedId = radioSexGroup.getCheckedRadioButtonId();
                radioSexButton = (RadioButton) findViewById(selectedId);
                Toast.makeText(HelloWorldActivity.this,radioSexButton.getText(), Toast.LENGTH_SHORT).show();

            }
        });
    }
