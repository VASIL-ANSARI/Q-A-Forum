Seems like you're using two directives with a isolate scope (scope: {...}) not just in the same view but on the same element. your element probably looks something like this:

<p mw-confirm-click="..." translate="..." ...></p>

you either have to remove one of the directives with the isolate scope (mw-confirm-click or the translate directive) or you have to remove the isolate scope from your directive and use the variables from the parent scope...
