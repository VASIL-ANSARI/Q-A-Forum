You can't. Childs in CSS are counted since the parent element. So doesn't matter how classnames you are define. The way to achieve this with css is wrapping the second group of elements into a parent.

With jQuery yes, you can, but not with the CSS selector that have the behaviour that I tell you in the first paragraph. You can make this:

$('.suggestedContentAmountRowMonthly').get(4)

Be careful, because it returns a DOM object and not a jQuery object. You can jqueryfy it like this:

var DOMobj = $('.suggestedContentAmountRowMonthly').get(4);
var jQueryObj = $(DOMobj);

Good luck

EDIT

As Rycochet said in the comments, you can use eq() function to return a jquery object directly:

$('.suggestedContentAmountRowMonthly').eq(4).css({clear:'left'});
