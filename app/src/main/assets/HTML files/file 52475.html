The default implementation of the ArrayAdapter is to automatically call notifyDataSetChanged() when an item has been add(), remove(), insert(), clear(), etc. In your case you're adding an item to the list which is used by the adapter, so the adapter is notified automatically.

You can change this behaviour by calling adapter.setNotifyOnChange(false). Just keep in mind that notifyDataSetChanged() overrides the notifyOnChange back to true, so if the former method is called you'll have to set notifyOnChange again.
