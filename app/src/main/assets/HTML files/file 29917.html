You need 3 foreach loops: 1 for $adata, 1 for $sdata & 1 for $qdata.

$sql='insert into assignment_answers
            (a_id, s_id, q_id)
            values (:aid, :sid, :qid)';
$command1=$connection->createCommand($sql);
foreach($adata as $ad)
{
    foreach($sdata as $sd)
    {
        foreach($qdata as $qd)
        {
            $command1->bindParam(":aid",$ad->id,PDO::PARAM_STR);
            $command1->bindParam(":sid",$sd->id,PDO::PARAM_STR);
            $command1->bindParam(":qid",$qd->id,PDO::PARAM_STR);
            $command1->execute();
        }
    }
}

I assumed you also have a $adata set like $sdata & $qdata.
