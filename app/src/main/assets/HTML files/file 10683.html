You can do it like this:

if(value1 > value2)
    printf("value1 is greater");       
else if (value2 > value1)
    printf("value2 is greater");
else
    printf("Invalid output");

Or preferably with brackets:

if(value1 > value2)
{
    printf("value1 is greater");       
}
else if (value2 > value1)
{
    printf("value2 is greater");
}
else
{
    printf("Invalid output");
}

If you wish to put more than one statement in your if-else blocks, you will need to use brackets.
