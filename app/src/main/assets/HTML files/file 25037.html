If your property has one of standard validation attributes and you enable client validation, the view engine generates markup and javascript for client validation.

There is some built in dataannotations that works out of the box: Compare, Range, RegularExpression, Required, StringLength.

If you want, you can create your own dataannotation attribute that will work on the client. For this, you should add to attribute method:

public IEnumerable<modelclientvalidationrule /> GetClientValidationRules
(ModelMetadata metadata, ControllerContext context)
{
    ...
}

That will return validation rules. Based on this rules ViewEngine will generate markup and javascript.

You can read about this in details here: Custom Client Side Validation in ASP.NET MVC3
