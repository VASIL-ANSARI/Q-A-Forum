You can use background-position to set this. Using margin will affect the entire containing element. In the shorthand background property you've set, it's the second to last two values. Also, strictly speaking the background-color property should be set first. Try this:

background: white url("../Images/DatePicker.png") no-repeat scroll 35px center;

This will position the background image 35px from the left edge of the containing element. You may need to amend the value - I just guessed.
