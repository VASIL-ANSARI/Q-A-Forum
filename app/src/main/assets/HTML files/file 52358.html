$('input.Stylednumber').keyup function binds event to elements that are currently in dom. If you want to bind an event to dynamically created elements, you must specify global scope like body or in this situation it can be table#tbl.

$('table#tbl').on('keyup', 'input.Stylednumber', function(){ ... });
