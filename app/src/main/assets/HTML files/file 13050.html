You can achieve with the combination of white-space & display:inline-block like this:

HTML:

<div class="parent">
    <div class="child">1</div>
    <div class="child">2</div>
    <div class="child">2</div>
</div>

CSS:

.child{
    display:inline-block;
    *display:inline; /* for IE7*/
    *zoom:1;/* for IE7*/
    min-width:100px;
    min-height:50px;
    margin-right:10px;
    background:red;
    white-space:normal;
}
.parent{
    white-space:nowrap;
}
