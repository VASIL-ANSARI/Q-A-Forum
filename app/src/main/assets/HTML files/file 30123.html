1. The keyword template is lower-case.
  2. I'm guessing you want to use your template argument as part of the node type.
  3. You're missing a semicolon at the end of your class declaration.
  4. You're missing the template argument in your variable declaration.

Put it all together, and you get:

template <class T>
class linkedList
{
public:
    T *mStart; // or node<T> *mStart; (hat tip: @chris)
    linkedList():mStart(NULL) {}
};

linkedList<int> que;
