first of all, you don't need temporary tables (especially 2nd one, as its not used at all):

SELECT
A.from, A.to
FROM booking as A
LEFT JOIN
turnostomados as B
ON (A.from = B.to)
WHERE A.day = ".$weekday."
AND B.confirmed IS NULL

then, run explain extended to find out execution plan and add appropriate indexes to these two tables

probably you will need index on bookings (day, from, to)
