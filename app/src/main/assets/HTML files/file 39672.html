Mergesort, as the name suggests, is based on merging: combining two sorted lists into one. The algorithm divides the list to be sorted into two (roughyly) halves, and sorts each half (this is done by the recursive calls to merge sort). The next two loops each copy half of the results from array a into array b; the last loop re-fills a from b, walking along the two halves of b and, at each step, taking the smaller of the two (or, once one half has ben used up, just taking it form the other) for the next value to go into a.
