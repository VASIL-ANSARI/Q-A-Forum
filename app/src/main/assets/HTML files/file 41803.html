scanf("%s",n); does not read a string with spaces (or any white-space) in it.

1) To read a movie title with spaces, use fgets()

printf("Insert name of the movie:");

// remove trailing \n
size_t len = strlen(n);
if (len && n[len-1] == '\n') n[--len] = 0;
