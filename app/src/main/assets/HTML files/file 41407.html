replace symboolconstante with a call to a symboolconstante() method. Each sub-class can override this method.

In the abstract class you can define it to be abstract :

protected abstract String symboolconstante();

In each sub-class you override it :

@Override
protected String symboolconstante()
{
    return "something";
}

And the abstract class's toString() method would call this method :

public String toString() {
    String res ="(" + Character.toString(propositieletter1) + " " + symboolconstante() + " " + Character.toString(propositieletter2) + ")";
