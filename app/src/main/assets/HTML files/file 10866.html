To get a string array from the data string just go

string[] data = accNos.Split(',');

You might want to clean up your array values (e.g. any leading spaces):

for (int i = 0; i < data.Length; i++)
{
    data[i] = data[i].Trim();
}

Then bind your data to your dropdown list

cmbAccounts.DataSource = data;
cmbAccounts.DataBind();

Or, if you are using a BindingSource

bindingSource.DataSource = data;
bindingSource.ResetBindings(false);
