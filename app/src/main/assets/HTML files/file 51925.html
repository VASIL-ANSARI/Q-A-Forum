I can help more with exactly how to do this so feel free to message me, but the short answer is in this great package of filters:

https://github.com/a8m/angular-filter#filterby

Filter by helps with exactly what you are trying to do, however I suggest converting your top level object into an array first.

collection | filterBy: [prop, nested.prop, etc..]: search: strict[optional]
