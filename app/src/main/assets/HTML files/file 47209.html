You just need to call this script for using datable after adding jquery and datatable js :-

$(document).ready(function() {
$('#example').dataTable( {
    "scrollY": 200,
    "scrollX": true,
    "order": [[ 1, "desc" ]]

} );
} );

Here, 1 is column no. needed to sort , you can change it as per need.
