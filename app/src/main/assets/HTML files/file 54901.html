From the look of your setColors function you are making visual modifications to the blocks using JavaScript. The general consensus in front end web development is to leave those tasks to CSS.

In your case you are checking of your DIVs have the classes block and blck--bottom. So rather than using JavaScript perhaps you could attempt a stylesheet with the following selector:

.block.blck--top {
    background-color: '#51616F';
}

.block.blck--middle {
    background-color: '#51616F';
}

.block.blck--bottom {
    background-color: '#51616F';
}

In contrast to this however, if you need to perform further actions that are not visual based you can take on the following approach.

function setColors(target) {

target.forEach(function (element) {

    if(element.classList.contains('blck--top'))
    {
        element.style['background-color'] = 'red';
        /* - Extra code here - */
    }
    else if(element.classList.contains('blck--middle'))
    {
        element.style['background-color'] = '#51616F';
        /* - Extra code here - */
    }
    else if(element.classList.contains('blck--bottom'))
    {
        element.style['background-color'] = '#394B5A';
        /* - Extra code here - */
    }
});

But if you really wish to use switch case statements then:

function setColors(target) {

target.forEach(function (element) {

    switch (element.getAttribute('class')) {
        case 'block blck--top':
            $(element).css({
                'background-color': 'red'
            });
            break;

        case 'block blck--middle':
            $(element).css({
                'background-color': '#51616F'
            });
            break;

        case 'block blck--bottom':
            $(element).css({
                'background-color': '#394B5A'
            });
            break;
    }
});

Please note that I have also provided a way for you perform the same tasks using plain old JavaScript (JQuery-less if you will).
