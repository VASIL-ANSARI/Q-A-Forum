So if you want to use javascript to write from right to left, you can use following code:

HTML

<div id="test">Hello Germany!</div>

Javascript

var el = document.getElementById('test');

var text = el.innerHTML;
text = text.split("");
text.reverse();
text = text.join("");

el.innerHTML = text;

OUTPUT

!ynamreG olleH

JSFiddle

Link: http://jsfiddle.net/8xo9qde8/

So it's simply split the string into an array, reverse it, joins the array back to a string and insert it back to the div.

So it's literally written from right to left.
