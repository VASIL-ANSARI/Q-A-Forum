The JVM loads the classes from a JAR file. With the default classloader it is not possible reflect the change in your jar file to the program running. The web servers uses their own classloader to reflect the changes in the class files in runtime. You can write your own classloader, look here http://en.wikipedia.org/wiki/Java_Classloader and http://www.javablogging.com/java-classloader-2-write-your-own-classloader/
