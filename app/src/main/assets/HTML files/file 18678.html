This sounds really like some memory leaks or not releasing memory which is not used anymore by your app.

There is no general solution to be given, because the cause of those problems could be anywhere and be anything.

What you could do is:

  1. ensure that you have defined didReceiveMemoryWarning methods in your classes and that you try to free not used memory there;

  2. running your app under the Leaks and Memory Allocation profiling tools and see what you can to about what they report to you;

  3. if you do not have memory leaks, try and use snapshot analysis to find abandoned memory.
