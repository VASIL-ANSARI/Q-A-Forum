Typically you would use some form of Dependency Injection to inject the mock AuthenticationService into RestService. Try rewriting your RestService like,

public class RestService {
    private final AuthenticateService auth;
    public RestService(AuthenticateService auth) {
       this.auth = auth;
    } 
    public string authenticate(String user, String pass){
        return auth.authenticate(user,pass);
    }
}

And then in your test do

 rs = new RestService(authenticate_object);
