You can save the date into the table as DATETIME format and while retrieving back use the below query to format in a right way

SELECT DATE_FORMAT(datetime, '%d/%b/%Y') datetime FROM table

or after you get the actual value you can convert to the format like

$php_timestamp_date = date("d F Y", $php_timestamp);

or 

$php_timestamp_date = date("d F Y H:i:s", $php_timestamp);
