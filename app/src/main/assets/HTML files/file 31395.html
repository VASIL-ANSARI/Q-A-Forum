That second line has a number of issues. You have unescaped double quotes inside a double-quoted string, and appear to be trying to concatenate without using the concatenate operator.

Change your second line to this:

echo "<td><img src='$src' height='30px' width='30px'></td>";

This now has double quotes only at the beginning and end, with single quotes inside. The $src variable will be parsed and work properly since it is in a double-quoted string.

This may be some helpful reading: http://www.php.net/manual/en/language.types.string.php
