As you can see from the documentation, the parameters you need to pass with the request in order to obtain customers created within a specific range are created_at_min and created_at_max. Unless you want to type the timestamps manually, we're going to need the datetime object from the datetime module.

from datetime import datetime
time_format = "%Y-%m-%dT%H:%M:%S+00:00"
min_date = datetime(year=2016, month=5, day=1).strftime(time_format)
max_date = datetime(year=2016, month=6, day=1).strftime(time_format)

The formatting string we're using with the strftime method will give us our time in the format required by the Shopify documentation, with UTC timezone hard-coded in. If you want to use a different timezone, you can either hard-code in a different time offset, or use the pytz module.

Now, for actually calling the API. When using Shopify resources' find methods, you pass property name/value pairs as keyword arguments, like so:

customer_list = shopify.Customer.find(
    created_at_min = date1,
    created_at_max = date2
)

And voila, this should return you a list of Customer resources, or an empty list if there are no matches.
