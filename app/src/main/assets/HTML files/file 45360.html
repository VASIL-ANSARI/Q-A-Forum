All of the above aside, you might be able to achieve what you want with the new .dt accessor attribute for a Series of datetime64[ns] objects (which the Date column above is). This was introduced in pandas 0.15.0. It allows you to do things like df.Date.dt.year to get the year from the datetime64[ns] object. Also month, day, hour, minute, second, etc., etc.
