Yes. It can be done but there need to be some elements before or after the list view. Thus there will be a relative layout inside which there will be a list view and a textView.

<RelativeLayout
xmlns:android="http://schemas.android.com/apk/res/android"
android:orientation="vertical"
android:layout_width="fill_parent"
android:layout_height="fill_parent">
    <ListView
        android:id="@+id/listview"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        />
    <Button
            android:id="@+id/btnButton"
            android:text="Hello"
            android:layout_width="match_parent"
            android:layout_height="wrap_content"
            android:layout_gravity="bottom" />
</RelativeLayout>

In such a case where there are many elements above and below a listView you will be able to scroll out of the listView frame.
