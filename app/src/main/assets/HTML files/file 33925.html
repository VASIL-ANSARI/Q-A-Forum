If they are both within the same container, just apply your :hover pseudo class to the parent element and apply the styles using that.

HTML

<a class="hoverable">
    <div id="a">Foo</div>
    <dvi id="b">Bar</div>
</a>

CSS

.hoverable:hover #a, .hoverable:hover #b {
    visibility:visible;
}

Should bear something similar. If however the items are not in a common parent this won't work for you. Please post your code so we can get a better understanding :)

AN ALTERNATE CONCEPT

If both elements are direct siblings you can also try something along the lines of 

#b:hover + #a {
    visibility:visible
}

This would require b to be drawn before a however. you could manipulate this ordering using other css.
