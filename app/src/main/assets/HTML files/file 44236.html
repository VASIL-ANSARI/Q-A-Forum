This will prevent the footer from overflowing from the top of the page. You can change the max-height attribute below to fit your need.

footer.container {
   max-height: 160px;
   overflow-y: scroll;
}

This must be added in your mobile media query part.

@media screen and (max-width: 480px) {
  footer.container {
       max-height: 160px;
       overflow-y: scroll;
    }
}

Pure Jquery based solution,

// Footer
$(".footToggle").click(function(){
    $("footer div:nth-child(2)").slideToggle();
    $("footer").css({ "maxHeight" : $(window).height(), "overflowY" : "scroll" });
});
