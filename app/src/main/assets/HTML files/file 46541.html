connect are additive, they do not replace each other. With your code, if I click on "ordered" then "random", the finished signal will be connected to two slots: self.ordered and self.random. I don't think it's the behaviour you're expected.

Instead, you could have a string value, let's call it whatToDoNext. A click on the menu set the string to the action's text:

def sortedEvent(self,action):       
    self.whatToDoNext=action.text()

In your case, whatToDoNext can be "ordered", "random" or "replay".
The finished signal of mediaObject is always connected to the same function:

self.mediaObject.finished.connect(self.on_finish)

def on_finish(self):
    if self.whatToDoNext=="ordered":
        #read the next song
    elif self.whatToDoNext=="random":
        #read a random song
    else:
        #replay
