Two way data binding is what you are looking for. Use angular for example. Very simple example is here: 

<div ng-app>
    Name:
    <input type="text" ng-model="name"/>
    <div>Hello {{name}}!</div>
</div>

http://jsfiddle.net/anandmanisankar/L6nofag6/.

There are other ways to achieve what you want, but two way data binding is IMHO the easiest.
