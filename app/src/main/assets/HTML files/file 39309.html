It looks like it's buggy:

Stackoverflow: Singleton Scope binding not working as intended
Stackoverflow: Ninject InSingletonScope with Web Api RC

Both posts are old, but you seem encountering the same issues. Maybe you should use this approach instead:

kernel.Bind<System.Runtime.Caching.ObjectCache>()
        .To<System.Runtime.Caching.MemoryCache>()
        .ToConstant(new MemoryCache("Enquiries"));

or just

 kernel.Bind<System.Runtime.Caching.ObjectCache>()
        .To<System.Runtime.Caching.MemoryCache>()
        .ToConstant(MemoryCache.Default);
