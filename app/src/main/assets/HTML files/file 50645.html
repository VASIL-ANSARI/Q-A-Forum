So I did not know that in order to rollback the DB I need to enable the Db module:

class_name: ApiTester
modules:
    enabled:
        - Laravel5
        - Db
        - REST:
            url: http://localhost:8000/api/
            depends: PhpBrowser
    config:
        Laravel5:
            cleanup: true
            environment_file: .env.testing

And that did it.
