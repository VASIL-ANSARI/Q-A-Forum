To "connect" two scrollable container's scroll event, you can do something like

document.getElementById('div1').onscroll = function(e) {

  var x = ? /* do your calculation to find the right 'scrollTop' for the 2nd div, according to your logic */

  document.getElementById('div2').scrollTop = x;
}

Here is a simple jsfiddle demo with two <div>s and an onscroll event handler attached to the 1st one. In the handler function, we calculate the proportion of the height to the scroll position and set the target <div>'s scroll position so its height has the same proportion to it. The result is that regardless of how much content is in each container, the 2nd <div>'s scroll position is synchronised with the 1st one.
