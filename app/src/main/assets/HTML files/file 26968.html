This should do what you want:

Enable responsiveness

<meta name="viewport" content="width=device-width, initial-scale=1.0">
<!-- Bootstrap -->
<link href="css/bootstrap.min.css" rel="stylesheet" media="screen">

Add the container with the row and span (bootstrap spans a total of 12 so the number of spans needs to add up to 12

<div class="container">
  <div class="row">
    <div class="span6">Column 1</div>
    <div class="span6">Column 2</div>
  </div>
</div>

This should work is the responsive layout is enabled.

A full working example:

<!DOCTYPE html>
<html lang="en">
 <head>
  <title>Bootstrap, from Twitter</title>
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link href="css/bootstrap.css" rel="stylesheet">
  <link href="css/bootstrap-responsive.css" rel="stylesheet">
  <style>
   body {
    padding-top: 60px;
   }


  </style>

 </head>
 <body>

<div class="container">
  <div class="row">
    <div class="span6">Column 1</div>
    <div class="span6">Column 2</div>
  </div>
</div>

  <script src="js/jquery-2.0.3.min.js"></script>
  <script src="js/bootstrap.min.js"></script>

 </body>
</html>
