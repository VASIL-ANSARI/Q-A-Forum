You need to import the generate method from where its coming from in the js file.

Add an import statement to the to of the file pointing to the module supporting this generate method.

Looking at the file, this is currently commented out: 

// var ecc = require('../dist/0.1/ecc');

Try un-commenting it.

Note, this has to be commented when used in HTML as a JS file.

Edit Just saw your changes. Maybe you're using an outdated ecc module. As the error states there is no generate method in the ecc file.

Alternatively, install the latest ecc via npm using npm install eccjs and replacement var ecc = require('../dist/0.1/ecc'); with var ecc = require('eccjs');
