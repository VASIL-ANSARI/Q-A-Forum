You should get rid of the '' around LABEL etc.

Use this:

' Select DECODE (VERSION_JPA,
               1, ''Abierto'',
               2, ''Cerrado'',
                  ''Desconocido'') as LABEL,
       ''Simulacion'' AS SERIE,
       count(VERSION_JPA) as VALUE 
FROM EVENTOS 
WHERE TIPO = ''simulacion''
group by VERSION_JPA '
