What your JS code produces is

a    b   c 
 d   e  f

But the valid should be

a,b,c
d,e,f

in code

var data = "a,b,c\r\nd,e,f";

or in tab-delimited format

a    b    c
d    e    f

in code

var data = "a\tb\tc\r\nd\te\tf";

CSV is a short of Comma-separated values
