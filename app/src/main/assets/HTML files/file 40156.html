I dont think you need "whereHas", but "Eager Load Constraints". This is found under eager loading in the laravel docs.

For Example:

$users = User::with(array('posts' => function($query)
{
    $query->where('title', 'like', '%first%');
}))->get();

You however need to nest it and load more relations.

So it's like this:

$message = Message::where('m_to','=',12345)
->with(array('messageVideos.languages' => function($query)
{
    $query->where('language_id', '=', 2);
},
'messageUser'))->first();
