You created an array (a holder for several Tablet objects), but you didn't actually create any Tablets to go inside it. Right now, T (which really should be lowercase; T looks like both a constant and a type parameter) has the following contents:

T: {null, null, null, null, null, null}

You need to create new Tablets and put them inside your array, perhaps like this:

for(int i = 0; i < array.length /* don't hardcode the size twice */; i++) {
    array[i] = new Tablet();
    array[i].set("111");
}
