Ok here is something you're looking for:

<?php
$array = array(
        'name' => 'CMS Logo',
        'id' => 'cms_icon_image',
        'type' => 'select',
        'options' => array(
            'Wordpress'   => 'Wordpress',
            'Magento'     => 'Magento',
            'OpenCart'    => 'OpenCart',
        )
    ) ;

foreach ($array as $key => $value) {
    $array[] = $key;

    if (is_array($array[$key])) {
        $results = array_keys($array[$key]);
        echo "<select>";
        foreach ($results as $result => $val){
            echo "<option class='" . $val . "'>" . $val . "</option>";
        }
        echo "</select>";
    }
}
?>

This will output a select box with an <option> for each option defined in the array.
