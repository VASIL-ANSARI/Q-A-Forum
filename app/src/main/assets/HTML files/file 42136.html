You need to iterate over key, value pairs and then append both key and sum of values in list as tuple.

y = []
for key, item in d.iteritems():
    y.append((key,sum(item))) # total is a function that sums the integers
print(y)

Gives Result :

[('Clare', 18), ('Eva', 13), ('Bob', 20), ('Dennis', 16), ('Alice', 15)]

Can be done in simpler way using list comprehension and calculating total of values using built-in sum function:

>>> d = {"Alice": [1, 2, 3, 1, 3, 5],
...      "Bob": [5, 2, 4, 1, 3, 5],
...      "Clare": [1, 2, 3, 3, 4, 5],
...      "Dennis": [2, 2, 3, 3, 3, 3],
...      "Eva": [2, 2, 2, 1, 3, 3]}
>>> print  [(k, sum(v)) for k,v in d.iteritems()]
[('Clare', 18), ('Eva', 13), ('Bob', 20), ('Dennis', 16), ('Alice', 15)]
