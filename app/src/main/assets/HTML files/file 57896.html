Q: For the Oid, do I only need to input ONE Oid(the first item of the Bulk)?

A: Yes, you only need to set the first OID.

Q: If I wish to get a bulk request starting from the stated Oid, and the following 4 items after the first, do I simply set the "Max-Repetition" to 5?

A: Yes. 5 is the minimum that you need to set for max-repetition, but you can set to a bigger value (e.g. 10). The way you would work with the response is to traverse over the response PDU and get out each OID at a time.
