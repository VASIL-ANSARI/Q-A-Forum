You have to remove the space between the attributes, so this

$('button[data-path="' + sortpath1 + '"] [data-order="desc"]')

becomes this

$('button[data-path="' + sortpath1 + '"][data-order="desc"]')
//                                     ^^ look ma, no space

otherwise you're looking for descendant elements, just as you would with

$('div span')
