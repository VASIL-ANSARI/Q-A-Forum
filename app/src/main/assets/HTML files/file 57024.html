Use jquery attr API :

$("a[data-module ='*']").click(function() {
    var dataModule = $(this).attr('data-module');
});

EDIT

Currently your jquery selector is wrong, it is interpreted as :

Find all <a> tags whose data-module attribute has a value of *

So in your example, there is no link that matches this condition.

Example: I will get all tags with data-module that starts with rooms. Then my valid selector would be $('a[data-module^="rooms"]')
