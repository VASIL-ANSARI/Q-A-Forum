How about using the following:

Test: {0:15} Something {1}".format(Variable,AnotherVar)

You can calculate the numbers based on the length.

Adding more example:

>>> a="test"
>>> b="another_one"
>>> "A: {0:20} B {1}".format(a,b) 'A: test                 B another_one'
>>> "A: {0:15} B {1}".format(a,b) 'A: test            B another_one'
>>> "A: {0:4} B {1}".format(a,b) 'A: test B another_one'

You can also use a variable for the second number in the first {} based on the length of strings you want to print.
