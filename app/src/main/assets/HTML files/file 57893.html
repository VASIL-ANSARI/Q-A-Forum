There are CSS methods for creating tables and accomplishing the commands. However, at times one wants a purely in-line HTML method to create a table, and the provided code offers suggestions.

One method in which centering can be accomplished is through the align tag. The align command in the table tag centers the table on the page, and "left" or "right" would change the placement as well.

TR align="center" will justify text in the row. Or text could be centered individually using TH / TD tags (center, left, right).

Standalone div commands could be used as well.

TH tags are optional heading rows. Background color can be set at the TH level, in any row or cell, and in the table tag itself. 'transparent' is a safe color to ensure use of the page's background color.

Column width can be set in TH and TD cells using percents or fixed values such as pixels.

There are a few ways to place images/captions on separate lines:

~ The laziest is to use the BR tag, which creates a line break and is comparable to Shift-Enter keyboard. BR has its place in coding but should not be overused. Adjust image padding in the img style tag to create buffer space for the caption text. (Row 15:00)

http://www.w3schools.com/tags/tag_br.asp

One could alternately use 'p' or 'h' tags around the text; they define text styling and create new lines (roughly comparable to Enter keyboard).

~ A recommended approach is to use a small table within the cell that contains separate rows for image and caption. Image padding and table cell padding & spacing can be adjusted accordingly. (Row 16:00)

~ Or combine the two methods to create rows for image1 +'br' caption1, and image2 +'br' caption2. Image padding and table cell padding & spacing can be adjusted accordingly. (Row 17:00)

Table borders have been applied for ease of viewing.

Fonts have been set within the larger table as an example; however they could be defined easily in the stylesheet and implemented using a variety of command tags.

Hope this helps!
