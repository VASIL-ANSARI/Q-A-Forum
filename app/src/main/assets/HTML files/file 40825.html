Try it with numpy.

>>> import numpy as np
>>> a = np.array([['M', 'A', 'R', 'K'],
... [1,    3,   5,    4],
... [2,    6,   7,    8]], dtype=object)
>>> a[:,np.argsort(a[0])]
array([['A', 'K', 'M', 'R'],
       [3, 4, 1, 5],
       [6, 8, 2, 7]], dtype=object)

Without numpy:

>>> map(list,zip(*sorted(zip(*a))))
[['A', 'K', 'M', 'R'], [3, 4, 1, 5], [6, 8, 2, 7]]
