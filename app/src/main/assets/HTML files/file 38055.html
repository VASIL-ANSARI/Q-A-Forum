One way to do this is to use the '$stateChangeStart' event provided by ui-router This event is fired when the transition begins. You could inject the service handling your user and redirect it if he is not logged. This would looks like this :

app.run(['$state', 'Session' function($state, Session){

$rootScope.$on('$stateChangeStart', function(event, toState, toParams, from, fromParams){
    if (toState.name !== "login" && !Session.is_connected()) {
        $state.go("login");
        event.preventDefault();
    }

});

}]);

Additionally, you could also save the last state he was trying to access and redirect him to it once he is logged in.
