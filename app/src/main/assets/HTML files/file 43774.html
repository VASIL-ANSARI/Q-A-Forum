In python 3.X, urllib.response.readlines returns a byte string, and python 3 supposedly being more type safe and encoding friendly doesn't support all different encoding string within methods.

So your split method is actually been called on a byte string which expects a byte rather than a string.

So either you convert the data back to string 

lineSplit = str(lineSplit )

or pass a byte string delimiter

lineSplit = line.split(b',')
