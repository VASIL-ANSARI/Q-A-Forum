DECLARE @Table AS TABLE (Label VARCHAR(25), Location VARCHAR(25))

INSERT INTO @Table (Label, Location) VALUES ('OLR-2873-SSA/GTA','GTA')
   ,('OLR-2873-SSA/GTA','GTA')
   ,('OLR-2873-SSA/POW','POW')
   ,('OLR-2873-SSA/POWGTA','POWGTA')

SELECT
    Label = STUFF(
        (SELECT ',' + Label
        FROM
            @Table t
        FOR XML PATH(''))
        ,1,1,'')
    ,Location = STUFF(
        (SELECT ',' + Location
        FROM
            @Table t
        FOR XML PATH(''))
        ,1,1,'')

Basically you appear to be looking for a concatenation of strings in rows, there are a few ways if you look around the internet. One way I use is converting the rows to an xml string with a comma prefacing the value and no column name, then use stuff to tidy up the first comma. So the FOR XML PATH('') basically means no root element, then because there is no column name you won't get the other xml tags either e.g. no <label></label>. Adding the delimiter to the value of the column you now have a long delimited string that you just need to get rid of the first unneeded delimiter.
