There's no module that fit your needs, a little development should be done to retrieve the result you are looking for.

You could use the app/design/frontend/base/default/template/catalog/navigation/left.phtml block for this. In that block subcategories from the current category are shown, normally this block only is shown on listpages with subcategories. You can add this block to the homepage and replace <?php $_categories = $this->getCurrentChildCategories() ?> with custom code which retrieves the categories/menu structure.
