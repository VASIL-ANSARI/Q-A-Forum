There are several ways to center an element within it's parent. The simplest is to set the child to the value margin: 0 auto; what this does is set the top/bottom margin to zero and the right/left value to equal each other. Another way is to use flexbox. Set the parent to display:flex; and justify-content:center

Here are two sites which should help- https://css-tricks.com/centering-css-complete-guide/ https://css-tricks.com/snippets/css/a-guide-to-flexbox/
