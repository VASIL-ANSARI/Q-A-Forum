After reading an excellent blog post on the topic, I've implemented my solution. It ended up being fairly simple.

I provided a similar method to the existing one:

public static string GetStringValue(this IUnvalidatedValueProvider provider, string key, performValidation)
{
    return provider.getValue(key, !performValidation).AttemptedValue;
}

In the calling code, for the particular property that I didn't want validated, I made the following call:

var content = (provider as IUnvalidatedProvider).GetValue(key, false);
