To get this working I used a Handler and Runnable:

    public void todayPositionFound(final int position) {
        Handler h = new Handler();
        Runnable r =new Runnable() {
            public void run() {
                View v = calendarView.getChildAt(position);
                if (v!=null) {
                    v.setSelected(true);
                }
            }
        };
        h.postDelayed(r, 500);
   }
