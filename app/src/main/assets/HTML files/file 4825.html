You should probably be able to call new3 on you subclass

class MyClient < XMLRPC::Client
end
MyClient.new3({})

Or overwrite it if you need to do extra stuff:

class MyClient < XMLRPC::Client
  def self.new3(args)
    client = super(args)
    # do some more stuff
    client
  end
end
MyClient.new3({})
