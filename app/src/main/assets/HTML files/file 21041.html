Try using date_add

$d1 = '2012-11-08';
$d2 = date_add($d1, date_interval_create_from_date_string('3 days'));
echo $d2;

http://www.php.net/manual/en/datetime.add.php

If you're not running PHP 5.3, this should work:

$d1 = '2012-11-08';
$d2 = strtotime('+3 days', strtotime($d1));    
echo date('Y-m-d', $d2);
