It is performing bitwise OR | between IEEE80211_FCTL_FTYPE and IEEE80211_STYPE_QOS_DATA. And then sending the result to the function htole16().

Later the result of htole16() is bitwise and:ed with fc.

Just like calling a function foo(a + b);, the expression(a + b) will be evaluated first and then the result will be sent to the function foo();
