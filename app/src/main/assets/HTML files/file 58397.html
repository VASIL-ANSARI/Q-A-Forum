To get the CPU Utilization and Memory Usage you should go to your VM on the portal and enable Diagnostics then you'll be able to retrieve those data.

After that, you need to access WADPerformanceCountersTable in the storage account that is setup to store diagnostic data.

Here is the help page to help with connecting Java to table storage: http://www.windowsazure.com/en-us/develop/java/how-to-guides/table-service/
