for(indexHand = 1; indexHand <= numOfHands; indexHand++){
...
for(indexCard = 1; indexCard <= numOfCards; indexCard++){

These loops go from 1 to n(max index including). But indexing should be from 0 to n-1 . Therefore , these loops access index out of bounds .

Modify you loops to this -

for(indexHand = 0; indexHand < numOfHands; indexHand++){
...
for(indexCard = 0; indexCard < numOfCards; indexCard++){

Edit -

 totalHands[indexHand].cards[indexCard]->face = shuffledDeck[indexCard].face;
totalHands[indexHand].cards[indexCard]->suit = shuffledDeck[indexCard].suit;

Before these both statements you don't allocate memory to array of pointer cards and yet you dereference it (uninitialized pointers).

Allocate memory pointers of this array and then access array members.
