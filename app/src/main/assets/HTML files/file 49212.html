As they can't follow the same canonical format, in my opinion you can make work around it by making each of these tags with its allowed attributes as JSON or multi-dimensional arrays

This JS Fiddle shows a simple example

var DOMarray = {

  'div': ['id', 'class'],

  'a': ['href', 'id', 'class'],

  'img': ['src', 'id']

};





document.getElementById('test').innerHTML = '<strong>for the img tag, allowed attributes: </strong>' + DOMarray['img'];
<div id="test"></div>
