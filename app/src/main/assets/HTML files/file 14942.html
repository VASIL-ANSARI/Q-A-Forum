It seems that you've write to much code to validate your form... Validating select with jquery-validation plugin is very simple. You have to make sure that there is an option with an empty value. And then you just add the "required" class to the select.

<select id="mySelect" class="required">
  <option value="">--select--</option>
  <option value="value1">option 1</option>
  <option value="value2">option 2</option>
  <option value="value3">option 3</option>
</select>

See the exemple : select validation example

Try to do your validation like this, it should solve your problem.
