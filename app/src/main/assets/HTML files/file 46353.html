You should take Java out of the equation -- raw SQL can do this. Best approach is to add a new column, set the new column to a decimal-converted value of the existing string column, then (if you wish) drop the old string column and rename the new column.

If you're using MySQL, for instance, this would look something like:

ALTER TABLE mytable
  ADD COLUMN newcolumn DECIMAL(10,2);

UPDATE mytable
   SET newcolumn = CAST(oldcolumn AS DECIMAL(10,2))
 WHERE true;

ALTER TABLE mytable
 DROP COLUMN oldcolumn,
 CHANGE newcolumn oldcolumn DECIMAL(10,2);
