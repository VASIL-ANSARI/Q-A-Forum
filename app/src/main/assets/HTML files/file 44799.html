I would guess the main problem is you have whitespace in your comma-separated string.

This:

$parseLocations = "sioux city_ia, alabaster_al, boise_id, atlanta_ga";

will explode() to this:

array("sioux city_ia", " alabaster_al"," boise_id", " atlanta_ga")

Note the space at the beginning of all but the first item in the array.

If you just do this:

$parseLocations = "sioux city_ia,alabaster_al,boise_id,atlanta_ga";

You should get the array values you expect.

Now that being said, I don't know why, if you are hardcoding the city/state values to begin with, that you wouldn't just hard code into a usable array like:

$locations = array(
    array('city' => 'some value', 'state' => 'some other value'),
    ...
);

and save yourself all the exploding and other string manipulations to get to a workable data structure.
