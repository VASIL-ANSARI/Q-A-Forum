There are two ways I can think of to accomplish your goal:

  1. You could place each element of your column in its own div tag. Make the column that contains these smaller divs "position: relative;" and each of the smaller divs inside "position: absolute;"

    From there, you assign "top: 200px;" or whatever you wanted for each small div. This is starting at top of screen, working downward. Then set "left: 50px;" or whatever distance you want it from the left-hand side of the screen.

    I'm not saying this is the most elegant solution, but it will be an exact solution because you control down to the pixel where every element in your component ends up.

    See: www.w3schools.com/css/css_positioning.asp for more on that solution.
  2. You're alternative is to create specific height values to these smaller divs (ie "height: 50px;") and then use "margin-top: 10px;" (numbers are for you to decide) to place where those tags will start in reference to the div above it.

    See: www.w3schools.com/cssref/pr_margin.asp for more on this property.



Hope this helps.
