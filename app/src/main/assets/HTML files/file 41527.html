The best way to tackle this is to set the link to another element above it. Like this:

<a href="#intro">Introduction</a>

<br id="intro">
<br>
<h2>Introduction</a>

Now, you jump to just ABOVE the desired top of the page that it jumps to.

p.s. <br> is just an example. You could also make an empty div, an empty span, or just about anything you want. <br> is good though because it's a valid and functional element which means the page will have better SEO than having an empty div.
