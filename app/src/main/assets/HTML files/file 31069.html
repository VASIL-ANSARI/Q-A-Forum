If you want something to scale with the window and always be centered, use percentages, not fixed pixels. It can be a pain to setup, but always works. 

#appcontainer {
position: absolute;
left:50%; top:50%;
margin:-10% 0 0 -8%;
width:80%; height:auto;
border:1px solid blue;
background-color: rgba(146,255,92,0.74);
color:white;
}

#main {
width:100%; height: 100%;
border:1px solid red;
background-color: white;
color:black;
}
