If it will help someone, I achieved my goal by adding some delay to the opening of the notification bar. this way the onWindowFocusChanged() method always called as needed. So now my code implemented this way:

findViewById(R.id.status_bar).setOnTouchListener(new View.OnTouchListener() {
            @Override
            public boolean onTouch(View view, MotionEvent e) {
                Utils.showBelTitleBar(ctx, false, updateClkBtry, handler);
                ctx.handler.postDelayed(new Runnable() {
                    @Override
                    public void run() {
                        Utils.setNotification(ctx);
                    }
                }, 100);
                return false;
            }
        });
