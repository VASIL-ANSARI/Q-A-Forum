You can do something like this and only use a single table since they will have the same content. 

table, tr, td {

    display:block;

}



@media only screen and (min-width: 600px){

    table {

        display:table;

    }

    tr {

        display:table-row;

    }

    td {

        display:table-cell;

    }

}
<table>

    <tr>

        <td>Stuff 1</td>

        <td>Stuff 2</td>

    </tr>

    <tr>

        <td>Stuff 3</td>

        <td>Stuff 4</td>

    </tr>

</table>

This is probably best done without tables.

div {box-sizing:border-box; width:100%;}



@media only screen and (min-width:600px) {

   .half {

       float: left;

       margin: 0;

       width: 50%;

   }

   .row:after {

     content:"";

     display:table;

     clear:both;

   }

}
<div class="container">

    <div class="row">

        <div class="half">Stuff 1</div>

        <div class="half">Stuff 2</div>

    </div>

    <div class="row">

        <div class="half">Stuff 3</div>

        <div class="half">Stuff 4</div>

    </div>

</div>
