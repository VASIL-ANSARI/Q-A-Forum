Every elements in a column have the same dtype, so when you get a column Series from a DataFrame, they share memory.

But if you get a row Series from a DataFrame, it may not contain the same dtype values, in this case you will get a copy of the data that converted to general dtype.

I the following code, df.apply(f, axis=1) pass every row to the function, that contains the copy of the data:

df = pd.DataFrame([[0,1.0,0],
                   [0,1.0,0],
                   [0,1.0,0]],
                   columns=['a', 'b', 'c'])

def f(x):
    x.values[0] += 1
    return x

df.apply(f, axis=0) #pass every column to f, this will change the original dataframe
print df

df.apply(f, axis=1) #pass every row to f, this will not change the original dataframe
print df
