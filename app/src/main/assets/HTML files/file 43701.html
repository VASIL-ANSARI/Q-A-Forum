You seem to pass the parameters incorrectly for storeUser:

You pass:

 $user = $db->storeUser($zica_number, $first_name, $last_name, $email, $password);

but the method signature is:

storeUser( $first_name, $last_name, $email, $zica_number, $password)

try changing it to:

$user = $db->storeUser($first_name, $last_name, $email, $zica_number, $password);
