The gem includes the ActiveRecord::Base model SimpleHashtag which can be used to search hashtags. Your form could look like this:

<%= form_for :search, :url => your_result_path do |f| %>
  <%= f.text_field :q %><br />
  <%= f.submit "Search" %>
<% end %>

Of cause you have to replace your_result_path with a path helper that actually exists. Then in the controller action which receives the data you can use this code to provice the search results:

@results = SimpleHashtag.where("name LIKE ?", "%#{params[:search][:q]}%")

Then in the view you can loop through the results which are accessable as @results. 

<ul>
<% @results.each do |result| %>
  <li><%= link_to result.name, result %></li>
<% end %>
</ul>
