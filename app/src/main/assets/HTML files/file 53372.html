What you are looking to do is give the text inside an textarea. to do this you need to replace your textarea code:

<textarea
 style="width:500px;height:200px;"
 name="PageContent"
 id="PageContent"
 class="date"
 value="<?php echo $row['PageContent']; ?>">
</textarea>

With this:

<textarea
 style="width:500px;height:200px;"
 name="PageContent"
 id="PageContent"
 class="date">
  <?=htmlentities($PageContent)?>
</textarea>

You can also see that the code is corrected by http://stackoverflow.com 's code color system.

The htmlentities code escapes the text so if there is another </textarea> inside, it will not mess up the code.

Also, a short way of using the echo(""); command is to use the tags like this: <?=$variable?> or for functions: <?=customFunction("hello")?>
