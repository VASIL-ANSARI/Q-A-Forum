You can use Process.GetProcessesByName to check to see if a given process is running. When this returns results, disable your button.

var timer = new DispatcherTimer
            {
                 Interval = TimeSpan.FromSeconds(5)
            };
timer.Tick += (o,e) =>
            this.button1.IsEnabled =
                 !Process.GetProcessesByName("TheExternalProgramName").Any();
timer.Start();
