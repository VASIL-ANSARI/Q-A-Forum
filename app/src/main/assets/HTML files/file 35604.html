Phonegap / Cordova project structures and workflows have changed quite a bit from 2.x to 3.x versions. One of the recent changes consists in including the CordovaLib project within each newly created project instead of linking to pre-compiled .jar classes. Thus, you will not find the cordova.jar you are looking for in 3.3 version.

The most straightforward way to upgrade might be creating a new 3.x project from scratch and populate it with your existing assets/resources. It is explained in detail here (for Android):

http://docs.phonegap.com/en/3.3.0/guide_platforms_android_upgrading.md.html#Upgrading%20Android
