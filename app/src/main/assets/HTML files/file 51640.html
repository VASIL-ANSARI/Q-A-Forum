EWS is limited in what it can do with Accessing to the Directory there are 2 EWS operations ResolveName https://msdn.microsoft.com/en-us/library/office/dn645423%28v=exchg.150%29.aspx and ExpandGroup https://msdn.microsoft.com/en-us/library/office/dn629392(v=exchg.150).aspx that give limited address book function. On 2013 there is FindPeople https://msdn.microsoft.com/en-us/library/office/jj191039(v=exchg.150).aspx which can do a lot more. I would suggest you start with these write some code yourself to see how they work.

All the directory information that Exchange accesses in the Global Address list is in Active Directory so I you will most probably need to use LDAP directory to Access Active Directory if you application needs more complex directory access.
