I understand you want a dropdown in which you cannot select any option?

Then just disable the options individually using <option disabled="disabled">...</option>

Example: http://jsfiddle.net/mwvdlee/d5w51ut3/

<select>
    <option disabled="disabled">1</option>
    <option selected="selected">2</option>
    <option disabled="disabled">3</option>
    <option disabled="disabled">4</option>
</select>
