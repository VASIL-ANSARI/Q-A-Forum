The reason why your cannot get value from other Activity is because that is not a correct way to get the extra. When you are storing Intent extra like this:

String hall1=String.valueOf((fileUri.getPath()));
Intent i1 = new Intent(recordhall1.this, bhk1.class);
i1.putExtra("hall1", hall1);
startActivity(i1);

Then you should use below code to get the extra:

value = getIntent().getStringExtra("hall1");
System.out.println(value);
hall1.setText(value);

Your code returns null because you never put Bundle as an Intent extra.

EDIT:

Above code works if you are not using Bundle. However, if you use Bundle:

String hall1=String.valueOf((fileUri.getPath()));
Bundle extras = new Bundle();
extras.putString("hall1", hall1);
Intent i1 = new Intent(recordhall1.this, bhk1.class);
i1.putExtras(extras);
startActivity(i1);

Then you could get the extras with below code:

Bundle extras = getIntent().getExtras();
if (extras != null) {
    value  = extras.getString("hall1");
    System.out.println(value);
    hall1.setText(value);
}
