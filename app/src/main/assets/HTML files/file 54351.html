You should use IN as compare value, like this:

$args = array(
    'post_type' => 'product',
    'posts_per_page' => 2,
    'orderby'    => 'title',
    'order'      => 'ASC',
    'meta_query' => array(
        array(
            'key'     => 'key1',
            'value'   => array('val1','val2','val3'),
            'compare' => 'IN',
        ),
    ),
    'paged' => $paged
);

Hope it helps!
