The best way to achieve this is by using the CSS preprocessor like SASS, which helps to assign the values dynamically.

$font-color:#dea12c;
$background-color:white;

a {
    background-color: $background-color;
    border: 2px solid #dea12c;
    color: $font-color;
}

a:hover {
    background-color: $font-color;
    color: $background-color;
}
