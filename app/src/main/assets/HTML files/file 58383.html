You just need to do pair[:3] to get the first three elements from list. List comprehension is not needed in this case.

List comprehension is alternate and cleaner way to iterate over the list (or dict/tuple) and perform some action. For example, lets say you are having a loop in your code that creates new list with the square of element in the list as:

square_list = []
original list = [1, ,2 ,3 ,4]
for item in original_list:
    square_list.append(item*item)

Equivalent list comprehension implementation will be as:

square_list = [item*item for item in original_list]

There is a great blog Python List Comprehensions: Explained Visually explaining the concepts of list comprehension. You might find it useful.
