The only way to do this is to download the .deb file from the repository and install it as part of your script using dpkg -i. To do so, you require being able to run sudo as dpkg -i has to be run as root.

While doing this, remember that most Travis tests are run on Ubuntu 12.04, which only gets up to HHVM-3.6. You'll need to opt into the trusty beta to be able to run newer versions.
