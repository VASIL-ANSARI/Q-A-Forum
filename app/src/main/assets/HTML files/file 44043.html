If you are saying that you are executing that block of code several times, and expecting that you will get several alarms, that will not work. Your PendingIntent needs to differ by more than the extras on the underlying Intent. As it stands, you are replacing existing alarms, not adding new ones.

Replace the 0 in your getBroadcast() call with value. If your comment is correct, and value is unique for each alarm, that should be sufficient to give you separate PendingIntent objects for each alarm.
