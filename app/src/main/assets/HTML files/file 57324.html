I think your loop do is creating an infinite loop. I don't see any reason to add a loop here. Also, I think you need to call chomp on your word variable to remove the trailing newline character.

i.e. 

def compare(input)
  File.readlines('test.txt').each do |word|
    word = word.chomp # removes training '\n' character
    if word == input
      puts "Found a match! #{word} == #{input}"
    else
      puts "Comparing #{input} to #{word}"
    end
  end
end

In your code, you say that puts "Comparing #{input} to #{word}" will overwrite the current compared word with the new compared word, but this isn't true. You're not assigning variables or changing application state here at all - just logging something to console. Unless I'm unaware of some approach you're taking with $stdout.flush, I think it's unnecessary.

If all else, fails, a good way to debug is to install either pry or byebug and place a breakpoint in your code. Then you can inspect the values of your variables at runtime.
