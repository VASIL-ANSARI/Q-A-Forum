Looks like the field name should probably be userid, not useid.

Alternatively, try enclosing the values in single quotes, like this:

"UPDATE free_ebook SET math = '".$assign_math."' WHERE useid = '".$newuserid."';"

or even 

"UPDATE free_ebook SET math = '{$assign_math}' WHERE useid = '{$newuserid}';"
