I am not sure why you are mixing both query & method syntax, stick with one if possible. Here, I am using method syntax:-

var xdoc = XDocument.Load(p.text.ToString());
var names = xdoc.Descendants("Text")
                .Where(c => c.Attribute("group").Value.Equals(groupname));

Then you can use either SetAttributeValue or SetElementValue to update attribute or element value resp. like this:-

foreach (var item in names)
{
    item.SetElementValue("yourelementName","value");
}

Update: Finally save the xdoc Object:-

xdoc.Save(p.text.ToString());
