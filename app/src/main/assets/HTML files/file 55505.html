The problem is with the structure of your code. After the application starts a new form, it stops executing you code after the line Application.Run (NewForm); After that line is executed, the application renders the new form and waits for you to interact with the form. The only time program execution would get to the line 

string[][] list = new string[10][];

is when you exit the program and by that time your form is already gone. So you could put your logic in the constructor. Or look for a more efficient way to do it.
