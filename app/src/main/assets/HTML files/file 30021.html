You should get an existing Graphic object. You can get one from JLabel like this:

public void init(){
cgp = new CardGamePanel();
JPanel panel2 = new JPanel();
panel2.add(btnD);
btnD.addActionListener(new ActionListener(){

    public void actionPerformed(ActionEvent e){
        cgp.go(panel2.getGraphics());
    }
});
  add(cgp, BorderLayout.CENTER);
  add(panel2, BorderLayout.SOUTH);
  setSize(500,500);
}

And modify your other method in CardGamePanel class:

public void go(Graphic gr){
  if(count < 9 ){
    cards[count].display(gr); 
  }
  count ++;
}
