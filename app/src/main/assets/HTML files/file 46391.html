Eclipse has it's own built-in compiler, which is quite separate from the one you'd normally use from Ant or other external applications. That is precisely why you only need the JRE to run Eclipse and not a JDK to compile the code you write.

The page linked to above gives one way to use the Eclipse compiler from an Ant buildfile by defining build.compiler, which may make your Eclipse and Ant environments more compatible.

Other than that, you may want to think about why your strings are so large, there may be a better way.
