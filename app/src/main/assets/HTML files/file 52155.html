I suggest you to use the LexikJWTAuthenticationBundle.

To start use JWT, you'll just have some parameters to define (parameters.yml):

parameters:
    jwt_private_key_path: '%kernel.root_dir%/../var/jwt/private.pem'
    jwt_public_key_path: '%kernel.root_dir%/../var/jwt/public.pem'
    jwt_key_pass_phrase: ThisTokenIsNotSoSecretChangeIt
    jwt_token_ttl: 86400

and some configuration (config.yml):

lexik_jwt_authentication:
    private_key_path: %jwt_private_key_path%
    public_key_path:  %jwt_public_key_path%
    pass_phrase:      %jwt_key_pass_phrase%
    token_ttl:        %jwt_token_ttl%
    ...

More informations about configturation here

Then, still with bundle, you'll be able to provide authentication (security.yml)

firewalls:
    # ...
    api:
        # ...
        lexik_jwt: ~ # check token in Authorization Header, with a value prefix   of 'Bearer'
