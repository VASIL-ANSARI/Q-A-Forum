In javascript there is no concept of block scope, all variables will be declared in the respective function scope/global scope.

In your case you are declaring the same variables twice in the same functional scope, that is the reason for the warning.... even though you declare a variable at the bottom of the page, its scope will be the entire pages because of Hoisting also this

var firstFoo = 2;

$('div').click(function () {
    var var1, var2, var3;
    if (firstFoo == 1) {
        var1 = 'var1';
        var2 = 'var2';
        var3 = 'var3';
    } else {
        var1 = 'no var1';
        var2 = 'no var2';
        var3 = 'no var3';
    }

    $('<span></span>', {
        'data-var1': var1,
        'data-var2': var2,
        'data-var3': var3
    }).insertAfter(this);
});
