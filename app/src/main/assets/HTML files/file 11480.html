Use a Click event of ComboBox. You can use in a method, as on ComboBox Selection:

//this method you will get after you double click the ComboBox in the Form       
private void comboBox1_SelectedIndexChanged(object sender, EventArgs e)
{
    if (comboBox1.SelectedIndex > 0)
    {
        comboBox1.Click += new EventHandler(this.IWantToDisplayMessageBox);
    }
}

public void IWantToDisplayMessageBox(object sender, EventArgs e)
{
    MessageBox.Show("student ID and email");
}

Where, you may want to have "select" string at 0th index of ComboBox. Now only thing you need to do is, call the Database accessor you created and provide the ID and Email of that student selected in MessageBox
