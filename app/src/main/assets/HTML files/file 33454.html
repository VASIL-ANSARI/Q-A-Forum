$stmt->fetchAll() returns an array like this:

array(
    array("id"=>1,"name"=>"Dave"),
    array("id"=>2,"name"=>"John")
)

Namely, an array of records.

Using foreach iterates through this array. Using while does not. Since your array always exists, while will always be true!

foreach is used most commonly, but you could do this:

$result = $stmt->fetchAll();
while($row = array_shift($result)) {  ...  }

This works because the $result array will run out of rows, at which point the while will be false and end.
