Series is an array of objects so you should do like this:

options.series[0].data.push(["Service Ok", 45.0 ])

In this case you will get

series: [{

    type: 'pie',

    name: 'service status',

    data: [
             ["Service Ok", 45.0 ]
          ]

    }]
