Possible answer: make the .page-reordering .list-group::before element a block element, so it can inherit the width, from the parent element, then align text to the right.

.page-reordering .list-group::before {
content: 'Drag a page here to create a child page of this parent.';
color: #AEAEAE;
font-size: 25px;
font-style: italic;
padding-left: 10px;
text-align: right;
width: inherit;
display: block;
}
