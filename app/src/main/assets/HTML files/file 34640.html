Basically you would have to use JavaScript's scrollLeft() and scrollTop() methods on link click to take them to certain areas of the page. Be careful about this though. It can be quite slow on some devices if you have a lot of content.

For dragging certain objects on the page (or for dragging the entire page itself), you'll want to use a library like Hammer.js. You can bind a drag event to a certain element or to the page itself and then use scrollLeft() and scrollTop() to control the page.

To "show" movement as you scroll, use jQuery's animate() function.

$(window).animate({ scrollTop: $(document).height() }, 5000);
