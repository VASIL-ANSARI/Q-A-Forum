Third EDIT

As we talk in comments, all is solved with next parameter. The whole code:

$("#curtain-div").delay(15800).queue(function(next) {
     $('#home-learn').css('display', 'block');
     next();
}).toggle("slide", {
     direction: "right"
}, 1000);
