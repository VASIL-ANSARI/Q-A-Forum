You left off an equal sign.

This:

<form action="convertor_get.php" method "get">

Should be:

<form action="convertor_get.php" method="get">

Get is the default method normally. But putting a method attribute with no value is equivalent to setting method="" so I'm thinking you were sending to the convertor_get.php with no method defined at all. At least that theoretically would be the result in some browsers.

However on the url it says .php/amount=X

With the get method that should be ? not /. => .php?amount=X. So I think this confirms the request was going to the server with no method defined.
