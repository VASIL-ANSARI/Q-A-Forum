if response.list is an array/collection then you can use the following code to ierate it

$.each(response.list function(index, value) { 
  console.log(index + ': ' + value); 
});

The $.each() function can be used to iterate over any collection, whether it is a map (JavaScript object) or an array. In the case of an array, the callback is passed an array index and a corresponding array value each time. for more info on $.each() try this post
