The fundamental mistake you are making is to edit the file generated by pyuic. Do not ever do this. It's a static module that is meant to be imported at runtime. Put all of your program logic in the main file.

The reason why you are getting that error is because Ui_MainWindow is just a simple namespace class. It is only meant to be used for adding the widgets from Qt Designer to your main window class (i.e. Editor).

You should also avoid using QBasicTimer with timerEvent, and use QTimer with signals instead:
