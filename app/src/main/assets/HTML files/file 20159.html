If you're dealing with a Message<File>, using payload in the expression would be referring to the File. You can access any of File's methods in the expression. The following will route to outputChannelB if the file doesn't exist or is zero-length, or outputChannelA otherwise:

<int:router input-channel="inputChannel" expression="payload?.length() > 0 ? 'outputChannelA' : 'outputChannelB'" />

Note the use of the "safe-navigation" operator ?. for null safety.
