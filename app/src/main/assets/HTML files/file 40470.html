You may want to check your UID value: It looks like the original event is using a HTAB to do the line folding of the UID value. It is well possible that somewhere in your code path, this HTAB gets translated into a serie of simple spaces, hence totally changing the value itself.

See https://tools.ietf.org/html/rfc5545#section-2.1 about icalendar line folding.
