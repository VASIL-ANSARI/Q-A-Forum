Try to invoke .fadeIn() over the element which is having a class header(no need to grab all the elements inside of it) inside widow's load event,

$(window).load(function(){
 $(".header").hide().fadeIn("slow");
});

Suppose if you want to fadeIn the internal elements for sure then you have to write it like,

$(window).load(function(){
 $(".header > *").hide().fadeIn("slow");
});
