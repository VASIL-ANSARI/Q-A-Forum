It is duplicate removing problem. Maps, catalogs are dealing with it. The are many solutions to this problem with different success degree. Most simple one is to identify synonyms and useless words(Great, fun) etc and do items matching using this information. Some unclear case you should send to human for review.

Example "new Apple Ipad" vs "Great ipad" =remove usless word=> "Apple ipad" vs "ipad" =equal by substring/to ambiguous=> equal
