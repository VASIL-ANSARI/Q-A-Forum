Subject has already been partially treated here. Partially because it doesn't talk about React. It says:

Microsoft's Internet Explorer supports onresize on all HTML elements. In all other Browsers the onresize is only available at the window object. https://developer.mozilla.org/en-US/docs/Web/API/Window.onresize If you wanna have onresize at a div in all browsers check this: http://marcj.github.io/css-element-queries/ This library has a class ResizeSensor which can be used for resize detection.

Knowing that, there is few chances that setting onResize work on a React component. Read how React advises to attach onResize event. As a result, some modules were created to handle so cases, like react-component-resizable that allow you to set a wrap your component and set a props.

I advice you to follow React example.
