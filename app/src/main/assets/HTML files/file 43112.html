Consider

changeq +=1;
document.getElementById("element").src = OS[changeq].image ;

You are increasing changeq first, before accessing OS[changeq].image. You are not checking whether changeq is still a valid index into the array!

The simplest solution would be to switch the lines. If that is not possible because it changes the desired behavior, move the check in between:

changeq +=1;
if (changeq>=OS.length){
    window.location.assign("start.html");
    return;
}
document.getElementById("element").src = OS[changeq].image ;
