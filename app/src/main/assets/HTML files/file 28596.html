You're redefining the same keys over and over.

$items[] = array(
    "text" => "", "value" => null,
    "text" => "YAMASAKI", "value" => "YAMASAKI"
)

Gives:

array(array(
    "text" => "YAMASAKI",
    "value" => "YAMASAKI"
));

In think you intend:

//    v-- no [] here
$items = array(
    array("text" => "", "value" => null),
    array("text" => "YAMASAKI", "value" => "YAMASAKI"),
    // etc.
