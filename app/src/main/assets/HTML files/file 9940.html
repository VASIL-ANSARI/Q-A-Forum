Concatenation operator is || However, there is not short form of the concatenation that you are looking for (i.e. +=).

You can try this:

DECLARE
 lvOutPut VARCHAR2(2000);
BEGIN
    lvOutPut := 'BLA';
    FOR i in 1..10 LOOP
        lvOutPut := lvOutPut || i;
    END LOOP;
    DBMS_OUTPUT.PUT_LINE(lvOutPut);
END;
