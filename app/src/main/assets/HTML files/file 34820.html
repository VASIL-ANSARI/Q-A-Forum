You need to end your case first then use where clause

UPDATE openor 
    SET cur2 = CASE                
    WHEN $val1e<cur2 THEN cur2-$val1e
    WHEN $val1e>cur2 THEN $val1e-cur2
    WHEN $val1e=cur2 THEN '0'
    ELSE cur2 END
    WHERE id='2'

Also you can skip your else part
