override show() inside your DialogFramgent, and when it is invoked disable the orientation of your Activity. Let your DilagFragment implements the DismissListener, and when the onDismiss is called, re enable the orientation

@Override
public void show(FragmentManager fm, String tag) {
 getActivity().setRequestedOrientation(ActivityInfo.SCREEN_ORIENTATION_PORTRAIT);
 super.show(fm, tag);
}
