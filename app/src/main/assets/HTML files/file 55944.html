If my understanding of your question is correct, you can do this using inset box-shadow like in below snippet. One shadow (the red colored one) has a blur radius because it seems to go from red color to transparent whereas the other doesn't have the blur radius as it seems to be a solid line all the way.

.fancy-border {

  height: 200px;

  width: 200px;

  border: 10px solid rgb(0, 53, 107);

  padding: 20px;

  box-shadow: inset 0px 0px 10px 10px rgb(252, 1, 2), inset 0px 0px 0px 20px rgb(0, 53, 107);

}
<div class='fancy-border'>Some content</div>
