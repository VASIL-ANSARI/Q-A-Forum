The "tree" item in your request data needs to be an array of items rather than a single item since a tree is a collection of items.

$treeArr = array(
    "base_tree" => "d7126bd6c559ab461e851e96ef2c33675d851c5e",
    "tree" => array(
        array(
            "path" => "resources/blahTest.txt",
            "mode" => "100644",
            "type" => "blob",
            "sha" => "38d15319d3ee8a7292be0ec0da65fe111660a94d"
        )
    )
);
