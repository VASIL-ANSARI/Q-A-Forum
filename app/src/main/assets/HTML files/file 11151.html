Hm, it is several ways to achieve your goal, one of them is to include your module with methods direct in ActiveRecord::Base, 'ApplicationController' and Rspec. Not sure about RSpec (I don't quite sign with it's class/module structure) but you can do something like this in some initializer file:

class ActiveRecord::Base
 include MyModuleWithAwesomeMethods
end

class ApplicationController
 include MyModuleWithAwesomeMethods
end

etc. for other classes/modules
Hope it will help you.
