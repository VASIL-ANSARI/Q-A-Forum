You're calling AmCharts.stringToDate incorrectly. The second parameter is the format of the string date the function is trying to parse in the first parameter. Your dates are in YYYY-MM-DD format, not DD-MM-YYYY. Once you fix the format, the chart will render correctly. Ex: AmCharts.stringToDate("2016-10-03T09:33:00+00:00", "YYYY-MM-DD HH:NN")
