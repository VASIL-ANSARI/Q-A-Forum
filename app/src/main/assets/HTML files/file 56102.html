The reason the boxes are sized differently is because your tags are mismatched in the second box. Change:

    <p>OVERALL HP SERVERS DOWN: 26
    <p>
    <button type="button" class="btns  sky-skin btns-bot sml-btn sml-radius">Goal: 2%</button>
</div>

to:

    <p>OVERALL HP SERVERS DOWN: 26</p>
    <button type="button" class="btns  sky-skin btns-bot sml-btn sml-radius">Goal: 2%</button>
</div>

As for adding a border, it should be simple with CSS. Don't forget to set box-sizing properly or your borders may make things appear off again.

.hvr-grow-shadow {
    /* Border Stuff here, example:
     * border: 1px solid black;
     */
    box-sizing: border-box;
}
