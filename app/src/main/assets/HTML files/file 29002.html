Follow the Devise Wiki Page on Approvals for the initial set-up of an approval process, which will notify you (or whomever you choose) when new users have registered via an AdminMailer mailer.

Then, by making use of a before_create callback, you can check your custom validations ie:

def approved_via_linked_in?
    # perform logic here...
end

And, you can set a condition on the email message which would send from your AdminMailer to only send if the user has not been approved yet by your custom validations.
