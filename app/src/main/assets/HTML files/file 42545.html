In the documentation you provided, it says the string returned by getWeights() is a string containing the weights (doubles separated by blanks).

So this should work:

String a = classifier.getWeights();
String[] weightsString = a.split(' ');
double[] weights = new double[a.length];
for(int i =0; i< a.length; i++)
    weights[i] = Double.valueOf(weightsString[i]);
