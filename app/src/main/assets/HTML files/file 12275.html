If all of the content you want to change has a class of "highlighted" you can do something like this:

JavaScript

$(".highlighted").addClass("coloured-text");

CSS

.coloured-text {
    color: red;
}

If you need to add the classes by hand have a look at this jsFiddle that I've knocked up. This uses a regular expression to wrap pre-defined punctuation with a <span> which has the highlighted class.
