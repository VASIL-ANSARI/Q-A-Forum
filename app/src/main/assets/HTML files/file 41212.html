Multiple ways (depends on the larger context of how you use it):

You can assign it to a variable:

spinner = d3.selet(this).append("svg:image")
  .attr('class', "circle")
  ...

and later remove it with spinner.remove(). The var spinner will have to be declared somewhere where it's accessible to all the places in your code where you use it (i.e. towards the top of the file).

You can assign it an id, using .attr("id", "my_image") and you can select it based on that id later using d3.select("#my_image"). Or you can select it by its class d3.select('circle').

Ids won't work if you have more than one image with the same id, and with class, if there are multiple, they would all be selected unless you first d3.select(...) the parent of one of them.
