You can use public static Criterion sqlRestriction(String sql, Object value, Type type) if you only need to pass one value or public static Criterion sqlRestriction(String sql, Object[] values, Type[] types).

For example: 

Type[] tipos = {IntegerType.INSTANCE, IntegerType.INSTANCE};
Integer[] values = {1, 2};

criteria.add(Restrictions.sqlRestriction("SELECT ({alias}.id * {alias}.code * "+ ? + " * " + ? + ") 
as number from city HAVING number < 10", values, tipos ));
