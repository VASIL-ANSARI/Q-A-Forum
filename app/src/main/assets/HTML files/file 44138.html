You can get a simple start by using absolute positioning and a 2D transform.

.container {

  position: relative;

  display: inline-block;

}

.info {

  background: black;

  color: white;

  position: absolute;

  bottom: 0;

  left: 50%;

  transform: translateX(-50%);

  white-space: nowarp;

}
