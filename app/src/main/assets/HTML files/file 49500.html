If you have a pre-defined set of locations (EG New York), you could put them into their own model:

#app/models/location.rb
class Location < ActiveRecord::Base
   has_many :users
   has_many :events
   has_many :groups

   validates :x, uniqueness: true
end

#app/models/user.rb
class User < ActiveRecord::Base
   belongs_to :location
end

#app/models/event.rb
class Event < ActiveRecord::Base
   belongs_to :location
end

#app/models/group.rb
class Group < ActiveRecord::Base
   belongs_to :location
end

The above will allow you to call the following:

@location = Location.create({name: "New York"})

@event = Event.create({location: @location})
@group = Group.create({location: @location})
@user = User.create({location: @location})
