You could do the following:

  • Determine width of the map (in pixels).
  • Determine how many tiles your grid should have on this map
  • Divide the width of the map by the number of tiles
  • Hold this number
  • Divide the maximum longitude by the same number of tiles (point 2)
  • Divide the yet to place longitude by the result
  • Multiply the result of that by the number you held before
