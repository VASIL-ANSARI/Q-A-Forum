you must cancel alarm manager first and restart when edit string on notification because alarm once set you can not editing in that so use like below:

Intent intent = new Intent(getActivity(), AlarmReceiver.class);
                                            PendingIntent sender = PendingIntent.getBroadcast(getActivity(), id, intent, 0);
                                            AlarmManager alarmManager = (AlarmManager) context.getSystemService(Context.ALARM_SERVICE);

                                            alarmManager.cancel(sender);

and then reset alarm like below:

Intent startServiceIntent = new Intent(getActivity(), AlarmReceiver.class);
                                        startServiceIntent.putExtra("deviceid",idmydevice);


                                            AlarmManager alarmMgr = (AlarmManager)context.getSystemService(Context.ALARM_SERVICE);
                                            PendingIntent alarmIntent = PendingIntent.getBroadcast(context, id, startServiceIntent, 0);
                                            alarmMgr.setInexactRepeating(AlarmManager.RTC_WAKEUP,  
                                                    System.currentTimeMillis()+10000, 10000, alarmIntent);
