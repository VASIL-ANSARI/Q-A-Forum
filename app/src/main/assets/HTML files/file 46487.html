You have two models Route and Stop, you need to define associations between these classes to solve your problem

class Route < ActiveRecord::Base
  has_many :stops
end

class Stop < ActiveRecord::Base
  belongs_to :route
end

So stops table will have a route_id column, to fetch all the stops that belong to a route, you can do this

route = Route.where(id: params[:route_id]).first
stops = route.stops
