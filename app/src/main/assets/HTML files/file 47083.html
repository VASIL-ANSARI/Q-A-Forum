Use some javascript and an ajax request!

$(document).ready(function() {
  $( ".tweet" ).click(function() {
    $.ajax({
      type: "POST",
      format: "js",
      url: <url for tweets_show_replies_path>,
      data: {
        parent: <%= tweet %>,
        active: false
      },
      dataType: "script"
    });
  });
});

Then in your controller, do whatever you need to and render the partial.
