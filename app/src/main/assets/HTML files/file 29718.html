The type of jQuery event handlers you are using install themselves on DOM objects that EXIST at the time you run the initial code. They do not work on DOM elements you might create in the future so thus they don't work on content loaded with .load().

You have two choices:

  1. You can rerun the code to install event handlers on the new content.
  2. You can switch to delegated event handlers which will work on dynamically created content.
