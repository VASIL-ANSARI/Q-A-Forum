You have to use document.createElement to create new element appendChild to append the a child content.

Here li is child of ul which in turn is also a child of div#output

Hope this snippet will be useful

var _ul = document.createElement('ul');
var _li=  document.createElement('ul');

_li.innerHTML="Something";
_ul.appendChild(_li);
document.getElementById("output").appendChild(_ul);
