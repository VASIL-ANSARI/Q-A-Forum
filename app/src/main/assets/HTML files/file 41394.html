You got your logic backwards :

for (int j = 0; j < p; j++){
  if (!data.get(j).equals(tempTable)){
    data.add(tempTable);
    break;
  }
}

This will add tempTable if any of the elements in data is not equal to tempTable. You want to add it only if all the elements are not equal :

boolean found = false;
for (int j = 0; j < p; j++){
  if (data.get(j).equals(tempTable)){
    found = true;
    break;
  }
}
if (!found)
  data.add(tempTable);
