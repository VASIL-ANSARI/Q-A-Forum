Autoresizing works only when the parentView's size being changed.
And they don't remember the old frame of its subviews.

So, you have to manually adjust the frame of childView after sizeToFit:

CGRect oldFrame = childView.frame;
CGFloat bottomY = oldFrame.position.y + oldFrame.size.height;
[childView setText:@"A long text that's bigger than the view."];
[childView sizeToFit];
CGRect newFrame = childView.frame;
newFrame.position.y = bottomY - newFrame.size.height;
childView.frame = newFrame;
