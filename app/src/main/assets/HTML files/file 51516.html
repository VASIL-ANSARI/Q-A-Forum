You could use Math.min(int, int) and Math.max(int, int) to replace your input > maxValue and input < minValue checks like

maxValue = Math.max(maxValue, input);
minValue = Math.min(minValue, input);

And you should declare them like

int maxValue = Integer.MIN_VALUE;
int minValue = Integer.MAX_VALUE;

 if (Character.toUpperCase(sentinel.charAt(0)) == 'Q')

and you might also remove the done boolean, and use

System.exit(0);
