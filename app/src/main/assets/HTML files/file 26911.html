The access_token generated to manage a user is not the same as the access_token needed to manage a page.

To get an access token for a page:

me/accounts?fields=access_token,name

Then, use the page access token to run the query: 

$facebook->api("/MYALBUMID/photos","POST",
                array('url'=>$picUrl,'message'=>"status message","access_token"=>$token));
