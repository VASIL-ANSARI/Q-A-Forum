You can do that using ajax.

You need to create a function that receives the category/subcategory and does a WP_Query based in that, then returns a list of the post.

To make the ajax call I sugest using the $.get();

For example:

var data = {
   category : you_category;
};

$.get(ajaxurl, data, function(response) {
   ... update the list of posts
});
