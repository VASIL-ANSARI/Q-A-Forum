If you mean bootstrap popover, you can add data-content attribute which would hold popover content for each button, e.g if you are also fetching it from database it would be something like this

<?php
// Start Loop
echo '<td><a href="#" class="information btn" data-content="'.$row->PopoverInfo.'" id="'.$row->IdUser.'">Show Info</a></td>';
...
// End loop
?>

then initialize it with javascript on dom ready

$(function(){
    $(".information").popover();
})
