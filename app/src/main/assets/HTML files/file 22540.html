Well if you use the actual Propel objects you can catch messages by catching PropelException

try {
    $propelObject
        ->setSomething('foo')
        ->save()
    ;
} catch (PropelException $pe) {
    someLoggingFunction($pe->getMessage());
}

Otherwise you're simply limited to the PDO error handling functions:

http://www.php.net/manual/en/pdo.error-handling.php
