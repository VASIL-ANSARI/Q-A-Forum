First make sure no firewall/antivirus is preventing eclipse from connecting to internet.

Add this node to pom.xml 

<repositories>
    <repository>
        <id>xuggle repo</id>
        <url>http://xuggle.googlecode.com/svn/trunk/repo/share/java/</url>
    </repository>
</repositories>

After that right click project -> Run as->Maven clean
it will clean project then
right click project -> Run as->Maven install

This was causing because There is no artifact in central repository of maven named with xuggle.

OR If this fails too.

Download the xuggle-utils.jar and use maven:install command to install it your local .m2 repo.

This will definately solve the issue.
