First off, your array declaration syntax is incorrect.

Since you are using C++, you should be using the built-in array class for static (fixed-length) arrays.

The correct way to initialize an array is with the following syntax:

std::array<char, 5> vowels = { 'a', 'e', 'i', 'o', 'u' };

Next, this syntax is also not doing what you think it is:

if (stringArr == 'a'|'e'|'i'|'o'|'u')

The | is actually a binary or, meaning it's doing bit manipulation, which isn't what you want. Also, it's odd that you declare the vowels and then re-state all the vowels in this if statement. Don't repeat yourself! Use the list you already made and search that. It makes your code much easier to maintain.

The standard way to search an array (and many other container classes) is to use the find function.

c = 'a'; // some character
find(vowels.begin(), vowels.end(), c) != vowels.end()

What this means is that it will return true if c is found in vowels.

So now we just need to go through all the characters of the string the user entered. In order to do this you need to use a loop. In C++, there is something called a range-based for loop. The syntax is as follows:

for (auto c : user_string)
{
    // do something with c
}

This will iterate through the entirety of the string, and execute the code inside the block ({ }) for each character.

Here is working code that does exactly what you want:

#include <iostream>
#include <cstdlib>
#include <string>
#include <iomanip>
#include <array>
#include <algorithm>

using namespace std;

int main()
{
    bool vowels_found = false;
    string user_string = "yea man";
    array<char, 5> vowels = { 'a', 'e', 'i', 'o', 'u' };

    for (auto c : user_string)
    {
        if (find(vowels.begin(), vowels.end(), c) != vowels.end())
        {
            cout << c;

            vowels_found = true;
        }
    }

    if (!vowels_found)
    {
        cout << "No vowels found.";
    }

    return 0;
}
