as per your table schema you provided in edit . you dont have Plays column in plays table thats why you getting this error try removing this p.Plays

SELECT *,  

            m.plays AS mp,
            COUNT(p.sid) frequency,    
FROM        music m
            JOIN plays p
            ON p.sid = m.sid
            INNER JOIN users u
            ON u.uid = m.uid
            INNER JOIN downloads d
            ON d.sid = m.sid        
WHERE       p.time > NOW() - INTERVAL 3 DAY 
            AND p.sid != '' 
            AND m.perms != 'c' 
GROUP       BY m.sid
ORDER       BY frequency DESC
            LIMIT 50
