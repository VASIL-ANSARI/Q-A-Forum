The File.Copy method accepts full file paths for the source and destination, so if you're using FileInfo, you'll want to use the Name and FullName properties.

File.Copy(file.FullName, Path.Combine(targetPath, file.Name), true);

Also, as a general rule, you don't want to just call ToString() on an object and hope it returns something meaningful. A class may override ToString() and return some value, or it may not and the value you get back is simply the class type. Check the documentation first.
