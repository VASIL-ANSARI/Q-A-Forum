On Android, according to Sensor.TYPE_MAGNETIC_FIELD, the value array corresponds to the micro-tesla readings of the magnetometer in the X, Y and Z directions. If you look at the MSDN docs for the CompassReading Structure the MagnetometerReading will give you a Vector3 providing the raw magnetometer readings from the device.

One thing to note is that the MSDN docs specifically call out that these are raw readings where as the Android docs don't specify this. As such you'll probably get very different readings between the two systems.

MSDN has a good article on How to: Get Data from the Compass Sensor for Windows Phone which would be a good starting point.
