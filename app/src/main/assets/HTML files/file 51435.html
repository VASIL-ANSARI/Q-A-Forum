If you want to preserve the data in fields, you need to push instead of presenting the VC modally.

Create a navigation controller and init with your VC and then present the VC like:

   UIStoryboard *storyBorad = [UIStoryboard storyboardWithName: @"Main_iphone" bundle:nil];
   UIViewController *secondController = [Storyboard instantiateViewControllerWithIdentifier:@"ControllerB"];

   UINavigationController *navController = [[UINavigationController alloc] initWithRootViewController:firstViewController];

   [navController pushViewController:secondController animated:YES];

In this case, when you click on back button, you will be taken back to your previous VC and you won't loose any data!!!!

If you don't want o have back button, you can hide the navigation bar, and if you want to dismiss the secondController,you can simply call:

     [self.navigationController popViewControllerAnimated:YES];

Also make sure you don't clear all the data in the fields when you move away from your VC!!!! (Make sure to check viewDidDisappear/viewWillDisappear if data is getting nil or not)
