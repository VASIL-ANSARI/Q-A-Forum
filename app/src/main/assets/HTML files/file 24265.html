If you are referring to centering the entire menu within the menu bar, I suggest the following adjustments to your CSS:

Push entire UL list right by 50% of container width:

ul {
  ...
  left: 50%;
}

Push each LI element left by 50% of the UL width:

li {
  ...
  left: -50%;
}

Prevent horizontal scroll bar caused by UL overflowing on the right:

.lavalamp {
  ...
  overflow:hidden;
}
