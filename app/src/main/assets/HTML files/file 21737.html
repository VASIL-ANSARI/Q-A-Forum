I believe you can use HTML5 data- attributes to achieve your goal:

So your a.room-edit nodes could be like:

<a href="#" class="room-edit" data-roomID="room_12345">Edit</a>

And your click function could be like:

$( 'a.room-edit').click(function() {
    var roomID = $(this).attr('data-roomID');

    // do something spectacular
});
