There is no direct way to do this, so you have to implement a function bound to the onDrop event.

When this event fires, you have to get a count of all <li> elements in the sortable list. If the count is less than the max of 4, allow the recieve event to continue. Otherwise, trigger the cancel method.

$("ol.nested_with_switch").sortable({

  ... //previous code

  over : function(event, ui){

    if ($("ol.nested_with_switch li").length > 4){
      $("ol.nested_with_switch").sortable("cancel);
    } 
  } 
});
