In your code, you refer to Groups(1), but there are no groups in your regular expression! Just return the matched value directly:

Dim rs As String = Regex.Match(theusercp, pattern, RegexOptions.Multiline).Value

This will yield the desired output.

As a side note: Your pattern will also match http://example.com/xcom in http://example.com/xcom/123. I'm not sure this is what you indended...
