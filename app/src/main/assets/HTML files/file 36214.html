You don't read values out of ViewData in controller. These structures are for controller to share some little data with view (ideally view should get data through model i.e. strongly typed view).

You can send data from view to controller in QueryString or FormBody. As an example, If you write controller like this:

public ActionResult ExportLevyToExcell(string data)
{
    var result = data;

    return HttpNotFound();
}

and change your JS code a bit like this:

function ExportExcell() {

    var data = '@Html.Raw(new JavaScriptSerializer().Serialize(ViewData["html"]))';

    window.location.href = "Home/ExportLevyToExcell?data=" + data;
}

When you click on button, you will see controller receive the value of

ViewData["html"]
