In a UIView sizeToFit calls sizeThatFits: with the view's current size.

The default implementation of sizeThatFits: returns the current value of the view, in other words it does pretty much nothing in most cases (some subclasses of UIView override this method so it returns something more useful, ex: UILabel)

In other words sizeThatFits it is a "Template Method", a method that is made for extending the functionality of the class. It's main purpose is to be overridden.
