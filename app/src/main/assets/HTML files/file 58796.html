Explanation :

CSS . i floated the aside and the section giving them 20% and 80% ( that's purely my choice, you can change these values how you want but they need to add up to 100% )

JQ

on scroll , for each article i got the offset().top values( distance from the top of the div to the top of the document ) , height() values and found the aside and section divs inside each of the articles , and all of this stored in variables.

also stored the scrolled distance ( scrollTop() )

then, i've made an if statement with 2 conditions.

condition 1 b > a , that means that the distance scrolled must be greater than the top offset of the article

condition 2 b < a+c , which means that the distance scrolled must be lesser than the top offset + height of article

if these 2 conditions are met , then i add class fixed to the aside and class giveMargin to the section . when these 2 conditions return FALSE, these 2 custom classes are removed.

of course the JQ can be easily adjusted if you want changes

CSS part2

added styles to the 2 custom classes. position:fixed to aside.fixed and margin-left:20% to section.giveMargin

added that margin-left because by giving position:fixed to the aside , it's width does not influence the structure, so because the 20% width ' disappears ` it needs to be replaced with something else. adding the margin does that and keeps the same structure as before

hope my explanation is clear :) let me know if it helped :D

$(window).scroll(function() {

  

       $("article").each(function(){

             var a = $(this).offset().top ,

             b = $(window).scrollTop(),

             c = $(this).height(),

             title = $(this).children("aside"),

             text = $(this).children("section")

           if (b > a && b < a+c ) {

                 $(title).addClass("fixed")

                 $(text).addClass("giveMargin")

           }else{

                $(title).removeClass("fixed")

                $(text).removeClass("giveMargin")

           }

       })

       })

section.giveMargin { margin-left:20%}
