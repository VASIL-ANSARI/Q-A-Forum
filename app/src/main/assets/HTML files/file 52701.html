First of all, you need to update your jQuery selector.

$("logout").submit(function(){

It should be $("#logout") since you're trying to get the element by ID. Just like in CSS, you need to include the hashtag when you want to select an element by it's id.

<script>
    $(document).ready(function(){
        $('#logout').on('click', function(e){
            if(confirm("Are you sure you want to logout?"))
                window.location.href = "http://localhost/login.php";

            return false;
        });
    });
</script>
