First let's take a look at the answer to the question C# What does List<List<string>> mean?

The accepted answer shows how you can output the contents of this 2D list:

List<List<string>> lists;
...
foreach (List<string> list in lists)
{
    foreach (string s in list)
    {
        Console.WriteLine(s);
    }
}

Now look at the answer to the question How to create a table based on a two-dimensional array?

It explains how to use the data of a List<List<string>> to construct a PdfPTable:

PdfPTable table = new PdfPTable(numColumns);
foreach (List<string> question in questions) {
    foreach (string field in question) {
        table.AddCell(field);
    }
}

Now all you have to do, is to add the table to the Document instance:

document.add(table);

Important: I don't know the value of numColumns. You should replace numColumns by the number of string values of a question object. As you create the questions object, you (and only you) know the answer to that question. Actually, you could ask the first element of the questions list for its size; that way you don't have to guess. Note that it is assumed that every question has the same amount of elements.

UPDATE: if you don't want a table, you should wrap the string values in paragraphs. For instance:

foreach (List<string> question in questions) {
    foreach (string field in question) {
        document.Add(new Paragraph(field));
    }
}
