The issue is your list items have a css class called "ui-li-has-arrow" â this is adding padding-right: 40px to the links, pushing them to the left.

You can override this in the head of test.html.

Where you currently have:

<style type="text/css">

.ui-listview .ui-btn-inner {
  text-align: center;
}

</style>

Change it to: 

<style type="text/css">

.ui-listview .ui-btn-inner {
  text-align: center;
}
.ui-li-has-arrow .ui-btn-inner a.ui-link-inherit, .ui-li-static.ui-li-has-arrow {
  padding-right: 15px;
}


</style>
