You should use setInterval() method. Try this 

$(function(){
    var colors = ["red","green","blue","yellow"];

    setInterval(function() {
           $('#block').css('background', colors[Math.floor(Math.random() * colors.length)]);
    }, 500);
});

To get color code randomly without using array of color. You can generate dynamically hexadecimal value for color codes. 

$('#block').css('background', '#'+Math.floor(Math.random()*16777215).toString(16));

In terms of multiple div block try below code :

$(function(){
   $("div").each(function(){
      $(this).css('background','#'+Math.floor(Math.random()*16777215).toString(16));
   });
});
