So what happens in this program is:

a = 0, b = 1

c is set to their sum, = 1

a is set to b, = 1

b is set to c, = 1

a = 1, b = 1

c is set to their sum, = 2

a is set to b, = 1

b is set to c, = 2

a = 1, b = 2

c is set to their sum, = 3

a is set to b, = 2

b is set to c, = 3

a = 2, b = 3

c is set to their sum, = 5

a is set to b, = 3

b is set to c, = 5

a = 3, b = 5

... And so on. You should get the idea :)
