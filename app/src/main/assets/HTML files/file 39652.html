Here is a piece of javascript catching the right-click on any <img/> element and sending an AJAX request. The script your_delete_script.php then has to read the posted filename and delete it on the server.

$(function() {    
      $('img').mouseup(function(e){ 
        if( e.button == 2 ) { 
          $.post('your_delete_script.php', {filename: $(this).attr('src')}, function() {});
      }); 
});

If there are any other <img/> elements, you have to assign a special class to those you wish to be deletable and assign the handler to that class.

Note that it might be unsafe to delete the file without checking whether the file is really an image which you wish to be deleted. Furthermore, it's not recommended to intrude into the user's browser handling by catching the right click which should open the context menu by default. Maybe you'll prefer a delete button or a double click event, depending on whether your site will be publicly available.
