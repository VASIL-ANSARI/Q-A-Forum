The way I managed to resolve this problem is setting CATALINA_OPTS in the DockerFile like this:

ENV CATALINA_OPTS="-Dkey=value"

Also you could have edited some config files, but this way is not that complex as for me. So the whole DockerFile for my app would look like this:

FROM tomcat:8.0-jre8
ENV spring.profiles.active=dev 
ENV CATALINA_OPTS="-Dkey=value"
ADD myWar.war /usr/local/tomcat/webapps/
CMD ["catalina.sh", "run"]
