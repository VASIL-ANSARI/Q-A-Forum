If I'm interpreting "size" correctly as at a certain width: Add paddingRight or marginRight to the last TextView. That will shift the whole text area to the left and thus the ellipsis, too.

If you want the ellipsis to be after a specific line, e.g. line 3, add these to your TextViews:

android:maxLines="3"
android:minLines="3"
android:ellipsize="end"
