Just in case this is useful for someone:

1) Create a new keychain using keychain-access. Let's call it 'jenkins', for example.
2) Copy the keychain to the jenkins library directory ($HOME/Library/Keychains/jenkins.keychain)
3) run chown jenkins:jenkins on the jenkins.keychain
4) run sudo su jenkins to switch to jenkins user.
5) add the jenkins.keychain to the keychain list security list-keychains -s ~/Library/Keychains/jenkins.keychain
6) add a bash script call to the jenkins build process to unlock the keychain. ideally, the call should use the password stored in the jenkins build via the mask password plugin. Something like security unlock-keychain -p "${jenkins_keychain_password}" "$HOME/Library/Keychains/jenkins.keychain"

This is a good guide for more details: http://www.egeek.me/2015/04/12/jenkins-setting-up-ios-code-signing/
