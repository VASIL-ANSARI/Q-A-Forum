You can use $("body").position() which gives you top and left coordinate. Then use the $("body").width() and $("body").height(), and you'll be able to calculate the center of the eleyment (the body, here)

var mouseX, mouseY // with e.pageX and e.pageY    
var $e = $("body");

var center_x = $e.width() / 2;
var center_y = $e.height() / 2;

mouseX -= $e.position().left + center_x;
mousey -= $e.position().top + center_y;
