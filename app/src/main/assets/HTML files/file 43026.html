In the FAQ at https://realm.io/docs/java/0.80.0/#faq explains why the file size of a Realm might increase (see section "Large Realm file size").

I would recommend that you compact your Realm file when the app either starts or resumes (and before it opens a Realm instance). The method Realm.compactRealmFile() is the method you should be looking for (see https://realm.io/docs/java/0.80.0/api/io/realm/Realm.html#compactRealmFile-android.content.Context-).
