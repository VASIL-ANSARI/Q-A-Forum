You can use Tabulate for that.

e.g. 

>>> table = [["spam",42],["eggs",451],["bacon",0]]
>>> headers = ["item", "qty"]
>>> print tabulate(table, headers, tablefmt="simple")
item      qty
------  -----
spam       42
eggs      451
bacon       0

You can pass tablefmt="latex" as an argument to generate Latex tables.
