You are receiving that error because the response that comes from the server is empty and the NSJSONSerialization class cannot work with a nil NSData.

To avoid the crash, sorrund the json parsing with the following condition:

NSError* error;
if (responseData != nil) {
     resposeDataDict =  [NSJSONSerialization JSONObjectWithData: responseData options: NSJSONReadingMutableContainers error: &error];
}

Then you have to investigate about why you get a nil response. The main cause with this kind of problem is in a server side exception.
