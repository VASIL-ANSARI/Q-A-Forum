You have this element .clr that causes the problem, you have a width of 1000px, set a width:auto. The element is on the sidebar. On the footer you have a width:100% with padding, you need to set box-sizing: border-box to set the 100% with padding inside

CSS

.clr {
    clear: both;
    font-size: 0;
    line-height: 0;
    margin: 0;
    padding: 4px;
    width: auto; /* Set this rule */
}

.footer p {
    background: #d00000 none repeat scroll 0 0;
    box-sizing: border-box; /* Set this rule */
    color: #ffffff;
    font: 12px/1.5em Open Sans,Helvetica,sans-serif;
    padding: 4px 0 0 30px;
    width: 100%;
}
