You need only digits then replace all non-digits using \D

String str = "\"sampleDay\": newDate(1402027200000)";
System.out.println(str.replaceAll("\\D+", ""));  // print 1402027200000

OR use Character Classes or Character Sets to negate any digits using [^\d]

String str = "\"sampleDay\": newDate(1402027200000)";
System.out.println(str.replaceAll("[^\\d]+", ""));
