The top three reasons that i've seen this happen is due to the following:

  1. The TCP port is blocked on the application server due to security policy
  2. The TCP port is blocked by firewall or other network device due to security policy
  3. The target SMTP server requires you to "whitelist" your application server to allow the connection.

All three cases happens because a company wants to prevent spammers from using their email server has spam delivery systems.

Talk with your support personnel.
