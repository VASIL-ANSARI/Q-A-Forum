Depending on your version of react-router how to exactly access and change this might differ but the gist should be the same.

You're currently using a hash history. This means your url's will be built using a hash. An other option is the use the browser history. This enables the clean urls www.example.com/about.

import { browserHistory } from 'react-router'
render((
  <Router history={browserHistory }>
    <Route path="/" component={Layout}>
      <Route path="/about" component={About}/>
    </Route>
  </Router>
  ),

https://github.com/reactjs/react-router/blob/master/docs/guides/Histories.md

Ps. If above doesn't work please consult the docs for react-router and the version you are using. They've had some API changes and package changes in this department (the history package is effectively powering this behavior).
