Since you have not provided any code, I can only help you with design. So, Here is some design tips for you:

  1. Store your matrices in two-dimensional arrays

    Ex: int[][] matrix = new int[5][3]; this gives you a 5x3 matrix.

  2. Since you don't know how many matrices you will have, add them into an arraylist.

    Ex: ArrayList<int[][]> matrices = new ArrayList<>();

  3. I don't know what you mean with comparison but, you can do it using a nested for loop by retrieving the matrices from the arraylist one by one to compare.
