You're using GROUP BY clause without selecting that field column, Add $did in your select fields column.

SELECT $did, AVG( driver_rating.rate ) 
FROM  `driver_rating` 
JOIN  `order` ON order.id = driver_rating.order_id
JOIN  `driver` ON order.did = $did
GROUP BY $did
