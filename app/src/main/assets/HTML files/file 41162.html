You're setting your map height as 100%. Try setting a fixed height, instead.

If it's too wide, add this to your CSS files.

.gmnoprint img {
    max-width: none; 
}

Alternatively, try this if that doesn't work:

html,body {
    height: 100%; 
    width: 100%;
}
.whateveryourmapis {
    height: auto; 
    position: absolute; 
    top: 0; 
    bottom: 0; 
    left: 0; 
    right: 0;
    margin-top: -50px;
}
