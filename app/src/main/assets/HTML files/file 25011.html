Try this (See the fiddle):

CSS

<style type="text/css">
    .center_outer
    {
        position: relative;
        left: 50%;
        float: left;
    }
    .center_inner
    {
        position: relative;
        left: -50%;
    }
</style>

HTML

<div class="center_outer">
    <div class="center_inner">
               <!-- Put your contents (which needs to be centered) here --!>
        Whatever contents here get centered.
    </div>
</div>

This will center any contents inside the div.center_inner. Even with floated (no width mentioned) divs. Wrap the div inside #Content2 with these two divs.
