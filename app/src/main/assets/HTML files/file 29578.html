The problem you have is that you cannot have mixed formatting within certain controls. So for instance if this control was rendered as a span you cannot have this one word rendered differently to all of the other text, you would instead need to wrap this one word in it's own tag.

e.g. <span>Your xyzProductname was developed on the principles of intensive research</span>

Cannot have the formatting you want. Instead you would have to do this:

<span>Your </span><span>xyzProductname </span><span>was developed on the principles of intensive research</span>

This is so that a CSS rule can be applied to the second span. If this was not considered at the start then you will have to go back through and alter the controls to allow this to happen.

This will all depend on what kind of controls you want this done to, buttons, textboxes, labels etc. all require different approaches to get the same effect.
