You can just identify the selected row using a css class.

<tr height="50px" onclick="this.className='selected_row';">   
    <td style="text-align:left" id="Q1B" height="50"></td>
</tr>

Elsewhere

<stlye>
    .selected_row{background-color: yellow;}
</style>

To exclusively select one row at a time,

<tr height="50px" onclick="select_row(this);">   
    <td style="text-align:left" id="Q1B" height="50"></td>
</tr>

<script type="text/javascript">
function select_row(target){
    var prevs = document.getElementsByClassName("selected_row");
    for(var i=0; i<prevs.length; i++){
        prevs[i].className = "";
    }
    target.className='selected_row';
}
</script>

IMPORTANT NOTE: This is just for example and this only works if you're only using one class name for your table rows. This is NOT recommended for any useful implementation.
