You want to make the page 100% in height, and also ensure the div is at least as tall as the page. So, use this:

html, body {
    height: 100%;
}

#fullheight {
    min-height: 100% !important;
    width: 250px;
    background: blue;
}

This ensures that the fullheight div is not constrained to only 100%, and that the page is at least full height.
