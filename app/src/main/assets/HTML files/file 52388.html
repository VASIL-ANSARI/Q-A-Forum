PHP provides a way to track your upload progress, read more about it here - http://php.net/manual/en/session.upload-progress.php

The upload progress will be available in the $_SESSION when an upload is in progress. You can get the key name for your upload using this, 

$key =  ini_get("session.upload_progress.name");
print_r($_SESSION[$key]);

Now to cancel your upload, simply call the $_SESSION with the key and set cancel_upload to TRUE.

$_SESSION[$key]["cancel_upload"] = true;

This will only abort the in-progress uploads and not the completed ones.
