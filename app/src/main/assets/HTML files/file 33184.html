Right, inline script is not allowed. You'll need to move all your script into a separate file, e.g. script.js, and then reference that with a <script> tag:

<script src="script.js"></script>

You'll likely need to set the button's onclick in your script as well, e.g.

document.querySelector('button[value=go]').onclick = join_ym;
