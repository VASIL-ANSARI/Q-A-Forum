If you have any watchers in your controller, you might want to unwatch them in $destroy listener. Like `

var $unwatch = scope.$watch(something, somethingElse);

$scope.$on('$destroy', function () {
    $unwatch();
});

Also, I would recommend not using $scope.$$listener = []; in your code, at least since $$listener is a local variable of angularjs, and it's name might change with upcoming versions of angular.
