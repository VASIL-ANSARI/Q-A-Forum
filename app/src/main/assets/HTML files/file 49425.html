You just have to create a second rectangular element on top of the image with a CSS blur filter:

filter: blur(20px);

CSS

body, html {
    margin: 0;
    padding: 0;
}

#header {
    width: 100%;
    height: 200px;
    background-image: url(http://www.joomlaworks.net/images/demos/galleries/abstract/7.jpg);
    background-position: left bottom;
}

#gloss {
    position: absolute;
    top: 100px;
    height: 100px;
    width: 100%;
}

#gloss::before {
    display: block;
    width: 100%;
    height: 100px;
    background-image: url(http://www.joomlaworks.net/images/demos/galleries/abstract/7.jpg);
    background-position: left bottom;
    -webkit-filter: blur(20px);
    content: ' ';
}

HTML

<div id="header">
    <div id="gloss"></div>
</div>

This creates this gloss effect on the lower half:
