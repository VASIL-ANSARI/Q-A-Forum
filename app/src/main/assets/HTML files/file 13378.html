You don't need that trick any more in iOS5 because of the new appearance APIs. You can set properties like backgroundImage and tintColor on tab bars in iOS 5, so just do something like this:

if ([self.tabBarController.tabBar respondsToSelect:@selector(setBackgroundImage:)])
{
    self.tabBarController.tabBar.backgroundImage = someImage;
}
else
{
    //use the old iOS4 subview hack
}
