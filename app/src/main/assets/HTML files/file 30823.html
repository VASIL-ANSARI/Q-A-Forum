Give a PictureBox array pictures you may use these functions to change visibility.

This one will set visibility for item with given index and all its children:

void SetVisible(PictureBox[] pictures, int index, bool visible) {
    for (int i=index; i < pictures.Length; ++i)
        pictures[i].Visible = visible;
}

Same as above but using object reference instead of index:

void SetVisible(PictureBox[] pictures, PictureBox item, bool visible) {
    SetVisible(pictures, Array.IndexOf(item), visible);
}

To find first visible ancestor of item with a given index:

int FindFirstVisibleAncestorIndex(PictureBox[] pictures, int index) {
    for (int i=index; i >= 0; i--) {
        if (pictures[i].Visible)
            return i;
    }

    return -1;
}

The other overloads are pretty straightforward.
