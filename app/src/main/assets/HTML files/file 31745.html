You can achieve this result using by various means:

  • Include Excel tables in Word.
  • Use VBA or VSTO as described above. There are various articles described already how to add code.
  • A (free) version of Invantive Composition as an add-in to Word (but I work there).
  • Maybe even RTF generation on a Samba share that automatically fills the RTF using the magic statement. Sorry, it is really called that way! See Samba book.

The option chosen depends on some variables I can think about:

  • Roll out to how many client PCs?
  • Size in pages of the filled out document, such as 1 page or 1.500?
  • Controlled environment or PC's somewhere else in the world using a web frontend?
  • Number of document templates to be set up?
  • Support for both 32 or 64 bit Word?
  • Are database drivers available at all PC's?
  • May all users access the database (credentials)?
  • How frequently does the result of the SQL change?
