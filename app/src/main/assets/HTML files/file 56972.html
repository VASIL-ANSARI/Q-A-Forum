If you need more distance between the field-set and the button just do something like: 

<style>
.form-inline .form-control{
  margin-right:1em;
}
</style>

If you need distance between the label and the field as well, do: 

<style>
.form-control{
  margin-right:2em;
}
.form-group label{
  margin-right:1em;
}      
</style>
