You can use something like the sample code in this guide here.

The example uses a CursorLoader which may be used with a ContentProvider ideally, but you can use just the SimpleCursorAdapter passing in the cursor from your query without the CursorLoader. It will show the content of the cursor in a ListView all on its own.

Your code will look like:

@Override
public View onCreateView(LayoutInflater inflater, ViewGroup container,
                         Bundle savedInstanceState) {
    View view = inflater.inflate(R.layout.fragment_visa_utgift, container, false);

    tvDatabase = (TextView)view.findViewById(R.id.tvDatabaseTest);
    dbHandler = new MyDBHandler(getActivity(), null, null, 1);
    tvDatabase.setText(dbHandler.databaseToString());

    lvUtgift = (ListView)view.findViewById(R.id.lvUtgift);
    lvUtgift.setAdapter(new SimpleCursorAdapter(getActivity(),
            android.R.layout.simple_list_item_1,
            dbHandler().getAllItemsInCursor(),
            new String[]{"_titel"},
            new int[]{android.R.id.text1},
            0));
    lvUtgift.setOnItemClickListener(new ListViewClicked());

    return view;
}

And you will additionally have to add the below method in your MyDBHandler class:

public Cursor getAllItemsInCursor() {
    return getWritableDatabase()
            .rawQuery("SELECT * FROM " + TABLE_UTGIFTER, null);
}

android.R.layout.simple_list_item_1 is the resource ID of a predefined layout that has just one TextView with resource ID android.R.id.text1.

You can replace android.R.layout.simple_list_item_1 with a layout of your own that probably has more TextView to be able to show all the details in your table. Your own layout may look like:

utgifter_item.xml in res/layout directory:

<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical">

    <TextView
        android:id="@+id/item_id"
        android:layout_width="match_parent"
        android:layout_height="wrap_content" />

    <TextView
        android:id="@+id/item_title"
        android:layout_width="match_parent"
        android:layout_height="wrap_content" />

    <TextView
        android:id="@+id/item_category"
        android:layout_width="match_parent"
        android:layout_height="wrap_content" />

        android:id="@+id/item_price"

        android:layout_height="wrap_content" />

    <TextView
        android:id="@+id/item_datum"
        android:layout_width="match_parent"
        android:layout_height="wrap_content" />

</LinearLayout>

And call to the constructor of SimpleCursorAdapter will be changed to:

lvUtgift.setAdapter(new SimpleCursorAdapter(getActivity(),
            R.layout.utgifter_item,
            dbHandler().getAllItemsInCursor(),
            new String[]{"_id", "_titel", "_kategori", "_pris", "_datum"},
            new int[]{R.id.item_id, R.id.item_title, R.id.item_category, R.id.item_price, R.id.item_datum},
            0));
