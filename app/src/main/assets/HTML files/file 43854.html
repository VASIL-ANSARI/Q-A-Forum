Yes, this is possible. I can confirm I have done this successfully on iOS. A couple of tips to get it running in the background:

  1. Get this working in the foreground first.

  2. You only have 5 seconds of background running time after entering/exiting a region, so make sure your web service returns quickly.

  3. Add NSLog statements to your callbacks to figure out what is and is not completing in the background.

If the above does not help, post your code in the callback.
