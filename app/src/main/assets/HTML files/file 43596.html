Firstly you need to deserialize JSON into model. Model is object that represents JSON string (see example below). For deserialization you can use default .Net JavaScriptSerializer or third party library like Newtonsoft.Json (You can download third-party libraries using NuGet package manager (Right click on project => Manage NuGet Packages => search what you want)). After deserialization you can bind collections to grid / list views, like this:

    GridView_List.DataSource = response.VerifiedUsers;
    GridView_List.DataBind();

Here is example of your models and databinding using Newtonsfoft.Json

Models: 

// Response model
public class DataResponse
{
    [JsonProperty("data")]
    public string Data { get; set; }

    [JsonProperty("total_user")]
    public int TotalUsersCount { get; set; }

    [JsonProperty("unverified_count")]
    public int UnverifiedCount { get; set; }

    [JsonProperty("verified_count")]
    public int VerifiedCount { get; set; }

    [JsonProperty("deactivate_count")]
    public int DeactivateCount { get; set; }

    [JsonProperty("verified")]
    public List<UserInfo> VerifiedUsers { get; set; }

    [JsonProperty("unverified")]
    public List<UserInfo> UnverifiedUsers { get; set; }

    [JsonProperty("deactivate")]
    public List<UserInfo> DeactivateUsers { get; set; }
}

// User model
public class UserInfo
{
    [JsonProperty("user_id")]
    public int Id { get; set; }

    [JsonProperty("user_name")]
    public string Name { get; set; }

    [JsonProperty("money")]
    public double? Money { get; set; }
}

Deserialization and binding:

    public BindData(string json)
    {
        var response = Newtonsoft.Json.JsonConvert.DeserializeObject<DataResponse>(json);
        GridView_List.DataSource = response.VerifiedUsers;
        GridView_List.DataBind();
    }
