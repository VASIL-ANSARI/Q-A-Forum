If all the values are zero I return a true else false is returned.

So, as soon you see a non-zero value, you can return false immediately. If you get out of the loop, that must mean that everything was zero, and you can return true.

public boolean update()
{
    for(int i=0; i< this.numOfObjects; i++)
    {
         if (BagOfWordsAverage[i] > 0) return false;
    }
    return true;
}

Btw a better way to iterate over arrays in Java:

public boolean update()
{
    for (float value : BagOfWordsAverage)
    {
         if (value > 0) return false;
    }
    return true;
}

Also, the naming convention in Java is camelCase for variable names, so I suggest to rename BagOfWordsAverage to bagOfWordsAverage.
