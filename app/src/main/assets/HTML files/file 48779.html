It's hard to tell for sure based on what you've posted, but one possible solution is to pass an instance of whatever class contains saveFile to your SendFile (and then callback to the instance that was passed in). Something like,

class Example {
    public void saveFile() {
         System.out.println("Do Save");
    }
}

class SendFile {
    Example e;
    public SendFile(Example e) {
        this.e = e;
    }
    public void sendFile() {
        // ...
        e.saveFile();
    }
}
