By requesting http://127.0.0.1:3000/ from http://127.0.0.1:8080 (and vice versa) you violate the same origin policy, because

An origin is defined by the scheme, host, and port of a URL.

Obviously, you haven't configured cross-origin resource sharing on your server.

So you should configure the server to return the Access-Control-Allow-Origin HTTP response header. For instance,

Access-Control-Allow-Origin: http://127.0.0.1:8080

or even (all origins)

Access-Control-Allow-Origin: *
