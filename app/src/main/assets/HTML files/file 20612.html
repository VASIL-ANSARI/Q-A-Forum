setStyleSheet ("YourQWidgetBasedPageClass#nameOfYourPageWidget {background: white;}");

if you're overriding the paintEvent of that custom widget class, remember to call the default QWidget::paintEvent somewhere in your implementation or the css will not kick in

note that the # name is only necessary if you want to select a specific widget. Also, the color can be any of the html style names: white, red, purple, etc... as well as #XXXXXX style colors or using RGB(...)
