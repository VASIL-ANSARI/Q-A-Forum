You can put the two footer contents next to each other using the below code and mark-up.

Demo is available here.

HTML:

<footer>
    <h2 id="contact">Contact us</h2>
    <h2 id="career">Careers</h2>
</footer>

CSS:

footer {
    clear: both;
    position: relative;
    z-index: 15;
    bottom:0;
    height: 5em;
    margin-top: 8em;
    background-color:#98FB98;
}
#contact {
    width:100px;
    padding-right: 10px;
    display: inline;
}
#career {
    width:50px;
    display: inline;
}
