I was able to rework this with just knockout, no jQuery (also, the optionsAfterRender was unnecessary):

http://jsfiddle.net/ZP4E9/36/

I had to add a custom bindingHandler definition, but otherwise it's pretty straightforward. I also had to reformat the opt data object to lend itself to the nested foreach loops, but it's essentially the same.

<select id="mySelect" size="9" data-bind="foreach: opt, value: selectedOption">
    <optgroup data-bind="attr: {label: label}, foreach: options">
        <option data-bind="text: name, option: name"></option>
    </optgroup>
</select>
