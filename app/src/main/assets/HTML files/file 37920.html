What you could do is add the imageview to the row xml file and make it invisible like this:

<ImageView
        android:id="@+id/myImageView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:visibility="gone" />

and then when you want that ImageView to be shown, you can do it in the Base Adapter class like this:

ImageView imgView = (ImageView)findViewById(R.id.myImageView);
imgView .setVisibility(View.VISIBLE);
