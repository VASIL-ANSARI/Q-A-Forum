You're only checking if VerticalOffset is equal to ScrollableHeight. User can avoid this one single spot by moving the scroll bar very fast. Events are not fired that frequent, so you should check if VerticalOffset is past half. You can achieve this by changing operator from == to >=:

if (MyContent.VerticalOffset >= MyContent.ScrollableHeight / 2 && MyContent.VerticalOffset != 0)
{
    MessageBox.Show("Past Half");
}
