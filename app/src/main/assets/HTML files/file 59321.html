Okay, I figured out a way and it is to read the fields and files properties when parsing a form.

let form = new multiparty.Form()
form.parse(req, (err, fields, files) => {
  console.log(fields)
})

Fields is a object with the name of the attribute as a key and it's value as the value. The result is

{ todoname: [ 'Post the answer' ], tododesc: [ 'Post the answer you figured out on your StackOverflow question' ] }
