I've also encountered that error. I encountered that error when almost thousands of user simultaneously accessing the web service.

I put this piece of code..

 ServiceClient = new ServiceClient ();

 var customBinding = new CustomBinding(ServiceClient.Endpoint.Binding);
 var transportElement = customBinding.Elements.Find<HttpTransportBindingElement>();
 transportElement.KeepAliveEnabled = false;

 ServiceClient.Endpoint.Binding = customBinding;
 ServiceClient.Open();
