You can create two dimensional vector like so

std::vector<std::vector<float> > table(N, std::vector<float>(N));

don`t forget to initialize it like this, it reserves memory for N members, so it does not need to reallocate all the members then you are adding more. And does not fragment the memory. you can access its members like so

table[1][2] = 2.01;

it does not uses copy constructors all the time because vector index operator returns a reference to a member; so it is pretty efficient if N does not need to change.
