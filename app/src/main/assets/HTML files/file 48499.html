The best way to authenticate and Android app, is to

  1. Authenticate the user in the Android App with the Google Signin for Android. Link: https://developers.google.com/identity/sign-in/android/start

  2. The Google SignIn above will generate a google ID token. The google ID token should be sent to your server. The server can get user details using the this ID. You can read more on https://developers.google.com/identity/sign-in/android/backend-auth

This part can however be implemented using the strategy passport-google-token.

Create a route and strategy to validate the token on your server, and call the API from android passing the access_token as a variable.

Hope this helps and saves you time.
