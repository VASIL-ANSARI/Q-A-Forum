There's currently no way of doing this with CakePHP. However, you can get around it by piping a series of answers into the script. I found the answer here.

If you want to answer 'y' to every prompt, you can use:

echo 'y' | cake schema create

If you just want to use defaults, you can use:

printf "\n\n" | cake schema create

If you want to use specific responses, you can list them, separated by newline characters. This will send 'y' and then 'n', dropping the tables, and not recreating them:

printf "y\nn\n" | cake schema create
