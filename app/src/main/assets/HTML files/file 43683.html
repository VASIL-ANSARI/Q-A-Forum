You need to add a footer to your UITableView which will have your view that has the comment UITextField and UIButton.

To add a footer to your UITableView, implement the following method in the UITableView's delegate:

- (UIView *)tableView:(UITableView *)tableView viewForFooterInSection:(NSInteger)section

It will return the view you want to use as your footer (your UIView with your UITextField, etc).

Documentation: https://developer.apple.com/library/ios/documentation/UIKit/Reference/UITableViewDelegate_Protocol/#//apple_ref/occ/intfm/UITableViewDelegate/tableView:viewForFooterInSection:
