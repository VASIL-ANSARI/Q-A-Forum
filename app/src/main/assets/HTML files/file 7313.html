Add another class name (I used class="text") for the main body paragraphs, to distinguish them from the note paragraphs.

<p class="text">

Then use that new class name to define the increment:

p.text {
   counter-increment: paragraph;  
}

http://jsfiddle.net/Keh65/3/

Note that 'paragraph' is an arbitrary name for the counter of your choosing, and is not necessarily tied to the html <p> element.
