Based on the error messages it could be you didn't pass in an $id? CodeIgniter controller work like this:

Say you have this URL:

example.com/index.php/stack/your/url/42

The two segments (url, 42) after your will be passed to the function as parameters:

class Stack extends CI_Controller {
    public function your($url, $id) {
        echo $url;
        echo $id;
    }
}
