Your code looks good except for the larger_than? method. Think again about how the method is called:

square.larger_than?(rectangle)

It is called on a Shape object, with another Shape object as the first parameter.

So, the method expects a parameter (the other shape). Within the method, you can access the current shape with self and the other shape (using the parameter) to compare both areas.

One more little thing: You are doing this:

if Shape.new.area > self.class.area
  true
else
  false
end

The test-expression in your if statement already returns a Boolean (true in the first case and false in the second). So you can directly return the value of the test-expression instead of writing that if statement.

Update:

The larger_than? method should look similar to this:

def larger_than?(otherShape)
  self.area > otherShape.area
end

self is the Shape where the larger_than? method is called on. To compare both shapes, you have to compare the areas of those shapes.
