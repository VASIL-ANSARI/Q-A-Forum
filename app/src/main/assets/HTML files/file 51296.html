You could do it this way: 

def create
  r = Record.create(record_params)
  if r.save
    render json: r.to_json
  else
    render json: errors: r.errors, status: 422
  end
end

If and error is returned, it will return this:

{
  errors:
    {
      attr_1:["can't be blank"],
      attr_2:["can't be blank"]
    }
}

Related to the first example you showed, ruby's Exception#message http://ruby-doc.org/core-2.2.0/Exception.html#method-i-message returns the errors in the way you described. I generally see rescue_from being used to return generic 404 pages.
