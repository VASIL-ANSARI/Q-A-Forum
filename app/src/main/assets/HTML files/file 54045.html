Firstly, the concatenation is incorrect. You use . in php not + to concatenate. Next, with prepared statements, you should be binding user input through parameterized queries. Let's have a look:

$dbExec = $dbConnect->conn->prepare("SELECT nome, direc FROM condoInfo WHERE idCondo = :id");
$dbExec->bindParam(':id', $condoInfo, PDO::PARAM_INT);
$dbExec->execute();

This will make sure your user data is sanitized correctly.
