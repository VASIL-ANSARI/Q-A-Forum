I suspect the problem is with the locale used in the SimpleDateFormat. By default, SimpleDateFormat takes the default locale of the system... so if you run this code in France, "Mon" won't be recognized as a day-of-week abbreviation, for example.

The simplest fix for this is just to call the SimpleDateFormat constructor which specifies a locale:

SimpleDateFormat formatter =
    new SimpleDateFormat("EEE MMM dd HH:mm:ss zzz yyyy", Locale.US);
