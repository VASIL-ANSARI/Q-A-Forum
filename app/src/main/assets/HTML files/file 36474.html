Without introducing the additional complexity of regular expressions, you could just use indexOf to determine if the path name starts with "/auto/make/":

if (window.location.pathname.indexOf('/auto/make/') === 0) {
    // do something
}

If you just want to make sure that the path name includes "auto/make/", but not necessarily at the beginning, you would do:

if (window.location.pathname.indexOf('auto/make/') >= 0) {
    // do something
}
