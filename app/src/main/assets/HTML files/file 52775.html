You can either use jQuery's built in fadeIn() function, or you can simply add a fade in transition on the class you are adding/removing.

.parallax_bg {
  transition: opacity .5s, visibility .5s;
  opacity: 0;
  visibility: hidden;
}

.parallax_bg.bg_change {
  opacity: 1;
  visibility: visible;
}
