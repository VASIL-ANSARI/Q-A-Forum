Use $_REQUEST instead of $_GET and $_POST. Because $_POST will be overwritten by $_GET when you are using two in one request.

Code Suggestion:

<form action="<?php echo site_url();?>" method="post">
<input type="text" name="name"  /> <input type="submit" value="Send"  />
<input type="hidden" name="page_id" value="8" />
</form>

Now $_POST should work for you.
