If you want to highlight when user press on the row, then just use selectors in the background of your row's parent layout. row_selector.xml

<?xml version="1.0" encoding="utf-8"?>
<selector xmlns:android="http://schemas.android.com/apk/res/android" >
      <item android:state_selected="true"
           android:drawable="@color/red" /> <!-- pressed -->
     <item android:state_pressed="true"
           android:drawable="@color/red" /> <!-- pressed -->
    <item android:drawable="@color/gray" /> <!-- default -->
</selector>

put this xml in drawable and set as background of your parent layout.

Hope it will work :)
