You are not flushing (saving) the file, thus nothing is saved. To put it out there, it works like the following:

First the Formatter writes to a buffer, and when you flush the formatter it will write that buffer to the file and then clear this buffer.

So in short, call f.flush(); before f.close();

When you try to flush the formatter after it has been flushed, you will get a FormatterClosedException. Everything what is stated here is in the documentation of Formatter
