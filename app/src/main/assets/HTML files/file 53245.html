Looks like you need http://php.net/manual/en/book.iconv.php If you have it you can try:

$fileName = iconv("cp1251", "utf-8", $_FILES["upload"]["name"]);
// or

$baseDir = '/some/dir/';

move_uploaded_file(
    $_FILES["upload"]["tmp_name"],
    $baseDir.$fileName
);
