This can be achiever quite easily with absolute positioning in CSS.

* {
  margin:0;
}
header {
  position:absolute;
  width:100%;
  height:64px;
  top:0;
}
footer {
  position:absolute;
  width:100%;
  height:64px;
  bottom:0;
}
div {
  position:absolute;
  width:100%;
  top:64px;
  bottom:64px;
}

Note, that in real code you'll want to use a class for the inner div otherwise you'll be styling all the div elements.
