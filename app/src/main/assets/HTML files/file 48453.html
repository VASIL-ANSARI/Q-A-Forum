You may create the link with inline base64 content that will ask user to download the content by setting the download HTML5 attribute. For example:

<a title="Download PDF" download="Sample.PDF" href="data:application/pdf;base64,%PDFCONTENT%">Click here to download</a>

About the printing PDF in Firefox: if you use the latest Firefox then it may use PDF.js based viewer. Try to switch to Adobe Reader plugin instead as it still works best when need to view or print PDF.
