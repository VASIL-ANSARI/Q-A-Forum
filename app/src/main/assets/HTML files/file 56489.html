Adding a reference to Microsoft.Office.Interop.Outlook and then adding the following using statement

using Microsoft.Office.Interop.Outlook;

Should give you a start to work with Outlook programmatically. From there the documentation for the Interop library(Which can be found here https://msdn.microsoft.com/en-us/library/office/microsoft.office.interop.outlook.aspx) should be of use to you.
