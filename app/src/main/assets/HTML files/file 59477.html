In your user model you can either define a scope or a search function. For example 

def self.search(name)
  where("name LIKE ?", "%#{name}%")
end

Then in your controller assuming you are going to search through your index view the code will probably be something like this 

def index
  @user = User.all
  @user = User.search(params[:search]) if params[:search].present? 
end

Then just do up a simple form in your views to get the data 

<%= form_tag(users_path, method: :get) do %>
