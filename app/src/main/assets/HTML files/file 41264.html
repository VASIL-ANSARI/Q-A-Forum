Your adapter holds all the items of your list. In your case one instance of such an item should at least contain a String (for the textView) and a Bool (for the CheckBox). Whenever you want a list of all the checkboxes, just iterate over your list items and create a temporary list of all the items that have true boolean value.

List<YourListItemObject> checkedItems = new ArrayList<YourListItemObject>();

for (YourListItemObject listItem : adapter.getItems()) {
  if (listItem.getBool()) {
     checkedItems.Add(listItem);
  }
}
