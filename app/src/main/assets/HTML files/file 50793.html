The problem is that your renderScene method needs to know which component to render but you are always sending it the LoginScreen component. Do something like this:

renderScene: (route, navigator) {
  Component = route.component
  <Component navigator={navigator} route={route} />
}

This means that you move to a new scene by passing a component key to the navigator.

this.props.navigator.replace({
  component: MainScreen
});
