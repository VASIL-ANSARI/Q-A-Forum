You can hide text past a certain point by limiting the height of your description in css, and setting your text-overflow behavior in css.

.ellipsis {
  white-space: nowrap;
  text-overflow: ellipsis;
  overflow: hidden;
}

Note that this will only work for a single line description. If you want to extend your description to multiple lines, there are a few ways you can do that.

See: http://html5hub.com/ellipse-my-text/

Alternatively, if you don't care if the overflow is indicated with an ellipsis, you can simply set

.thumbnail .caption p {
  height: 40px;
  overflow: hidden;
}
