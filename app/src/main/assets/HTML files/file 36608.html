I don't know what exactly are you trying to do. But If you can use php, definitely use it! Something like this:

function get_data($url) {
   $ch = curl_init();
   $timeout = 10;
    curl_setopt($ch, CURLOPT_URL, $url);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
    curl_setopt($ch, CURLOPT_CONNECTTIMEOUT, $timeout);


  $data = curl_exec($ch);
  curl_close($ch);
  return $data; 
}

$page = "http://yourwebsite.whatever";    
$html =  get_data($page);

And pars what you need from $html.
