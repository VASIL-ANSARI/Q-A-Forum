You should use SQL parameters:

cuDatabase.execute ("""
    SELECT Result
    FROM resultstable
    WHERE QuizID = ?
    AND UserID = ?""", (Quizno, UserID))

The ? placeholders will be replaced by your values, automatically quoted to prevent SQL injection attacks (and operational errors).
