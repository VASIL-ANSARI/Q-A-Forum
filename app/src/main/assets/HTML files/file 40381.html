Just use absolute positioning with offsets of 100% to stick to the sides of the originator.

HTML:

<div id="a">
    <div id="b">
    </div>
    <div id="c">
    </div>
</div>

CSS:

div {
    position:absolute;
    height:50px;
    width:50px;
}
#a {
    left:100px;
    background:red;
}
#b {
    right:100%;
    background:blue;
}
#c {
    background:green;
    left:100%;
}

Will give you exactly what you want.
