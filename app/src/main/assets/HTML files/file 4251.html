Try this

<Style TargetType="Label">
    <Setter Property="Effect">
        <Setter.Value>
            <BlurEffect Radius="8.0" KernelType="Box"/>
        </Setter.Value>
    </Setter>
</Style>

Update

For the Label that should't have this effect you can use

<Label Style="{x:Null}" ...>

or any other style that you may use
