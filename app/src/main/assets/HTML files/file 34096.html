You can not put comma to there between "email" and "type:string, required:true, length:140, regex:[_A-Za-z0-9-+]". You can check the right syntax and fix that.

For example this would work:

  create_admin:
  http_method:  POST
  requires_user_auth: true
  param_names:
    - { email: "type:string, required:true, length:140, regex:[_A-Za-z0-9-\\+]" }

    - "password_confirmation"
