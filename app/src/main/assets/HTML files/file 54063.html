Your code is not very readable if you insert ternary operators inside a for loop declaration as you're attempting.

It would be far more readable to produce two for loops, one of which is entered depending on the condition.

if (bool)
{
    for (int i = 0; i < NB_MAX; i++)
    {
        body(i);
    }
}
else
{
    for (int i = NB_MAX - 1; i >= 0; i--)
    {
        body(i);
    }
}
