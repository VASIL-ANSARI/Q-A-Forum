This is the way I do it:

int[] selectedRows = table.getSelectedRows();
if (selectedRows.length != 0) {
    for (int counter : selectedRows) {
        String value= table.getModel().getValueAt(counter, Constants.COLUMN).toString();
        /*
        * do stuff
        */
    }
}

And to give some context, I used this with a JTable that had info about the files in a folder. The app permitted me to delete files so, I used that piece of code to retrieve the name of the files selected and to delete them one by one.

In your case (i think) this line:

String value= table.getModel().getValueAt(counter, Constants.COLUMN).toString();

will turn into this:

Gerar ger = modelo.getGerar(counter);

NB: I strongly recommend the use of a lower case letter as the first letter of an attribute( ex: private Integer Nr; -> private Integer nr;)
