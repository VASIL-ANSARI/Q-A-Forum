This line:

$check = $this->_db->get($rule_value, array($item, '=', $value));

is returning false, instead of the expected instance of $this->_db. I'm not familiar with the tutorials you mention, but you should be checking the value returned at this point. 

case 'unique':
$check = $this->_db->get($rule_value, array($item, '=', $value));
if($check != false) {
    if ($check->count()) {
        $this->addError("This {$item} already exists.");
    }
} else {
    // Failed database call
    echo $this->_db->error();
}
break;

This echoes the database error - you shouldn't display database errors in production, of course.
