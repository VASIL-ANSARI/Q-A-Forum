*++p->name equivalent to *(++(p->name)).

At that point, p is still pointing to the first element of b so its contents are { 18, "Peter" }.

It takes the name pointer, applies the prefix increment to it and the expression evaluates to a pointer to the second character of "Peter". Finally the dereferencing operator gives the character 'e'.

At *(++p)->name, p is pointing to the last element of b (after the increment) so its contents are { 20, "John" }.

That takes the name pointer and dereferences it to give us the first character of "John" which is 'J'.
