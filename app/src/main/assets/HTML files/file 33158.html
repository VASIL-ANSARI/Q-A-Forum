You have to use more data types than just String. As you are working with prices, the best primitive type is float and the best object is BigDecimal. For convenience, lets just work with floats:

You will want to add some values for the prices like this:

private static final float TYPE_LAMINATE_PRICE = 10.0f;
private static final float TYPE_PLASTIC_PRICE = 5.0f;

You then want a way to detect what the user entered, by doing something like:

float price;
if (customerCoat.equalsIgnoreCase("laminate") {
    price = TYPE_LAMINATE_PRICE;
} else if (customerCoat.equalsIgnoreCase("plastic") {
    price = TYPE_PLASTIC_PRICE;
}

You also need to detect the quantity needed. Integer.parseInt(String) will do this for you.

This should get you started, I'm sure you'll be able to work out how to get things working how you want them from there.
