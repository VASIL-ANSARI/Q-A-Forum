Technically solution that can take you 5mins to implement:

In /app/code/community/AW/Blog/Block/Collection.php replace public function getPosts(); with:

public function getPosts()
{

    $collection = parent::_prepareCollection();

    $tag = $this->getRequest()->getParam('tag');

    if ($tag) {
        $collection->addTagFilter(urldecode($tag));
    }


    //this part here is responsible for search

    $search = $this->getRequest()->getParam('search'); //get request "search" parameter from url

    if ($search) {
        $collection->searchBy(urldecode($search));
    }

    //end

    parent::_processCollection($collection);

    return $collection;
}

After that, add following code in: /app/code/community/AW/Blog/Model/Mysql4/Blog/Collection.php CREATE FUNCTION searchBy()

public function searchBy($key)
{         
    $this->getSelect()
            ->where("(title) LIKE '%$key%' OR (short_content) LIKE '%$key%'");

    return $this;
}

Though it's not perfect solution but it's doing it's job you can improve MySql query in way you want it.

Now when you go to: http://example.com/blog?search=keywords

Blog will display with posts which includes particular keywords.
