You have to export the SomethingElse explicite:

[Export(typeof(SomethingElse))]
[PartCreationPolicy(CreationPolicy.NonShared)]
public class SomethingElse : ISomething { }

Then the _container.GetExportedValue<SomethingElse> works. It is also possible to export the same class with by giving a name the export.

[Export("SomethingElse", typeof(ISomething))]
