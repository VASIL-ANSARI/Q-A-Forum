Try to call label.sizeToFit() and scrollView.layoutSubviews().

If this isn't working try to add a height constraint to your label and manually set its constant to the size you need :

let expectedSize = label.sizeThatFits(CGSize(width: scrollView.frame.size.width, height: CGFloat.max))
labelHeightConstraint.constant = expectedSize.height
self.view.layoutSubviews()

In the controller's updateViewConstraints().
