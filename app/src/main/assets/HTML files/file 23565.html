It looks like you're confusing the rows with the spans. A row with a document width of 1200 will have the same width even if you nest it. The thing you have to track is your spans.

row - 1200px width
  span12 - 1170px width
    row - 1200px width
      span12 - 1170px width

The reason being that the span12 has a hard coded width of 1170px and the row resets the left margin with -30px

This part of the css code may help explain

@media (min-width:1200px) {
    .row {
        margin-left: -30px;
        *zoom: 1;
    }

    .container {
        width: 1170px;
    }

    .span12 {
        width: 1170px;
    }
}
