Since you are retrieving data every time a new child appears on /Course/ (CourseRef.observeEventType(FIRDataEventType.ChildAdded), when calling the first .setValue(courseField.text) your code is triggering the event listener and logging the entry with only the CourseName.

So make sure that when adding a new Course you call setValue() one time only.

Your code will look like this:

    let Course = self.ref.child("Course").childByAutoId()

    let course = ["AddedDate": dateString,
        "CourseName": courseField.text,
        "AmountOfHoles": holesField.text]

    Course.setValue(course)
