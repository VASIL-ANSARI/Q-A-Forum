That should have worked.... but anyway you have other options to try. You can return a RedirectView instead, or just a String 'redirect:/'.

//...
@RequestMapping(value = "/public_html", method = RequestMethod.GET)
public View homePagePub() {
    return new RedirectView("/", true);
}
// - OR -
@RequestMapping(value = "/public_html", method = RequestMethod.GET)
public String homePagePub() {
    return "redirect:/";
}
//...
