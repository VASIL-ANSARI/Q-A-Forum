The tech-list is not a value resource. You need to put it under res/xml/ instead. Thus, the path of your tech_filter.xml needs to be

app/src/main/res/xml/tech_filter.xml

and the manifest entry for it would look like this:

<meta-data
    android:name="android.nfc.action.TECH_DISCOVERED"
    android:resource="@xml/tech_filter" />
