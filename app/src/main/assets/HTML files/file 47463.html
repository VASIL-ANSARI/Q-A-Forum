If all you are trying to do is delay running a function you can try a function like :

$('.play-button').click(function(e){
    e.preventDefault();
    $('html, body').animate({ scrollTop: 0 }, 500);
    setInterval(function () {$('#video-frame').attr("src", $(this).attr("href"));}, 650);        
});

The set interval will run a function after the given time 650 in our case meaning 650ms. Try it out and let me know if it doesn't work
