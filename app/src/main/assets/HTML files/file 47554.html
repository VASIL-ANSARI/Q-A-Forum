One difference between jQuery promises and bluebird promises (which are Promises/A+ compliant), is that your then function gets only one parameter. If you want to retain all the information that jQuery gives you, you have to wrap it up in a single object:

Promise.resolve($.get(...).then(function(data, status, xhr) {
    return {
        data: data,
        status: status,
        xhr: xhr
    };
})).then(function (data) {
    // data.data has the response data
    // data.xhr contains the jqXHR object
});
