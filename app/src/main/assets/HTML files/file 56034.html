You are calling new Clipboard inside a self-invoking function, which won't work because it is executed before the DOM is loaded. Instead you should wrap it in the "document ready" event:

$(function() {

    new Clipboard('#copy-button');

});

I would also like to criticize your HTML markup:

  • There is basically no semantics in your code. For example, you should use <label> tags to indicate input labels, not <b> tags (which have been deprecated for more than a decade).

  • You should specify one global font, not repeat yourself on each tag. This is as simple as:

    body {
    font: 12px 'Trebuchet MS', Arial, Helvetica, sans-serif;
}

    Also notice font names with spaces require quotes around them.

  • Do not use inline styles.
  • Don't mix HTML with Javascript. Why? For maintainability of the code, and to make it easier to read. This is most likely the reason why nobody took the time to answer your question.
  • Close your HTML tags properly.
