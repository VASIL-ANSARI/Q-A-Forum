Short answer is yes, all your network requests should be asynchronous.

Maybe it helps to imagine it like a row of dominoes with each request a domino, and the completion handler is what you use to push the next domino in the row.

Your -DoAPI_A_WithData... method hides the completion handler, that's why you are not seeing how it should work.

You could rework it like

- (int) doAPI_A_WithData: (NSData *)data onCompletion: (ResponseBlock)resultHandler

where you call the resultHandler, that does whatever you need with the response, from the task's completionHandler.
