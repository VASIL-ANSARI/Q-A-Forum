You cannot. The C# code in your action method will always be executed well before client-side scripts are executed. Your best bet will be simply injecting the loop in the script:

[HttpPost]
public ActionResult PerformDiagnostic(TestsModel model)
{
    var script = "for (var i = 0; i < 10; i++ ) $('#divStatus').append('<h1>hello World' + i + '</h1>');";
    return JavaScript(script);
}
