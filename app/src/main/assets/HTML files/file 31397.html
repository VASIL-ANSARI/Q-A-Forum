You should check if those checkboxes are not empty like this $_POST[$key] != '' and if they are empty unset them from $_SESSION array like this unset($_SESSION[$key]).
So the whole code is:

session_start();
foreach ($_POST as $key => $value) { 
    if($value == ''){
        $_SESSION[$key] = false;
        unset($_SESSION[$key]);
    }
    else {
        $_SESSION[$key] = $value;
    }
}
