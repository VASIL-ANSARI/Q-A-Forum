Just for posterity, I finally figured out what was wrong:

the error handlers fired correctly but the redirect to the next page still happened because there was still a window.location = "next.html" available. Similar to removing the comments from the window.location line in the code above.

So basic setup, taking into account that the code above only works in FireFox (not in Chrome or Safari):

$(document).ajaxError(genericAjaxErrorHandler);

$("#prev").click(function() {
  window.location = "prev.html";
});

$("#next").click(function() {
  doPostData();
});

function doPostData() {
  // actions here

  if (allIsOk) {
      window.location = 'next.html';
  }
}

I added a 

$('form').submit( function(e) {
  e.preventDefault();
});

to be sure, but it might not be necessary.
