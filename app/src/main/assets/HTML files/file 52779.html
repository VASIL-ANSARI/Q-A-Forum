You can prevent evaluation of HTML tags simply by converting all instances of < to &lt;. Example:

<?php
$rawHTML = '<html><h1>This is a Title</h1></html>';
$encodedHTML = str_replace('<','&lt;',$rawHTML);
?>

<html>
    <body>
        <h3>
            The Encoded HTML
        </h3>
        <div style="border: 1px solid gray; padding: 12px">
            <pre><?php echo $encodedHTML; ?></pre>
        </div>
    </body>
</html>

Alternatively, there's the <xmp> tag, but it is obsolete. The Mozilla documentation on <xmp> recommends using <pre> and <code> in its place.
