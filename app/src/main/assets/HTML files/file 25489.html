You have to understand few things here:

  1. bootstrap modal is already loaded in a view or rather its already rendered - its just hidden (do view source you will see it there).

  2. No matter how much you click on it and what you do it will always display same data

  3. you can't parse data to bootstrap modal via link unless you do some post procesing.
