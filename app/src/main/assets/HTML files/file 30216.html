You're assigning the new array (JsonConvert.DeserializeObject<Skill[]>(blah)) to allArray, so the reference to old instance is replaced by new instance! if you wanna fill the old array with values of new array just use CopyTo method:

JsonConvert.DeserializeObject<Skill[]>(blah).CopyTo(allSkills, 0);

Note that this way you can't apply all values to allSkills[1], because it's a one dimensional array. For that you can use a two dimensional array if you want an array of arrays!

Edit: If you wanna use an array of arrays you have two options, use jagged array or a simple two dimensional array, with jagged array you have to specify the number of rows but the number of columns can vary. But if you know the number of columns then go for a simple two dimensional array, though I recommend you to use List instead if Array. Here is jagged array code sample:

public Skill[][] = new Skill[5][]; // Instantiate a jagged array with 5 rows

...

allSkills[1] = JsonConvert.DeserializeObject<Skill[]>(blah);
