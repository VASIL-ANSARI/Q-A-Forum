The purpose of this answer is not to provide a ready chunk of code but rather the instructions for you to create the script that you need.

What you could do is:

Loop through the string using the charAt(int index) function until you find the space * . When you do, you can use the substring(int beginIndex, int endIndex) function to get the first word and then the substring(int beginIndex) function to get the second word. If you do not want to use substring() then you could just add each character as you go through the loop.

When you get both strings you just need to set the String output to the second String, add a space and then add the first String.

* The ascii value for space is 32, you do not need to know this but it is better practice in my opinion to have a field called space instead of using " " all the time.
