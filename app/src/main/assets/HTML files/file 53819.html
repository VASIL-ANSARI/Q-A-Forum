As far as I understand, you can specify the positions as the dataSource configuration object. It allows you to set filter for the lookup data source:

var positions = {
    store: [{
        "ID": 1,
        "Name": "CEO"
    }, {
        "ID": 2,
        "Name": "   Sales Assistant"
    }, {
        "ID": 3,
       "Name": "    HR Manager"
    }, {
        "ID": 4,
        "Name": "   IT Manager"
    }, {
        "ID": 5,
        "Name": "   Shipping Manager"
    }],
    filter: null
};

Then, you will be able to configure the filter according to your case.
