The AttributeError occurs due to the fact that labels is not part of the original DataFrame. You can however access the data by using the following method:

df1['labels']

This will give you the following output:

0    1
1    1
2    0
Name: labels, dtype: int64

for your plotting error also replace df1.labels with the method suggested above as follows:

plt.scatter(df1.TowerLat, df1.TowerLon, c=df1['labels'])

As for the error with the groupby method change it to:

bylevel=df1.groupby(df1["labels"])

as levels is not part of the DataFrame
