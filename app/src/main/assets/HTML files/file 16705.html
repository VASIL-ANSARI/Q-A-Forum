As @Chris said, there are no shortcuts with JPEG. However - and I realise that this is a long shot - you can specify the compression rate directly if you are willing to use JPEG2000 instead of JPEG. If you look at the Encoder class in this JJ2000 library, you'll see how to fix the bitrate of the encoded image.

Unfortunately, as JPEG2000 support is nowhere near as widespread as JPEG, I fear this may not be much use to you.
