From your question and comments I take it that you are implementing some kind of CI system (otherwise Xcode has excellent built-in support for this as I am sure you have seen). To have the static analyzer run when building from the command line, see this question.

To generate the report, you need to parse the standard output/error from clang (this is what Xcode does for you). You may want to try scan-build (there is also a Jenkins plugin for this if you use Jenkins for CI).
