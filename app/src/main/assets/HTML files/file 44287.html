UChar* u_strFromUTF8    (   UChar *     dest,
int32_t     destCapacity,
int32_t *   pDestLength,
const char *    src,
int32_t     srcLength,
UErrorCode *    pErrorCode 
)       
Convert a UTF-8 string to UTF-16.

If the input string is not well-formed, then the U_INVALID_CHAR_FOUND error code is set
