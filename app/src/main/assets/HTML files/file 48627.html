I found the answer : What I did was to forget the methode lookup(...)

ctx = new InitialContext(...);
int maxThreadsTemp = (int) ctx.lookup("maxThreads");

Instead I used a hashtable (much simpler): 

ctx = new InitialContext(...);
Hashtable<?, ?> environment =  ctx.getEnvironment();
            String nbrThread = (String)  environment.get("maxThreads");

And in jndi.properties : maxThreads = 5 (thx to @Claus Ibsen)
