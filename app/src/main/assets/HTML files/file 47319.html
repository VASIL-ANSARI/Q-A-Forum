You could use direction: rtl; applied to the columns container (and use the opposite value for the inner elements), e.g.

Codepen : http://codepen.io/anon/pen/bdxKbw

Markup

<main>
    <p>1</p>
    <p>2</p>
    <p>3</p>
    <p>4</p>
</main>

CSS

main {
  -webkit-column-count: 2;
     -moz-column-count: 2;
          column-count: 2;
  height    : 600px;
  direction : rtl; 
}

p {
  direction : ltr; 
   -webkit-column-break-inside: avoid;
             page-break-inside: avoid;
                  break-inside: avoid;
   height   : 150px;
   margin   : 0 0 2em 0;
   padding  : 0;
   border   : 1px #ccc dashed;
}

Result (tested on Chrome)
