It would be simple to write an extension method..

public static IQueryable<T> WhereAll<T>(this IQueryable<T> q,IEnumerable<Expression<Func<T, bool>>> conditions)
{
  foreach(var condition in conditions)
    q=q.Where(condition);
  return q;
}

You could also use a PredicateBuilder. Initialize to True, and AND in all the conditions, then just use the regular .Where with the compound predicate you've built.
