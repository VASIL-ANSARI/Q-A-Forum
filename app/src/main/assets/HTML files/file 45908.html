Have a look at Hamcrest.

Assume the following test class:

public class TestObject {

    private byte[] content;

    public byte[] getContent() {
        return content;
    }

    public void setContent(byte[] content) {
        this.content = content;
    }
}

You can compare the content field like this:

import static org.hamcrest.Matchers.equalTo;
import static org.junit.Assert.assertThat;

[...]

PropertyDescriptor propertyDescriptor = new PropertyDescriptor("content", TestObject.class);
Method readMethod = propertyDescriptor.getReadMethod();

assertThat(readMethod.invoke(testObject1), equalTo(readMethod.invoke(testObject2)));

This will work with array and non-array fields, Hamcrest does the check for you.
