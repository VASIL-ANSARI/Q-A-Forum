Yes, you need to surround the comma, or use JavaScript to achieve this.

JS is something like this:

Html:

<div id="myText">Visit, Stackoverflow!</div>

jQuery + JS:

var myText = $("#myText").html();
myText = myText.replace(",", "<span>,</span>");
$("#myText").html(myText);
