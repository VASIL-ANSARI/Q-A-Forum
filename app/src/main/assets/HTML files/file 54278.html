You can use Skip() and Take() on your query :

        var qry = (from i in dt.Questions
                  where i.TechnologyId == ddlTechnology.SelectedIndex
                  select i).Skip(currentPage).Take(1);

Then you add the currentPage variable to your class and increment it in the Next page button handler. And of course the Next page button handler must reexecute the code you have in the btnSchedule_Click method.

Note that in my example currentPage must be zero-based (first page is 0, not 1)
