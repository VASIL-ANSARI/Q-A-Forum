HTML tags can have data attributes that specify certain data for that tag. In this case, I suggest adding two data attributes to each image, one each for low-res and high-res.

<img src="assets/images/front/grupo/grupo-bignardi-2.png" data-high-res="grupo-bignardi-2.png" data-low-res="grupo-bignardi-2-low-res.png"/>

And from there you can swap out the src attribute with whichever image you need.
