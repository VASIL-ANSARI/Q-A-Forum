Try scanf("%d", &phoneNumber); and use %d for the printf as well. In C and its derivatives, functions are all pass-by-value. In order for a function to modify a variable, you need to pass in the address of that variable rather than its value. That's why you prefix with & to get the address of phoneNumber into the scanf function. However, you only use the & for the scanf argument, not for declaring or using the variable locally.
