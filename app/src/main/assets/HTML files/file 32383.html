Run the JS after the HTML has loaded. When you call the JS in the head, the button doesn't exist yet, it's rendered after the JS is called.

This will work:

<!DOCTYPE html>
<html lang="en">
<head>

</head>
<body>
    <button id="next">Say hi</button>
    <script type="text/javascript">
        ...
    </script>
</body>
</html>
