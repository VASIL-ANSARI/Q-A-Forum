Android is Linux-based operating system. If we will take a look at Android source here, we will see a lot of C source files.

DVM is virtual machine developed by Google and different from JVM. It should work on devices with low memory amount and designed for them. If we will take alook at Dalvik sources here, we will see a lot of C/C++ and Java source files.

Application for Android are written in Java but compiled byte-code is different from Java byte-code and is not compatible with JVM.

Python is used to build Android, but (As far as i understood) not used to implement Android or its parts.
