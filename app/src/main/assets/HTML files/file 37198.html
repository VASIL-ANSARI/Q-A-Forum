You need to create and add a leftBarButtonItem to your navigation bar. You can assign whatever action you want to it (presumably, popViewControllerAnimated:NO). You can do this in IB by dragging a barButtonItem to the left side of the navigation bar, or in code like so,

-(void)viewDidLoad {
    [super viewDidLoad];
    UIBarButtonItem *left = [[UIBarButtonItem alloc] initWithTitle:@"Zap" style:UIBarButtonItemStyleBordered target:self action:@selector(zap:)];
    self.navigationItem.leftBarButtonItem = left;
}


-(void)zap:(UIBarButtonItem *)sender {
    [self.navigationController popViewControllerAnimated:NO];
}
