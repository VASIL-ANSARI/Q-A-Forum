I'll just write the basics I'm using here.

You can use javascript object structure to mimic a class-like structure in javascript. Or use different frameworks such as angular.js and use their structuring method.

The javascript object structure example:

var app = app || {}; // if it exists, don't overwrite. 

var app.toolset = {
   globals : {
       is_ready: false,
       will_update: true
   },
   tool1 : {
      add: function(){
         if(app.toolset.globals.is_ready){
             // do something
         }
      },
      remove: function(){
         app.toolset.globals.will_update = false;
      }
   },
   tool2: {
      add: function(item){
          // do another thing 
          app.toolset.tool1.remove();
      },
      remove: function(item)
      {

      }
   }
}

and then you'll have a structured access to methods under a namespace-like pattern.

you can extend it with different files with different keys below app.

// make sure every file starts with this line
var app = app || {}; 

// app.collections.js file
app.collections = { };

// app.tools.js file
app.tools = {};

// app.sessions.js file
app.sessions = {}; // ... etc.
