There is the code what you need to do with your accordion element:

$(function() {
   $( "#accordion" ).accordion({
      collapsible: true,
      active: false
   });
});

With collapsible: true (documentation) we set your accordion to allow us to close sections, and with active: false (documentation) we said to accordion "We don't want any section open at the beginning"
