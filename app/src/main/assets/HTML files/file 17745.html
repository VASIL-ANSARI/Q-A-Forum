This would be a complete reflection. If you are going for a more realistic way you would like to do something like this (Pseudocode):

  1. Check if ball and player collide
  2. Determine if it is colliding on top/bottom, left/right, or exact corner
    1. left/right : ball.Velocity.X*= -1;
    2. top/bottom : ball.Velocity.Y *= -1;
    3. corner : ball.Velocity.X *= -1; && ball.Velocity.Y *= -1;
