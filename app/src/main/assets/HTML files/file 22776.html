You can do this easily with lxml module with a xpath expression :

from lxml import etree

file = "file://path/to/file.xml"
doc = etree.parse(file)

print doc.xpath('//cat[id=2]/name/text()')[0]

See http://lxml.de/

and a Xpath tutorial
