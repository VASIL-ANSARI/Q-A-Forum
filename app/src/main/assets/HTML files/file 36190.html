The message fatal error: windows.h: No such file or directory means that your C compiler cannot find your Windows SDK, and in particular the windows.h header file.

If you have that header file, you need to fix your C compiler's configuration.

If not, you need to obtain the Windows SDK.

This might help:

http://www.microsoft.com/en-us/download/details.aspx?id=8279

See also http://msdn.microsoft.com/en-us/library/windows/desktop/aa383745(v=vs.85).aspx

windows.h is in several places in Debian-derived GNU/Linux distributions. One such place is the mingw-w64-i686-dev package, but I don't know which package, if any, you need to run green_shoes video.
