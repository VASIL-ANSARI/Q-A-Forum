This is to do with the way Auth component works, if the "unauthorizedRedirect" property isn't set then it seems to redirect to "/{app-directory" which is your domain - hence why you end up at "yourDomain/youDomain".

Easiest way to fix it, in your Auth component in AppController add

public $componens = array(
    //other components
    'Auth' => array(
        //other properties
        'unauthorizedRedirect' => 'url to redirect to' //e.g. '/'
    )
);
