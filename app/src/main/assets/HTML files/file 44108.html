I'm not really sure what you're trying to accomplish in the end, but file_exists() generally checks an absolute path in the root filesystem of the machine the webserver is running on, while you seem to try to check a path relative to the root directoty of the webserver (which is a subdirectory of the root filesystem).

You can access this subdirectory in PHP using the predefined variable $_SERVER['DOCUMENT_ROOT'].

So something like

if (!file_exists($_SERVER['DOCUMENT_ROOT'] . "/featured/JksCreations" . $JKSP)) {
    $JKSP = '/Pages/JKS-404.php';
}

might work for what you try to do, but it might also still fail due to missing permissions.

Also, be careful what you do with your result, especially in an untrusted environment. You should check that the parameter p only consists of valid characters for a filename (especially no dots and slashes).
