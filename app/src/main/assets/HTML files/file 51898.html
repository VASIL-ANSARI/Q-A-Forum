There are at least one mistake, but I can't assure that's the root of the problem.

To pass a callback function pass a delegate, not a void*:

delegate void PSIGN_FUNC(int* buf, int rev, void* pParam);

Then you declare the dll call as this:

[DllImport(_dllLocation)]
public static extern byte uSign300_AddPointHandle(PSIGN_FUNC func, void* pParam);
