You could get the innerHTML and change that

document.getElementById("button_unlock").innerHTML ="<img src='unlock.jpg'/>" ;

But better would be to add a css class to your button and define a background-image. Then when clicking it change the class to a class with another image.

.lock {
    background-image: url(lock.jpg);
    }

.unlock {
    background-image: url(unlock.jpg);
    }


<button id="button_unlock" type="button" onClick="unlock(this);" class="lock"></button>
