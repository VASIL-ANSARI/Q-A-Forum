Edit:

I saw from comments that your code is inside a p with style display:none. This won't work, since the entire block will be hidden/not displayed, including any elements inside. Skip the display:none in the wrapper (the outer p)!

<p class="txt_h"><!-- code here is fine --></p>

As an alternative, you can toggle the visibility of the outer p when an item is clicked. See snippet below for that.

$('#showthings').click(function() {

  $('.txt_h').show();

});



$("#img1").on('click', function() {

  $("#div1").fadeIn();

  $("#div2,#div3").fadeOut();

});

$("#img2").on('click', function() {

  $("#div2").fadeIn();

  $("#div1,#div3").fadeOut();

});

$("#img3").on('click', function() {

  $("#div3").fadeIn();

  $("#div1,#div2").fadeOut();

});
