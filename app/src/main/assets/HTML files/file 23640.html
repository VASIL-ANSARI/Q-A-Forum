If your case is limited to unique character the following will allow you to get the previous or next character:

def next_char(c)
  (c.chr.ord + 1).chr
end

def prev_char(c)
  (c.chr.ord - 1). chr
end

puts next_char('A')
puts prev_char('B')

However, as you deal with grades, I'd go for a kind a Enumeration of grade. For instance, take a look at that blog post for possible implementation.
