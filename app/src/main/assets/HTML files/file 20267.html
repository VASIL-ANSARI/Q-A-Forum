On OS X, the symbols are prefixed with an underscore.

call        printf

and

call        scanf

need to be

call        _printf

and

call        _scanf

respectively; also

global calcAndPrint

and

calcAndPrint:

should read

global _calcAndPrint

and

_calcAndPrint:

instead.

(But hey, you could have deduced this from the fact that your calcAndPrint() function was symbolicated into _calcAndPrint).
