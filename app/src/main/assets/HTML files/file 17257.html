It sounds like you may be using jQuery 1.8.0.

jQuery 1.8.0 introduced a bug that caused the ready method to fire too early in IE9.

The bug was fixed in 1.8.1, and is still fixed in 1.8.2. Upgrading to either of those two versions should fix that problem if you are currently using 1.8.0.

The bug was mentioned in the release notes for 1.8.1 http://blog.jquery.com/2012/08/30/jquery-1-8-1-released/

Here's the ticket: http://bugs.jquery.com/ticket/12282
