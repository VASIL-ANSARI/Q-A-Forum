Here i did it simply by writing 2 line coding in model of given controller .I have found this the best way to do it using "Virtualfield" concept in cakephp.

   var $virtualFields = array(
    'review_count' => 'SELECT COUNT(*) FROM reviews as Review WHERE Review.venue_id = Venue.id'
);
