Google Apps Script is a web based script which is very akin to Java Script. With GAS, you can of course create Google Docs, which Google handles as blobs.

You can do something like this:

// Open a document by ID.
var doc = DocumentApp.openById('DOCUMENT_ID_GOES_HERE');

// Create and open a document.
doc = DocumentApp.create('Document Name');

You can also programatically edit documents. Please refer to : Document Service
