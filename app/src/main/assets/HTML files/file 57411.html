Unity does not use or support .Net framework v4.6.01038.

This problem can easily be solved if you use Visual Studio that comes with Unity.

1.Uninstall the current Visual Studio.

2.Uninstall Unity and restart computer.

3.Install Unity. Make sure to select Microsoft Visual Studio Community 2015 when installing Unity. That will automatically install Unity and Visual Studio and every other thing required to make them work together. You won't lose your project if you uninstall Unity or VS.
