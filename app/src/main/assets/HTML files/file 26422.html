That's because your binary string has (unintended) invalid UTF-8 inside; to make it valid, you can encode the string using utf8_encode():

$arr = array('password' => 1, 'salt' => utf8_encode($salt));

Later you will have to perform the operation in reverse:

$options = json_decode('...', true);
$salt = utf8_decode($options['salt']);

If you don't feel comfortable with such binary treatment, you could also perform a textual encoding like base64_encode() which you can later decode using base64_decode():

$arr = array('password' => 1, 'salt' => base64_encode($salt));
