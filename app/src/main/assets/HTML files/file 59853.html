It looks like your destroy.js.erb file is wrong. You are attaching an event handler when you just need to simply remove the line. Change your file to:

$('div#item_<%= @item.id %>').fadeOut();

Now, you could also make a check to see if the item truly has been deleted:

<% if @item.destroyed? %>
  $('div#item_<%= @item.id %>').fadeOut();
<% else %>
  #Do something
<% end %>
