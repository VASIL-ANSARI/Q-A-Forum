Using .width() will return px, so you would need to calculate its width somehow. One way is to check it against its parent element.

Something like this could do the trick:

$.fn.Percentage = function(){
    var parent = this.parent();
    return ~~((this.width()/parent.width())*100)+"%";
}

and then call it using:

if ($("#progress_bar").Percentage() == "100%" ){
 $('#black_div').fadeOut('fast'); 
}
