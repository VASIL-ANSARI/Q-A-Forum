Due to the new design changes introduced in Android 5.0, you need to use to the notifications only white icons. You can read more about it here: http://developer.android.com/intl/ko/design/patterns/notifications.html#guidelines

The system will automatically convert your icons to white. My suggestion is to you create white notification icons.

You can find more information about it here: http://stackoverflow.com/a/29207365/3482000
