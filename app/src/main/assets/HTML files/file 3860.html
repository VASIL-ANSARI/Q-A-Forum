Assuming I'm reading this correctly, you can replace your compatible method with a call to the Collections.disjoint static method.

Edit: Code sample:

public void addEdges(){
  for(int i =0; i < size()-1; i++){
   for(int j = i+1; j < size(); j++){
    if(Collections.disjoint(get(i),get(j))){
     get(i).linkTo(get(j));
     get(j).linkTo(get(i));
    }
   }
  }
 }
