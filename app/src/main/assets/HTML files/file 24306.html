If you make some search on php.net, you can easily find an example that will make what you need. YOu can also use my advice on comments of your question.

<?php

$src = fopen('http://www.example.com', 'r');
$dest1 = fopen('first1k.txt', 'w');
$dest2 = fopen('remainder.txt', 'w');

echo stream_copy_to_stream($src, $dest1, 1024) . " bytes copied to first1k.txt\n";
echo stream_copy_to_stream($src, $dest2) . " bytes copied to remainder.txt\n";

?>

But, depending on your PHP version, it seems to be quite a memory hog. The way with fopen, fread and fwrite can then be

<?php

    function customCopy($in, $out)
    {
        $size = 0;

        while (!feof($in))
            $size += fwrite($out, fread($in,8192));

        return $size;
    }

?>

Assuming $in and $out are file handler resources.
