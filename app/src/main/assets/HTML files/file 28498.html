You forgot to add source code of ADTbag.h (as I understand, both your source codes are the same). But basic theory is very simple:

  1. Put all three files (main.cpp, ADTbag.h and ADTbag.cpp) in one directory,
  2. Add line #include "ADTbag.h" in ADTbag.cpp and add implementation of all functions from (ADTbag.h) in ADTbag.cpp.

So you ADTbag.cpp will be like this:

#include "ADTbag.h"

ArrayBag::ArrayBag()
{
  size = 0; // initialize empty list
}

bool ArrayBag::isEmpty() const
{
  return (size == 0); // return true if list is empty
}

int ArrayBag::getCurrentSize() const
{
  return size; // return size of the list
}
