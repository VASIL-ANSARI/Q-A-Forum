you need to use a tag inside your button tag in order to send the data to your servlet, like this:

<body>
  <div>Click button to show message.</div>
  <p></p>
  <button type="button"><a href="helloworldservlet.do">Go to Servlet</a>
  </button>
</body>

Note that I am using the .do domain and lowercase, you can use @WebServlet("/helloworldservlet.do") intead of @WebServlet(urlPatterns = {"/helloworldservlet.do"}).
