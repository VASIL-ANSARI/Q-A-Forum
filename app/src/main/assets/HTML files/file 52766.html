Looks like you are trying to log in with basic auth credentials.

Your webpage is expecting form data in the form:

curl --data "param1=value1&param2=value2" http://example.com/

So in your case you should provide something similar to:

curl --data "username=aUserName&password=thePassword" http://example.com/index.asp
