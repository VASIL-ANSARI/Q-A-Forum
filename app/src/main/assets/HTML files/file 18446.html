$des was set to the next record of the array and you then while on it. This means it will always be true as the array has contents, which causes an infinite loop as you thought. The $des = mysql_fetch_array($result) will not get called again to move on to the next result.

Try removing the first $des = mysql_fetch_array($result) and replacing while($des) with:

while($des = mysql_fetch_array($result))
