How about something roughly like this:

class Dialogs {
    public static Dialogs INSTANCE = new Dialogs();

    public int confirm(String message) {
        return JOptionPane.showConfirmDialog(null, message);
    }
}

Then call Dialogs.INSTANCE.confirm() everywhere you would call JOptionPane.showConfirmDialog(). For testing, set Dialogs.INSTANCE to a mocked value.
