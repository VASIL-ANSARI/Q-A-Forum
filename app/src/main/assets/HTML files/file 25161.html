See documentation.
Class SoapFault is well documented.
There are many methods.
To access trace, use getTrace() method.

http://php.net/manual/en/class.soapfault.php

Try :

try {
  WebServices::create($this->nameWS);
}
catch (Exception $e) {                      
  $tr = $e->getTrace();
  var_dump($tr[3]);     
}
