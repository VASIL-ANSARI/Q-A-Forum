if you want to access the query params use URI and CGI libraries

url    = 'http://www.foo.com?id=4&empid=6'
uri    = URI.parse(url)
params = CGI.parse(uri.query)
# params is now {"id"=>["4"], "empid"=>["6"]}

id     = params['id'].first
# id is now "4"

That said I think this is a dirty solution to the problem you have but it is what you asked for.
