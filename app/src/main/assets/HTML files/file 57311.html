You need to evaluate the IEnumerable that is returned by:

eAuthor.Papers.Select(p => p.PaperID)

The string formatter is going to call ToString on the IEnumerable and that's why you're getting:

System.Linq.Enumerable+WhereSelectListIterator`2[DirichletProcessClustering.GraphData.Paper,System.Int32]

You could add an additional foreach statement and enumerate the above IEnumerable to write the paper IDs and years for the author like the following:

foreach (EvoObject eEvoObject in curCluster.GetClusterObjects())
{
      Author eAuthor = (Author)eEvoObject.GetOriginalObject();

      twObjClus.WriteLine(
          @"AuthorID: {0}, AuthorName: {1} \n",
           eAuthor.AuthorID,
           eAuthor.AuthorName);

       foreach (var paper in eAuthor.Papers)
       {
           twObjClus.WriteLine(
             @"PaperID: {0}, Year: {1}", paper.PaperID, paper.Year);
       }
}
