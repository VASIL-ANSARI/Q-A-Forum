use if condition in required. for example,

$("#form").validate({
        rules: {
            text1: {
                required: function (element) {
                    if ($("#selid").val() == '1') {
                        return true;
                    } else {
                        return false;
                    }
                }
            },
            text2: {
                required: function (element) {
                    if ($("#selid").val() == '2') {
                        return true;
                    } else {
                        return false;
                    }
                }
            }
        }
    });

consider #selid is id of your select control. If select value '1' only it will reflect text1 will be required true. other wise, it will required false. In the same way If select value '2' only it will reflect text2 will be required true. other wise, it will required false.

like this you can apply to your code...
