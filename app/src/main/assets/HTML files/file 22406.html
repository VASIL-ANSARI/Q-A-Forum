In the beginning of operator+, add:

if (amount <0) {
  operator-(-amount);
  return;
}

and similarly in operator- add: 

if (amount <0) {
  operator+(-amount);
  return;
}

Edit: by the way, be very careful about typos like:

if(amount = (-amount))

which assigns -amount to amount, then tests if amount is equal to zero!
