I managed to get it working. It's all about Google Tag settings. There is a trigger which fires when you send an event like this

dataLayer.push({
    'event':'GAevent', 
    'eventCategory':'Ad', 
    'eventAction':'click', 
    'eventLabel':'FJbanner'
});

trigger expression

Event contain GAevent

so my error was that I didn't set it up correctly. You should specify an event name in a trigger settings according to Event variable value, 'GAevent' in my case
