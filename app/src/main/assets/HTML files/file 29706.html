one way would be to animate the margin by the required number of pixels

$("element").animate({marginLeft : "-20px"}, 500);

NOTE: "slide" will generally be for the complete width/height of the element. You can try to set the distance parameter, but it will not give you the effect that you want.

     <script>
      $("#hide").click(function(){
         $(".target").hide( "slide", 
                     { direction: "down" ,
                       distance: "200px" }, 2000 );
      });

      $("#show").click(function(){
         $(".target").show( "slide", 
                      {direction: "up" }, 2000 );
      });



      });
       </script>
       <style>
          p {
               background-color:#bca;
               width:200px; 
               border:1px solid green; 
            }
         div{ width:100px; 
                height:100px; 
                background:red;  
            }
      </style>
    </head>
    <body>

   <p>Click on any of the buttons</p>
   <button id="hide"> Hide </button>
   <button id="show"> Show</button> 

   <div class="target">
   </div>
