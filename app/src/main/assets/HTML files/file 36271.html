However, note that in a RelativeLayout, the children are laid out in top left corner by default, overlapping each other unless told otherwise.

To make the RelativeLayout children lay out relative to each other, add some layout rules. For example, to get similar results as in a vertical LinearLayout, use

android:layout_below="@id/id_of_previous_child_view"

attribute in RelativeLayout child views.
