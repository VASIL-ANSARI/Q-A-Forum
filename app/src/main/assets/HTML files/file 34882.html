If you are using a grid you would probably have to put each image in a new "cell" or section of the grid. To do so you would add grid column and then put the images in their respective grids by using grid.column="#"

example:

<Grid>
    <Grid.ColumnDefinitions>
        <ColumnDefinition Width="100" />
        <ColumnDefinition Width="Auto" />
        <ColumnDefinition Width="*" />
        <ColumnDefinition Width="5*" />
    </Grid.ColumnDefinitions>
    <Image grid.column="0" />
    <Image grid.column="1" />
    <Image grid.column="2" />
</Grid>

I would probably recomend using a stack panel instead though. You can make the orientation horizontal and then all elements in the stack panel will show up in a horizontal orientation.

example:

<StackPanel Orientation="Horizontal" Height="100" Margin="0,0" VerticalAlignment="Top"     Width="100">
     <Image />
     <Image />
     <Image />
</StackPanel>

On second thought a Wrap panel may be better for your phone application as it will display the images horizontally and then start a new line once the first line is full. It works almost exactly like the Stack Panel.

example:

<WrapPanel Orientation="Horizontal" Height="100" Margin="0,0" VerticalAlignment="Top"     Width="100">
     <Image />
     <Image />
     <Image />
</WrapPanel>

I hope that helps!
