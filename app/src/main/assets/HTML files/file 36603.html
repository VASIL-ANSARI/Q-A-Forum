Some advices if you are really new to magento :

  1. Create a module to override core items (there are plenty of tutos over the net my prefered is the inchoo's one)
  2. The architecture of adminhtml is specific and you'd better have a close look at it to moke it
  3. The cache and compilation are strong enemies in module development so deactivate them if is not yet done
