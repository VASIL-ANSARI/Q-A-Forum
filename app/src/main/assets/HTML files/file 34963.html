Try using [[_prescribingDoctorTableView superview] setNeedsLayout] instead of [_prescribingDoctorTableView setNeedsDisplay] to layout the subviews.

if this doesn't work, subclass your view and override the methods,

- (void)layoutSubviews (or) - (void)setFrame:(CGRect)frame

And debug by placing the break point in these methods so that you can come to know from where your frame being updated.
