.htaccess should be able to do the trick have a look at: https://httpd.apache.org/docs/current/mod/mod_rewrite.html#rewritecond. It would require checking the browser agent unless you have a mobile specific url setup. It would be done something like this.

...
RewriteCond %{HTTP_USER_AGENT} *user agent to redirect*
RewriteCond %{URL} *url to redirect from*
RewriteRule specificpage general page
...

Both the user agent and url are considered regexp. I don't know any other way besides javascript on the page. Also the answer to this question may be helpful.
