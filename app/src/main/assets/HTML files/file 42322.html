In your worker class, implement Comparable interface like:

public class Worker implements Cloneable, Comparable<Worker> {
....
    public int compareTo(Worker worker) {
        return Double.compare(this.salary, worker.getSalary());
    }
}

And then assuming you have list of workers already, you could sort like:

Collections.sort(listOfWorkers);

clone has nothing to do with sorting the collection or array but you could implement multiple interfaces.
