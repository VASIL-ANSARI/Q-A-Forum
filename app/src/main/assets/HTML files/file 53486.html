How about

^.*?(\b\w+\b)(.*?\1).*?$

It matches a word (captures it), and the matches anything up to that same word (\1 is a back reference to the first capture).

The result is two capture groups - first holds the actual word, and second - the rest up to and including the second word.

Edit:

If you want to make sure there's only one word in between, and (as I assume) are interested in only the two words, this is an alternative solution:

^.*?(\b\w+\b)[.,]\s*(\w+)\s*\1.*?$

It captures the first word in the same way, matches a , or a . followed by a word which in turn is followed by the first. It only captures the two words.
