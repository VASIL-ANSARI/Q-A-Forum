It sounds like you are wanting the table view cells to reload to the appropriate edit/non-edit version of the cells anytime the user enters or leaves edit mode. If I understand that correctly, then you could provide a setter for the editing property to reload the table view cells anytime edit mode is change.

override func setEditing(editing: Bool, animated: Bool) {
    tableView.reloadData()

    self.tableView.beginUpdates()
    super.setEditing(editing, animated: animated)
    self.tableView.endUpdates() 
}
