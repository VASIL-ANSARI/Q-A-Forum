I don't know if this counts as "better/ simpler", but you can write it as a regular SQL query:

SELECT MIN(sequence)
  FROM ( SELECT t1.sequence
           FROM table_name t1
           JOIN table_name t2
             ON t2.category = t1.category
            AND t2.sequence <= t1.sequence
          WHERE t1.category = ...
          GROUP
             BY t1.sequence
         HAVING SUM(t2.value) <= 0
       )
;
