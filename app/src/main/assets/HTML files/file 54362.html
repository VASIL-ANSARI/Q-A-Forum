You can add any view inside a fragment, just like you would do in an activity. For what I can tell, that button is inside the activity layout. So:

  • Go To com.example.android.fragments.ArticleFragment
  • Go to its method onCreateView (ctrl+F to look for it)
  • It should have a return statement, something like inflater.inflate(R.layout.fragment_layout, container, false);

At this point, you should understand that this method returns the Fragment's layout.

  • Go to the layout specified in that line (in this case, it would be R.layout.fragment_layout).
  • Put in that layout the button, run the app and you should see it.
