The value of norecursedirs is a glob pattern and to make it match on the full path your pattern must include a / character. Unfortunately the full path this is matched against is an absolute path so if you only want the base Tools you will have to include the parent directory name:

$ find root/
root/
root/Tools
root/Tools/test_foo.py
root/Tools/__init__.py
root/setup.cfg
root/ook
root/ook/Tools
root/ook/Tools/test_foo.py
root/ook/__init__.py
$ cat root/setup.cfg
[pytest]
norecursedirs = root/Tools
$ py.test --collectonly root/
