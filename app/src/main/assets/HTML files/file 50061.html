You can set column names by df_B.columns = ['Col_B', 'Col_A']. Then you can merge dataframes.

print df_A
   Col_A  Col_B
0     23     34
1     12     18

print df_B
    0   1
0  23  12
1  45  10

df_B.columns = ['Col_A', 'Col_B']
print df_B
   Col_A  Col_B
0     23     12
1     45     10

out = df_A.merge(df_B, on='Col_A')
print out
   Col_A  Col_B_x  Col_B_y
0     23       34       12
