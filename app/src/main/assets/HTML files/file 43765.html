Here is how I bring in my prebuilt *.so into my apps, just make sure your *.so are in the src/main/libs folder:

android {
    ...
    sourceSets {
        //noinspection GroovyAssignabilityCheck
        main {
            jni {
                srcDirs = []
            }

            jniLibs {
                srcDir 'src/main/libs'
            }
        }
    }
}
