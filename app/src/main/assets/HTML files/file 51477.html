An element's direct text nodes can have only one color, but its children can have different colors.

Add a div with the desired color each time you update results:

if (current > previous) {
  color= 'green';
}
else if (current == previous) {
  color= 'black';
}
else {
  color= 'red';
}

document.getElementById("results").innerHTML += 
  '<div style="color: '+color+'">' +current + " </div>";
