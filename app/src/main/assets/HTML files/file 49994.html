I think your core problem here is that you are trying to get the text contained in the ListViewItem, but you are calling ToString() which gets you nothing but the 'string representation' which in your case is System.Windows.Controls.ListViewItem: Joe

Depending on whether you want the name or value associated with the ListViewItem, you would use different methods.

Method 1

To get the name of a ListViewItem use the .Name property, e.g.:

ListViewItem i = new ListViewItem();
i.Name;

Method 2

If you want to get text value within the ListViewItem use the .Text property, e.g.:

ListViewItem i = new ListViewItem();
i.Text;
