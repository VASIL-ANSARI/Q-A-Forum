This is one way to accomplish it:

string number = "25"; //Assign the string to a variable
var x = Int32.Parse(number[0].ToString()); //Call Int32.Parse() on the first element of the array to convert to an int. Notice that Int32.Parse() only works on strings, so we convert the char back to a string
var y = Convert.ToInt32(number[1].ToString()); //This is a different way to convert, and works on various types.

This will result in x == 2 and y == 5.
