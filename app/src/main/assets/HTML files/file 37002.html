The query is wrong:

SELECT * FROM table 
WHERE timest >= :low OR timest = 0
ORDER BY id DESC, parent, cat

The query base "format"/grammar is :

select [...]
[from ... [inner | left | right ] join ... [on ... | using ...]]
where ...
group by ...
having ...
order by ...
limit ...

Except for select, all parts are optional.
