easiest way to redirect using htaccess

# permanently redirect specific IP request for single page RewriteEngine On RewriteBase / RewriteCond %{REMOTE_HOST} 123\.456\.789 RewriteCond %{REQUEST_URI} /requested-page\.html$ RewriteRule .* /just-for-you.html [R=301,L]
