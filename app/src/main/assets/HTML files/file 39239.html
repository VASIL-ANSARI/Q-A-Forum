Check if your XML does not contain zero sized non-breakable space UTF-8 symbol (also called utf encoding signature) with the code xFEFF; This symbol in the attribute value leads to the iOS8 parser string deallocation error and crash. To get rid of this string in PHP backend use:

$char = mb_convert_encoding('&#xFEFF;', 'UTF-8', 'HTML-ENTITIES');
$xml = str_replace(array($char, '&#FEFF;'),'',$xml);
