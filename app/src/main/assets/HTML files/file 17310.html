Generating schema from Hibernate is actually built into Hibernate itself. I'm not sure if there is a GUI for this, but all you need to do is just add a line to your configuration file. Start by add a configuration value to whatever you are using to your Hibernate configuration. For example:

hibernate.hbm2ddl.auto=create

Other configuration values include update, validate, and create-drop.

You can customize the generated schema quite a bit.
