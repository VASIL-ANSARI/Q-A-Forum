Make sure that you deploy the program correctly copy the needed .dlls from bin folder from Qt Folder and copy all folders with all the files(.dlls) from plugins folder

My paths to folders for example:

C:\Qt\Qt5.4.0\5.4\msvc2012_opengl\bin (copy needed needed dlls from bin)

C:\Qt\Qt5.4.0\5.4\msvc2012_opengl\plugins (copy everything from plugins)

After you paste the files run you program and try to delete all the files the unnecessary files will be deleted.
