This is actually a little tricky, since I assume you're asking for the scroll view to only include that portion of the web view, correct? If you just want to start at that place, then look at webview.scrollview.contentOffset.

To do what you're talking about, you'd have to take a picture of the webview, and display that. Of course this would invalidate any links, javascript, or other interactive features.

If you needed those things back, you could handle touch events and forward them to the real webview, as long as the page isn't too dynamic.

Personally, I'd probably modify the HTML itself to display the content you want to display rather than trying to snip out a portion of it.

EDIT: As I think about it, setting webView.scrollView.contentOffset actually might give you exactly what you want, if you also disable scrolling using webView.scrollview.scrollEnabled = NO.
