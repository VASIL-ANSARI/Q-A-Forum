You'd have to use the getResourceAsStream() and write the contents to a known location. You could use something like the following to copy the template in your deployment to a temporary file.

final InputStream in = Thread.currentThread().getContextClassLoader().getResourceAsStream(resourceFile);
final Path f = Files.createTempFile("template", ".txt");
Files.copy(in, f);
final String absoluteFileName = f.toAbsolutePath().toString();

If you have a set of known templates you could use a @javax.ejb.Singleton @javax.ejb.Startup EJB.
