Solved in a non-fashion way.

Basically, the DateRangePicker plugin creates a div which contains an list with Today, Yesterday, etc. 

<div class="ranges">
    <ul>
        <li>Today</li>
        <li>Yesterday</li>
    </ul>
</div>

So after the plugin has done being initialized I change the textof the li manually.

$('.ranges ul li').eq(0).text(lang[0]);
$('.ranges ul li').eq(1).text(lang[1]);
