Using the .val() function on a multi-select list will return an array of the selected values:

var selectedValues = $('#multipleSelect').val();

After that you can loop through this array and create multiple images like this and add every image to a specific point in the DOM:

var img = $('<img />',
             { id: 'ImageId',
               src: 'Image.png', 
               width: 300
             })
              .appendTo($('#YourDiv'));
