Voice Control

Description

It's the feature through which user can give voice commands to navigate through the app.
here is an overview of how it works :
you can say "search on basis of [ any] language" which will filter up questions on based on that language .(refer to tags section to know more...)
also you can sort questions based on title,body,date, time,votes count,answers count by simply giving voice command to "sort by title" or so
also you can navigate to app settings,tags,users by giving voice commands as " go to settings or show all users or show all tags"
also you can see your own posted questions by saying "show my questions" or simply "my questions"
and can refresh things up by just saying "reset or reset all "
as well as can post questions by just commanding "i want post or add a new question"
now in app settings also you have this feature in which you can try such commands
"turn on notifications"
"turn off notifications"
"log me in or sign me in using google account"
"log me in or sign me in using github account"
"sign me out or log out"
"i want to give some rating to this app"

so enjoy this feature now...

Note : you have to tap the icon for sometime and then give voice commands.
Sometimes commands may not work as needed but don't worry it will work correctly in 90% of cases.
