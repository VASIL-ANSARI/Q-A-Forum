Why can't you make an immutable struct? Consider the following:

public struct Coordinate
{
    public int X
    {
        get { return _x; }
    }
    private int _x;

    public int Y
    {
        get { return _y; }
    }
    private int _y;

    public Coordinate(int x, int y)
    {
        _x = x;
        _y = y;
    }
}

It's an immutable value type.
