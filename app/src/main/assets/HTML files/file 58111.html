I don't think there's a built-in way to do this, but you could use usort to sort it according to your preferred order:

usort($array, function($a, $b) {
    $order = ['white', 'black', 'red', 'orange', 'yellow', 'green'];
    $k1 = array_search($a, $order);
    $k2 = array_search($b, $order);
    return $k1 < $k2 ? -1 : +1;
});
