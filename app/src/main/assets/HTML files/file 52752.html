It makes much more sense to store your board in a 2-dimensional list:

with open(filename, 'r') as f:
    board = [list(line) for line in f.read().splitlines()]

for row in board:
    print(row)

Output:

['A', 'B', 'C']
['C', 'A', 'B']
['B', 'C', 'A']

It is then possible to pick a value based on its coordinates:

board[0][0] # top left 'A'
board[2][2] # bottom right 'A'
