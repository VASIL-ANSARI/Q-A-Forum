Based on your comments, error is in this line:

Bitmap coverImage = BitmapFactory.decodeStream(book.getCoverImage().getInputStream());

What is being thrown is NullPointerException, which means that:

  1. book is null
  2. book.getCoverImage() returns null
  3. book.getCoverImage().getInputStream() returns null

You have to check which one of those is happening (use Log.d() and simply print out those values, or use debugger).

Possible reasons I can think of:

  1. book could be null on example because "sample.epub" is missing, or it's format is invalid and EpubReader fails in decoding it.
  2. Book does not have cover image?
