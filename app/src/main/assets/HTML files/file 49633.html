Position your circle absolutely and then use the following code for an animated effect

$("#blinkandresize").fadeOut(300).fadeIn(300).fadeOut(300).fadeIn(300).animate({
    'width': '+=400px',
    'left': '-=200px',
    'top': '-=200px',
    'height': '+=400px'
  }, 600);

CSS:

#blinkandresize {
    position:absolute;
    top: 250px;
    border-radius: 1000px;
    left: 250px;
    border-radius: 50%;
    background-color: #334;
    height: 50px;
    width: 50px;
}
