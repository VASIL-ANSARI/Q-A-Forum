You can' bind it because $index is not defined for ngOptions, it's only ngRepeat thing. If you want to know the index of the option you can calculate it with Array.prototype.indexOf method:

<select ng-model="testModel" 
        ng-options="labels.indexOf(label) as label for label in labels">
</select>
