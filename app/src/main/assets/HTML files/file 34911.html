Installing with source

Go to this link:https://raw.github.com/pypa/pip/master/contrib/get-pip.py

and save it as get-pip.py(right click on page),and open your cmd as admin and go to the file path and then enter(you need internet) the following command,

python get-pip.py install

You can refer the link for installing on different operating systems

http://www.pip-installer.org/en/latest/installing.html

Or you can install with exe file,

click here to get the file,and install it and you need to set path ot pip directory,you will find a source folder in c drive and you can find the pip file set the folder path.
