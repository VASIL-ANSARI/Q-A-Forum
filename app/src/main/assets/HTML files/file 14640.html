Using trigger('create') should do the trick.

function form(){ 
    data = '<form action="index.html" method="post"><div data-role="fieldcontain"><label      for="username">Username<label><input type="text" name="username" id="username"></div></form>';
    $('#textcontent').html(data);
    $('.ui-page').trigger('create');
    $('#textcontent').show();

}

A couple of other things that may help you would be

  1. You have an html error in the form. Your div must close before your form does. i.e. </div></form> is how it should be. Also a <li> element should be wrapped in either a <ul> or an <ol>.
  2. Get rid of data + in your from variable. I'm not sure what that is intended for.
  3. Just put the form inside a div that is set to display:none. That way the trigger('create') method will be unnecessary. Then you can just show() or toggle() the visibility.
