To solve this problem, you need to figure out when the last quarter was. Then it's as easy as going back from there. Here's how I would do it:

$minute = idate('i');
$qdiff = $minute % 15;
$lastQuarter = time() - $qdiff*60;
echo date('Y-m-d H:i', $lastQuarter);
