You have two options

Option 1 -- Ask for length ahead of time

I don't think this option requires much explanation. It would look something like this

Scanner in = new Scanner(System.in);
int length = in.nextInt();
//... use "length" as you wish

Option 2 -- Use an ArrayList!

Your code would then look a little something like this:

int inputNum = in.nextInt();
ArrayList<Integer> a = new ArrayList<Integer>();
while (inputNum != 1){
    a.add(inputNum);
}
//you can use "a" as you wish now
