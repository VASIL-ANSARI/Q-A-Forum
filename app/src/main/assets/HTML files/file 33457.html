Firstly, are you sure your SQL query works? Usually SQL likes you to use ' ' around table names with space in them. In general bad idea to have spaces in table and field names. Use underscore or something like TabelName.

Now on to your PHP code: In PHP when using SQL you should use the field/variable names as indexes. So instead of writing:

while ($row = mysql_fetch_array($result)) {
 echo "<h2>";
 echo $row[1]."";
 echo "</h2>";
 echo "<p>";
 echo $row[2]."";
 echo "</p>";
 }

You can write:

while($row = mysql_fetch_array($result)){
 echo "<h2>";
 echo $row['MPN']."";
 echo "</h2>";
 echo "<p>";
 echo $row['Model']."";
 echo "</p>";
}

I assume your field names are 'MPN' and 'Model' in your database. To make your code more concise you can write: 

while($row = mysql_fetch_array($result)){
     echo "<h2>".$row['MPN'].".</h2> <p>".$row['Model']."</p>";
    }

However, got with what you find is easier to read. Hope this helps.
