You can't because you are using dynamic memory allocation; meaning the elements will not be stored in a continuous memory location.

But maybe that is not what you want....notice that table[10] and &table[10] are different things.

Since you are using pointer to pointer you can use offsets to get the index of any &tab[index] element. They are the columns in your 2D array, allocated by table = new tab*[n+1], stored in an continuous memory location. Each of these locations holds a tab*, so rows of your matrix.
