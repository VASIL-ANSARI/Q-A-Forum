You are getting :

Parse error: syntax error, unexpected T_FUNCTION in [...][...] on line 4

becasue you are using PHP version less than PHP 5.3.0 upgrade your php or try using globals

PHP 5.3.0+

$app = "Hello Wolrd";
$authCheck = function () use($app) {
    echo $app;
};
$authCheck();

Below PHP 5.3.0

$app = "Hello Wolrd";
$authCheck = function () {
    global $app;
    echo $app;
};
$authCheck();

Both would output 

  Hello Wolrd
