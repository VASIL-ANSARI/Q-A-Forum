Replace the following line:

home_link = driver.find_element_by_partial_link_text('##')

with

home_link = driver.find_element_by_link_text('home')

More info here: http://selenium-python.readthedocs.org/en/latest/locating-elements.html#locating-hyperlinks-by-link-text

EDIT: please note that this will return one element or throw NoSuchElementException. So you shouldn't iterate through the results, it doesn't make sense.
