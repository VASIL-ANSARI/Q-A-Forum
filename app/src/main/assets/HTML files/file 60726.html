You are getting 404 because /Rest is no where registered in any paths

  1. Your context path is /Example
  2. Then you controller is mapped to /controller

So You don't have a mapping to resolve /Rest since your controller is is bound to /controllerURI path

Either remove the /Rest for your URL and update the servlet mapping in web.xml

<servlet-mapping>
 <servlet-name>Example</servlet-name>
 <url-pattern>/*</url-pattern>
</servlet-mapping>

OR

Add the /Rest to your controller path mapping

@Path("/Rest/controller")
public class Controller {
