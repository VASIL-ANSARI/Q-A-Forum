I have mentioned that the browsers other than Internet explorer 11 are all working fine. To solve this problem we have to extract the register value of Internet explorer 11 "C:\Program Files\Internet Explorer\iexplore.exe\"%1"

eliminate %1 and get C:\Program Files\Internet Explorer\iexplore.exe

This can be done by changing the 'extractIECmd()' function as

protected static String extractIECmd(String qq) { 

  String browser_cmd = "";

  *int index = qq.toString().lastIndexOf ("");*
  if (index != -1) {
      qq = qq.substring(0, index);

     if (qq.charAt(0) == '"') {
         index = qq.toLowerCase().lastIndexOf ("\""); 
          if (index != -1) {
              browser_cmd = qq.substring(1, index);


          }
      }
  }
  return browser_cmd;

}
