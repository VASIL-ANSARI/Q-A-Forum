You are not using Threads correctly. Invoking the run() method is not how to start a Thread. Also, the Thread.sleep() causes the Event Dispatch Thread to sleep so the GUI can't repaint itself until all the code in the loop has been executed.

Some suggestions:

  1. Don't keep an array or JLabels. Instead keep an Array of Icons. Then you just create a single label that you display on the GUI. For your animation you simple change the Icon of the label by using the setIcon(...) method.

  2. Don't attempt to use Threads. Instead you should use a Swing Timer for the animation. You start the Timer and every time the Timer fires you change the Icon of the label. Read the section from the Swing tutorial on How to Use Timers for a simple animation example.
