Are you getting results for the query and it's not sorting? Or are you not getting results at all?

Try this: 

@orders = Order.where("hotel_user_id = ?", cookies[:hotel_id_for_login_user]).order("#{params[:sort]} ASC").paginate(:per_page => 5, :page => params[:page])
  1. maybe your sort condition either needs to be a symbol or put in quotes
  2. The "where" part of your query didn't set a condition
  3. It's not safe to put in a user-provided variable directly into your conditions string because of potential SQL injection.

I'd recommend reading this for more info: http://guides.rubyonrails.org/active_record_querying.html
