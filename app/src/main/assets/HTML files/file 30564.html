Found a way. This may be rough but it should get you going. Set the element's height to zero and then use a percentage for the padding.

jsFiddle example

For example, 4:3:

div {
    height: 0;
    padding-bottom: 75%;
    background: #999;
}

For example, 16:9

div {
    height: 0;
    padding-bottom: 56%;
    background: #999;
}
