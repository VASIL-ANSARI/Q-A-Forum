Your best bet is going to be to use the bootstrap-sass gem, which you can include by using gem 'bootstrap-sass', '~> 2.0.4.0' in your gemfile. Then you don't need to explicitly state it in your stylesheets. Instead, use a file for your custom styles:

<%= stylesheet_link_tag "application", :media => "all" %>
<%= stylesheet_link_tag "custom", :media => "all" %>

And then in custom.css.sass, use the @import directive to include bootstrap:

@import "bootstrap"

That'll be the easiest way to include Bootstrap in your app.
