If I'm understanding your question correctly you are trying to get a count of user actions per day.

To accomplish that you are going to use a group by in conjunction with an aggregate function: count

Select Date("datetime"), "user", count("action")
From tablename 
Where Date("datetime") > Curdate()
Group By Date("datetime"), user

If you want to offset it by 12 hours you'll need to do som date math. date function docs
