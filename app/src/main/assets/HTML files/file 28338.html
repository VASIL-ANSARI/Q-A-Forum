I'm not exactly certain what it is you want. Based of the question title, if you want to find certain words within strings, you can use java.util.StringTokenizer to split the string into words:

for (java.util.StringTokenizer tokenizer = new java.util.StringTokenizer(INPUT_STRING); tokenizer.hasMoreTokens(); ) {
    String word = tokenizer.nextToken();
    //do something with the word
}

Checking to see if the string matches certain criteria can be done using String.equals(...) or String.matches(...), the second one being for regexps.
