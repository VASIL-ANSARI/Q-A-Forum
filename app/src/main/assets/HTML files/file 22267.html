OLD ANSWER

I think you are doing things backward here:

  1. LocationManager.requestLocationUpdates() can take a long time, it should be done in a background thread.
  2. Starting/finishing an activity or modifying the UI from a background thread is unsafe. You should use the UI thread for that

You can use a background thread to request for location, and use runOnUiThread to update your TextViews or start a new activity.

Important: Make sure to stop the thread in the onPause method of your activity!
