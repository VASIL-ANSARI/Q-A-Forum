If you want to change the labels on html page, you should make changes in template file(.phtml). To find the template file which renders the output, turn on template path hints.In admin panel navigate to

system->configuration->developer->->debug->template path hints

Select the store view and turn on. Now reload the page and see the path of template file which is used to render those 2 labels. If path is from your theme, you can modify those labels there, either copy same directory structure in your new theme and make modifications there.
