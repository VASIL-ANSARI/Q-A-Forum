You make the Logout text into an hyperlink that calls a php script which unsets the user_id attribute from the session or destroys it using session_destroy().

<a href= <?php  
if(isset($_SESSION['user_id'])) 
    echo"index.php?logout=true";
else 
    echo"login.php"; ?>>
<?php 
if(isset($_SESSION['user_id']))
    echo"Logout"; 
else
    echo"Login";
?>
</a>

And in index.php, on the top of the script add the folowing:

if (isset$_GET['logout']))
    session_destroy();
