ok, the thing is that jcropper will look for 'POST' values to detect where to do cropping (x, y, height and width). In order to provide them you have to send them encoded like 'somestring_x', 'somestring_y', 'somestring_h' and 'somestring_w'.

In this case 'somestring' is exactly what you put in the getCoordsFromPost method: in your case "idOrAttributeName".

If you take a look at the sourcecode on github it will be clear (it's indeed very simple).
