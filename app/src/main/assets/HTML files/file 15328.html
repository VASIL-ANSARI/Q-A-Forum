You can specify an external compiler in mocha with --compilers option

--compilers

coffee-script is no longer supported out of the box. CS and similar transpilers may be used by mapping the file extensions (for use with âwatch) and the module name. For example --compilers coffee:coffee-script.

So, create mocha.opts in your app/tests path and add :

Then every extension .tjs will be compiled automatically when starting mocha
