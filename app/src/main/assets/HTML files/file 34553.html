setListAdapter(new ArrayAdapter(getActivity(), R.layout.fragment_news_feed, rssFeed));

As the crash log shows the ArrayAdapter expects the list item to have at least one TextView to update the values in rssFeed.

You need to create a simple custom Adapter and just Override the getView() like this

@Override
public View getView(int position, View contentView , ViewGroup parent)
{
    View row = contentView;
    if (row == null){
        row = mContext.getLayoutInflater().inflate(R.layout.fragment_news_feed, null);
    }
    //  update your inner views value here
    return row;
}

Now you can set the custom Adapter to the ListFragment

setAdapter(new CustomAdapter(context));
