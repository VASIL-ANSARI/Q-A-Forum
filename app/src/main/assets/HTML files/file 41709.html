If i get i right your if (Modernizr.mq('only all and (max-width: 599px)')) { only checks for the mediaquery and has no event attached?!

If you want to check your width on resize of your browser try this:

$(window).resize(function() {
    if (Modernizr.mq('only all and (max-width: 599px)')) {
        $('body').removeClass('desktop').addClass('mobile');
    }

    if (Modernizr.mq('only all and (min-width: 600px)')) {
        $('body').removeClass('mobile').addClass('desktop');
    }
});
