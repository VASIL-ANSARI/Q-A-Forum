I just did something like this last week, I used a tool called FPDF (http://www.fpdf.org/) just download the v1.7 zip. then copy the folder over to your server.

then you will need to include a file, and then write something like this:

require('../include/fpdf/fpdf.php');

$pdf = new FPDF();

$pdf->AddPage();

$pdf->SetFont('Arial','B',16);

$pdf->Cell(40,10,"$id_user");

$pdf->image("https://chart.googleapis.com/chart?chs=250x250&cht=qr&chl=$i",50,50,0,0,"PNG");

$pdf->Output();

and the chart.googleapis is nothing to worry about. I used the $pdf->image to generate a QR code. so you can delete that line, unless you need it.
