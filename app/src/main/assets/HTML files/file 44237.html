When your page loads, please check the JS console for any CORS errors. If you see these errors, then do the following: By the term "other.server", we are referring to a cross-domain request. In simple words, requesting from the domain which is different from the domain of the site which you are developing/using. There are a couple of things to be taken care of before making a cross domain request. Here is the list : https://jvaneyck.wordpress.com/2014/01/07/cross-domain-requests-in-javascript/

Also, if you need to understand what CORS ( Cross Origin Resource Sharing) means, please read this http://www.eriwen.com/javascript/how-to-cors/
