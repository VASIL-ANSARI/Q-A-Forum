The zoom property is not of type boolean. Instead it consists of several sub-properties

var board = JXG.JSXGraph.initBoard('jxgbox', {
     boundingbox:[-6,2,6,-2], 
     showNavigation: true,  
     zoom: {
            factorX: 1.25,
            factorY: 1.25,
            wheel: false,
            needshift: false,
            eps: 0.1
        }
});

But it is available only in newer version of JSXGraph, so my suggestion is to use the latest version 0.99.1:

http://cdnjs.cloudflare.com/ajax/libs/jsxgraph/0.99.1/jsxgraphcore.js
