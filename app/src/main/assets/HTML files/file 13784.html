There is no method in jQuery that will do what you are trying to do. It would require either DOM Mutation Events which may not be supported in all browsers or a setInterval which is not efficient.

Instead, bind a custom event to the window and trigger the event when you change the content of the page.

$(window).bind("contentChanged load",function(){
    console.log("window loaded or changed");
    $("#gallery").slideViewerPro({ // div#gallery is very inefficient
        autoslide: true,
        thumbsVis: false,
        galBorderWidth: 1,
        galBorderColor: "#003f00"
    });
});

and later on:

$("#content").load("somepage.php",function(){
    $(window).trigger("contentChanged");
});
