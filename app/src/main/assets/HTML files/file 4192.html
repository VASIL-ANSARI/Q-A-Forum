no regex needed:

l = []

with open("name_file", "r") as f:
   for line in f:
       l.append(line.split(", "))

if you want to remove first space and to have tuple you can do:

l = []

with open("name_file", "r") as f:
   for line in f:
       data = line.split(", ")
       l.append((data[0].strip(), data[1].strip()))
