Alternative solution to the create table xxx_2018 as select * from xxx_2017;' is to usecreate table ... like ...` construct (it creates table only, also gives option for copying constraints etc; does not copies data):

create table xxx_2018 like xxx_2017 including all;
insert into xxx_2018(<<4 columns you would like to fill>>) select <<4 columns you would like to fill>> from xxx_2017;
create trigger xxx_2018_trigger ... on xxx_2018 execute procedure trigger_procedure();

Reference: https://www.postgresql.org/docs/9.1/static/sql-createtable.html ("LIKE" paragraph)
