Check the URL in Facebook Debug Tool, https://developers.facebook.com/tools/debug/. Scrape your URL again and again and check. It will display the error that you are missing.

The minimum image size is 200 x 200 pixels. If you try to use an image smaller than this you will see an error in the Sharing Debugger.

Refer the link,

https://developers.facebook.com/docs/sharing/best-practices#images
