Actually there is no real minimum to the like box. Just generate a like box code from facebook developers here, and then paste into a regular wordpress text widget.

Now in order to resize it you will have to add inline styling like so:

<div style="width: 150px; overflow: hidden;" class="fb-like-box" data-href="https://www.facebook.com/FacebookDevelopers" data-colorscheme="light" data-show-faces="true" data-header="true" data-stream="false" data-show-border="true"></div>

I add a width of 150px to their demo code. You can also wrap this code in another div and give it a fixed height / width with an overflow: hidden;.

Just make sure you "cut it" in the width that doesn't cut the photos and this way it looks perfect.
