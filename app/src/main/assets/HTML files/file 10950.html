You can use the OrientationChanged event to detect orientation changes on your PhoneApplicationPage.

For XAML:

<phone:PhoneApplicationPage OrientationChanged="YourOrientationChangedEventHandler" />

If you want to just get the current orientiation, use the Orientation property.

For C#:

public class MyPage : PhoneApplicationPage
{
    public void SomeMethod()
    {
        var currentOrientation = this.Orientation;
    }
}

For more information about handling orientiation in Windows Phone 7, you can refer to this MSDN article.
