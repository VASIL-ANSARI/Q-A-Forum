Whenever yo use an ? in your statement, means that it will be "replaced" by a parameter.

You have to bind those paramters when executing the query.

Using standard JDBC syntax, it will be something like :

    PreparedStatement prepStmt = connection.prepareStatement(sql);
    prepStmt.setString(1, paramValue1);
    prepStmt.setInt(2, paramValue2);
    ....
    prepStmt.setInt(n, paramValueN);
    prepStmt.execute();

Hope it helps
