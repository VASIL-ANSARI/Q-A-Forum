You can't have hover event on mobile. I belive you understand that.

Instead you can create InfoWindow with data you need and set it to open on click. That way it will be opened when you touch marker on mobile device.

Update

Because you take external json, you can just push new object to it with:

  g.StoreLocatorResp.locationList.push({
    nodename: "My Location",
    address: "My Location address",
    longitude: "28.407081",
    latitude: "-24.704055",
    icon: "http://maps.google.com/mapfiles/ms/icons/red-dot.png"
  });
