In your function (below) you're iterating over a collection and returning a bool on the first object, that should ring some bells. 

$scope.disableInput = function (todos) {
todos.forEach(function (item) {

    if (item.toLowerCase().indexOf('clean') > -1) {

        return true;
    } else {

        return false;
    }
});

};

'I wanted to check the todos object every time when a new item is submitted'

Sounds like a sensible place to add that check is in the function executed when you add a todo.

Now you don't have to iterate over your todos.

var disableInput = function (input) {
    if(input.toLowerCase().indexOf('clean') > -1){
        return true;
    }
    return false;
};

So you'll want to add a variable to the scope to keep track of whether the input is disabled. 

$scope.disabled;

Now in your addTodo function (grabbed from your fiddle) you should add the check for the 'clean' string on what's just been input. 

 $scope.addTodo = function() {
    $scope.todos.push({
       text:$scope.todoText,
       done:false
     });

     //Set the scope variable you just made to the result of the above function
     $scope.disabled = disableInput($scope.todoText);

     $scope.todoText = '';
  };

So finally in your mark-up; assign the disabled variable to ng-disabled.

<input type="text" ng-model="todoText"  size="30"
 placeholder="add new todo here" ng-disabled="disabled">
