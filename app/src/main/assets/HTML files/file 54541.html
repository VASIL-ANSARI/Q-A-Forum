I believe that a better approach to this is to add a margin-right to the icon. To do this add a class to the icon (I'm using .app-signup-icon here)

%a{ href: "#" }
  %i.fa.fa-user.app-signup-icon
    Sign Up

Then define the css rule with a margin-right in your css file:

.app-signup-icon {
  margin-right: 10px;
}
