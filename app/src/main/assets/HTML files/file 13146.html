$rows[$i] = array( gmdate( strtotime($row['date2']) )*1000, $total);

You're using gmdate, which returns the time in GMT. If you're storing 00:00 in your database, gmdate will return 05:00.

Just changing it to date should fix this:

$rows[$i] = array( date( strtotime($row['date2']) )*1000, $total);
