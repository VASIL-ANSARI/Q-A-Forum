You can use the display_errors() method to show you what's not working correctly:

if(!$this->upload->do_upload('pan_card'))
{
    echo $this->upload->display_errors();
    exit;
}

However, I generally have issues with the upload class. The allowed_types always seems buggy on my installation of CodeIgniter. If I specified a list of allowed file types, it always comes up with an error saying the uploaded file type isn't allowed.

I would suggest changing it to this:

$config = array(
    'allowed_types' => '*',
    'upload_path' => $upload_path . '/'
);

Check to see if that works.
