The compiler doesn't know that the values inside the switch cannot exceed 3 or be less than 0.

Hence, you have to tell the compiler what to do in situations where it is. The easiest way is to add a default clause to your switch. I prefer an Exception over returning a default value, since it makes clear that actually something went wrong:

case 3:
{
    ...
}
default:
{
    throw new Exception("I totally didn't expect this.");
}
