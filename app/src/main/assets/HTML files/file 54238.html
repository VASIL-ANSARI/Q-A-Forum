Python provides a cycle function in itertools that is helpful with this:

from itertools import cycle

i = 0
n = 10
l = []
step = cycle([1, -2, 3])

for _ in range(n):
    i += next(step)
    l.append(i)

print(l)

Giving you:

[1, -1, 2, 3, 1, 4, 5, 3, 6, 7]

Or alternatively you could do the following one liner:

from itertools import accumulate, islice, cycle
import operator

l = list(accumulate(islice(cycle([1, -2, 3]), 10), operator.add))
print(l)

accumulate was added in version 3.2
