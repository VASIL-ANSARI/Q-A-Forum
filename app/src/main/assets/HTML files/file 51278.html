When you change the playerCount parameter in the ProcessInsert method, it will not affect the variable outside the method. Parameters are sent by value, so the parameter playerCount has a copy of the value from the playerCount variable outside the method.

To make the parameter reference the variable, you need the ref keyword:

static void ProcessInsert(int[] playerNumbers, string[] playerLastNames, ref int playerCount, int MAX_PLAYERS)

You also need to use if in every place you call the method:

ProcessInsert(playerNumbers, playerLastNames, ref playerCount, MAX_PLAYERS);

You are getting a value from the array instead of putting a value in it. This line:

numberOfPlayer = playerNumbers[insertIndex];

should be:

playerNumbers[insertIndex] = numberOfPlayer;
