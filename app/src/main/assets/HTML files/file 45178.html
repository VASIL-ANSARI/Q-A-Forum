You can use psudeo-element ::before to clear float. Take a look to the following example using last-child:

ul li {

  width: 100px;

  height: 50px;

  display: inline-block;

  background: red;

}

ul li:last-child {

  float: right;

}

ul li:last-child:before {

  content: '';

  clear: both;

}
