You can just loop over the permutations and write each of them to the file, like this:

import itertools
f = open('File.txt','w')
for key, value in dict.items():
    for i in itertools.permutations((str(counter-value))):
        print(i, file=f)

As it is a generator, the items are retrieved one-by-one so your memory won't be exhausted.
