I think there are 2 issues:

  1. your query should use "<=" rather than "=".

  2. instead of %U, use %i in the argument to STR_TO_DATE (%U is for the week number)

    SELECT * FROM pos_takeaways WHERE STR_TO_DATE(time,'%d/%m/%y %H:%i:%S') <= STR_TO_DATE('20/6/2011 22:30:00','%d/%m/%Y %H:%i:%S');
