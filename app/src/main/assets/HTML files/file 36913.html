Yes. In fact, clicking #btn2 will not alert "span" unless you first clicked #btn1. If you wanted for the <span>span</span> to be loaded automatically, without requiring a button click, you could do like:

$(document).ready(function() {
    $('#parent').append('<span>span</span>');

    $('#btn2').click(function(){
        if ($('#parent span').length > 0) {
            alert($('#parent span').text());
        } else {
            alert('element doesn\'t exists yet!');
        }
    });
});

Then your <span>span</span> will be appended automatically and clicking #btn2 should always alert "span".
