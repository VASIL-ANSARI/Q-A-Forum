Unpack the object into an array of strings?

Let's say you are passing an Object called "Sport" with the following properties:

.name: "Football" .ball: "round" .periods: 2 .periodTiming: 45

You could unpack this object into ["Football", "round", 2, 45], and, modify your Java code to pick up each parameter:

    String name = args.getString(0);
    String ball = args.getString(1);
    int periods = args.getInt(2);
    int timing = args.getInt(3);
