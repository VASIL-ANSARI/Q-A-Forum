I'm pretty new to this stuff myself. With styling being the main consideration here, the possibilities and combinations are endless.

I would consider making a few divs within the main contact form wrapper. Kind of try and keep the contact form whole. You may want to use a div for the first 3 text inputs on the left, the textarea input "message" as another one and lastly the submit button "contact". With that all in mind, the styling should be nothing more than adding some margin and padding between inputs and giving the proper width/height through specificity in CSS. Maybe give each div it's own id or class to distinguish specific input fields. Border radius can give that nice rounded edge on the input areas. Something like 8-10% is usually enough. Use some text align to get the input "value" (CONTACT) to center.

The responsiveness portion of your form would very much depend on how you want it laid out on another devices or resolutions. If you wanted them to stack on top of one another for smartphones and such; I believe you could do that with the display: inline and they should wrap appropriately with your breaks.
