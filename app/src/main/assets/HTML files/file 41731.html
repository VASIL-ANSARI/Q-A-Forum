if you are using jquery i use this plugin http://malsup.com/jquery/block/#overview and then setup a loading gif image to be set in the blocking html. my example is like

$(element).block({ message: '<h5><img src="/images/loader.gif" style="margin-right: 5px;" />loading...</h5>" });

this site can create an image for you to download and use. http://www.ajaxload.info/ this allows you to specify exact area to show loading image
