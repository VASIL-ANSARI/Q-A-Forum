Do like this:

mysql> SELECT LENGTH("boo"
    -> "m");
+-------------------+
| LENGTH("boo"
"m") |
+-------------------+
|                 4 |
+-------------------+
1 row in set (0.00 sec)

To understand this see this example:

mysql> SELECT "boo" "m";
+------+
| boo  |
+------+
| boom |
+------+
1 row in set (0.00 sec)

You can see same behavior in many programming languages.
