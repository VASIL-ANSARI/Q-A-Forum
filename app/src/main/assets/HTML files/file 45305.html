There are multiple problems:

insertFirst pushes elements inserted earlier to the back. If you want to use insertFirst in main, you need to reverse the order: 

addElements.insertFirst(29);
addElements.insertFirst(27);
addElements.insertFirst(23);
addElements.insertFirst(22);
addElements.insertFirst(21);

There are some issues with insertNodeAtParticularLocation:

  • The first == null case doesn't work as it does not assign first
  • The while loop looks like it may insert multiple elements in the wrong locations

I'd write insertNodeAtParticularLocation as follows:

    public void insertNodeAtParticularLocation(int x) {
        if (first == null || x < first.data) {
            // if the value that needs to be inserted
            // is less than the first element, or it does not exist,
            // then insert it at the very beginning
            insertFirst(x);
        } else {
            Link previous = first;
            Link current = first.next;
            while(current != null && current.data > x) {
                previous = current;
                current = current.next;
            }
            Link newNode = new Link(x);
            previous.next = newNode;
            newNode.next = current;
        }
    }
