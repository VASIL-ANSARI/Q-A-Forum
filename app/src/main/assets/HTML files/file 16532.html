This is about the closest I could come without modifying your html structure.

<center class="some_class"><h1 class="ui-title" tabindex="0" role="heading" aria-level="1">Advento</h1></center>

Then use the following css, alter as needed:

center.some_class {
    margin: 5px 15% -28px 0 !important;
}
