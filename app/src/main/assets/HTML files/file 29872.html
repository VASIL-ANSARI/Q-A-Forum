This feature of array derefernce was introduced in PHP 5.4. Therefor you cannot use it on a PHP 5.3 Production server.

There are at least two solutions. One is the idea of upgrading your PHP version on the server (good idea).

The other one is just a change in the code base:

$tmp_logged = $this->users_model->get_logged();
$param['info'] = $tmp_logged[0];
