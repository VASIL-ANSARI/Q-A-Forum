If I understand your question correctly, then MyEntity is not an entity (it is not saved inside database), it represents a data than needs to be computed using stored procedure.

Your code should looks like this:

class MyEntity
{
    public int Id {get; set;}

    public int Data {
        get {
            using(var context = new DatabaseContext())
            {   
                var idParam = new SqlParameter("@MyEntityId", Id);

                return context.Database
                    .SqlQuery<int>("GetDataForMyEntity @MyEntityId", idParam).SingleOrDefault();
            }
        };

        // Data should not have a setter!!!
    }
}
