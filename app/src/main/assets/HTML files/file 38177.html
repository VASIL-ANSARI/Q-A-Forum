z-index requires certain things to work. First, you need to create a stacking context on an element that is a "root" of both elements that need re-ordered. You can create a stacking context by adding position: relative;. in css (or use something in this list).

Then add a z-index to each branch off the root that needs to be set. and give them position: absolute | relative;. For example, this fiddle: http://jsfiddle.net/6fy1rccm/

edit: Given the website you linked, it works if you add

z-index: 1;
position: relative;

to the role=banner css class.

and

position: relative;
z-index: 0;

to the container css class.

And I encourage you to read the first link I posted, lots of good stuff on stacking contexts in there without being too long or boring.
