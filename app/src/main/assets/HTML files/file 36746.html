I think what you want is cos(x*pi) instead of cos(x/pi). That way you get a function of similar shape that is [-1,1]-periodic. So simply redefine

def en(k):
    return -2*cos(pi*k)

Your expression to transform k into k2 can be more simple written as k2 = [i*pi for i in k]. This is called a "list comprehension" and your code would look much nicer with some of them (maybe reduced to 4-5 lines).
