I believe you're looking for a masonry layout.

Check out Isotope: http://isotope.metafizzy.co/

Here's a simple example

    <div id="container">
         <div class="item">...</div>
         <div class="item">...</div>
         <div class="item">...</div>
         ...
    </div>

$('#container').isotope({
  // options
  itemSelector : '.item',
  layoutMode : 'fitRows'
});
