You are getting that error cause either your gridview don't have 5 rows or 5 columns. How many rows/columns does your gridview has? Remember that array indexing starts with 0. Make sure, the row number (or) column number you are accessing does exists.

Moreover, why are you accessing multiple rows at same time like below. That don't make sense cause at any time only 1 row will be selected. ... Right?

dgvList.SelectedRows[0]
dgvList.SelectedRows[1]

It rather should be probably be

 private void dgvList_MouseClick(object sender, MouseEventArgs e)
        {
                tbName.Text = dgvList.SelectedRows[0].Cells[0].Value.ToString();
                tbSurname.Text = dgvList.SelectedRows[0].Cells[1].Value.ToString();
                tbMobile.Text = dgvList.SelectedRows[0].Cells[2].Value.ToString();
                tbEmail.Text = dgvList.SelectedRows[0].Cells[3].Value.ToString();
                cbCategory.Text = dgvList.SelectedRows[0].Cells[4].Value.ToString();
        }
