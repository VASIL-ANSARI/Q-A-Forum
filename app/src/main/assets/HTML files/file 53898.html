On button click event, create a new System.Windows.Forms.OpenFileDialog()

Here is a sample:

        System.Windows.Forms.OpenFileDialog openFileDialog1 = new System.Windows.Forms.OpenFileDialog();
        openFileDialog1.Filter = "MP4|*.mp4|AVI|*.avi"; //use as you require.
        openFileDialog1.FileName = "";
        if (openFileDialog1.ShowDialog() == System.Windows.Forms.DialogResult.OK)
        {
             string[] fileNames = openFileDialog1.FileNames;
        }

Then you can do anything with the file names. If there is a file, you can use FileName.

Then you can save it to the server database.
