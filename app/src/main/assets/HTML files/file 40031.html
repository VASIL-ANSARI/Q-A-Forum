You can merge query arguments to the existing query.

global $wp_query;
$args = array_merge( $wp_query->query_vars, array( 'posts_per_page' => '12', 'paged' => $paged  ) );
query_posts( $args );

Or you can add to the query

global $query_string;
query_posts( $query_string . '&posts_per_page=12' );

ect. You can read more about it under "Preserve Existing Queries" on WP Codex. http://codex.wordpress.org/Function_Reference/query_posts
