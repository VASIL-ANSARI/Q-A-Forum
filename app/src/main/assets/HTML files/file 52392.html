Wildcards can only be used for extension mappings.

But the last parameter of method FilterRegistration.addMappingForUrlPatterns is a var arg so you can write a little bit shorter

sce.getServletContext().addFilter("adminFilter", adminGuar).addMappingForUrlPatterns(null, false, 
    "/admin_menu", "/admin_users", "/admin_banana", "/admin_orange");

Additionally the current URL schema cries out for the following redesign:

/admin/menu
/admin/users
/admin/banana
/admin/orange

which then allows

 sce.getServletContext().addFilter("adminFilter", adminGuar)
    .addMappingForUrlPatterns(null, false, "/admin/*");
