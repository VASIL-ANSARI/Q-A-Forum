You don't need to read what your program outputed to know the correct answer count. Just use a variable instead.

int correctCount = 0;

int ans1 = Convert.ToInt32 (Console.ReadLine ());
if (ans1 == 36) {
    ++correctCount;
    Console.WriteLine ("Correct.");
}
else if (ans1 != 36) {
    Console.WriteLine ("Incorrect. Try again.");
}
...

This way you will know how many answers are correct without relying on the output.

BTW: I strongly urge you not to use goto for your program flow as it can lead to multiple headaches.
