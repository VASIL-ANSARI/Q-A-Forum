After reading this http://laravel.io/forum/08-31-2014-invalidargumentexception-for-postgresql-database I decided to create model that extends default model class and replaces getDateFormat with custom code.
Because changing classes in vendor directory is bad idea.

I hope it helps:

class FixedModel extends Illuminate\Database\Eloquent\Model {
    public function getDateFormat()
    {
        return 'Y-m-d H:i:s.u';
    }
}

class User extends FixedModel {}
