EDIT If you are just looking to redirect from /School to /web/schools.html you could just do hte following:

    [Route("~/School")]
    public ActionResult YourActionThatReturnsSchoolDotHtml()
    {
        return View();
    }

Alternatively, you could do a simple RedirectToAction:

    public ActionResult YourActionThatReturnsSchoolDotHtml()
    {
        return View();
    }

    [Route("~/School")]
    public ActionResult Schools()
    {
        return RedirectToRoute(YourActionThatReturnsSchoolDotHtml());
    }

I'm not sure if this will help you, but if you are using ASP MVC, you could setup your routes via the App_Start/RouteConfig.cs file like this:

routes.MapMvcAttributeRoutes();
routes.MapRoute(
    name: "Schools",
    url: "Schools/{*url}",
    defaults: new {controller = "Schools", action = "Index" }
);

And then, in your SchoolsController.cs:

[Route("Schools/{*url}")]
public ActionResult Index()
{
    return View();
}

Provided you also have an index.cshtml file under Views/Schools directory

This will basically redirect every calls to /Schools/* MVC action to your angular router
