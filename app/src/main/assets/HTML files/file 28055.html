This must work for you. I tested in my local mysql database with the data you provided and it worked.

select dth.toy_id, dth.created_date as start_date, 
       ifnull((select min(dth2.created_date) from dog_toy_history dth2 where dth2.created_date > dth.created_date and dth2.dog_id='2'),'2013-09-01') as end_date 
       from dog_toy_history dth
       where (dth.created_date between '2013-07-01' and '2013-09-01') and dog_id='2' ORDER by start_date;

I will give some explanation for those who are not very familiar with mysql:

  • the inner select brings the created_date field of the next toy if existent. This will represent the end_date of the current toy.

  • ifnull(exp1,exp2) specifies a default if a next toy is inexistent. In this case the end_date will be the end date from the input.
