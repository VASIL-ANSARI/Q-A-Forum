Do it in the resolve.

$stateProvider.state('home', {
    url: '/',
    controller: 'MainController',
    resolve: {

      /* @ngInject */
      someResolve: (ConfigService) =>
        ConfigService.config()
    }
});

Then your 'ConfigService' can be injected into the controller, and used (the result of the config()). Edited to use the correct service name and so on.
