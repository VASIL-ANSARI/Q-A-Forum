To my mind there's two viable option out there.

I use both actually.

Eclipse + Aptana Studio + Pydev or Aptana Studio

Pros

  1. Free
  2. Decent auto completion (IntelliSense like system)
  3. More plug-ins (since it's based on eclipse)

  4. Support django template Cons

  5. relatively poor html editor

  6. no mako or jinja2 support (as far as I know)

Pycharm

Pros

  1. better auto completion
  2. Support mako,jinja2 and django template
  3. Good HTML edtitor

Cons

  1. Not Free

Both support debug without too many problems.
