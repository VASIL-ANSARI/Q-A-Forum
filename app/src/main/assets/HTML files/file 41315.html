One way I can think of is to set ClientID on each TableRow. 

tr = New TableRow
tr.ClientID = "tableRowID"

Then on the ASP page, add attribute to set static ClientIdMode 

<%@ Page Title="" Language="VB" MasterPageFile="~/UserMast.master" .... ClientIDMode="Static" %>

Now you should be able to use the ID of each row with jQuery for click event.

$('#tableRowID').click(function() {
        var href = $(this).find("a").attr("href");
        if(href) {
            window.location = href;
        }
    });
