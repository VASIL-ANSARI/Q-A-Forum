Your error will be fixed by addressing these statements:

std::string UserChoice = CARDS;
std::string GeneratedCard = CARDS;

You need to index the array to obtain the proper value for these variables.

Your function parameters are intended to be the arrays containing the five guesses and generated cards, so your function then becomes:

void DisplayCards(int* user, int* generatedCards)
{
    cout << "Users Guess" << "\t\t\t" << "Generated Cards" << endl;
    for (int i = 0; i < 5; ++i)
    {
        // Get the names of the choices from the deck
        std::string UserChoice = CARDS[user[i]];
        std::string GeneratedCard = CARDS[generatedCards[i]];
        // print the names side by side
        cout << UserChoice << "\t\t\t" << GeneratedCard << endl;
    }
}

The function parameters are pointers to the first element of the array that is presumably being passed in via the call to this function I cannot see. You'll likely want to include some more error checking, but the gist of it is you can index your input arguments the same way you would a normal array, using your i counter as the index number. You'll get bogus data and perhaps a segmentation fault if your arrays are not the proper size when you run the code, but that's up to you to learn on your own.
