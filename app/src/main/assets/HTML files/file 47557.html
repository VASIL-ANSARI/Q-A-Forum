You can have list objects as value for your dict. So it is as simple as:

content = """a;1
b;2
c;3
d;4
e;5
a;7
b;3
c;5
d;8
e;9
a;9
b;3
y;5"""

csv_data = []
for line in content.split('\n'):
    csv_data.append( [x.strip() for x in line.split(';')] )

s = csv_data
dic = {}
for n in range(0,len(s)):
    if (len(s[n]) == 2):
        key = s[n][0]
        val = s[n][1]
        if key not in dic:
            dic[key] = []
        dic[key].append(val)

And now dic is:

{'a': ['1', '7', '9'],
 'b': ['2', '3', '3'],
 'c': ['3', '5'],
 'd': ['4', '8'],
 'e': ['5', '9'],
 'y': ['5']}

Which is what you wanted I guess.
