According to this, the accountId property is the Id of the merchant center account that published the product. That sounds an awful lot like the store.

Create an array of merchant id numbers you would like to have, for example:

$merchants = array('Amazon' => '123456',
                   'HMV' => '789012');

Then generate your request and pass those values as the accountId. I'm not sure if you can pass multiple values at a time.

One thing to note, is it may be difficult to find the accountId of some merchants. See this: https://groups.google.com/group/google-content-api-for-shopping/browse_thread/thread/e0c4d098e25f97c8
