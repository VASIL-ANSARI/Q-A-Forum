Firstly, you really don't need to put PHP tags on every line, just keep it all in one block. I think the best thing you can do, since the forums table probably won't contain too many rows is to select them all, then add condition as to what's displayed. Also, I'm assuming a forum with no group has an id of 0.

$table = '';

$sql = "SELECT * FROM forums_forums ORDER BY order_number ASC";
$run = mysql_query($sql);
while($row = $mysql_fetch_assoc($run))
{
    if(($_GLOBALS['users']['team'] == 1 && $row['admin_only']) || ($row['team'] > 1 && $_GLOBALS['users']['team'] == $row['team']) || ($row['team'] == 0 && $row['admin_only']))
    {
        $table .= '
            <tr>
                <td>'.$row['forum_name'].'</td>
                <td>'.$row['replies'].'</td>
                <td>'.$row['posts'].'</td>
                <td>'.$row['forum_description'].'</td>
            </tr>';

    }
}

echo '<table>'.$table.'</table>';

Bear in mind though, this is a really nasty way of doing things. If you want cleaner, more navigatable code then look into templating. You can write a template class yourself without much difficultly. What you should be looking to do then is to have a template with all the HTML in a row, when repeat that row as often as you need it.

Also, you should take a look at some of the data types for your table. If admin_only can only be 1 or 0 for example, it should be a TINYINT with length of 1.
