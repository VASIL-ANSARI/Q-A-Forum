Captain Typo strikes again. Replace unlick with unlike and use live() for the second handler, too:

$('#like').live('click',function(){
    alert('like');
    $(this).prop('id', 'unlike').val('unlike');
});

$('#unlike').live('click',function(){
    alert('unlike');
    $(this).prop('id', 'like').val('like');
});

You should also use .prop() to set the id and .val() to set the value.

Oh, and consider not changing IDs but storing the flag (like/unlike) in a data attribute and check it inside the event handler. Then you don't need live events or delegates but can simply bind a regular event.
