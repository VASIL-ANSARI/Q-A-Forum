Some options:

Add security through your container using the standard Java API: http://docs.oracle.com/cd/E19226-01/820-7627/bncby/index.html

Use Spring security and enable authentication to the pages you want to protect. Samples: http://www.mkyong.com/tutorials/spring-security-tutorials/

Use Apache Shiro: https://shiro.apache.org/tutorial.html

I would usually go with using Spring Security and configure it through the xml config.
