In broad terms, you would create an instance of an Entry widget for each square in the puzzle. You should then use the grid geometry manager to lay the widgets out in a grid. You can do both the creation and the layout in a loop for the columns nested in a loop for the rows.

You can use the insert and get methods to insert and retrieve the data in each widget. Or, you can associate a Tkinter.IntVar variable with each entry and then use the get and set methods to get and set the value.
