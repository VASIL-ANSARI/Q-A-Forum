Your false positives are likely the result of an improper while loop in your isPalindrome function. The function is returning true in the first iteration of the loop...

while (x<=(length/2)) {
    if(number[x]==number[length-y]) {
        x++;
        y++;
    } else {
        return false;
    }
    return true; // this is in the wrong place, I believe
}

I think what you intended to do was:

while (x<=(length/2)) {
    if(number[x]==number[length-y]) {
        x++;
        y++;
    } else {
        return false;
    }
}
return true;
