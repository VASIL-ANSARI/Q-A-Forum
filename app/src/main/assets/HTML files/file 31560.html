If you need to refresh a current window you can call a function say mylocation() 

 select: function(start, end, allDay) {

       mylocation(start,end,allDay);
   },

and can pass params according to your need. it will refresh the current location.

 function mylocation(start,end,allDay)
  {
    window.location.assign("/index.php?start"+start)
  }
