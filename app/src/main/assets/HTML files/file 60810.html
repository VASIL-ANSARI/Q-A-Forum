jQuery DataTables doesn't escape HTML so that if field contains <b>Text</b> it will be shown in bold in a table by default.

You can encode HTML entities in your response using the code below:

return response()->json(array_map('e', $contacts->toArray()));

If you're using Yajra DataTables for Laravel, it has escapeColumns() method which you can use to escape HTML entities in some or all columns. For example:

return Datatables::of($contacts)
   ->escapeColumns(['first_name', 'last_name')
   ->make(true);
