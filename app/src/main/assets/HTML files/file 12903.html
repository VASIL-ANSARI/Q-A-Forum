The DateTime class constructor accepts valid datetime strings. If it's passed an invalid datetime it throws an exception. Once you have the DateTime object, you can format it in any way you want using DateTime::format

$date = 'bad date string'

try {
  $obj = new DateTime($date);
  echo 'valid date: '. $obj->format('Y-m-d');
} catch (Exception $e) {
  echo 'invalid date';
}
