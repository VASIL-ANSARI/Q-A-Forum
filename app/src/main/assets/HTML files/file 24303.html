Depending on where you have rvm installed, if you're using rvm.

If rvm is installed in your home directory:

In terminal, echo $HOME this will return your home directory.

cd /whatever/returned or cd ~/. will also change directory into your home directory

ls -a

will list all directories.

cd .rvm
cd gems
cd ruby-2.0.0-p195
cd gems
cd whatevergemname-version#

Find the file you are looking for inside of that gem and then pwd will display the current location.

It should look in your dependencies though, so maybe you have conflicts between ruby installations.

Hope this helps.
