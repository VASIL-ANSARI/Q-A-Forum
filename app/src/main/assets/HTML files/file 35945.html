Nevertheless, as I understand, the MainActivity takes its object graph from the Application. So, that is the place where you have to insert your MockModule.

In order to do this, you should create a subclass of PhoneApplication and override getModules() method to return your MockModule. After that, you should mock application with ActivityUnitTestCase.setApplication() (your test should subclass ActivityUnitTestCase first). That should fix the issue.
