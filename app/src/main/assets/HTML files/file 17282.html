You want to set the lpstrDefExt member:

From MSDN:

Type: LPCTSTR

The default extension.

GetOpenFileName and GetSaveFileName append this extension to the file name if the user fails to type an extension. This string can be any length, but only the first three characters are appended. The string should not contain a period (.). If this member is NULL and the user fails to type an extension, no extension is appended.
