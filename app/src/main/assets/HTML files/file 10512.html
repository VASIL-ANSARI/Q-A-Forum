You can try this CSS:

.language-selector > ul {
    position: absolute;
    clear: both;
    top: 35px;
    left: 37px;
}

#header-container > ul > li {
    display: inline;
    float: left;
    position: relative;
}

It should do the trick!

The position: relative; is particularly important on every list element as that is what is used to position the dropdown menu.
