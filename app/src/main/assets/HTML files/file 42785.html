Use a BorderLayout for the main layout.

  1. add the "header" to the NORTH.
  2. add the "footer" to the SOUTH.
  3. Create another panel and add it to the CENTER. Then set the layout manager of this panel to an appropriate layout manager. We can't advise you what layout manager to use for this panel since we don't know how the two components area related. Then add "area1" and "area2" to this panel.

The key to this answer is that you don't need to use a single layout manager, you can nest panels with different layout managers.
