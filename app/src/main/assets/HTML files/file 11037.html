Your syntax is incorrect. Which column has to be LIKE \'%".$s."%\'? Say that column is colname. Your query should be:

$sql = "SELECT * FROM test WHERE colname LIKE \'%".$s."%\'";

Looking back over your code you're also doing something /really weird/. That PHP there is being executed /before the javascript ever is sent to the client/. PHP runs serverside, JS runs clientside, so JS can't pass parameters to PHP.
