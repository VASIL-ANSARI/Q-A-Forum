It's probably because of your project's user data which contains your editing context. Those files changes while using Xcode.

Stop tracking everything bellow following folders inside your project's package

  • yourproject.xcodeproj/project.xcworkspace/xcuserdata
  • yourproject.xcodeproj/xcuserdata

For information, yourproject.xcodeproj is a package, if you right click it and click "Show Package Contents" you will see mentioned directories.

And then add all files bellow those folders to ignore (.gitignore if you are using git) Your .gitignore file will contains those 2 lines: yourproject.xcodeproj/project.xcworkspace/xcuserdata yourproject.xcodeproj/xcuserdata

Hope it helps
