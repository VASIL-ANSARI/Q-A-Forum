There would generally be a server language used with a website that has a login feature, most commonly PHP.

With PHP your server can save variables, that contain information, such as if a user is currently logged in.

Then you can write conditional statements that offer different output, depending on the condition. For example:

if (user_logged_in()) { echo "Hello ".$user."!"; } else { echo "Login"; }

This line basically says, if the user is logged in, then display Hello and their username. If they are not logged in, then display 'Login'.

That's a very simple form, but that's how it would be done, essentially all using PHP IF statements - http://php.net/manual/en/control-structures.else.php
