s is Student Type But How and Why Do we Need to Make Data Type Like

In java Array is Object. In Student[] s = new Student[100] is array of Student Object which are null right now by default.Means every single index of Student[] from 0 to 99 will have Object reference stored at index.

As you can see in image every box contains value which is nothing but value stored at specific memory address allocated to the array.

Why Do we Need to Make Data Type Like? Yes.Array is very older than collection as before collection array was the option to store the collection of variables of same data type and access it by index.So still now array is there for storing values/references of same type.

To use s you have to first initialize it s[0]=new Student() so it's Object now before this s[0] was null
