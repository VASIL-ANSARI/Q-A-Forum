You can absolutely position the child div and use top:0; right:0;bottom:0; left:0; to center it:

.modal-dialog {
    width: 210px;
    height: 400px;
    margin: auto;
    border: 1px solid white;
    position:absolute;
    top:0;
    right:0;
    bottom:0;
    left:0;
}

jsFiddle example

Note also that I removed the display and vertical-align rules for the parent since they're not needed.
