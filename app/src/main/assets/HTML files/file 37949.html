< intent-filter > can have a label attribute. Using this, you can set a label(name) for the launcher icon, while still having the Activity with it's own title. Modify your manifest as follows:

<meta-data android:name="com.google.android.gms.version"
       android:value="@integer/google_play_services_version"/>
<activity
    android:name=".MainActivity"
    android:label="@string/yukleniyor" >
    <intent-filter android:label="@string/app_name">
        <action android:name="android.intent.action.MAIN" />

        <category android:name="android.intent.category.LAUNCHER" />
    </intent-filter>
</activity>
