I would suggest putting this into a function, so you can make this happen on whatever trigger you want:

function addClass() {
    $('.grid-block').each(function(i){
        var expHeight = 58;            
        var gridHeight = $(this).height();
        if(gridHeight <= expHeight){
            $(this).find('h1').addClass('truncate-text');
        }
    });
}

Then you can call this whenever. For example, if you want it to happen on page load:

$(document).ready(function(){
    addClass();
});

or in a timed event:

setTimeout(function(){
    addClass();
}, 1000);

As Kevin B mentioned below - I should have said when the document is ready - if you want it to fire when the page LOADS, it would be 

$(window).on("load", function(){
    addClass();
});
