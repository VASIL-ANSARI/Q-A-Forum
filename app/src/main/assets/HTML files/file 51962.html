There are a couple of ways to achieve this, but generally speaking you'll need to generate the file_name before calling blah or blah2. (BTW, your md5 is always going to be the same value, which makes it pretty useless, imo. I would change what you hash or remove it entirely.)

<?php

public static function generateFilename ($id) {
    self::$file_name = $id . '_' . md5(time()) . '_' . uniqid();
}

Then to use it:

ImageUploadModel::generateFilename(Session::get('id'));
ImageUploadModel::blah();
ImageUploadModel::blah2();

and ImageUploadModel::$file_name will be the same between both "blah" calls.
