You can make your custom deserialize of your viewstate, that contains the commas that break its ',' using the LoadPageStateFromPersistenceMedium() function of the System.Web.UI.Page as:

private ObjectStateFormatter _formatter = new ObjectStateFormatter();

protected override object LoadPageStateFromPersistenceMedium()
{
    // fix here and load your ViewState on correct way
    string vsString = Request.Form["__VIEWSTATE"];

    return _formatter.Deserialize(vsString);
}

This is just an idea that you need to debug, and work with it to make it work for your issue here.
