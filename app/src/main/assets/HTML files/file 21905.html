Active Admin uses devise to handle its logins, which is probably why you couldn't find anything.

From the instructions here:

Create this class in lib/custom_failure.rb:

class CustomFailure < Devise::FailureApp
  def redirect_url
    #return super unless [:worker, :employer, :user].include?(scope) #make it specific to a scope
     new_user_session_url(:subdomain => 'secure')
  end

  # You need to override respond to eliminate recall
  def respond
    if http_auth?
      http_auth
    else
      redirect
    end
  end
end

And add this to config/initializers/devise.rb:

config.warden do |manager|
  manager.failure_app = CustomFailure
end
