Try increasing the maximum length allowed for your query string in both your application and IIS:

Application: 

<httpRuntime maxQueryStringLength="32768" />

IIS:

<system.webServer>
  <security>
    <requestFiltering>
      <requestLimits maxQueryString="32768"/>
    </requestFiltering>
  </security>
</system.webServer>
