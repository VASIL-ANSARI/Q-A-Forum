I doesn't see a code of your adapter for GridView. At first you should write a separate class - adapter for GridView, where method getView() must be overrided (see similar questions about adapters for GridView). When you retrieve a data, u should call notifyDataSetChanged() method for your adapter - this will refresh the GridView with new data.

Good example how to custom gridview http://www.mkyong.com/android/android-gridview-example/

And don't forget call adapter.notifyDataSetChanged();
