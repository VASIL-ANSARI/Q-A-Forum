You're searching for a non-existing ID.

First you're setting the Id of the control to just a number (j) and later you'll try to find it with with a search to "chk"+the number j.

2 options here:

Change chk.ID = j.ToString(); to chk.ID ="chk" + j.ToString();

or

Change CheckBox cb = (CheckBox)Page.FindControl("chk" + j); to CheckBox cb = (CheckBox)Page.FindControl(j);

Personally I would go for the first option since naming a control with just a number wouldn't be a good idea.
