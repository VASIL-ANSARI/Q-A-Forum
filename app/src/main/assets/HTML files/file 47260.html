Your code should be like below. For double you should use %f. %d is for integers.

public class Assignment6 {

    public static void main(String[] args) {
        double count = 1;
        double kilo = 1;
        double pound = 2.2;

        System.out.println("Kilograms    Pounds");
        while(count<100)
        {
          System.out.printf("%f  %f", (kilo*count), (pound*kilo));
          System.out.println("");
          count++;
        }
    }
}
