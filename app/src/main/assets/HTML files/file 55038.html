If you're just talking IE (I know nothing of Typescript) and Visual Studio, you can use the following in VS:

    WinPath = System.Environment.GetEnvironmentVariable("SystemRoot")   ''MsgBox(WinPath)

    On Error Resume Next

    Dim ver = FileVersionInfo.GetVersionInfo(WinPath & "\system32\ieframe.dll")
    Dim IE = Val(ver.ProductVersion) ''MsgBox(IE)

This will return the local machine IE version - is this what you want?

If not, Javascript can detect which browser it is being used by with this script:

var browser=navigator.userAgent;
var IE=browser.search(/(MSIE ?[7891]|Trident|Edge\/)/i);    //=-1: not MS
if (IE!=-1) {
    T=browser.substr(IE,7);
    if (T.search(/^M/)==0) IE=parseInt(T.substr(4));
    else if (T.search(/^T/)==0) IE=11;
    else IE=12; //Edge
}
else IE=13; //not IE,Edge

Put it your onload function and react accordingly (OK, so it only does IE version, Edge, or other... but ...)
