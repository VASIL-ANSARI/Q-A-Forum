It looks like you are not completing the OAuth flow.

Leave off access_token in your initializer, this token is unique to the user that is logging in/authorizing your application with Instagram.

The redirect_uri should be your callback uri:

http://localhost:3000/auth/callback

This is where the temp code is exchanged for a access_token. It should not be result/index. The redirect_uri must also match the one entered for your OAuth Application on the Instagram Developer page.

http://instagram.com/developer/clients/manage/

Everything else seems about right. I recommend visiting the Instagram Ruby Gem github page, there is some great information there.

https://github.com/Instagram/instagram-ruby-gem
