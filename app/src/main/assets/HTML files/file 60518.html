See this excerpt from the pywinauto source code:

title_regex = re.compile(title_re)
def _title_match(w):
    t = handleprops.text(w)
    if t is not None:
        return title_regex.match(t)
    return False

The return title_regex.match(t) line means that the regex is passed to the re.match method, so, the regex pattern is anchored at the string start.

To match for a partial match, you need to start the pattern with .*: ".* FireFox ".
