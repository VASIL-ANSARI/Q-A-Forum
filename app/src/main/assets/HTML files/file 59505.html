You should check if parent is null or not before defreferencing.

convert signature of your getParent as:

Node* Node::getParent()

Node* Node::getParent() {
    return parent;
}

And in your application whenever you access it, check first.

Node * parent = getParent();
if(parent==nullptr){
    cout << "parent is null\n";
    raise error;
}else{
    // do whatever you want
}
