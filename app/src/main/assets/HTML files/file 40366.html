pattern is a two-dimensional array. So you need to provide two indices to access it's values.

If you want to populate pattern with random values you can use two for loops:

for(int i = 0; i<pattern.GetLength(0); i++) 
{
     for(int j=0; j<pattern.GetLength(1); j++)
     {
          pattern[i,j] = gen.Next(0,5);
     }
}
