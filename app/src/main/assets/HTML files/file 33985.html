Use min-height: 100% instead of height: 100% to allow the wrapper to grow beyond the browser viewport height:

#content-wrapper {
  width: 100%;
  float: none;
  position: absolute;
  box-shadow: none;
  overflow: visible;
  min-height: 100%;
  background: red;
}
