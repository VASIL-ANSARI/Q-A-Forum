If I understand your question correctly, you want "Voorraad" text box to be filled with a random number when there is a selection in the "Kies een product" list box correct?

If so, you don't need to use an inline js trigger to engage it. Here is a jquery script that will do the job for you:

$('.medicijnen').change(function(){
    var stock = Math.floor(Math.random() * 50) + 1;
    $('#medicijnen').val(stock);
});

Also, here is the JSlint of the code working (note I had to change some of the text to understand what was going on) - http://jsfiddle.net/t2D6S/
