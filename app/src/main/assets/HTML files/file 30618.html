Price in the given json (http://services.runescape.com/m=itemdb_rs/api/catalogue/detail.json?item=385) contains ,.

... "current":{"trend":"neutral","price":"1,844"},...
                                           ^

Remove , before call String#to_i.

"1,844".to_i
# => 1
"1,844".gsub(',', '').to_i
# => 1844
