You keep overwriting message in this loop:

for (int i = 0; i < scoresArray.Length; i++)
{
    message = scoresArray[i].ToString() + "\n";
}

So it's only ever going to show the last value. You probably want to append to it instead:

for (int i = 0; i < scoresArray.Length; i++)
{
    message += scoresArray[i].ToString() + "\n";
}
