This is because your CSS styles refer to .pika-image, which is a <div> dynamically created by your plugin. When the page loads, the images are contained within a <ul> of id="pikame", which is not affected by your CSS.

To get this "Flash of Unstyled Content" to go away, you'll need to add in CSS to target these images before the slideshow loads. I suggest turning off Javascript and then writing CSS to make them look the way you want.

#pikame {
    list-style: none;
}

is a good start.
