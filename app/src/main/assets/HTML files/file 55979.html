If you need to do it in CSS only, how about changing the image opacity to 0 and adding the background image to the logo link instead?

body.page-id-90 a.fusion-logo-link {
    background: url('http://praktijkvanginkel.nl/new/wp-content/uploads/2016/06/huidvitaal.png') top left no-repeat;
    background-size: 100%;
}

body.page-id-90 .fusion-logo img {
    opacity: 0;
}

Setting the background size for the link should also fix your sticky header problem too. Hope that helps!
