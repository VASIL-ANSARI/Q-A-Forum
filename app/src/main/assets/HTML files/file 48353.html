You were very close! You just need to set the value of the select with the abs value of 11 like so:

$("#map_abs").click(function (event) {
   $('#strength_exercise_muscle_group_id').val(11);
});

If you are listening for change events on the selects, you can even trigger one when you change the value by doing $('#strength_exercise_muscle_group_id').trigger('change'). Hope this helps.
