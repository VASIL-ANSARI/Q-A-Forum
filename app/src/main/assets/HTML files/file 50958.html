Since list items render as html options, you could add a bit of CSS. 

<head runat="server">
    <title>My Page Title</title>

    <style>
        option {
            padding-bottom: 15px;
            padding-top: 15px;
        }
    </style>

If you want to apply it to a specific listbox you can do this:

  <style>
       #ListBox1 option {
            padding-bottom: 15px;
            padding-top: 15px;
        }
    </style>
