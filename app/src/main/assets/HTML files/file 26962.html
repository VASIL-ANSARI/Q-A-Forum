Changing the text-anchor set by xcharts as a style in javascript :

This works on the markup in the question, changing text-anchor from middle to start - try it yourself :

var nodes=document.getElementsByTagName('g')[0].childNodes;
for (var i=0;i<nodes.length;i++) {
    if (nodes[i].tagName=='text') {
        nodes[i].style.textAnchor='start';
    }
}

Yes, color doesnt work on <text> elements - use stroke instead :

nodes[i].style.stroke='#ff0000';

-- makes the text red. <text> uses stroke and fill
