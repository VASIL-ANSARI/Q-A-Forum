Since you want to append 3 new elements to your HTML page, you will anyway need to either create 3 new elements or clone them. Instead of creating an element and cloning it, you can simply instruct jQuery to generate a new element every time:

var pattern = '<img src="img.png"/>';
$(pattern).appendTo('#ulSelector1 li');
$(pattern).appendTo('#ulSelector2 li');
$(pattern).appendTo('#ulSelector3 li');

It will create 3 new DOM elements and append them to corresponding elements.
