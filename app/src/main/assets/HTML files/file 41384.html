Reference & is used in declarations while operator & is used in expressions.:)

Consider example

#include <iostream>


int main()
{
    int x = 10;
    int * const &r = &x;

    std::cout << "x  = " << x << std::endl;
    std::cout << "*r = " << *r << std::endl;

}

The output is

x  = 10
*r = 10

In statement

    int * const &r = &x;

in its left part there is declaration of reference &r and in the right part there is initialization expression &x
