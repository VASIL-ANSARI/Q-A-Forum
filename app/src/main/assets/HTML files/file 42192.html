The VertexClustering object returned by cl.as_clustering() has a crossing() method - this gives you a Boolean vector which contains True for edges that are between clusters and False for edges that are within a cluster. You can easily extract the indices of the crossing edges like this:

cl = g.community_fastgreedy().as_clustering()
crossing_edges = [index for index, is_crossing in enumerate(cl.crossing()) if is_crossing]

Then you can simply take each edge and put its endpoints in a set:

boundary = set()
for edge in g.es[crossing_edges]:
    boundary.update(edge.tuple)
