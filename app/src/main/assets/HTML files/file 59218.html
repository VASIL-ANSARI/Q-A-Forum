I assume you are talking about responsive web design

http://www.w3schools.com/html/html_responsive.asp

https://www.youtube.com/watch?v=BIz02qY5BRA (I learned a lot from here)

This basiclly makes the website respond to the window/device size. You can do this by making a website then using media queries. For example

Html file

<p class="color"> Hello </p>

CSS file

color {color: blue;}

@media only screen and (min-width: 800px) {
color {color: red;}
       }

This would make it so when the width of device/window will reach 800px it will change "Hello" to red instead of blue. But this is just a small example, you can do much more after knowing the basics!
