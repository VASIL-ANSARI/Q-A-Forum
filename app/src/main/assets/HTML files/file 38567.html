I use a similar template engine, and i encountered (some time) similar problems... are you sure that the values that you are replacing are strings? If not cast them as strings.

if ( isset ( $this->parameter ) )
{
    foreach ($this->parameter as $key => $value) {
        str_replace('{' . $key . '}', (string)$value,  $this->content);
    }
}
