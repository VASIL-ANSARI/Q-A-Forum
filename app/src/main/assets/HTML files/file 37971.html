What you need is a scope:

scope : find_by_new_style_id, -> (new_style_id){ where( 'style_id=?', new_style_id ) }

If you need to use it in more than one class, you can use a concern.

module StyleScope
  extend ActiveSupport::Concern
  included do
    scope : find_by_new_style_id, -> (new_style_id){ where( 'style_id=?', new_style_id ) }
  end
end
