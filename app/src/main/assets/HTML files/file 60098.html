Use XML, there is lots of support and documentation on how to do this online. Another arguably 'cleaner' result would be JSON. This uses a simple key value pair relationship which is similar to the example you posted. But both are good.

XML Example:

<?xml version="1.0"?>
<catalog>
   <book id="bk101">
      <author>Gambardella, Matthew</author>
      <title>XML Developer's Guide</title>
      <genre>Computer</genre>
      <price>44.95</price>
      <publish_date>2000-10-01</publish_date>
      <description>XML example</description>
 </book>
</catalog>

JSON Example:

{
   "id": 1,
   "name": "A green door",
   "price": 12.50,
   "tags": ["home", "green"]
}
