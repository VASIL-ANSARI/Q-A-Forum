The reason the image button is not aligned, is because the content inside it is larger than text, there are two solutions to this:

  1. Decrease the height of the image inside the button
  2. Increase the height of the text buttons by manually calculating the height of the image button and matching it (which can be achieved by height:100px in .mybuttons.

Regarding the second problem (same width), set width: 90px;.

In .myButton can achieve same width across buttons (change the 90 to whatever number you feel is best).
