Because if use:

df[[True, False, True]]

it is called boolean indexing by mask:

[True, False, True]

Sample:

df = pd.DataFrame({'A':[1,2,3],
                   'B':[4,5,6],
                   'C':[7,8,9]})

print (df)
   A  B  C
0  1  4  7
1  2  5  8
2  3  6  9

print (df[[True, False, True]])
   A  B  C
0  1  4  7
2  3  6  9

Boolean mask is same as:

print (df.B != 5)
0     True
1    False
2     True
Name: B, dtype: bool

print (df[df.B != 5])
   A  B  C
0  1  4  7
2  3  6  9
