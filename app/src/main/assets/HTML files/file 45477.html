I've done it with a simple storeEval command:

<!-- jump to the footer, and get the scroll position's Y co-ordinate offset -->
<tr>
    <td>click</td>
    <td>link=Skip to footer</td>
    <td></td>
</tr>
<tr>
    <td>storeEval</td>
    <td>this.browserbot.getUserWindow().pageYOffset</td>
    <td>footerScrollPosition</td>
</tr>

In the eval commands, this.browserbot gets you the browser instance that is running your site, rather than the Selenium test runner browser instance.

You can then perform assertions on the resultant value with an assertEval command:

<!-- assert that the footer Y position is greater than the main content Y position -->
<tr>
    <td>assertEval</td>
    <td>${footerScrollPosition} > ${contentScrollPosition}</td>
    <td>true</td>
</tr>

(where I'd previously performed another storeEval to create the variable currentScrollPosition)
