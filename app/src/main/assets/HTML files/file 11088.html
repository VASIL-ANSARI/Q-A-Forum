Assuming your link is being rendered like this:

link_to "Delete", user_path(user), method: :delete

You'll need to ensure that jQuery and jQuery UJS are being required.

This can be done with require calls from your application.js file

*= require jquery
*= require jquery_ujs

To add a confirm message.

link_to "Delete", user_path(user), method: :delete, confirm: "Are you sure?"
