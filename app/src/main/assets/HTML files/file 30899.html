First, how to include regular expressions in a Python program:

import re

if re.match(pattern, string_to_match):
    # there is a match
else:
    # there is no match

Then, the first regular expression you need. As you haven't given any code, and this question smells of homework I won't give you the others.

 re.match(r"^[aeiou].{,3}[aeiou]$", string_to_match)

Explanation:

The ^ and $ ensure that the match is from the beginning to end of the string. The [aeiou] checks for a character that is a vowel. The .{,3} checks for up to 3 repetitions of any character (represented by .).
