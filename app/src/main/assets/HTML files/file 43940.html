The reason is that hiding the element fire its mouse leave function because it's leaving the element itself while disappearing.

You can add a wrapping element like, 

<div id="demo">
    <div id="test"></div>
</div>

and attach the event handlers on it.

Code:

$('#demo').mouseover(function () {
    $('#test').hide();
});
$('#demo').mouseout(function () {
    $('#test').fadeIn(500);
});
