Change your rules so that they look like this:

RewriteCond %{THE_REQUEST} ^\w+\ /(.*)\.php(?:\?page=([^&\ ]+)|)\ HTTP/
RewriteRule ^ http://%{HTTP_HOST}/%1/%2? [R=301,L]

RewriteCond %{DOCUMENT_ROOT}/$1.php -f
RewriteRule ^(.*)/([^/]+)$ $1.php?page=$2 [L]

RewriteCond %{DOCUMENT_ROOT}/$1.php -f
RewriteRule ^(.*)$ $1.php [L]
