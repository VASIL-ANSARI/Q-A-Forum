There are a number of things wrong here.

  1. There is no span element in your buttons. button span style isn't doing anything for you. If you want to hide the text, I suggest using color: transparent;.

  2. You need to apply a different image for each button, which means you need different selectors.

    button#previous { background-image: url('prev.jpg'); }

    button#next { background-image: url('next.jpg'); }

  3. Generally, applying styles to a button won't affect listeners added in js. Something must be wrong in your js if the buttons aren't working anymore. Please post the relevant code if you'd like help with this.
