--EDIT-- In your .html file, in the <head> tag, make a <style> </style> tag. Next edit your Iframe and include a class="iframe".

Next add the following lines in <style>.

.html{
width:100%;
height:100%;
}
.body{
width:100%;
height:100%;
}

This sets the width and height of the page to 100%. Now add this underneath the .body{} attribute

.iframe{
width:80%;
height:600px;
margin:100px auto 0 auto;
}
