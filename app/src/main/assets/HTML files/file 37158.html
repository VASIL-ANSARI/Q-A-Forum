I think you have a problem with redefining your functions - I forget the term, but as all functions defined in $rootScope are available in $scope your are probably screwing up the original $rootScope version.

For example, a function defined here in $rootScope like this:

$rootScope.hi = function() {
    console.log('hi');
}

is available in $scope like this:

$scope.hi();

You can either totally leave out the $scope version or rename it. I think this will fix your issue.
