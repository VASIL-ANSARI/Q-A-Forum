Here is an example (using VBA)

1- Create a UserForm1 - custom error message

2- Have the following code in the userform

Private Sub CommandButton1_Click() 'Raise Error
    On Error Resume Next
    Call Err.Raise(666, "Test", "Test")
    Unload UserForm1
End Sub

Private Sub CommandButton2_Click() 'Ignore Error
    On Error Resume Next
    Call Err.Raise(667, "Test", "Test")
    Unload UserForm1
End Sub

3- In Module1

Sub Test()
    If Err.Number = 666 Then
        MsgBox "error happened"
    End If
End Sub

4- In 'ThisWorkbook'

Private Sub Workbook_Open()
    UserForm1.Show
    Call Test
End Sub
