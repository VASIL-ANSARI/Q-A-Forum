"putting the code: srand(timeDelay) in a for loop". You don't put srand() in a loop. You call it once at the beginning of your program, typically with a time argument, so the random sequence you get from rand() is different each time you run the program.

#include<stdio.h>
#include<stdlib.h>
#include<time.h>

int main()
{
    int i;
    srand ((unsigned)time(NULL));
    for (i=0; i<5; i++)
        printf ("%6d", rand());
    printf ("\n");
    return 0;
}
