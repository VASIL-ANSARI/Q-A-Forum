Here's the right way to do this. You'd use jQueryUI's built-in callback functionality.

http://jsfiddle.net/isherwood/73B4f

<style>
.sidebar-img {
    display: none;
}
</style>

<div id="container">
    <div id="tabs">
        <div class="sidebar">
            <p>Basic description</p>
            <div class="sidebar-img" style="display: block;">Image 1</div>
            <div class="sidebar-img">Image 2</div>
            <div class="sidebar-img">Image 3</div>

            ...
        </div><!-- close content -->
    </div><!-- close tabs -->
</div><!-- close container -->

<script>
$('#tabs').tabs({
    activate: function (event, ui) {
        $('.sidebar-img').hide();
        $('.sidebar-img').eq( ui.newTab.index() ).show();
    }
});
</script>

http://api.jqueryui.com/tabs/#event-activate
