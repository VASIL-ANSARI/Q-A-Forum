This plugin has an awesome documentation where you can find exactly what you are looking for here and maybe here. The MomentJS-Docs might help you too.

$('#calendar').fullCalendar({
    defaultView: 'timelineYear',
    defaultDate: moment()
});

or

$('#calendar').fullCalendar({
    defaultView: 'timelineYear',
}).fullCalendar('today');

(both untested)
