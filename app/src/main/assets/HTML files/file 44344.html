You need to select the adjacent label element:

Updated Example

input[name*="xr"] + label {
    max-width: 40px;
    color: #fff;
}

The attribute selector was also wrong since you were trying to select an input element with a name of xr. You could use the attribue selector [name*="xr"] to select name attributes that contains xr.

I would suggest changing your name attributes, though.

Updated Example

<input type="radio" id="radio2" name="radios-xr" value="false">

input[name="radios-xr"] + label {
    max-width: 40px;
    color: #fff;
}
