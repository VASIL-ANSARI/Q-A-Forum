It is completely possible to distribute a Framework using Cocoapods. Look at how PSPDFKit is doing it.

Basically, instead of linking to a Github repo, their source url point to a .zip archive containing the Framework (with obviously a static library inside of it). The only source files distributed are the headers.

The interesting part in the podspec is the vendored_frameworks variable, which will tell Cocoapods to keep the Framework as part of the pod.
