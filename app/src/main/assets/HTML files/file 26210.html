Bit shifting is very easy in any arbitrary number of bits. Just remember to shift the overflow bits to the next element. Thats all. Below is a shift left 3 bit example

unsigned __int64 i1, i2, i3, o1, o2, o3; // {o3, o2, o1} = {i3, i2, i1} << 3;

o3 = i3 << 3 | i2 >> (32 - 3);
o2 = i2 << 3 | i1 >> (32 - 3);
o1 = i1 << 3;

Similar for shift right

Edit:

It seems that you're using base 109 for your large number, so binary shifting does not apply here. "Shifting" left/right N digits in a base B is equivalent to multiplying the number by BN and B-N respectively. You have only one solution, that's multiplying the number by 2591. If you want to shift like binary, use a base that is a power of 2 like base 232
