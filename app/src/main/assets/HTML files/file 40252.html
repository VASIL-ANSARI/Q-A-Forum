When you are using a table layout for the parent div, you should use the display:table-cell for the child divs. This will give you an equal spacing by default for any amount of children.

See jsFiddle here: http://jsfiddle.net/83gdw0uq/

HTML:

<div id="all">
    <div class="sub_all"></div>
    <div class="sub_all"></div>
    <div class="sub_all"></div>
    <div class="sub_all"></div>
    <div class="sub_all"></div>
    <div class="sub_all"></div>
    <div class="sub_all"></div>
</div>

CSS:

#all {
    background:#ccc;
    width: 100%;  
    display:table;
    border-spacing: 10px 5px;   /*first parameter is horizontal spacing / second is vertical spacing*/
}

.sub_all {
    background:#ff0000;
    display:table-cell;
    height:30px;
}
