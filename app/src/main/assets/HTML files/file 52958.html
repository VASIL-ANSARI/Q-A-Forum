Setting 'numberposts' => -1 means to get all records. You have to set for the first 10 records (0-9 records)

'numberposts'       => 10,
'offset'            => 0,

And for next 10, (10-19 records)

'numberposts'       => 10,
'offset'            => 10,

And for next 10, (20-29 records)

'numberposts'       => 10,
'offset'            => 20,
