You can run the pages on Google Chrome and just observe the console for errors(if there are any). This is because if the pages built in phonegap, run properly on Chrome browser then they will run fine on emulator or actual device. You can activate the console by pressing the F12 key.

Also you can try using rel="external" at the end of <data role=button> like
<data role=button.....rel=external>. It may help to open the pages properly.
