The "problem" with the displaytag is that it tries to do in memory pagination. It means that you are going to store everything in the session. Using displaytag you could do something called partial list support. In that case you have to send a list of PaginatedList as the doc says:

If the object passed to the displaytag as a list is in fact an instance of org.displaytag.pagination.PaginatedList (and not a collection), displaytag will assume that the list is externally sorted and paginated. Implementing this interface is trivial, and an adapter over existing classes can easily be written. See the org.displaytag.pagination.PaginatedList javadocs for more details.

you will have urls like this

http://foo.bar.com/context/requestUri?sort=name&dir=asc&page=5&originalParameters=originalValues

for further details take a look at the doc:

http://www.displaytag.org/1.2/tut_externalSortAndPage.html

If you don't want to use this technique you could use another alternative with more freedom. You could fetch directly what you want from the db. Take a look at this page (it is not exactly as you wanted it), it will give a basic idea of how to do it: http://theopentutorials.com/examples/java-ee/jsp/pagination-in-servlet-and-jsp/ . The main points are:

  1. Having parameter that tells you in which page you are:

    if(request.getParameter("page") != null) page = Integer.parseInt(request.getParameter("page"));

  2. With that parameter you fetch the proper data

    List list = dao.viewAllEmployees((page-1)*recordsPerPage, recordsPerPage); int noOfRecords = dao.getNoOfRecords(); int noOfPages = (int) Math.ceil(noOfRecords * 1.0 / recordsPerPage); request.setAttribute("employeeList", list); request.setAttribute("noOfPages", noOfPages); request.setAttribute("currentPage", page);

  3. Then, in your jsp you controls your links for navigation
    to the previous page

    <c:if test="${currentPage != 1}">
    <td><a href="employee.do?page=${currentPage - 1}">Previous</a></td>
</c:if>

the rendering of the table is exactly as you had it before: <display:table ...> </display:table>

and the 'next one' link 

    <c:if test="${currentPage lt noOfPages}">
        <td><a href="employee.do?page=${currentPage + 1}">Next</a></td>
    </c:if>
