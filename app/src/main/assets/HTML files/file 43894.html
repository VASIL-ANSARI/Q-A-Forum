Static class variables may not be initialized with expressions in PHP.

Here is one way to split the string via HEREDOC-Syntax, so long as it may contain additional spaces and linebreaks:

<?php
class JKWerte {
   public static $purifyoptions = array(
     'HTML.Allowed' => <<<EOT
     table, thead, tbody, 
     tfoot, th, tr, td,
     img[src|alt], div,
     span, p, br, ul, ol,
     li, *[class], *[style],
     *[height], *[width], h1,
     h2, h3, h4, a[href|title],
     b, i, em, strong
EOT
   );

   public function test() { }
}

var_dump(JKWerte::$purifyoptions);
?>

Outputs:

array(1) {
  ["HTML.Allowed"]=>
  string(213) "     table, thead, tbody, 
     tfoot, th, tr, td,
     img[src|alt], div,
     span, p, br, ul, ol,
     li, *[class], *[style],
     *[height], *[width], h1,
     h2, h3, h4, a[href|title],
     b, i, em, strong"
}

Alternatively you can define the content of your static variable later, e.g. after the class definition:

<?php
class JKWerte {
   public static $purifyoptions = array(
     'HTML.Allowed' => ''
   );

   public function test() { }
}

JKWerte::$purifyoptions['HTML.Allowed'] = 'table, thead, tbody, '
     .'tfoot, th, tr, td, '
     .'img[src|alt], div, '
     .'span, p, br, ul, ol, '
     .'li, *[class], *[style], '
     .'*[height], *[width], h1, '
     .'h2, h3, h4, a[href|title], '
     .'b, i, em, strong';

var_dump(JKWerte::$purifyoptions)
?>

Outputs:

array(1) {
  ["HTML.Allowed"]=>
  string(172) "table, thead, tbody, tfoot, th, tr, td, img[src|alt], div, span, p, br, ul, ol, li, *[class], *[style], *[height], *[width], h1, h2, h3, h4, a[href|title], b, i, em, strong"
}
