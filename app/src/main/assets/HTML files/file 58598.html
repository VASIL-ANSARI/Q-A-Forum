I'm not so clear what you want for an end product, but I can tell you that's not how custom filters are defined. the format is something like

    var app = angular.module('buttonApp',[]);
    app.filter('myFilter', function() {
    return function(x) {
        //filter goodness in here
      }    
    })

You can find the official documentation for filters here https://docs.angularjs.org/tutorial/step_11
