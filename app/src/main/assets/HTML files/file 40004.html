<wicket:message key="messageKey">
   <a wicket:id="link">...</a>
</wicket:message>

doesn't look correct. It should rather be:

<a wicket:id="link"><wicket:message key="messageKey"/></a>

because I guess you want the label to be clickable. A label is a text (i.e. HTML text node), so it cannot have children HTML elements.

Same is valid for the Java side - the Link should be the parent of the Label.
