private Map<Class, Factory> services = new HashMap<>();
private Map<Class, Object> constants = new HashMap<>();

If you wanna use this structures you can declare it like that:

private Map<Class<?>, Factory<T>> services = new HashMap<>();
private Map<Class<?>, Object> constants = new HashMap<>();

Remember to put Factory<T> to catch the type. Using Class provides you to use AnyClass.class as Key

Btw you can put all in one Map like private Map<Class<?>, Object> and at the get function cast it if it's a Factory or not ;)
