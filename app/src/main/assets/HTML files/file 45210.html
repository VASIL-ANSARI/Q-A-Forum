The foreach loop should be:

foreach($myMenu as $row){
    $tr = "<tr><td>". $row->name ."</td><td>". $row->price."</td><td>". $row->multiplier[1]. "</td></tr>";
    $body .= $tr;
}

You were using the wrong syntax for accessing an object property (=> instead of ->), and you were also accessing the wrong variable -- $myMenu is the array of all menu items, while $row is the current item in the loop. And you were also using the same variable $row for both the iteration and the temporary string being added to $body.
