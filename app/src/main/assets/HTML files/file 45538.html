Is there any particular reason for you to submit a form using jQuery?

Eliminate the jQuery part, add an "action" into your form and see what'll happen.

Your form should now be:

<form action="/api/sondages/" method="POST" id="formsondages" name="formsondages" style="text-align:center;">

Edit: if that's not working, you can add a GET method into your route. 

@app.route('/api/sondages/', methods=['GET', 'POST'])
