Actually the class like row, col does not contain margin-bottom or padding-bottom

Inside the <div class="col-sm-6 "> you were using <h3> for assigning values, or you can use any other tags also. In that assigning tag use the style="margin-bottom:0px;padding-bottom:0px;" This may solve your problem

Like this

<div class="col-sm-4 col-lg-4">
  <div class="row">
     <div class="col-sm-6 ">
         <h3 style="margin-bottom:0px;padding-bottom:0px;">Total</h3>
     </div>
     <div class="col-sm-6 ">
         <h3 style="margin-bottom:0px;padding-bottom:0px;">$1,618.20</h3>
     </div>
  </div>
</div>

or you can use this class

h3 {
margin-bottom:0px;
padding-bottom:0px;
}
