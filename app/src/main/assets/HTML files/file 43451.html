When you send an Ajax request, you need to listen for the state change.

Your code is asynchronous, hence listening for state change is required. xml.open("GET", "setverfile.txt",true); // The true parametter is for async

If you did xml.open("GET", "setverfile.txt",false); // The false parametter is for non async.

Then your code should work.
