If I understand correctly, you stored the data like:

id  spinner_data    comment
--  ------------    -------
0   A1,A2,A3        my 'A' data
1   B2,B4,B6        my 'B' data
2   C3,C6,C9        my 'C' data


So when you load and iterate over the cursor:

Cursor cursor = dbHelper.getAllRows();

It's giving you exactly what you asked for...a single row:

0   A1,A2,A3        my 'A' data


And your spinner is showing the text as "A1,A2,A3"

If you want your existing code to work, you'd have to store each string as separate record:

id  spinner_data    comment
--  ------------    -------
0   A1              my 'A1' data
1   A2              my 'A2' data
2   A3              my 'A3' data


Alternatively, a (relatively) simple fix would be to switch to an ArrayAdapter. What you could do is get the column data as a string

String data = cursor.getString(COLUMN_SPINNER_DATA); // A1,A2,A3

Parse that string (data) into a list of strings using some string tokenizer, I prefer guava's Splitter:

// Split using the comma (,) as a separator
Splitter spl = Splitter.onPattern(",");     
List<String> lstData = spl.split(data);

And now, instead of the SimpleCursorAdapter, instantiate an ArrayAdapter:

ArrayAdapter<String> adapter =
            new ArrayAdapter<>(SomeActivity.this, // This is your context
                    android.R.layout.simple_dropdown_item_1line, // This is your layout
                    lstData); // This is your data
