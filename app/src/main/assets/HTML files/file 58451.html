You should disable the form from submitting by capturing its state. You can do this by adding a class to the form and if it has that class then stop the event.

 $(document).on('submit', '.js-form', function(e) {
     if ($(this).hasClass('disabled')) {
         // Form is disabled, stop the event
         e.preventDefault();
         e.stopPropagation();
         return false;
     } else {
         // Allow first submit and add class to stop subsequent
         $(this).addClass('disabled');
     }
 });

This then does not clear or disable any button, allowing all the data to come through.
