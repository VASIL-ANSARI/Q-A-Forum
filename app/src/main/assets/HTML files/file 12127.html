You might try expressing the clef as a list of lines and spaces.

def print_chord(note_list):
  clef = ['-----',
          '     ',
          '-----',
          '     ',
          '-----',
          '     ',
          '-----',
          '     ',
          '-----']

  for note in note_list:
    # insert the notes into the clef
    clef[note] = clef[note][:2] + 'o' + clef[note][3:]

  # lower notes have lower values, so reverse the list
  clef.reverse()

  for line in clef:
    # show the result to the user
    print line

Then you could pass in a list of notes like so:

print_chord([0, 4, 5])

and get the following output:

-----

-----
  o   
--o--

-----

--o--
