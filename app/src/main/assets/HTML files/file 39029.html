The problem is the way you are querying in the find part of the find_or_create. I suspect what you are trying to do is only create the ShippingCategory if it doesn't exist. Is the name unique? If so you could do:

ShippingCategory.
  create_with(name: name, size: size, price_transport: price_transport, 
    price_storage: price_storage).
  find_or_create_by(name: name)

Refer to the documentation of find_or_create_by for more info.
