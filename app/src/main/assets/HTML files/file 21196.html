Here is a cross-browsers method to catch the ENTER key:

$('#condition_pop').keydown( function(e) {
    var key = e.charCode ? e.charCode : e.keyCode ? e.keyCode : 0;
    if(key == 13) {
        e.preventDefault();
        alert("enter pressed");
    }
});

Tested on Chrome 24 : http://jsfiddle.net/PTauw/1/
