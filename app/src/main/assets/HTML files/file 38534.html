This is basic C syntax.

The statement does two things:

  1. Declare the type struct elementT.
  2. Create a type alias so that you can say element to mean struct elementT.

It's commonly written like this for brevity and convenience, but you can split it into its two declarations and make them separate:

struct elementT
{
    int data;
    struct elementT *link;
};

and

typedef struct elementT element;
