This is how you check for a null String value in java:

String a = ...;
if (a == null) {
    // ...
}

The following throws a NullPointerException because a is null:

String a = null;
if (a.length() == 0) {
    // ...
}

The following checks for an empty string, which is different than null! (And if a is null, it will also throw a NullPointerException)

String a = ...;
if (a.equals("")) {
    // ...
}

The following (similar to above), checks for a one character string containing a single space character. (And if a is null, it will also throw a NullPointerException)

String a = ...;
if (a.equals(" ")) {
    // ...
}
