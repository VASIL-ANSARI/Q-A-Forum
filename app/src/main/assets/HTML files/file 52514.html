An initial recommendation, is to change this:

var modelList = Model.ToList();

for this:

var modelList = Model.Where(x => x.SeasonTeamID == t.SeasonTeamID).ToList();

Therefore, modelList now will only have the players you are interested in, so you won't have to iterate every single one of them, and you will be able to eliminate this 'if':

if (modelList[i].SeasonTeamID == t.SeasonTeamID)
