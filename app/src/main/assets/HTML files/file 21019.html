Knockout exposes a function ko.applyBindingsToNode that does what you need. After creating the input element, you should set up the binding like this:

ko.applyBindingsToNode(input, function() {
    return {
        value: valueAccessor().value
    };
});

Since version 3.0 of Knockout, you can use this method also:

ko.applyBindingAccessorsToNode(input, {
    value: function() {
        return valueAccessor().value;
    }
});
