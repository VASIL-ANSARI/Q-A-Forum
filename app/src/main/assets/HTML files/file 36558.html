Couple of ways you can do this, either with a RewriteRule as you've attempted:

RewriteEngine On
RewriteBase /
RewriteRule ^(getData\/.*) http://www.siteB.com/$1 [P]

Or you can use ProxyPass:

ProxyPass /getData http://www.siteB.com/getData

Note that this would work for anything following getData, e.g. it would match getData/mydata.php or getData/some/other/path/file.php
