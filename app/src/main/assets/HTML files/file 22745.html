There are two regular expressions. The first is just /xyz/, which is used to test whether the function (coerced to a string) matches xyz. It does. Then, one of two regexen are returned (in this case the first because the test passes).

The first of them is word boundary followed by _super followed by another word boundary which can match at least "_super", " _super " and word._super.

The second matches zero or more characters (i.e. everything).
