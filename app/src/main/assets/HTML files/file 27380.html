As ButtonGroup is not a component, you cannot add your ButtonGroup to a JPanel. Instead add directly your buttons to your JPanel, e.g:

JPanel jPanel = new JPanel();
ButtonGroup group = new ButtonGroup();
btn1 = new JRadioButton("btn1 ");btn1.setSelected(true);
btn2 = new JRadioButton("btn2 ");
group.add(btn1 );
group.add(btn2 );

jPanel.add(btn1);
jPanel.add(btn2).

Hope it will be useful.
