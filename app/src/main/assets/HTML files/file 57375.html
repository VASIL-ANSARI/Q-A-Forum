You had width: auto on the .filter element, which make the element grow with content and not create a scroll. I changed it to width: 100% and now it does scroll.

I also added box-sizing: border-box; to make the padding: 17px 10px; value be counted within the width.

html, body {

  margin: 0;

}

.header{

  height: 180px;

  width: 100%;

}

.header .filters {

  background-color: whitesmoke;

  position: fixed;

  width:100%;

  margin-top: 130px;

  max-height: 50px;

  z-index: 1000;

  overflow-x: scroll;

  overflow-y: hidden;

  padding: 17px 10px;

  white-space: nowrap;

  box-sizing: border-box;

}



.header .filters a{

  margin: 0 10px;

  font-family: LatoWebLight;

  font-size: 16px;

  width: 150%;

  cursor: pointer;

}
