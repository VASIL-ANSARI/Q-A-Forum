For pre-iOS 6, looks like their website hardcodes a banner when you visit it. You can see it in Safari on Mac if you set your user agent to an iPxx agent (in the Developer menu.)

For iOS 6, they use a meta tag introduced by Apple: <meta name="apple-itunes-app" content="app-id=YOUR_APP_ID_HERE">. If you view the page source, you can see it.
