Your ListView should have some sort of data adapter, probably an ArrayAdapter, associated with it (you need to create this, it isn't automatic).

If you're using an ArrayAdapter, in the listener for your Volley request, add()/addAll() your results into the adapter. The adapter will automatically cause the ListView to refresh.

If you've created your own adapter type that doesn't derive from ArrayAdapter, you'll have to handle the notification of changes manually. That's what notifyDataSetChanged() does -- just call it after making your changes in the adapter, and it will refresh the view.
