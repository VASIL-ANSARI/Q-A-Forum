First off, make sure that you aren't running anything like Application.RequestUserAuthorization(UserAuthorization.Microphone) anywhere.

Second, according to this guy on the Unity forums, the only way to permanently disable asking for microphone permissions is by unchecking the Prepare iOS for Recording box in XCode and renaming the game. I believe that the renaming portion only applies if you have submitted to Apple, in which case you would have to do a factory reset on your device (or just rename), but I cannot be completely sure.
