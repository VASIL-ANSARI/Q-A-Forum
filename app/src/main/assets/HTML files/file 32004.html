1. getString() in an activity gets a string from resources. Referring to a resource id with the name of an index number looks suspicious. If I understood your code and question, this

    Cursor cursor = ourDatabase2.rawQuery(sql, new String[] {aFindex});

    should be

    Cursor cursor = ourDatabase2.rawQuery(sql, new String[] { String.valueOf(aFindexnum) });

    and you can remove the first problematic getString().
