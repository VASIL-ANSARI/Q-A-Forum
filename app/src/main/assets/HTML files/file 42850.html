jQuery doesn't have cookie support built in. If you grabbed that sample code from somewhere you will also need to include the cookie plugin:

  • https://github.com/carhartl/jquery-cookie

When you set the cookie in lines like these:

$.cookie('currentToggle', $("#secondaryfilter").hasClass('no-filters')?'on':'off' );

You are just creating a session cookie. This means that the browser will throw the cookie away when you close it. If you want this to be persisted next time they come back then you need to add in an expiry date like:

var currentToggle = $("#secondaryfilter").hasClass('no-filters')?'on':'off');
$.cookie('currentToggle', currentToggle, { expires: 7 });

Other than that I can't see any problem with your code, although I am wondering if you worded your question correctly as these two lines seem to contradict each other:
