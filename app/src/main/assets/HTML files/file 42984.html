This goes in your template index.php file:

<link href='http://fonts.googleapis.com/css?family=Open+Sans&subset=latin,greek' rel='stylesheet' type='text/css'>

And this goes in your template css file:

body {font-family: 'Open Sans', sans-serif;}

Also make sure you don't have any other fonts overriding your css such as h1, h2, h3, p, etc...
