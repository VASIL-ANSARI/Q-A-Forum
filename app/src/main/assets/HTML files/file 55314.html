The setHandles function is never running because it has no reason to; it's not called, and functions in JSX without () at the end don't run unless they're asked to.

I would suggest to throw your desired function into the componentDidMount() React method, so it would look like this:

class Register extends Component {
  componentDidMount() {
    // Do the thing
  }

  render() {
    ... 
  }
}

Here's React's doc on it: https://facebook.github.io/react/docs/component-specs.html#mounting-componentdidmount
