For a delay you specify in milliseconds, use setTimeout:

$("#item-1").animate({left:100},1000,function(){
   $("#item-2").animate({left:250},2000, function(){
      setTimeout(function(){
          $("#item-3").animate({left:100},1250)
      }, 2000);
   })
});

setTimeout basically says: "perform this function (first parameter), after the specified amount of time (second parameter)"
