Are you sure your teacher wanted the function to take the indexes and not values?

If it's indexes, your solution is right.

If it's values, a quick and dirty way would be to this:

def slice(alist, start, end):
    s = alist.index(start)
    e = alist.index(end)
    return alist[s:e+1]

You can add the check for inputs, and also modify the function if you don't want the greedy approach, as the index function returns the index of the first element it can find in the list starting from left (which mightn't be what you want for the end value).
