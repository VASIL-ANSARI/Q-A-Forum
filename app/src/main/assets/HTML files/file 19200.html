"...

I fixed this by increasing memory_limit & max_execution_time settings in both php.ini & .htaccess:

php.ini 

memory_limit = 256M

max_execution_time = 18000

.htaccess 

php_value memory_limit 256M

php_value max_execution_time 18000

"
