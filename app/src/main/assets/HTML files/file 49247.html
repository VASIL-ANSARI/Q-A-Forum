Somewhere in the app there must be a collection of Content objects. In the first code snippet, there is a variable allContent that presumably represents this full list of objects.

What you can do is iterate through all of these content objects to find the one whose beacon identifiers matches the ranged beacon. Something like this:

func locationManager(manager: CLLocationManager, didRangeBeacons beacons: [CLBeacon], inRegion region: CLBeaconRegion) {
  for rangedBeacon in beacons {
    for content in allContent {
      if let contentBeacon = content.beacon {

        if rangedBeacon.proximityUUID.UUIDString == contentBeacon.proximityUUID.UUIDString &&
           rangedBeacon.major.integerValue == contentBeacon.major.integerValue &&
           rangedBeacon.minor.integerValue == contentBeacon.minor.integerValue {
            // content now contains the content for rangedBeacon
            // TODO: do something with the matching content here
        }
      }
    }
  }
}

The above code is pretty brute force, and there may be a much more efficient way of doing this if you have your Content objects stored in CoreData or some other structure that is queryable by the beacon UUID/major/minor.
