As smdsgn commented on your question, without code you've already tried, it is difficult for others to completely understand what you're trying to accomplish. Regardless, presuming I understand your question correctly I can provide you a couple of pointers.

If you simply want to add an image to your menu bar link when it's hovered over, utilize CSS' :hover selector with the background attribute (or the background-image attribute, if you prefer) as I've done in this jsfiddle.

nav ul li:hover {
  background: #someColour url(yourImage.file) no-repeat left;
  background-size: 50px;
  cursor: pointer;
}

If you want more customization, such as hiding the link text when it's hovered over, you can implement some jQuery as I've done in this jsfiddle.

var hiddenText;

$('li').hover(function() {
  hiddenText = $(this).text();
  $(this).css({'background': '#someColour url(yourImage.file) no-repeat center','background-size': '50px'})
  $(this).text('');
}, function() {
  $(this).css({'background': '#someOtherColour'})
  $(this).text(hiddenText);
})
