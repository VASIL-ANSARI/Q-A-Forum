R.id.nextpage is in xml layout of the fragment not activity you cannot find it in activity directly.

best way would be to find this button inside fragment it self. and attach an onClickListener to it inside fragment only.

Now create an Interface like this:

interface FragmentCallback{
   void nextButtonClicked();
}

Make your activity implement it and write code to change view inside nextButtonClicked function inside activity.

Now inside Fragment and an instance variable of type FragmentCallback:

FragmentCallback callback;

and write a setter for it:

public void setFragmentCallback(FragmentCallback callback){
    this.callback=callback;
}

now in your activity when you create fragment call setFragmentCallback.

<FragmentObject>.setFragmentCallback(this);

onClick of the nextButton inside Fragment just call:

callback.nextButtonClicked();

Fragment:

public class FragmentViewOne extends Fragment implements OnClickListener{
    FragmentCallback callback;
    public void setFragmentCallback(FragmentCallback callback){
        this.callback=callback;
    }
    @Override
    public void onCreate(Bundle savedInstanceState)
    {
        super.onCreate(savedInstanceState);
    }

        @Override
    public View onCreateView(LayoutInflater inflater, ViewGroup container,
                             Bundle savedInstanceState) {
        // TODO Auto-generated method stub
        View myFragmentView = inflater.inflate(R.layout.fragmentview_one, container, false);
        myFragmentView.findViewById(R.id.nextpage).setOnClickListener(this);
        return myFragmentView;
    }
    @Override
public void onClick(View v) {
    callback.nextButtonClicked();
    }
}
