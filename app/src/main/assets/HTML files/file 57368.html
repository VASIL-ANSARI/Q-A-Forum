Try to declare con object as static in main activity where you getting connection, and then use it anywhere as per your requirement,

See below structure,

In Main Activity

   public static ApiConnection con;

On Getting Connection

    con = ApiConnection.connect(host);

Crate one method to get this connection in main activity as below

   public static ApiConnection getCon()
    {
        if(con != null)
           return con;
         else
           return null;
    }

Now use this object in any activity by below code

   ApiConnection con = = MainActivity.getCon();
   if(con != null)
    {
       // Use Your Logic Here
    }
