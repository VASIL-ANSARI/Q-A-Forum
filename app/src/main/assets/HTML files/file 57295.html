In C, arrays pass "as reference" (as a pointer, for more advanced C programmers) to functions. That means that any change the function makes to an array affects the array in the caller function.

That means, that my_array (after calling the function) now holds the sorted array (because the function modified it).

So, instead of a return value, you can just print my_array, like that:

int main()
{

    int i;
    int my_array[1000];

    for (i = 0; i < 1000; i++) {
        my_array[i] = rand();
    }

    shellsort(my_array, 1000);

    for (i = 0; i < 1000; i++) {
        printf("%d\n", my_array[i]);
    }

    return 0;
}
