You should not have any problem using session_start() in the landing page . You would just need to make sure your header script is included at the top of the page so that nothing else is sent to be browser before the call to session_start() . It is because :

Note : To use cookie-based sessions, session_start() must be called before outputing anything to the browser.

Above note is copied from : http://php.net/manual/en/function.session-start.php
