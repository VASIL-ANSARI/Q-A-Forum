if it's a one shot action, you may do

select 'update ' + table_name + ' set ' + column_name + ' = UPPPER(' + column_name + ');')
from information_schema.columns
where data_type in ('char', 'nchar', 'varchar', 'nvarchar') AND table_name = 'plats';

execute this in your dbms administrator tool (ssms probably), copy/paste the result and execute it.

Else, you'll need a stored procedure with dynamic sql.
