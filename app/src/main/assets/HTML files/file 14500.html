I don't think that you can create hashes in a resource file, you can only create arrays.

The best way to do this then would be to create a string array resource whose values would need to be parsed when you load them.

E.g. 

<string-array name="my_hash_hack">
   <item>"KEY1:VALUE1,VALUE2,VALUE3"</item>
   <item>"KEY2:VALUE1,VALUE2,VALUE3"</item>
</string-array>

I'm sure there are also other ways you could achieve the same thing but this gives you the general idea.
