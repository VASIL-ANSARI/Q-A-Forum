You cannot have a class literal based on a type argument such as T.class. To work around this, add a parameter to the test method that is of type Class<T>, so you can use a class typed at runtime.

Try:

public void test(Class<T> clazz) {
    // ...
    myDao = getDao(clazz);
    // ...
}

The caller of this method will pass in the Class of the type needed.
