Ok, so you want to save state for each div separately, which is logical. I can suggest you two ways:

1) You assign a unique id to each div.well

<div class='well ...' id='wellState_0'>
...
<div class='well ...' id='wellState_1'>
...

And then you save those ids in cookies:

$.cookie($(this).parents('.well').attr('id'), 'show')

2) You generate ids automatically with javascript via divs' numeration:

$.cookie(
    'wellState_' +
    $('.well').index($(this).parents('.well')[0]),
    'show'
)
