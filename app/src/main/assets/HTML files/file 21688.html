You have several elements with the same ID dd-mainnav. That's not allowed. Use multiple classes instead:

<li class="leaf7 dd-mainnav">

Once you do that (and rewrite your JS and CSS accordingly), add a test for the window's width inside the click handlers:

$('#categories-dd').on('click',function(){
    if ($(window).width > 657){
        $(this).toggleClass('active-main');
    }
});

But it might be easier to leave your JS as-is and just add new styles to your mobile stylesheet:

#categories-dd, #categories-dd.active-main {
    display: block; /* or whatever */
}
