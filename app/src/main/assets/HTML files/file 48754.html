There is one major problem: you're trying to determine the length of the input before receiving any input.

so switch the lines like this:

printf ("input\n");
fgets (input, sizeof (input), stdin);
n = strlen(input);

Then your for-loop ... why so heavy testing, wouldn't

for ( front = 0, back = n-1; front < back; ++front, --back)

be sufficient?

And at last: the rest of your code. Please think about it again. You want to:

  • print all characters from the beginning up to and including front
  • print all characters from back till the end
  • print a newline

Your code does nothing about this. But we're not here to solve your homework ;)

A last hint: there are two different types of stop-condition, and they depend on if your input length is even or odd. (My suggested test should be ok to catch'em both, tho).
