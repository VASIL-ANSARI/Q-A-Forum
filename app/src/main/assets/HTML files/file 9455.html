As you probably have found, this is a common bug in IE. One solution would be to wrap the faded element in an absolutely positioned element and have the faded element display inline-block. Here's the updated fiddle that uses this method.

UPDATE

Another option based on your comment is to do just that: iterate over the different elements you want to fade in. Here is an example where you add the fadeIn class to each element and then iterate over them individually. In that case, the javascript would be as follows: 

$("#hello .fadeIn").each(function(){
    $(this).hide();
    $(this).fadeIn();
});
