You're looking to use joins since you want to pull info from multiple tables.

If $starring is an array of names, you can do this:

if (!empty($starring)) {
    $query .= " INNER JOIN actors on actors.dvdId = dvds.dvdId WHERE";
    foreach( $starring as $name )
         $query .= " starring LIKE '%$name%' OR";
    substr( $query, 0, -3 );  //Remove last OR
}

The logic here isn't perfect (will match all DVDs with any of the supplied actors), but I know this will get you going in the right direction.
