You idea is basically correct, but remember that any header can be set only while there is nothing written to the output yet. Be sure that function with this header command will be called before any other html code will go.

And also, use double quotes for header contents if you want it to work correctly 

header("Location: messages.php?page=3&conversation_id=$conversation_id");

or

header('Location: messages.php?page=3&conversation_id='.$conversation_id);
