Visual Studio project files (.csproj) are just XML files. In your patch builder, you can then read the XML file to read the dependent DLL or projects and if it references the modified library (BaseProject in this case), build the project. You can find the dependent DLL under ItemGroup as shown below

<ItemGroup>
<ProjectReference Include="..\ClassLibrary1\ClassLibrary1.csproj">
  <Project>{323f9761-89f4-4405-b8b1-d6feb5b1ec89}</Project>
  <Name>ClassLibrary1</Name>
</ProjectReference>

The above is an excerpt from a .csproj file.
