The getImage method should return an Image, not an ImageIcon. You can get an ImageIcon from an Image easily.

ImageIcon imageIcon = new ImageIcon(image);

The getDie1Image should just be getDieImage. You can call the method twice. The getDieImage method should look like this.

// Obtain die image by calling result of the roll
public static Image getDieImage(Die die) {
    int dieVal = die.roll();
    Image face = die.getImage(dieVal);
    return face;
}
