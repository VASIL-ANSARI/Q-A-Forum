If I understand your query correctly, you want to get all users from a certain school, who are not being educated...

If so, I'd go the following route - given that your relations are all properly set up:

class User < ActiveRecord::Base
  def self.find_educated_by_school_name(school_name)
    Education.pluck(:user_id).joins(:schools).where('schools.name = ?', school_name)
  end
end

User.where('users.id NOT IN (?)', User.find_educated_by_school_name('school_name'))

Also keep in mind, that by convention myUser.find_by_* would mean "find something on a given user instance". Hence, it should make use of the user instance to find something.

Whereas User.find_by_* means "find users in general, by a certain criteria".

Hope it's clear what I mean!
