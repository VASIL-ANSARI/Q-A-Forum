Follow these steps.

  1. Open the project for your main program (your web project or your exe project ) in Visual Studio.

  2. Locate the "Solution Explorer" pane

  3. Right click on the "References" node and choose "Add Reference"

  4. Click "Browse"

  5. Select the DLL you wish to "import" and click OK

Congratulations, you have now "imported" all the non-internal, non-private classes and methods in your DLL.

Just instantiate and use.
