Two things:

$('#player').animate({
    top: e.pageY + 'px';
    left: e.pageX + 'px';
}, 800);

Has syntax errors. It should be:

$('#player').animate({
    top: e.pageY,
    left: e.pageX
}, 800);

Notice I left off 'px' since it isn't necessary.

You can see it working here: http://jsfiddle.net/VBzUw/
