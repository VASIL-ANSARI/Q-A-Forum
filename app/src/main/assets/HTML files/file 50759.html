So where exactly is the code for opening the popup?

You have to create a Javascript function which creates a new "window" class and opens it.

after that you have to add the onClick=">nameOfJSFunction<" attribute to element.

For example:

<a href="#" onClick="openPopup()">Open a new Popup</a>

and then the JS part:

<script>
function openPopup()
{
    newPOPUP = window.open('>LINKtoPOPUPfile<', '>TITLE<', '>OPTIONS<');
    newPOPUP.focus();
}
</script>

The file which the popup code is opening should contain the code/elements you want to see inside of the popup.
