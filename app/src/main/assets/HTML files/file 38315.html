Make sure you're using the same character set throughout your PHP and database interaction. A few things must be checked:

  1. Pick an appropriate character set. In this case I'll assume UTF8.
  2. Save your PHP file as UTF8.
  3. Add a query SET NAMES 'utf8' to your PHP immediately after opening the connection.
  4. Set your table and column char sets to UTF8. At the very least use CHARSET=utf8 in your table definitions.
