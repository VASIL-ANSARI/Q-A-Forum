That method is part of PFQueryTableViewController, a tableview subclass that Parse created to make it easier to display info in a tableview. However, the line [query orderByAscending:@"name"]; is not specific to that class, you can use that on any query. I would suggest adding a query to your collection view, ordering it how you wish, and displaying it in the collection. I don't think you should use the PFQTVC as it is meant to be used with tables.
