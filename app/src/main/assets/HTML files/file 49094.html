I rewrite your code in more portable way, it require c++11:

#include <cstdint>
#include <iostream>

int main()
{
    union {
        uint32_t t[2];
        uint64_t a;
    } val;
    val.t[0] = 1;
    val.t[1] = 2;
    std::cout << val.a << ", " << std::hex << val.a << '\n';
}

The output on my amd64 machine is:

8589934593, 200000001

and it seems to completely valid, equal to uint64_t(2) << 32 | uint64_t(1)
