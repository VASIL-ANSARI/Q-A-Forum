A couple of things you might like to think about.

For one, you're assuming that each token appears on a new line. That might be true for what you're doing; but it would be more flexible to use input.next() rather than input.nextLine(). That will read the next token, without caring whether it's on a new line or not.

To read in the next token and determine whether it's an integer or not, the easiest thing is to look at input.hasNextInt(). That will tell you, without advancing past the token, whether the next token can be parsed as an integer or not. If it is, you can read it with input.nextInt(); if not, read a String with input.next().
