No need of quotes at first parenthesis set in insert. Also note that md5 is not a good choise anymore to hash your passwords. Also try to use prepared statements to avoid SQL Injection. Furthermore, md5(md5($email)).$pass will produce a 32+ character string so check if you have a sufficient length at your password column.

Change:

$query = "INSERT INTO users ('fname', 'lname', 'email', 'password') VALUES ('".$fname."', '".$lname."', '".mysqli_real_escape_string($link, $email)."', '".md5(md5($email)).$pass."')";

To:

$query = "INSERT INTO users (fname, lname, email, password) VALUES ('".$fname."', '".$lname."', '".mysqli_real_escape_string($link, $email)."', '".md5(md5($email)).$pass."')";
