Actually it hides the navigation bar button in complete stack when you hide it in parentView. To hide navigation barButton, write your code in viewWillAppear of A viewController

self.navigationController.navigationBar.hidden = NO;
self.navigationItem.hidesBackButton = YES;

And in A viewController's viewWillDisappear, unhide the navigation button

- (void) viewWillDisappear:(BOOL)animated
{
    [super viewWillDisappear:animated];
    self.navigationItem.hidesBackButton = NO;
}

Now it will show up the navigation bar button in B viewController.

Update

Instead of hiding backButton on A viewController, set backButton nil 

self.navigationController.navigationBar.topItem.leftBarButtonItem = nil;

And add custom backButton on B viewController
