Put this in a file named fibonacci.py:

def fib(n):    # write Fibonacci series up to n
    a, b = 0, 1
    while b < n:
        print(b, end=' ')
        a, b = b, a+b
    print()

def fib2(n): # return Fibonacci series up to n
    result = []
    a, b = 0, 1
    while b < n:
        result.append(b)
        a, b = b, a+b
    return result

Then open an other file __main__.py in the same folder as fibonacci.py. The content of __main__.py should be:

import fibonacci
print(fibonacci.fib(3))

Then you can execute the __main__.py file and it should work.
