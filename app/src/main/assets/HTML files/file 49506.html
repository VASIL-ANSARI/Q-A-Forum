The code isbnNumber[100] checks the 101st element of the array. The loop will only exit if you enter 0 on the 100th iteration.

Also, it should be noted that isbnNumber[100] is not an element of the isbnNumber array. Array indexing begins at zero, so the last element in isbnNumber is isbnNumber[99]. Accessing elements past the end of the array is undefined behavior. If you were entering 0 on the 100th iteration, this is why the loop didn't exit.

Furthermore, you really should have something to prevent the loop from running past the end of the array. Here's how you would have the loop run until you enter 0 or you've hit the end of the array.

for (int i = 0; i < 99; i++) {
    printf("ISBN number: ");
    scanf("%d", &isbnNumber[i]);
    if (isbnNumber[i] == 0)
        break;
}
