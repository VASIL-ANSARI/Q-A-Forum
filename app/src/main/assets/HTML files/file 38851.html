You didn't specify a width style to the rod element that's why it becomes a round object (Absolute positioned elements are 0 width by default if it has no contents). Try adding a width or text inside the div and you will now see that it becomes a longer rod :)

Sample:

.rod {
    border-radius: 39px;
    border: 17px solid rgb(51, 51, 51);
    transform: rotate(-6deg);
    position: absolute;
    width: 300px;
}
