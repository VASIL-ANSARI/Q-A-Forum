You can use postsharp OnMethodBoundaryAspect to add additional call before and after each method, there are many ways to control where it should be applied.

For e.g. below code will add those additional call to only pubic class and public method on the namespace "MyNamespace.*" and only in debug mode.

#if DEBUG 
[assembly: Trace( AttributeTargetTypes = "MyNamespace.*", 
    AttributeTargetTypeAttributes = MulticastAttributes.Public, 
    AttributeTargetMemberAttributes = MulticastAttributes.Public )] 
#endif
