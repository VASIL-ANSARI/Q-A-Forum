You can use css calc for your propose. Here is the browser support for that.

#ContentPanel1 {
    text-align:center
}
#ContentLeft1 {
    background:Yellow;
    float: left;
    width: 80px;
}
#ContentRight1 {
    background:Blue;
    float: right;
    width: 80px;
}
#ContentMiddle1 {
    text-align:left;
    background:Orange;
    margin:0 auto; 
    display:inline-block
    width: calc(100% - 80px - 80px); // or width: calc(100% - 160px);
}
