opening PDF files is a little bit difficult. As I descriped in my request couple of weeks ago: Photoshop JavaScript: options to open a PDF as smart object

My workarround is to use an action (from the Actions panel) that opens the files (modal controle on, to set efficient PDF render) and also starts the script.

You could use the action to export as web JPG.

If you want to use the script, the File needs a full path.

var destFile = new File ("~/Desktop/" + "web-" + doc.name + ".jpg");

Also the parameters for exportDocument are needed (non optional).

doc.exportDocument(destFile, ExportType.SAVEFORWEB, options);
