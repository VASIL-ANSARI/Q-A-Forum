You should add id's to the anchors and then apply the functions with jquery

<p>Name : <a href="#" title="" id="yourName">Your Name</a></p>
<p>Gender : <a href="#" title="" id="myGender">My Gender</a></p>


 $('a#yourName').on('click', function() {
   $('#inputName').show()
 })

 $('a#myGender').on('click', function() {
   $('#inputGender').show()
 })

UPDATE To do inline edit you need to put the anchors and inputs into span tags with id's set to them and styling the input spans with display:none it should be hidden by default, then use the replaceWith() function to replace the anchors with the inputs html,

See the snippet below for an example.

$('a#yourName').on('click', function() {

  $('span#a1').replaceWith($('#span1').html());

})



$('a#myGender').on('click', function() {

     $('span#a2').replaceWith($('#span2').html());

    })
