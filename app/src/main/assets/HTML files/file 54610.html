One way under Windows is to start developing cordova Apps with Visual Studio TACO (Tools for Apache Cordova).

Start by downloading the free Visual Studio 2015 Community here and choose the "Cross Plattform Mobile Development" part to be installed. With that you have everything to start development.

Create a new cordova project in Visual Studio File->New Project and in the window choose on the left side Templates->Javascript->Apache Cordova Apps and give your first project a name and create it.

Now you have a "Hello World!" App which you can run on different plattforms.

This link will show you how to enhance your first App.

For sure you do not need Visual Studio and you can just install plain cordova here, but I found VS very helpful.

Hope this helps.
