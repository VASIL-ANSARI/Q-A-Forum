As you have discovered, the >= operator is not implemented for cv::Point instances. What you can do instead is construct a cv::Rect representing the bounding box of the person. Then use the Rect::contains() method to test each of your inner points. This should meet your comparison criterion.

// I assume that index 0 is top-left and index 2 is bottom-right
cv::Rect person_bounds(scene_corners[0], scene_corners[2]);
if (person_bounds.contains(r.tl()) && person_bounds.contains(r.br()))
{
    // do something...
}
