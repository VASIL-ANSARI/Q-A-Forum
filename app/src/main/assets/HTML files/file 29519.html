You should use printf or sprintf for this purpose. These functions apply a specific format (the first parameter) to a number of string, integer or float arguments (the subsequent parameter(s)) â printf outputs the formatted string and sprintf returns it so that you can store it in a variable.

$price = 4.4049494;
printf("%01.2f", $price);

An easier alternative is to use number_format but this function adds a comma between every group of thousands (makes no difference if your input number is less than 1000).

$price = 4.4049494;
echo number_format($price, 2);
