The onlu way to do this with CSS is to set them all a fixed height. Instead of your min-height : 100% use something like min-height: 300px. But since you do not know how much data each of the uls will contain, this is a very inflexible solution.

Now, If you want all your columns to have the same height, than i supose you want them all to be as tall as the tallest one.

This jQuery code should do the trick:

var equalHeight = function ( group ) {
    var tallest = 0;

    group.each(function() {
        var thisHeight = $(this).height();
        if( thisHeight > tallest ) {
            tallest = thisHeight;
        }
    });

    group.height( tallest );
}

And than just call it on document ready with your group of elements

$(function () {
    equalHeight($('.rows'));
});
