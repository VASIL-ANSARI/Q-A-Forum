That's not what the <h2> element is for. If you want to style an unordered list, give it a class:

<ul class="big">
   <li>thing one</li>
   <li>thing two</li>
   <li>thing three</li>
</ul>

And then add some CSS:

.big {
    font-weight: bold;
    font-size: 1.5em;
    display: block;
}
