You can use CASE in WHERE clause

SELECT 
  * 
FROM
  t 
WHERE 
  CASE
    WHEN user_type = 1 
    THEN photos_count > 0 
    ELSE TRUE 
  END

Fiddle Demo

While writing active record query you can do so

$result=$this->db->select('*')
             ->from('t')
             ->where('CASE
                        WHEN user_type = 1 
                        THEN photos_count > 0 
                        ELSE TRUE 
                      END ',null,FALSE)
             ->get()
             ->result();
