each() will do just that, iterate through each link. You want to alert when a specific link is clicked. The click function will be called when an element of class .high_res_link but not of class .no_pop is clicked. The this keyword refers to the element that triggered the function, so it will display the corresponding alert depending on which link was clicked.

   $('.high_res_link').not('.no_pop').click(function(e){
       var href = $(this).attr('href');
       href = href+"sizes/o/";
       alert(href);
       e.preventDefault();
   });
