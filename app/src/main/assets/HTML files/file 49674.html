You could use the dependent frequency:

f = f(w1 and w2) / f(w1 or w2)

where f(w1 and w2) is the number of slots where both words appear and f(w1 or w2) is the number of slots where at least one of both words appears. If both words only appear together, you get f = 1. If both words never appear together, you get f = 0.

You can easily map this to a distance-like value via distance = k * (1 - f) for some scaling factor k.
