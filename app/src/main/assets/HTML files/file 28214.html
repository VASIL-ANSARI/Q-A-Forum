Use parse_str():

$str = '4=Test&6=This is a test&9=19&15=Bla Bla Bla&appid=19746';
$result = array();
parse_str($str, $result);
print_r($result);

Output:

Array
(
    [4] => Test
    [6] => This is a test
    [9] => 19
    [15] => Bla Bla Bla
    [appid] => 19746
)

And now, to do the INSERT, you can loop through the splitted array:

foreach ($result as $part) {
    // do the INSERT
}
