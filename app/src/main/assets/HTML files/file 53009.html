Add 'dragover' event with preventDefault and it should work.

Inside your init():

element.addEventListener('dragover', over, false);

and over function:

function over(e) { 
   e = e || window.event; 
   if(e.preventDefault) {
     e.preventDefault();
   } 
}

Also add the same prevention to your other two functions ..
