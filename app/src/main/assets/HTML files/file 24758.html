It sounds like a self-join situation, something like:

SELECT * 
FROM table a
JOIN table b
   ON  a.id_BusLine = b.id_Busline
   AND a.id_NextBusStop = b.id_BusStop

That links them together, but you'd want ordering too, if your isFirstStop is 1/0 then:

SELECT * 
FROM table a
JOIN table b
   ON  a.id_BusLine = b.id_Busline
   AND a.id_NextBusStop = b.id_BusStop
ORDER BY isFirstStop, ....
