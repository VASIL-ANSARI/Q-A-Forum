Make also one array -

filteredImageViews = ImageViews.filter() { $0 != ImageViews[5] }

And fill collection view from filteredImageViews. So, when need remove image from collection view make this:

collectionView.performBatchUpdates(
{
    filteredImageViews = ImageViews.filter() { $0 != ImageViews[5] }
    collectionView.deleteItemsAtIndexPaths(NSIndexPath(forItem: 5, inSection: 0))
},
completion: nil)
