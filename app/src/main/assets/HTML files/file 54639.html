Try putting below code in your web.config file.

 <system.webServer>
   <httpProtocol>
         <customHeaders>
           <add name="Access-Control-Allow-Origin" value="*" />
           <add name="Access-Control-Allow-Headers" value="Origin, X-Requested-With, Content-Type, Accept" />
         </customHeaders>
  </httpProtocol>
 </system.webServer>

As error says Cross-Origin Request Blocked so this code allows the cross origin request here * means accept all request.
