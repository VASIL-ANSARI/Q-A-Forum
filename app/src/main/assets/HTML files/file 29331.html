Saving Multiple Records At Once

It seems your main problem is saving multiple records at once:

No handler found for {"0"=>

This shows that Rails is not able to compute the save function, as it's only designed for single params. You'll actually have to save the records with a manual SQL query, like these resources:

  • INSERT multiple records using ruby on rails active record

  • How to implement bulk insert in Rails 3

Essentially, you somehow have to loop through your params array, and save the data in one bulk way. Here is one way you could do it:

As per one of those links, you could use the activerecord-import gem

def create
assets = []
    params[:asset].each do |k,v| 
       assets << Asset.new(params[:asset][k])
    end
    Asset.import assets
end
