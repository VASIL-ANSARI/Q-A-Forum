In the case of 

ts_dic=dict.fromkeys(['a','b'],dict.fromkeys(['1','2'],0))

What the control flow is that python instantiates a dictionary from the internal call 

 dict.fromkeys(['1','2'],0)

This dictionary is then put as value for both the keys. The object reference is same for both, i.e. ts_dic['a'] and ts_dic['b'] are pointing to the same address in memory. Thus any modification to ts_dic['a'] will modify the ts_dic['b'] as they are actually at the same address in memory. You can check it by using id() function on both. id() gives the memory address of an object.

print(id(ts_dic['a'])) #1
print(id(ts_dic['b'])) #2
print(id(ts_dic1['a'])) #3
print(id(ts_dic1['b'])) #4

You will find that #1 and #2 will output the same value.
