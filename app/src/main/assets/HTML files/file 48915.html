You have properties that set itself:

public string cn
{
    get { return courseNumber; }
    set
    {
        if (value != null)
            cn = value;
    }
}

This will infinitely loop. You should have some private fields to assign:

private string _cn;
public string cn
{
    get { return _cn; }
    set
    {
        if (value != null)
            _cn = value;
    }
}
