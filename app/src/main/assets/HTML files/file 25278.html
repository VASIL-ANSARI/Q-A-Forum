You sound like you are building a reverse proxy. Code Project has a good post on how to accomplish this http://www.codeproject.com/Articles/18490/Reverse-Proxy-in-C-NET-v2-0 which included copying cookies from one request to the other.

Be aware you will only receive cookies for the domain you app is hosted on. If you are proxying requests to a different domain the browser sending the request will not send you cookies for the domain you are targeting.

For example if your app is at http://dev.local/ and you are forwarding the request to http://stackoverflow.com/ you will not have any cookies for the domain stackoverflow.com in your request.
