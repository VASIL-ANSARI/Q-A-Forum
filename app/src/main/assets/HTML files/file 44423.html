As @SatelliteSD stated; you're using the same JSONObject for each iteration. This is updating the values in THAT object each time and since the array has multiple references to the same object; it's outputting the same value multiple times.
Rewriting to something like this should resolve the issue.

jsArray = new JSONArray();
for (int i = 1; i < j; i++) {
    CheckBox checkBox = (CheckBox) findViewById(i);
    if (checkBox.isChecked()) {
        try {
            String ean = (String) checkBox.getText();
            JSONObject json = new JSONObject();
            json.put("ean", ean);
            jsArray.put(json);
            Log.v("jsArray", jsArray.toString());
        } catch (JSONException e) {
            // TODO Auto-generated catch block
            e.printStackTrace();
        }
    }
}
