Sure it's possibile. You can use jQuery map to translate your starting item array or object to new array of items, than you can apply the input filter using grep.

The current filter is stored in the request, source function parameter.

Ref:

Translate all items in an array or object to new array of items.

and

Finds the elements of an array which satisfy a filter function. The original array is not affected.

Code:

$('#personsearch').autocomplete({
    source: function (request, response) {
        var re = $.ui.autocomplete.escapeRegex(request.term);
        var matcher = new RegExp("^" + re, "i");
        response($.grep(($.map(people, function (v, i) {

            return {
                label: v.name,
                value: v.name
            };
        })), function (item) {
            return matcher.test(item.value);
        }))

    }
});
