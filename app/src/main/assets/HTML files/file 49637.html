You can use Window mangaer fo this, it is more convenient to use :

private WindowManager windowManager;
windowManager = (WindowManager) getSystemService(WINDOW_SERVICE);
WindowManager.LayoutParams mParams = (WindowManager.LayoutParams) <your-view>.getLayoutParams();

mParams.x = <value>;
mParams.y = <value>;

 windowManager.updateViewLayout(<your-view>, mParams);

Hope it helps :)
