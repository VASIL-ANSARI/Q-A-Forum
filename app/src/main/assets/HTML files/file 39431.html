You are finding checkbox with class="checkbox" but actually div containing checkbox has this class.

So find checkbox inside the div to bind change event. And all checkboxes are not siblings so you need to find them inside div and then take count.

See below code -

var limit = 1;
$('#pierwszy').on('change','div.checkbox input[type=checkbox]', function() {
   var count = $('div.checkbox input[type=checkbox]:checked').length;
  if(count > limit) {
   this.checked = false;
  }
});
