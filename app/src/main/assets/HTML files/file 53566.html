Table Name and Table Column Name should not contain a Space. So, either remove the space or replace it with an underscore (_).

If you want to keep the Space, wrap the names in double quotes ("). For example:

create table "abc xyz" ("first name" string);

EDIT: Your query string should be somewhat like this: 

String query = "CREATE TABLE " + TABLE_STUDENT + " ( " +
                    COLUMN_ID + " INTEGER PRIMARY KEY AUTOINCREMENT, " +
                    COLUMN_STUDENTNAME + " TEXT, " +
                    COLUMN_EMAIL + " TEXT, " +
                    COLUMN_PASSWORD + " TEXT, " +
                    COLUMN_CPASSWORD + " TEXT, " +
                    COLUMN_INSTITUTE + " TEXT, " +
                    COLUMN_CONTACTNO + " TEXT" +
                    ")";
