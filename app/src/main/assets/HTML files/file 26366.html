Give images a class and override border-radius in addition to border-top-left-radius and border-bottom-left-radius which inherit their values from parent element.

Demo

.image {
  -webkit-border-radius: 50px;
  -moz-border-radius: 50px;
  border-radius: 50px;
  border-top-left-radius: 50px !important;
  border-bottom-left-radius: 50px !important;
}
