When I was using eclipse I went through this. I never found a solution of using jar directly but I followed the following steps to include this library.

To add any library the steps are:

1) Eclipse -> Import -> Existing Android Code into workspace.

2) Add your_sdk_path\extras\android\support\v7\appcompat to your Eclipse workspace.

3) Right click on your project in Eclipse and select properties.(The window you have shown in the pic)

4) Select Android -> Library -> Add.

5) Select the imported project(appcompat) and Save.
