According to guides.rubyonrails.org seems like you have ensure that less-rails gem is installed and is present in Gemfile.

Currently not working properly with Sprockets, so if you have undefined method 'register_preprocessor' for nil:NilClass error, update your Gemfile with:

gem 'less-rails', github: 'metaskills/less-rails'

This creates files like app/assets/stylesheets/some_controller.css.less
