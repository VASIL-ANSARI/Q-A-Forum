Reffering to the docs https://dev.mysql.com/doc/refman/5.5/en/create-trigger.html it looks like that "CREATE OR REPLACE" syntax is not supported.

It should work

delimiter //
CREATE TRIGGER bookTakenDate
AFTER INSERT ON readers_books
for each row
begin
update readers_books
set date_supposed_taken = new.date_given
where given_book_id = new.given_book_id;
end;//
