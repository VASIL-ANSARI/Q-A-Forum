the problem here is that you are trying to compare a single value with an array so you'll get unexpected results:

In [8]:

next_day in df['Date']
Out[8]:
False
In [7]:

next_day not in df['Date']
Out[7]:
True

also I can't reproduce your other assertion:

In [17]:

So the correct method to use is isin and pass a series or list to check if the values in the passed in list exist in your Series as shown above, whatever result you are seeing is bogus and incorrect with respect to 41.106 not in hd['NAV'].

You can use the any along with the == operator to check for membership:

In [18]:

next_day == df['Date']
Out[18]:
0     True
1    False
2    False
3    False
4    False
Name: Date, dtype: bool
In [19]:

(next_day == df['Date']).any()
Out[19]:
True
