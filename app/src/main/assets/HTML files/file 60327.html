I got this to work. I checked my previous app, which used jQuery for the AJAX calls. When I replaced fetch with jquery-ujs, it worked! I saw my problem was that it had not been sending the session cookie with the request. I researched how to do this with isomorphic-fetch, and found the github/fetch documentation. When I structure my request like this, it works:

fetch('/my_resource.json?param=someparam', {
  'credentials': 'same-origin'
})

Gotta send the cookies.
