Yes, you can. ES6 destructuring assignment allows you to do this.

let a = 2, b = 3, d = 5, f = 6; //- these are the initial values



console.log({a, b, d, f});



[a, b, d, f] = [f, b, a, d];



console.log({a, b, d, f});

ES6, or ECMAScript 6, is the current standard for JavaScript that modern browsers are expected to comply to. Not all fully comply though, so if you want to do something like this in production code, you should transpile your code with Babel in order to make it compatible with older versions of Internet Explorer, and other browsers that still don't support ES6.
