I think the problem is in this block:

<% @post.uploads.each do |image| %>
    <div class="image-thumb">
        <%= image_tag image.upload.url(:thumb) %>  #here!
    </div>
<% end %>

You're calling each upload as image - and then calling image.upload.url - which is effectively calling upload.upload.url
