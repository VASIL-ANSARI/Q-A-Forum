That method does not exist on Windows Phone, because you are many to use async IO constantly. You need GetRequestStreamAsync or BeginGetRequestStream. The *Async method is usually easier to understand and use than Begin*, but requires an up-to-date compiler. The relevant example is here, in the phone a based documentation: http://msdn.microsoft.com/en-us/library/windowsphone/develop/System.Net.HttpWebRequest(v=vs.105).aspx
