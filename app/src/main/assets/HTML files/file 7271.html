Such usage of dashes in HTML comments causes unpredictable behaviour. See https://bugzilla.mozilla.org/show_bug.cgi?id=2749 for more information about differences in HTML comments parsing. (Source of http://www.hixie.ch/tests/evil/mixed/comments-evil.html is very good example)

I recommend using some server script or Find&Replace feature in your text editor to replace dashes in comments with equals signs or another symbol.
