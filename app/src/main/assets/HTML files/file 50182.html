If the height can be dynamic, then you can just switch classes on the alerts. I changed your html-markup a little bit in order to easier add/remove classes.

Here is a full screen example

Here is the JS-FIDDLE

function myAlert() {
    $(".jumbotron").addClass("alert alert-danger alert-dismissible fade in");
    $(".main").addClass("hidden");
    $(".myAlert").removeClass("hidden");
}

function danger() {
    alert("you just alerted some danger");
}

function cancel() {
    $(".jumbotron").removeClass("alert alert-danger alert-dismissible fade in");
    $(".main").removeClass("hidden");
    $(".myAlert").addClass("hidden");
}
