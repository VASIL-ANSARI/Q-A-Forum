Without knowing too much about your situation (like whether these are known conditions, or if the conditions are based on actions by the user), this is where I would start:

I would start with a ViewModel: 

public class ViewModel
{
    public bool ShowButton1 {get; set;}
    public bool ShowButton2 {get; set;}
    public bool ShowButton3 {get; set;}
    public bool ShowButton4 {get; set;}
}

And put the logic for the conditions in your controllers:

public ActionResult Index()
{
    var viewModel = new ViewModel();
    viewModel.ShowButton1 = condition1 && condition2;
    viewModel.ShowButton2 = condition3 && condition4;
}
