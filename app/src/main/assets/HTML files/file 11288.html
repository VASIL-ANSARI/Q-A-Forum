I believe the problem is that when you run sys.stdout.write it writes to the STDOUT buffer, but the buffer is not flushed until done so manually or closed (stdout is a File Object, which you can confirm yourself with type(sys.stdout)). Since your loop is infinite, the buffer is never flushed.

Changing your loop in echo.py to this should do the trick:

while 1:
    x = sys.stdin.readline()
    sys.stdout.write("got-" + x) # x already contains newline
    sys.stdout.flush() # flush the buffer
