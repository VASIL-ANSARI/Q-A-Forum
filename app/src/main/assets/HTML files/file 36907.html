Add a card background view whose backgroundColor has a transparency (alpha) set to less than 1 (maybe 0.5) when it is the top card. Something like:

View
    Card (back card outer view)
        Card Background view
        Card Icon view
        Card Label view
    Card (front card outer view)
        Card Background view
        Card Icon view
        Card label view

You'll then be able to see the back card through the front card.

The card outer view allows moving the card as a whole by just change the frame of the outer view.

The Card outer view backgroundColor should be clear (e.g., [UIColor clearColor]).
