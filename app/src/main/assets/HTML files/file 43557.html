a JS plugin called masonry allows me to achieve the desired result. it can be found here: masonry.desandro.com

after including the js file I followed HTML integration as shown on their website as:

<div id="container">
  <div class="item">...</div>
  <div class="item w2">...</div>
  <div class="item">...</div>
  ...
</div> 

<div id="container" class="js-masonry"
  data-masonry-options='{ "columnWidth": 200, "itemSelector": ".item" }'>

I was acutlly using Drupal Views to generate my code, so I overrode my view by creating a template override file views-view--myviewname.tpl.php to include a js-masonry class reference. and altered the .item to look for views-row.

Works like a charm.
