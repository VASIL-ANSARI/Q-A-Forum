To make the elements match the margin, you would need to change the values of the left and right attributes.

nav {
    position:fixed;
    left:15px;
    right:15px;
}

footer {
    position:fixed;
    left:15px;
    right:15px;
}

Or, perhaps, you want to keep those containing elements to the edge and you just want the content to match margin spacing. Then maybe you would like to simply add inner-padding:

nav {
        position:fixed;
        left:0;
        right:0;
        padding-left: 15px;
        padding-right: 15px;
}

footer {
        position:fixed;
        left:0;
        right:0;
        padding-left: 15px;
        padding-right: 15px;
}
