If it's really that simple, this should be quite enough:

my $xml_string = '<test>  <level>    <sub name="xyz">test</sub>   </level>  </test>';

# remove all whitespace before '<' symbol
$xml_string =~ s/\s+</</g;

print $xml_string; 
# echoes <test><level><sub name="xyz">test</sub></level></test>

I've used Perl syntax, but I guess it's quite easy to convert it into any language you want.

Be aware, though, that there are several caveats (as always in cases like that).

For example, you won't meet < symbol inside XML elements - but it can happily live within PCDATA sections, and that regex ignores this nuance.

UPDATE: the regex might be made even more concise with 'look-ahead' feature:

$xml_string =~ s/\s+(?=<)//g; # remove the sequence of whitespace, if followed by '<'

... but not all languages support that (Perl does, though )).
