You can just set overflow-x: hidden;, so scrolling in y direction will be nevertheless possible:

body {

  overflow-x: hidden;

}

  margin-left: -1000px;

  position: absolute;

  z-index: -1;

  overflow: hidden;

  -ms-transform: rotate(-30deg);

  /* IE 9 */

  -webkit-transform: rotate(-30deg);

  /* Safari */

  transform: rotate(-30deg);

  /* Standard syntax */

}
