Months in the Date constructor are 0-indexed, so 0 is January, 1 is February, etc. and moreover the parameters can overflow (new Date(2017,11,32) is actually Mon Jan 01 2018 00:00:00). So in your example the dates are in fact:

new Date(2017,1,31)
Fri Mar 03 2017 00:00:00
new Date(2017,2,10)
Fri Mar 10 2017 00:00:00

Therefore the result is correct.
