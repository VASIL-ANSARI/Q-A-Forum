I think you should be able to use ImageIO.read to parse the image:

InputStream in = new ByteArrayInputStream(imageDataByteArray);
BufferedImage bufImg = ImageIO.read(in);
ImageIcon icon = new ImageIcon(bufImg);
myButton.setIcon(icon);

As BufferedImage is a subclass of java.awt.Image, you should be able to use it as is. I haven't tested this myself, but I don't see why it wouldn't work.

Note also that you can't randomly change the byte array and expect it to represent a valid image. If you supply an array that doesn't contain a valid image data (or contains image of an unsupported format), ImageIO.read will return null. Have a look at ImageIO.read documentation for more info.
