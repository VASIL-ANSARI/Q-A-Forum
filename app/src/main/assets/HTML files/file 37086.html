1) You used 'SWITCH' instead of 'switch' inside 'ng-switch-when' directive. In your $scope this value is lower-case.

<div ng-switch-when="switch"> .. </div>

2) Ng-repeat directive now uses key-value form:

<option ng-repeat="(k,v) in input.inputValues" value="{{k}}">{{v}}</option>
