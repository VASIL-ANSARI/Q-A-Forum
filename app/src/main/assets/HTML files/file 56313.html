You have to add the @Path annotation to every method to specify different paths. Now, you are calling just /helloworld but there is no method associated to that path.

For example, for the first method try to do this:

@GET
@Produces(MediaType.TEXT_PLAIN)
@Path("/plainText")
public String sayPlainTextHello() {
    return "Hello World RESTful Jersey!";
}

and call it this way:

http://localhost:8080/jersey_helloworld/rest/helloworld/plainText
