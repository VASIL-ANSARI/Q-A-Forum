Try adding a callback function to your load() calls, then use your web browser's Javascript console to inspect the results. This should help you understand what's going wrong. Here's an example:

$("#prod_summary_377").load("http://your-website.com/template/Snippets/CategoryProductSummary.html #377", function(response, status, xhr){
  console.log(status);
  if (status == "error") {
    console.log(xhr.status + " " + xhr.statusText);
  }
});
