You need to customize your adapter (you can have an adapter per-model if you want, as far as I know).

In your specific example I think you want to customize the urlForFindAll method.

// maybe in app/organization/adapter.js (if using pods for a model specific  adapter)
urlForFindAll(modelName) {
    return 'urlYouWantToUseWithOrganizations';
}

A good article on this situation is on Ember Igniter
