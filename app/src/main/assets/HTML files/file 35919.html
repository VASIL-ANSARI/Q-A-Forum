The Image method can do that for you:

require ('fpdf.php');

$pdf = new FPDF();

for ($i = 0; $i < 5; $i++){
    $pdf->AddPage('L');
    $pdf->Image($i.'.gif', 0, 0, 250, 200);
}

$pdf->Output();

This will put each slide on a new page. You might want to adjust the image proportions (the last two parameter).
