You probably want this to say

SELECT `username`, `password` FROM Customers

to say this

SELECT `username`, `password` FROM Customers WHERE username = ?

or else it would give you all the users in the database... this is why your form is giving you problems... because it is checking the first user in the database...

The following is how I would change your code...

Change this:

$result = mysqli_query($con,'SELECT `username`, `password` FROM Customers');
while($row = mysqli_fetch_array($result))
{
if($row['username'] === $username && $row['password'] === $password)
{
$_SESSION['username']="$username";
setcookie('username' , $row['username']) ;
header("Location: http://localhost/loggedin.php") ;
}
else
{
header("Location: http://localhost/index.html") ;
}

to this:

$query = 'SELECT `username`, `password` FROM Customers where username = ?';

if($stmt = $mysqli->prepare($query)){
    $stmt->bind_param('s',$username);
    $stmt->execute();
    $stmt->bind_result($get_username, $get_password);
    $stmt->fetch();
    $stmt->free_result();
    $stmt->close();
}

if($get_username === $username && $get_password === $password){
    $_SESSION['username']="$username";
    setcookie('username' , $row['username']) ;
    header("Location: http://localhost/loggedin.php");
    exit();
} else {
    header("Location: http://localhost/index.html");
    exit();
}

There is a big flaw with the way you are handling passwords in the database... they should not be raw passwords... please hash them, with a salt and maybe even a pepper.
