By default, the httpOnly flag is set to to true in the cookie settings for express.session thus denying access to xhr requests, such as what jQuery does. You can enable it in the cookie configuration:

app.use(express.session({
  store: SessionStore,
  cookie: {
    maxAge: 36000000,
    httpOnly: false // <- set httpOnly to false
  },
  secret: 'MySecret'
}));

Now your client's cookies should be accessible to the server during an ajax request.
