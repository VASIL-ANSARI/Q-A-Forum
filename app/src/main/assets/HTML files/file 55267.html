Yes, consider using System.Timers.Timer class like below. Here, the interval is set to 1000 milliseconds (1 second). So after every 1 second it will raise Elapsed event when you can perform the task you want.

Timer tmr = new Timer(1000);
tmr.Elapsed += elapsedHandler;

private void elapsedHandler(Object source, ElapsedEventArgs e)
{
  //do your work
}
