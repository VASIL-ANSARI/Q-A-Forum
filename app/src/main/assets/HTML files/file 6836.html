You should edit AndroidManifest.xml and add this intent-filter to the activity you want started:

<intent-filter>
    <action android:name="android.intent.action.SENDTO" />
    <category android:name="android.intent.category.DEFAULT" />
    <data android:scheme="sms" />
    <data android:scheme="smsto" />
</intent-filter>

When the user send an SMS to a specific number, a chooser menu will be displayed. If she chooses your Application, your Activity will start.
