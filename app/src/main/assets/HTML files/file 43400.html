The problem is probably that time() returns the seconds that have past since 1970 and not a timestamp. (which I assume your db column is)

I suggest you use Carbon (the date library Laravel uses)

$now = Carbon::now();
$del = DB::table('event')->where('expired', '<', $now)->delete();

Or for soft delete:

$now = Carbon::now();
$del = DB::table('event')->where('expired', '<', $now)->whereNull('deleted_at')->update(['deleted_at' => $now]);
