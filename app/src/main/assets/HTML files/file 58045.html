My suggestion is to derive the test class from the WithApplication instead of handling manually the application life-cycle. This will look something like this

public class MyTestClass extends WithApplication {
    @Test
    public void testSomething() {
        Helpers.running(Helpers.fakeApplication(), () -> {
            // *whatever mocking*
            RequestBuilder mockActionRequest = Helpers.fakeRequest(
                                         controllers.routes.MyController.processSomething());
            Result result = Helpers.route(mockActionRequest);
            // *whatever assertions* 
       });
     }
}
