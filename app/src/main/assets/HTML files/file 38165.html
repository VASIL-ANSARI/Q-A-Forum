This can be achieved using list comprehension in python. 

a = [[1,2],[2,3],[3,4]]
b = [5,9]


[(x[0]+b[0], x[1]+b[1]) for x in a]

Explanation

for x in a - Iterate over the list a
x[0] - First element of the sub list
x[0]+b[0] - Add first element of each element in a with first element in b
x[1]+b[1] - Add second element of each element in a with second element in b

Output 

[(6, 11), (7, 12), (8, 13)]
